# Physics Sandbox Universe · 物理沙盒宇宙

> 纯 HTML / CSS / JavaScript 物理实验交互平台 — 零构建工具、零依赖、即开即用

46 个交互式物理实验 · 36 位科学家 · 6 大物理分类 · 17 个学科 · 5 套主题皮肤

## 项目概览

本项目是一个完全运行在前端浏览器的物理教学与探索平台。每个实验均为独立的单页 HTML 文件，通过 `shared-theme.js` / `shared-theme.css` 共享主题系统与导航。无需安装任何构建工具，用浏览器直接打开 `index.html` 即可使用。

### 核心特性

- **零依赖** — 除 Three.js（部分 3D 实验通过 CDN 加载）外，不依赖任何框架或库
- **五套主题** — 学术 (academic) · 液态玻璃 (glass) · 卡通 (cartoon) · 赛博朋克 (cyber) · 苏维埃 (soviet)，通过 localStorage 持久化、跨页面同步
- **知识体系** — 每个实验关联物理分类、学科分支与科学家，形成完整的知识图谱
- **教学辅助** — 内置知识图谱、知识追踪（关系网络）、教师备课三大教学页面

## 主题系统

五套主题通过 CSS 变量实现，由 `shared-theme.js` 统一管理：

| 主题 | 标识 | 风格描述 |
|------|------|---------|
| 学术 | `academic` | 简洁白底 · 适合课堂演示 |
| 液态玻璃 | `glass` | 暗色毛玻璃 · 现代质感 |
| 卡通 | `cartoon` | 暖色圆角 · 活泼亲和 |
| 赛博朋克 | `cyber` | 霓虹发光 · 深空黑底 |
| 苏维埃 | `soviet` | 红色硬核 · 复古工业 |

主题状态通过 `localStorage` (key: `psu-theme-v1`) 持久化，在页面间自动同步。

## 本地运行

```bash
# 方式一：Python 内置服务器
cd physics-sandbox-v2
python -m http.server 8000

# 方式二：Node.js
npx serve .

# 方式三：直接打开
# 用浏览器直接打开 index.html
```

访问 `http://localhost:8000` 即可。

## 技术栈

- **渲染**：原生 Canvas 2D API · Three.js (WebGL) · D3.js (知识图谱)
- **样式**：CSS 变量 + 主题覆盖 · 零 CSS 框架
- **交互**：原生 DOM 事件 · requestAnimationFrame 动画循环
- **3D 实验**：Three.js (通过 importmap 从 CDN 加载或本地 `lib/` 目录)
- **部署**：GitHub Pages 静态托管
