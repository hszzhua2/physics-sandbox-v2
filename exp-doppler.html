物理公式
多普勒效应（波源运动）
f' = f₀ · c / (c − vₛ·cosθ)
f₀：波源频率 · c：介质波速
vₛ：波源速度 · θ：波源运动方向与观察方向夹角
特殊情形
靠近: f' = f₀ · c / (c − vₛ) 远离: f' = f₀ · c / (c + vₛ) 激波: vₛ > c → 马赫锥 sinα = c/vₛ
当波源速度超过波速，波前叠加形成锥形激波面
应用领域

🚑 交通与声学

救护车警笛靠近时音调升高，远离时降低。这是最直观的多普勒效应体验。

⭐ 天体物理红移

星系远离地球时，光谱向红端偏移（波长变长）。哈勃通过红移测量发现宇宙膨胀。

🩺 医学超声

多普勒超声通过测量血流反射声波的频率变化，无创检测血流速度和方向。

🛰 雷达测速

交通雷达发射微波，通过反射波的频率偏移精确测量车辆速度。

历史背景
克里斯蒂安·多普勒 (1803-1853)
奥地利数学家、物理学家。1842年提出多普勒效应，最初用于解释双星颜色的差异。
1842
多普勒效应提出

1845
实验验证

1929
