公式讲解
波动方程
ψ(r, t) = A / √r · cos(kr - ωt + φ) 其中: k = 2π / λ (波数) ω = 2πf (角频率) r = √((x-x₀)² + (y-y₀)²) (到波源距离)
A 为振幅，λ 为波长，f 为频率。波从点源以圆形波前向外传播，振幅随 1/√r 衰减（二维波的几何衰减）。
叠加原理与干涉
ψtotal = ψ₁ + ψ₂ + ... + ψn 双源干涉: ψtotal = A₁/√r₁ cos(kr₁-ωt) + A₂/√r₂ cos(kr₂-ωt) 合成振幅: Atotal = √(A₁² + A₂² + 2A₁A₂cosΔφ) 其中 Δφ = k(r₁ - r₂) 为相位差
当 Δφ = 2nπ 时，两波相长干涉（亮纹）；当 Δφ = (2n+1)π 时，相消干涉（暗纹）。这就是明暗交替的干涉条纹的成因。
条纹间距公式
Δy = λD / d λ = 波长 D = 屏到波源的距离 d = 两波源间距
条纹间距与波长成正比，与波源间距成反比。红光（长波长）的条纹比蓝光（短波长）更宽，这正是白光干涉呈彩色条纹的原因。
拍频效应
两个频率相近的波叠加: ψ = A cos(ω₁t) + A cos(ω₂t) = 2A cos(Δωt/2) cos(ωavgt) 拍频: fbeat = |f₁ - f₂|
拍频是两个频率略有不同的波叠加后产生的振幅周期性变化。拍频等于两波频率之差。切换到「拍频效应」模式可实时观察此现象。
⚠ 本模拟中波的传播速度 v = f·λ，改变频率会自动调整波长以维持波速恒定（或反之，取决于当前锁定模式）。
