知识点讲解
波动方程
ψ(r, t) = A / √r · cos(kr - ωt + φ) 其中: k = 2π / λ (波数) ω = 2πf (角频率) r = √((x-x₀)² + (y-y₀)²)
A 为振幅，λ 为波长，f 为频率。波从点源以圆形波前向外传播，振幅随 1/√r 衰减（二维波的几何衰减）。
叠加原理与干涉
ψ_total = ψ₁ + ψ₂ + ... + ψₙ 合成振幅: A_total = √(A₁² + A₂² + 2A₁A₂cosΔφ) 其中 Δφ = k(r₁ - r₂) 为相位差
当 Δφ = 2nπ 时，两波相长干涉（亮纹）；当 Δφ = (2n+1)π 时，相消干涉（暗纹）。
条纹间距公式
Δy = λD / d λ = 波长, D = 屏到波源距离 d = 两波源间距
条纹间距与波长成正比，与波源间距成反比。红光（长波长）的条纹比蓝光（短波长）更宽。
拍频效应
ψ = A cos(ω₁t) + A cos(ω₂t) = 2A cos(Δωt/2) cos(ω_avg t) 拍频: f_beat = |f₁ - f₂|
拍频是两个频率相近的波叠加后产生的振幅周期性变化，等于两波频率之差。
⚠ 本模拟中波速 v = f·λ，改变频率会自动调整波长以维持波速恒定。
波动光学历史
1678
惠更斯原理
荷兰物理学家惠更斯提出波动说，认为光是以太中的弹性波动，波前上每一点都是新的子波源。
1801
杨氏双缝实验
托马斯·杨的双缝干涉实验首次以定量方式证实了光的波动性，否定了牛顿的微粒说主导地位。
1818
菲涅尔衍射理论
菲涅尔将惠更斯原理与干涉原理结合，建立了完整的衍射理论，成功解释了光的直线传播与衍射现象。
1865
麦克斯韦电磁理论
麦克斯韦证明光是电磁波，将光学与电磁学统一，光速 c = 1/√(μ₀ε₀)。
1905
波粒二象性
爱因斯坦提出光量子假说解释光电效应，揭示了光的波粒二象性，为量子力学奠定基础。
科学家档案
克里斯蒂安·惠更斯 (1629–1695)
荷兰物理学家、天文学家。提出光的波动说与惠更斯原理，发明摆钟，发现土星最大卫星泰坦。
托马斯·杨 (1773–1829)
英国博学者。1801年完成双缝干涉实验，首次证实光的波动性。还提出了杨氏模量，破解了罗塞塔石碑部分古埃及文字。
奥古斯丁·菲涅尔 (1788–1827)
法国物理学家。建立了衍射的完整数学理论——惠更斯-菲涅尔原理，发明菲涅尔透镜，广泛应用于灯塔照明。
