弗拉姆抛物面嵌入图 — 连接两个渐近平坦时空的拓扑通道

弗拉姆抛物面嵌入方程
z(r) = ±2√(rₛ(r − rₛ))

将施瓦西度规的空间部分嵌入三维欧氏空间，得到旋转抛物面。喉部位于 r = rₛ，z = 0 处。

史瓦西半径
rₛ = 2GM / c² ≈ 3 km × (M / M☉)

太阳质量黑洞的史瓦西半径约 3 km。rₛ 也是事件视界的边界。

爱因斯坦-罗森桥度规
ds² = −(1 − rₛ/r)dt² + (1 − rₛ/r)⁻¹dr² + r²dΩ²

1935 年爱因斯坦与罗森发现该度规在 r = rₛ 处存在坐标奇点，通过解析延拓可得到连接两个时空区域的桥。

潮汐力加速度 (径向)
Δa ≈ (2GM / r³) · Δr = (c²rₛ / r³) · Δr

在喉部 r = rₛ 处潮汐力最大。对于恒星质量黑洞，人体在喉部受到的拉伸力约 10⁹ g/m，瞬间撕裂。

⚠ 物理限制
1. 此嵌入图截断了时间维度和一个空间角，仅为数学可视化工具。
2. 原始爱因斯坦-罗森桥在动力学上不稳定，喉部会在光穿越时间内坍缩。
3. 维持可穿越虫洞（Morris-Thorne）需要违反类光能量条件的奇异物质。
4. 宏观可穿越虫洞在现有物理框架内尚无理论支持。
