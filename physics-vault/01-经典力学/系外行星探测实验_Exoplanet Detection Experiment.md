# 系外行星探测实验_Exoplanet Detection Experiment

## 1. 实验目的
通过高精度视向速度测量与凌星观测，在太阳系外寻找围绕恒星运行的行星，研究行星形成与轨道动力学，并验证开普勒定律在极端行星环境中的普适性。

## 2. 实验装置与方法
马约尔与奎洛兹使用上普罗旺斯天文台 ELODIE 高分辨阶梯光栅光谱仪，测量恒星谱线多普勒频移；凌星法则用开普勒空间望远镜监测恒星亮度。

## 3. 关键结果
1995 年发现首颗围绕类太阳恒星 51 Pegasi 的系外行星 51 Peg b：质量约 $0.46\,M_{J}$，轨道周期 4.23 天，视向速度振幅约 $56\,\mathrm{m/s}$，完全符合开普勒第三定律。

## 4. 历史意义
系外行星探测从根本上改变了人类对行星系统的认识，证实 [[开普勒定律]] 在宇宙中普适，催生了 "热木星" 等新行星类型，二人获 2019 年诺贝尔物理学奖。

## 5. 现代复现
现代以 HARPS、ESPRESSO 地面光谱仪和 TESS、CHEOPS、JWST 空间望远镜持续搜寻，已确认 5500 余颗系外行星，并向类地宜居行星推进。
