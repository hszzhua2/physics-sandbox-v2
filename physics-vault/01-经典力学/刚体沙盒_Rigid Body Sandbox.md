# 刚体沙盒 (Rigid Body Sandbox)

## 1. 实验目的与背景

刚体动力学是经典力学中研究不受形变影响之理想固体运动规律的分支。[[莱昂哈德·欧拉]]在18世纪中叶系统建立了刚体转动的完整理论框架，包括**欧拉角**描述方法、**转动惯量张量**概念和**欧拉动力学方程**。刚体沙盒作为一个理想化的实验平台，允许在自由空间中操纵任意形状刚体的初始角速度，观察其旋转演化，验证角动量守恒、转动惯量各向异性和中间轴不稳定性等核心原理。

本实验节点以三维自由刚体为研究对象，重点验证[[欧拉动力学方程]]的预测，特别是著名的**网球拍定理**（Tennis Racket Theorem）——绕中间主转动惯量轴的旋转是不稳定的，任何微小扰动都会导致翻转。这一现象深刻揭示了刚体转动中的非线性不稳定性，是经典力学中**动力学稳定性**理论的核心案例。

## 2. 实验装置与运行机制

### 2.1 装置结构

刚体沙盒的实验要素包括：

- **刚体**：具有固定质量分布的理想刚体，质量为 $M$，质心为 $O$。
- **主惯性轴**：三个相互正交的主轴 $e_1, e_2, e_3$，对应主转动惯量 $I_1, I_2, I_3$（按大小排列 $I_1 \leq I_2 \leq I_3$）。
- **角速度**：绕质心的角速度矢量 $\boldsymbol{\omega} = (\omega_1, \omega_2, \omega_3)$，在体坐标系中分量随时间变化。
- **外力矩**：在自由空间中，外力矩为零（$\boldsymbol{\tau} = 0$）。

### 2.2 运行机制

在无外力矩条件下，角动量 $\mathbf{L} = \mathbf{I} \cdot \boldsymbol{\omega}$ 和旋转动能 $T = \frac{1}{2}\boldsymbol{\omega} \cdot \mathbf{L}$ 均守恒。但体坐标系中的角速度分量 $\omega_i$ 并非各自守恒，因为体坐标系本身在旋转。角速度的演化由[[欧拉动力学方程]]描述，该方程在体坐标系主轴下的形式揭示了不同主轴方向上旋转运动的耦合与竞争关系。

### 2.3 欧拉角参数化

[[莱昂哈德·欧拉]]证明了任何刚体的取向可以用三个独立角度描述，即**欧拉角** $(\phi, \theta, \psi)$：

- **进动角**（precession）$\phi$：绕固定空间轴 $z$ 的旋转。
- **章动角**（nutation）$\theta$：节线与体轴之间的夹角。
- **自转角**（spin）$\psi$：绕体轴的旋转。

角速度在体坐标系中的分量为：

$$\begin{pmatrix} \omega_1 \\ \omega_2 \\ \omega_3 \end{pmatrix} = \begin{pmatrix} \dot{\phi}\sin\theta\sin\psi + \dot{\theta}\cos\psi \\ \dot{\phi}\sin\theta\cos\psi - \dot{\theta}\sin\psi \\ \dot{\phi}\cos\theta + \dot{\psi} \end{pmatrix}$$

## 3. 观测数据与数学模型

### 3.1 转动惯量张量

转动惯量张量 $\mathbf{I}$ 定义为：

$$I_{ij} = \int_V \rho(\mathbf{r})(\delta_{ij} r^2 - r_i r_j) \, dV$$

在主轴坐标系中，张量对角化为：

$$\mathbf{I} = \begin{pmatrix} I_1 & 0 & 0 \\ 0 & I_2 & 0 \\ 0 & 0 & I_3 \end{pmatrix}$$

角动量为 $\mathbf{L} = \mathbf{I}\boldsymbol{\omega}$，旋转动能为：

$$T = \frac{1}{2}\boldsymbol{\omega} \cdot \mathbf{L} = \frac{1}{2}(I_1\omega_1^2 + I_2\omega_2^2 + I_3\omega_3^2)$$

### 3.2 欧拉动力学方程

在无外力矩（$\boldsymbol{\tau} = 0$）条件下，[[欧拉动力学方程]]在主轴坐标系中的形式为：

$$I_1\dot{\omega}_1 = (I_2 - I_3)\omega_2\omega_3$$

$$I_2\dot{\omega}_2 = (I_3 - I_1)\omega_3\omega_1$$

$$I_3\dot{\omega}_3 = (I_1 - I_2)\omega_1\omega_2$$

这组非线性耦合常微分方程描述了自由刚体在体坐标系中的角速度演化。方程右侧的非线性项体现了不同主轴方向旋转运动的耦合。

### 3.3 角动量与能量守恒

无外力矩时，两个守恒量为：

**角动量大小守恒**（在空间系中固定，在体系中分量变化）：

$$L^2 = I_1^2\omega_1^2 + I_2^2\omega_2^2 + I_3^2\omega_3^2 = \text{const}$$

**旋转动能守恒**：

$$2T = I_1\omega_1^2 + I_2\omega_2^2 + I_3\omega_3^2 = \text{const}$$

这两个守恒约束将三维角速度空间限制在两个椭球面的交线上——即**能量椭球**与**角动量椭球**的交曲线，称为**泊松球面**上的轨迹。

### 3.4 网球拍定理（中间轴不稳定性）

考虑绕三个主轴各自的稳态旋转：

| 旋转轴 | 角速度 | 稳定性 |
|--------|--------|--------|
| 最大惯量轴 $I_3$ | $\omega_1 = \omega_2 = 0, \, \omega_3 = \Omega$ | 稳定 |
| 最小惯量轴 $I_1$ | $\omega_2 = \omega_3 = 0, \, \omega_1 = \Omega$ | 稳定 |
| 中间惯量轴 $I_2$ | $\omega_1 = \omega_3 = 0, \, \omega_2 = \Omega$ | **不稳定** |

**线性稳定性分析**：以绕 $I_2$ 轴的稳态旋转为基态，施加微小扰动 $\omega_1 = \epsilon_1, \omega_3 = \epsilon_3$。由欧拉方程线性化：

$$I_1\dot{\epsilon}_1 = (I_2 - I_3)\Omega\epsilon_3$$

$$I_3\dot{\epsilon}_3 = (I_1 - I_2)\Omega\epsilon_1$$

消去 $\epsilon_3$ 得到：

$$\ddot{\epsilon}_1 = \frac{(I_2 - I_3)(I_1 - I_2)}{I_1 I_3}\Omega^2 \epsilon_1$$

由于 $I_1 < I_2 < I_3$，有 $(I_2 - I_3) < 0$ 且 $(I_1 - I_2) < 0$，其乘积为正，因此：

$$\ddot{\epsilon}_1 = \lambda^2 \epsilon_1, \quad \lambda = \Omega\sqrt{\frac{(I_2 - I_3)(I_1 - I_2)}{I_1 I_3}} > 0$$

解为 $e^{\lambda t}$ 形式的指数增长，扰动指数放大——中间轴旋转**不稳定**。

而对绕 $I_1$ 和 $I_3$ 轴做同样分析，对应的系数乘积为负，得到 $\ddot{\epsilon} = -\omega^2 \epsilon$，扰动为振荡形式（正弦余弦），系统**稳定**。

### 3.5 典型实测现象

在太空或抛体自由飞行中，带手柄的刚体（如扳手或网球拍）绕中间轴抛出后，会在空中发生不可预测的周期性翻转，翻转周期由增长率 $\lambda$ 决定。这一现象在失重环境中被宇航员反复验证，是网球拍定理的直接实验体现。

## 4. 实验结论与影响

### 4.1 核心结论

1. **欧拉方程精确描述自由刚体转动**：理论预测的角速度演化轨迹与实测吻合，验证了[[欧拉动力学方程]]和转动惯量张量理论。
2. **角动量与旋转动能同时守恒**：自由刚体的角速度虽在体坐标系中不断变化，但角动量大小和旋转动能保持恒定，体现了[[角动量守恒]]和[[能量守恒定律]]的联合约束。
3. **中间轴旋转不稳定性是普遍现象**：网球拍定理表明，任何具有三个不同主转动惯量的刚体绕中间轴旋转时都会产生不稳定性，这一原理在卫星姿态控制中具有重要的工程意义。

### 4.2 科学影响

刚体动力学理论深刻影响了航天工程中的卫星姿态控制设计。卫星设计中通常选择绕最大或最小惯量轴进行自旋稳定，避免中间轴不稳定性导致的姿态失控。在物理学知识库中，刚体沙盒的转动概念与[[齿轮与自行车_Gears and Bicycle]]中的力矩传递、[[悬臂梁分析_Cantilever Beam Analysis]]中的应力分布共享力学理论基础，而其非线性不稳定性分析则与[[混沌双摆_Chaotic Double Pendulum]]和[[三体问题_Three-Body Problem]]所代表的复杂动力学系统形成了理论延伸。
