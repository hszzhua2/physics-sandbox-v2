# 胡克定律 (Hooke's Law)

## 1. 核心定义

在弹性极限内，弹簧的恢复力与形变成正比且方向相反，是描述弹性体小形变下应力-应变线性关系的本构定律。

## 2. 数学表达

### 一维胡克定律

$$F = -kx$$

弹性势能：

$$U = \frac{1}{2}kx^2$$

### 应力与应变

正应力 $\sigma = F/A$，正应变 $\epsilon = \Delta L / L_0$，胡克定律表述为：

$$\sigma = E\epsilon$$

其中 $E$ 为杨氏模量。横向应变与轴向应变之比为泊松比：

$$\nu = -\frac{\epsilon_{\text{横向}}}{\epsilon_{\text{轴向}}}$$

剪切胡克定律：

$$\tau = G\gamma$$

其中 $G = \frac{E}{2(1+\nu)}$ 为剪切模量，$\gamma$ 为剪应变。

### 广义胡克定律（张量形式）

各向同性弹性体：

$$\sigma_{ij} = \lambda \epsilon_{kk}\delta_{ij} + 2\mu\epsilon_{ij}$$

或逆形式：

$$\epsilon_{ij} = \frac{1}{E}\left[(1+\nu)\sigma_{ij} - \nu\sigma_{kk}\delta_{ij}\right]$$

其中 $\lambda, \mu$ 为[[拉梅系数]]：

$$\lambda = \frac{E\nu}{(1+\nu)(1-2\nu)}, \quad \mu = G = \frac{E}{2(1+\nu)}$$

**参数说明**：
- $k$：弹簧劲度系数（N/m）
- $E$：杨氏模量（Pa）
- $G$：剪切模量（Pa）
- $\nu$：泊松比（无量纲，$0 < \nu < 0.5$）
- $\lambda, \mu$：拉梅第一、第二参数
- $\sigma_{ij}, \epsilon_{ij}$：应力张量与应变张量分量

## 3. 物理学意义与推论

胡克定律是弹性力学的线性本构关系，其核心是"小形变"近似——将非线性的真实应力-应变关系在平衡态附近作线性展开。由此导出的弹性力学方程是线性的，满足叠加原理，为求解复杂结构的应力分布奠定基础。

**简谐运动**：$F=-kx$ 与牛顿第二定律结合给出 $\ddot{x} + \omega^2 x = 0$，其中 $\omega = \sqrt{k/m}$，是所有振动的原型，在[[量子力学]]的谐振子、声子、电磁场模态中反复出现。

**弹性波**：弹性体中的纵波波速 $c_L = \sqrt{(\lambda+2\mu)/\rho}$，横波波速 $c_T = \sqrt{\mu/\rho}$，是地震学的基础。

**能量与稳定性**：弹性势能 $U = \frac{1}{2}kx^2 \geq 0$ 保证弹性恢复的稳定性，是固体结构分析的基础。

**适用边界**：
- 仅在弹性极限内成立；超过弹性极限进入塑性区，应力-应变关系非线性且不可逆。
- 小应变假设（通常 $\epsilon \lesssim 0.01$），大形变需用非线性弹性或有限应变理论。
- 假设材料均匀各向同性；各向异性材料（如晶体、复合材料）需用全张量形式 $C_{ijkl}$（21个独立分量）。
- 不适用于粘弹性材料（时间依赖行为）与粘塑性材料。
- 在原子尺度，胡克定律对应原子间相互作用势在平衡位置附近的二次展开，揭示了其微观起源。

## 4. 相关验证实验

- [[弹簧振子实验_Spring Oscillator Experiment]]：直接验证 $F=-kx$ 与简谐周期 $T = 2\pi\sqrt{m/k}$。
- [[杨氏模量测量实验_Young's Modulus Measurement Experiment]]：用拉伸法或梁弯曲法测量 $E$。
- [[声速测量实验_Sound Velocity Experiment]]：通过弹性波速度反推弹性模量。
- [[扭摆实验_Torsion Pendulum Experiment]]：验证剪切模量与扭转胡克定律。
- 现代纳米压痕实验可测量微米尺度下的弹性模量。

## 5. 历史上下文

1676年，[[罗伯特·胡克]]以拉丁字谜 "ceiiinosssttuv"（*ut tensio, sic vis*，"如伸长，则力"）预告其弹性定律，1678年正式发表。胡克与[[艾萨克·牛顿]]之间的优先权与个人恩怨是科学史上著名的公案。

18—19世纪，[[托马斯·杨]]定义了杨氏模量概念，[[西梅翁·泊松]]引入泊松比，[[加布里埃尔·拉梅]]给出了广义胡克定律的现代张量形式。20世纪，[[欧拉-伯努利梁方程_Euler-Bernoulli Beam Equation]]与[[铁摩辛柯梁理论]]均以胡克定律为基础。胡克定律作为线性近似的原型，深刻影响了从声学、地震学到[[量子力学]]谐振子与[[量子场论]]中自由场展开的整个物理学体系。
