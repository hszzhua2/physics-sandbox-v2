# 梁振动模态实验_Beam Vibration Mode Experiment

## 1. 实验目的
测量梁的横向振动固有频率与振型，验证连续系统的模态分析与边界条件对固有频率的影响。

## 2. 实验装置与方法
悬臂或简支金属梁，用电磁激振器或压电片施加扫频激励，激光多普勒测振仪或加速度计扫描测振型，识别各阶模态频率。

## 3. 关键结果
悬臂梁各阶固有频率 $\omega_{n} = \beta_{n}^{2}\sqrt{\dfrac{EI}{\rho A}}$，其中 $\beta_{1}L = 1.875$、$\beta_{2}L = 4.694$；振型节点位置与理论一致，模态正交性得到验证。

## 4. 历史意义
梁振动模态是连续系统 [[欧拉-伯努利梁方程]] 的动态应用，建立了模态分析与正交性理论，是结构动力学与振动控制的基础。

## 5. 现代复现
现代用扫描激光测振仪全场获取振型，并通过主动压电控制抑制振动，应用于航天结构、桥梁监测和 MEMS 谐振器。
