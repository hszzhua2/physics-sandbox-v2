# 库埃特流实验_Couette Flow Experiment

## 1. 实验目的
研究两平行或同轴圆柱面间由相对运动引起的剪切流动，测量黏性流体的速度分布并验证牛顿黏性定律。

## 2. 实验装置与方法
库埃特 1890 年代使用两个同轴圆筒 (内筒旋转、外筒固定) 装置，筒间充满待测液体；用悬浮颗粒或扭矩传感器测速度剖面与剪切应力。

## 3. 关键结果
层流状态下两筒间速度分布 $v_{\theta}(r) = Ar + B/r$，剪切应力 $\tau = \eta\,r\,d(v_{\theta}/r)/dr$，扭矩与转速成正比，可定标流体黏度 $\eta$；高速时出现泰勒涡。

## 4. 历史意义
库埃特流是 [[纳维-斯托克斯方程]] 少数有精确解的流动之一，是黏度测量 (库埃特黏度计) 与剪切流稳定性的基础，催生了泰勒-库埃特湍流研究。

## 5. 现代复现
现代以高精度旋转黏度计和 PIV 研究泰勒-库埃特流动的二次失稳与湍流输运，用于润滑、聚合物加工与等离子体约束。
