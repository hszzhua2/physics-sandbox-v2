# 欧拉-伯努利梁方程 (Euler-Bernoulli Beam Equation)

## 1. 核心定义

描述细长弹性梁在横向载荷下小挠度弯曲的线性微分方程，假设截面保持平面且垂直于中性轴，是材料力学与结构工程的基础方程。

## 2. 数学表达

### 静态欧拉-伯努利梁方程

$$EI\frac{d^4 w(x)}{dx^4} = q(x)$$

其中 $w(x)$ 为挠度，$q(x)$ 为分布载荷。

### 逐阶积分形式

弯矩与剪力：

$$M(x) = -EI\frac{d^2 w}{dx^2}, \quad V(x) = \frac{dM}{dx} = -EI\frac{d^3 w}{dx^3}$$

载荷关系链：

$$q(x) = \frac{dV}{dx} = \frac{d^2 M}{dx^2} = -EI\frac{d^4 w}{dx^4}$$

### 动态梁方程（含惯性项）

$$EI\frac{\partial^4 w}{\partial x^4} + \rho A \frac{\partial^2 w}{\partial t^2} = q(x, t)$$

### 弯曲正应力

$$\sigma_x = -\frac{M y}{I}$$

其中 $y$ 为距中性轴的距离，$I$ 为截面惯性矩。

### 常见边界条件

- **固支端**：$w = 0$，$\frac{dw}{dx} = 0$
- **简支端**：$w = 0$，$M = 0$（即 $w'' = 0$）
- **自由端**：$M = 0$，$V = 0$（即 $w'' = 0$，$w''' = 0$）

### 屈曲方程（欧拉临界载荷）

$$EI\frac{d^2 w}{dx^2} + P w = 0$$

两端铰支柱的欧拉临界力：

$$P_{\text{cr}} = \frac{\pi^2 EI}{L^2}$$

**参数说明**：
- $E$：杨氏模量（Pa），参见[[胡克定律_Hooke's Law]]
- $I$：截面惯性矩（m⁴）
- $w(x)$：挠度（m）
- $q(x)$：分布载荷（N/m）
- $M(x)$：弯矩（N·m）
- $V(x)$：剪力（N）
- $\rho$：梁材料密度（kg/m³）
- $A$：横截面积（m²）
- $P_{\text{cr}}$：欧拉临界屈曲载荷（N）

## 3. 物理学意义与推论

欧拉-伯努利梁理论是弹性力学简化为"一维"问题的典范。其核心假设为：

1. **欧拉-伯努利假设**：变形前垂直于中性轴的横截面变形后仍保持平面且垂直于挠曲线（即忽略剪切变形）。
2. **小挠度假设**：$\frac{dw}{dx} \ll 1$，使曲率近似为 $\kappa \approx \frac{d^2 w}{dx^2}$。
3. **线弹性**：遵循[[胡克定律_Hooke's Law]]。

**弯矩-曲率关系**：$M = -EI\kappa$，其中 $EI$ 称为抗弯刚度，是梁抵抗弯曲能力的度量。

**模态与振动**：动态方程的本征模态对应梁的横向振动模式，频率：

$$\omega_n = \left(\frac{\beta_n L}{L}\right)^2 \sqrt{\frac{EI}{\rho A}}$$

其中 $\beta_n L$ 由边界条件决定（如两端简支时 $\beta_n L = n\pi$）。

**屈曲稳定性**：当轴向压力超过 $P_{\text{cr}}$ 时梁失稳，是结构稳定性的核心判据。

**适用边界**：
- 细长梁假设：长高比 $L/h \gtrsim 10$；短粗梁需用[[铁摩辛柯梁理论]]计入剪切变形与转动惯量。
- 小挠度假设；大挠度需用 elastica 方程（非线性梁理论）。
- 线弹性材料；塑性阶段不适用。
- 不考虑截面翘曲；薄壁开口截面需用翘曲理论修正。
- 复合材料与各向异性材料需扩展至广义梁理论。

- 现代数字图像相关（DIC）技术可全场测量梁表面应变分布。

## 5. 历史上下文

欧拉-伯努利梁理论的建立跨越一个世纪。1750年前后，[[丹尼尔·伯努利]]首先提出弯矩与曲率成正比的弹性线思想，并于1742年致信[[莱昂哈德·欧拉]]建议用变分法求解。欧拉在1744年的《曲线变分法》中推导出弹性线方程，并于1757年首次给出柱屈曲的临界载荷公式。

19世纪，[[纳维]]与[[巴雷·圣维南]]进一步完善了梁理论，使其成为工程教育的核心内容。20世纪，[[斯蒂芬·铁摩辛柯]]提出计入剪切变形的修正理论，扩展了梁方程的适用范围。现代有限元方法将梁方程离散化为单元，成为土木、机械、航空航天工程结构分析的基石。欧拉-伯努利梁理论作为"最简单而深刻"的连续体力学模型，也是[[量子力学]]中一维势阱与弦振动本征值问题的经典类比原型。
