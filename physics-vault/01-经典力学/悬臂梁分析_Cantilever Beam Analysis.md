# 悬臂梁分析 (Cantilever Beam Analysis)

## 1. 实验目的与背景

悬臂梁是一端固定、另一端自由的梁结构，是结构力学中最基本也是最重要的构件之一。[[莱昂哈德·欧拉]]与[[丹尼尔·伯努利]]在18世纪中叶共同发展了梁的弯曲理论，建立了[[欧拉-伯努利梁方程]]——一个描述细长梁在横向载荷作用下弯曲变形的四阶常微分方程。1750年前后，欧拉系统阐述了梁的挠度理论，使悬臂梁成为验证梁弯曲理论、应力分布和边界条件的经典实验模型。

悬臂梁分析在建筑结构（如阳台、桥梁挑臂）、机械工程（如悬臂支架、天线臂）和微纳米器件（如原子力显微镜悬臂梁）中具有广泛而深远的应用。本实验节点以悬臂梁为载体，系统验证梁弯曲的弹性理论、挠度方程和应力分布规律。

## 2. 实验装置与运行机制

### 2.1 装置结构

悬臂梁实验的典型设置包括：

- **梁体**：长度为 $L$ 的均匀截面梁，截面宽 $b$、高 $h$，杨氏模量为 $E$。
- **固定端**（$x=0$）：完全固支，位移和转角均为零。
- **自由端**（$x=L$）：自由，不受位移和转角约束。
- **载荷**：自由端施加集中力 $P$，或沿梁分布载荷 $q(x)$。
- **截面惯性矩**：$I = \frac{bh^3}{12}$（矩形截面）。

### 2.2 运行机制

当外力作用于梁时，梁发生弯曲变形。梁的中性轴（neutral axis）保持长度不变，上侧受压缩、下侧受拉伸（或反之），产生**弯矩**（bending moment）$M(x)$ 和**剪力**（shear force）$V(x)$。

弯矩与曲率的关系由梁的基本假设给出——即弯曲后梁截面保持平面且垂直于中性轴（**Euler-Bernoulli假设**，又称**平截面假定**）。弯矩产生的曲率与挠度的二阶导数成正比，由此建立挠度微分方程。

## 3. 观测数据与数学模型

### 3.1 欧拉-伯努利梁方程

[[欧拉-伯努利梁方程]]为：

$$EI\frac{d^4 w(x)}{dx^4} = q(x)$$

其中：
- $w(x)$：梁的挠度（向下为正），单位 $\text{m}$。
- $E$：杨氏模量，单位 $\text{Pa}$。
- $I$：截面惯性矩，单位 $\text{m}^4$。
- $q(x)$：分布载荷集度，单位 $\text{N/m}$。
- $EI$：**抗弯刚度**（flexural rigidity），衡量梁抵抗弯曲变形的能力。

各阶导数的物理意义：

| 导数 | 物理量 | 关系 |
|------|--------|------|
| $w(x)$ | 挠度 | 直接测量 |
| $w'(x) = \theta(x)$ | 转角 | $\theta \approx \frac{dw}{dx}$（小变形近似） |
| $w''(x)$ | 曲率 | $EIw'' = M(x)$（弯矩） |
| $w'''(x)$ | 三阶导 | $EIw''' = V(x)$（剪力） |
| $w''''(x)$ | 四阶导 | $EIw'''' = q(x)$（分布载荷） |

### 3.2 悬臂梁自由端集中力精确解

对于自由端施加集中力 $P$、无分布载荷（$q=0$）的悬臂梁，方程为：

$$EI\frac{d^4 w}{dx^4} = 0$$

边界条件为：

- 固定端（$x=0$）：$w(0) = 0$，$w'(0) = 0$
- 自由端（$x=L$）：弯矩 $M(L) = EIw''(L) = 0$，剪力 $V(L) = EIw'''(L) = -P$（符号约定）

积分四次并应用边界条件，得到挠度方程：

$$w(x) = \frac{P x^2}{6EI}(3L - x)$$

自由端（$x=L$）最大挠度为：

$$w_{\max} = w(L) = \frac{PL^3}{3EI}$$

自由端转角为：

$$\theta(L) = w'(L) = \frac{PL^2}{2EI}$$

### 3.3 弯矩与剪力分布

弯矩分布：

$$M(x) = EIw''(x) = P(L - x)$$

固定端弯矩最大：$M(0) = PL$（负弯矩，上侧受拉）。

剪力分布：

$$V(x) = EIw'''(x) = -P$$

剪力沿梁恒定，等于外力 $P$。

### 3.4 应力分布

梁截面上的弯曲正应力为：

$$\sigma(y) = \frac{M(x) \cdot y}{I}$$

其中 $y$ 为距中性轴的距离（$-h/2 \leq y \leq h/2$）。最大应力出现在固定端截面上下边缘：

$$\sigma_{\max} = \frac{M_{\max} \cdot (h/2)}{I} = \frac{PL \cdot h/2}{bh^3/12} = \frac{6PL}{bh^2}$$

当 $\sigma_{\max}$ 超过材料的屈服强度 $\sigma_y$ 时，梁发生塑性变形；超过极限强度 $\sigma_u$ 时发生断裂。

### 3.5 均布载荷精确解

对于全梁均布载荷 $q$（单位长度力）、自由端无集中力的悬臂梁，方程为：

$$EI\frac{d^4 w}{dx^4} = q$$

边界条件同上，但自由端剪力变为 $V(L) = 0$，弯矩 $M(L) = 0$。解得挠度方程：

$$w(x) = \frac{q x^2}{24EI}(6L^2 - 4Lx + x^2)$$

自由端最大挠度：

$$w_{\max} = w(L) = \frac{qL^4}{8EI}$$

固定端最大弯矩：

$$M(0) = \frac{qL^2}{2}$$

## 4. 实验结论与影响

### 4.1 核心结论

1. **欧拉-伯努利梁方程精确预测挠度**：在细长比 $L/h \geq 10$ 且小变形条件下，理论挠度 $w(x) = Px^2(3L-x)/(6EI)$ 与实测吻合度极高，验证了平截面假定的有效性。
2. **应力线性分布**：弯曲正应力沿截面高度线性分布 $\sigma(y) = My/I$，中性轴处应力为零，上下边缘最大，这一规律是梁强度设计的核心依据。
3. **边界条件决定挠度形态**：固支端的零位移零转角和自由端的零弯矩零剪力是悬臂梁特征挠曲线的根本约束。

### 4.2 适用边界

欧拉-伯努利梁方程在以下情形需要修正：

- **短粗梁**（$L/h < 10$）：剪切变形不可忽略，需使用[[铁木辛柯梁理论]]（Timoshenko beam theory）。
- **大挠度**：挠度与梁厚同量级时，几何非线性效应显著，需用非线性梁方程。
- **复合材料梁**：各向异性材料需修正截面惯性矩和耦合效应。

### 4.3 科学影响

悬臂梁理论是结构力学和材料力学的基石。在现代工程中，微纳米尺度的悬臂梁是[[原子力显微镜]]（AFM）的核心传感元件——探针微悬臂的弯曲量以纳米精度测量表面形貌和力学性质。在物理学知识库中，悬臂梁的弯曲振动（当施加周期性力时）与[[弹簧振子_Spring Oscillator]]的简谐运动理论相通，其高频振动模式的分析也与[[克拉尼图形_Chladni Patterns]]中的板振动模态理论共享弹性波动力学的理论基础。
