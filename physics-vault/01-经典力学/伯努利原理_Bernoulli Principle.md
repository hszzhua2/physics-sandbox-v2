# 伯努利原理 (Bernoulli Principle)

## 1. 核心定义

伯努利原理描述了[[理想流体]]在[[定常流动]]中，沿流线上各点的压强、流速和高度之间的能量守恒关系：流速越快，压强越低。

## 2. 数学表达

### 2.1 伯努利方程

对于沿流线的无黏性、不可压缩、定常流动，伯努利方程为：

$$p + \frac{1}{2}\rho v^2 + \rho g h = \text{const}$$

其中各参数含义为：

| 参数 | 物理意义 | 单位 |
|------|---------|------|
| $p$ | 流体静压（static pressure） | $\text{Pa} = \text{N/m}^2$ |
| $\rho$ | 流体密度 | $\text{kg/m}^3$ |
| $v$ | 流体流速 | $\text{m/s}$ |
| $g$ | 重力加速度（$\approx 9.81 \, \text{m/s}^2$） | $\text{m/s}^2$ |
| $h$ | 相对参考面的高度 | $\text{m}$ |

### 2.2 各项物理意义

方程中的三项分别对应流体的三种能量密度（单位体积能量）：

- $p$：**压强能**（pressure energy），单位体积流体所具有的压力势能。
- $\frac{1}{2}\rho v^2$：**动能**（dynamic pressure），单位体积流体的动能。
- $\rho g h$：**重力势能**（gravitational potential energy），单位体积流体的重力势能。

三者之和沿流线守恒，是[[能量守恒定律]]在流体力学中的具体体现。

### 2.3 推导过程

从[[欧拉方程]]出发，沿流线方向积分。欧拉方程为：

$$\rho\left(\frac{\partial \mathbf{v}}{\partial t} + (\mathbf{v} \cdot \nabla)\mathbf{v}\right) = -\nabla p + \rho \mathbf{g}$$

在定常流动（$\partial \mathbf{v} / \partial t = 0$）且不可压缩（$\rho = \text{const}$）条件下，沿流线 $s$ 积分。利用恒等式 $(\mathbf{v} \cdot \nabla)\mathbf{v} = \nabla(\frac{v^2}{2}) - \mathbf{v} \times (\nabla \times \mathbf{v})$，沿流线方向旋度项消失，得到：

$$\int \frac{d}{ds}\left(\frac{v^2}{2}\right) ds = -\int \frac{1}{\rho}\frac{dp}{ds} ds - \int g \, dh$$

积分后即得伯努利方程：

$$\frac{1}{2}v^2 + \frac{p}{\rho} + gh = \text{const}$$

或乘以 $\rho$ 后：

$$p + \frac{1}{2}\rho v^2 + \rho g h = \text{const}$$

## 3. 物理学意义与推论

### 3.1 压强-速度关系

在同一水平面上（$h$ 为常数），伯努利方程简化为：

$$p + \frac{1}{2}\rho v^2 = \text{const}$$

即压强与流速平方成反比关系。流速增大时压强减小，反之亦然。这是伯努利原理最核心的推论，解释了文丘里效应、翼型升力和球类运动中的弧线球等众多现象。

### 3.2 适用边界与限制条件

伯努利原理严格成立的条件为：

1. **无黏性流体**：忽略内摩擦，即 $\mu = 0$。实际流体存在黏性耗散，伯努利方程需修正。
2. **不可压缩**：密度 $\rho$ 为常数，适用于液体和低速气体（马赫数 $M \ll 1$）。高速气体需用可压缩流伯努利方程。
3. **定常流动**：流场不随时间变化，$\partial \mathbf{v}/\partial t = 0$。
4. **沿同一流线**：方程沿一条流线成立。在无旋流动（$\nabla \times \mathbf{v} = 0$）中，常数对所有流线相同。

当上述条件不满足时，需要使用更一般的[[纳维-斯托克斯方程]]。例如[[泰勒-库埃特流_Taylor-Couette Flow]]中考虑了黏性效应，其精确解超出了伯努利原理的适用范围。

### 3.3 可压缩流推广

对于可压缩、绝热（等熵）流动，伯努利方程推广为：

$$\frac{v^2}{2} + \int \frac{dp}{\rho(p)} + gh = \text{const}$$

对于理想气体等熵过程（$p/\rho^\gamma = \text{const}$，$\gamma$ 为比热比），积分后：

$$\frac{v^2}{2} + \frac{\gamma}{\gamma - 1}\frac{p}{\rho} + gh = \text{const}$$

## 4. 相关验证实验

### 4.1 皮托管（Pitot Tube）

皮托管利用伯努利原理测量流体流速。管前端迎流口（总压口）测得总压 $p_0 = p + \frac{1}{2}\rho v^2$，侧面静压孔测得静压 $p$。两者之差为动压：

$$\Delta p = p_0 - p = \frac{1}{2}\rho v^2$$

由此可求流速：

$$v = \sqrt{\frac{2\Delta p}{\rho}}$$

### 4.2 文丘里管（Venturi Tube）

文丘里管通过截面收缩测量流量。截面 $A_1$（大）到 $A_2$（小），由连续性方程 $A_1 v_1 = A_2 v_2$ 和水平伯努利方程：

$$p_1 + \frac{1}{2}\rho v_1^2 = p_2 + \frac{1}{2}\rho v_2^2$$

可导出流量公式：

$$Q = A_1 A_2 \sqrt{\frac{2(p_1 - p_2)}{\rho(A_1^2 - A_2^2)}}$$

### 4.3 翼型升力

机翼上表面弯曲使气流加速，压强降低；下表面平缓使气流减速，压强较高。上下表面的压强差产生升力。这是[[丹尼尔·伯努利]]原理最广为人知的工程应用之一，但其完整分析需结合[[库塔-儒科夫斯基条件]]和环量理论。

## 5. 历史上下文

[[丹尼尔·伯努利]]于1738年在其著作《流体动力学》（*Hydrodynamica*）中首次提出了这一原理。他是[[伯努利家族]]的成员，其父[[约翰·伯努利]]亦为著名数学家。伯努利原理的提出标志着流体动力学从纯经验走向理论化的关键一步，为后世[[欧拉方程]]和[[纳维-斯托克斯方程]]的建立奠定了基础。

伯努利原理与[[泰勒-库埃特流_Taylor-Couette Flow]]共享理想流体假设的理论根基，但后者进一步引入了黏性效应，展现了流体力学从理想到实际的演进脉络。在工程应用中，伯努利原理是飞机翼型设计、管道流量计和水利工程设计的理论基石，至今仍是流体力学教学的入门核心内容。
