# 谐振子相空间实验_Harmonic Oscillator Phase Space Experiment

## 1. 实验目的
以简谐振子为模型，直观展示其在相空间 $(q,p)$ 中的运动轨迹，验证哈密顿力学中等能量曲线与相空间体积守恒的概念。

## 2. 实验装置与方法
用弹簧-质量振子或气轨上的弹簧振子，配合位移传感器测 $q(t)$ 并由 $p = m\dot{q}$ 得动量。计算机实时绘制相空间轨迹与庞加莱截面。

## 3. 关键结果
无阻尼时相轨迹为椭圆 $\dfrac{q^{2}}{A^{2}} + \dfrac{p^{2}}{(m\omega A)^{2}} = 1$，椭圆面积 $\oint p\,dq = 2\pi E/\omega$ 守恒；有阻尼时相轨迹螺旋收缩为吸引子。

## 4. 历史意义
谐振子是 [[哈密顿力学]] 与作用量-角变量的最简范例，相空间方法为统计力学、[[混沌理论]] 及其量子化 (相干态) 提供统一的几何语言。

## 5. 现代复现
现代用 trapped ion 或原子干涉仪直接观测单粒子相空间分布 (Wigner 函数)，把经典相空间图景延伸到量子尺度。
