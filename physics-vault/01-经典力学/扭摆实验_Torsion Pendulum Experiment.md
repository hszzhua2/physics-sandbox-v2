# 扭摆实验_Torsion Pendulum Experiment

## 1. 实验目的
利用悬丝的微小恢复力矩测量极弱的作用力，是 18-19 世纪精密测量力学的核心工具，既用于验证电荷间力的规律，也用于测量万有引力。

## 2. 实验装置与方法
一根细银丝或钨丝上端固定，下端水平悬挂一根轻杆，杆两端装小球。水平面上的横向力使杆偏转，悬丝产生恢复力矩 $\tau = -\kappa\theta$。用固定在杆上的小镜反射光束实现光杠杆放大，可读出 $10^{-7}$ rad 的偏角。

## 3. 关键结果
扭摆周期 $T = 2\pi\sqrt{I/\kappa}$，由周期可定标悬丝扭转系数 $\kappa$。库仑 1785 年据此得到电荷力平方反比律，[[卡文迪什]] 1798 年据此测出万有引力常数。

## 4. 历史意义
扭摆是把 [[胡克定律]] 应用于角向位移的典范，催生了 [[库仑定律]] 与 [[万有引力常数]] 的首次精确测量，奠定了 [[精密测量]] 与实验物理的方法论。

## 5. 现代复现
现代扭摆用于 [[等效原理]] 检验 (如 MICROSCOPE 卫星) 和第五力搜寻，扭丝材料改用石英或镀膜钨丝，并采用光学读数与主动反馈控制。
