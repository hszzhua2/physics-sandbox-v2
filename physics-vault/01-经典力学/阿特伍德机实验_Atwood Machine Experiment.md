# 阿特伍德机实验_Atwood Machine Experiment

## 1. 实验目的
通过 "抵消" 部分重力来放慢落体加速度，精确测量匀加速运动并验证牛顿第二定律，测定重力加速度 $g$。

## 2. 实验装置与方法
乔治·阿特伍德 1784 年设计：轻质滑轮上跨过轻绳，两端挂质量 $m_{1}$、$m_{2}$ ($m_{1}>m_{2}$) 的物体，滑轮低摩擦；标尺与秒表测加速度。

## 3. 关键结果
系统加速度 $a = \dfrac{(m_{1}-m_{2})g}{m_{1}+m_{2}}$，绳张力 $T = \dfrac{2m_{1}m_{2}g}{m_{1}+m_{2}}$；测 $a$ 后可反推 $g$，与已知值偏差约 $1\%$ (考虑滑轮转动惯量后修正)。

## 4. 历史意义
阿特伍德机是 [[牛顿运动定律]] 的标准教学仪器，定量验证 $F=ma$ 并提供测定 $g$ 的便利方法，是 18 世纪精密力学实验的代表。

## 5. 现代复现
现代复现采用低摩擦气轴承滑轮、光电门计时和旋转编码器实时测速，并用以演示等效原理和非惯性系中的惯性力。
