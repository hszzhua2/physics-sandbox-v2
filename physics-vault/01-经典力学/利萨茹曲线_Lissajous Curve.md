# 利萨茹曲线 (Lissajous Curve)

## 1. 核心定义

利萨茹曲线是两个相互正交方向的[[简谐振动]]按[[叠加原理]]合成的运动轨迹，其形态由频率比和[[相位差]]决定，呈现丰富的闭合或非闭合几何图形。

## 2. 数学表达

### 2.1 参数方程

利萨茹曲线的参数方程为：

$$x(t) = A_1 \sin(\omega_1 t + \phi_1)$$

$$y(t) = A_2 \sin(\omega_2 t + \phi_2)$$

其中：

| 参数 | 物理意义 |
|------|---------|
| $A_1, A_2$ | 两个方向简谐振动的振幅 |
| $\omega_1, \omega_2$ | 两个方向简谐振动的角频率 |
| $\phi_1, \phi_2$ | 两个方向简谐振动的初相位 |
| $\Delta\phi = \phi_1 - \phi_2$ | 相位差 |

通过时间平移可消去 $\phi_1$，令 $\phi = \Delta\phi$，方程简化为：

$$x(t) = A_1 \sin(\omega_1 t + \phi)$$

$$y(t) = A_2 \sin(\omega_2 t)$$

### 2.2 频率比与曲线封闭性

定义频率比：

$$r = \frac{\omega_1}{\omega_2} = \frac{p}{q}$$

其中 $p, q$ 为互素正整数。

- 当 $r$ 为有理数（$r = p/q$）时，曲线为**闭合曲线**，经过周期 $T = 2\pi q / \omega_1 = 2\pi p / \omega_2$ 后回到起点。
- 当 $r$ 为无理数时，曲线**永不闭合**，轨迹逐渐填满整个矩形区域 $[-A_1, A_1] \times [-A_2, A_2]$，在矩形内**稠密**（dense）。

### 2.3 闭合曲线的拓扑特征

对于有理频率比 $r = p/q$，利萨茹曲线具有以下拓扑不变量：

- **切点数**（tangency points）：曲线与边界矩形的切点数为 $2(p+q)$。
- **交叉点数**（crossing points）：曲线自身的交叉点数为 $2(p-1)(q-1)$。

这些不变量仅依赖于频率比 $p/q$，不依赖于相位差 $\phi$。

## 3. 物理学意义与推论

### 3.1 频率比决定图形形态

不同频率比产生截然不同的利萨茹图形：

| 频率比 $r$ | 相位差 $\phi$ | 图形形态 |
|------------|-------------|---------|
| $1:1$ | $0$ | 对角线（退化为直线段） |
| $1:1$ | $\pi/2$ | 正椭圆（$A_1=A_2$ 时为圆） |
| $1:1$ | $\pi/4$ | 倾斜椭圆 |
| $2:1$ | $\pi/2$ | $\infty$ 形（八字形） |
| $3:2$ | $\pi/2$ | 五叶形 |
| $3:1$ | $\pi/2$ | 四叶花形 |
| $5:4$ | $\pi/2$ | 复杂多叶闭合曲线 |

当 $p$ 和 $q$ 较大时，曲线形态极其复杂，呈现精美的花纹图案。

### 3.2 相位差的影响

对于固定频率比 $r = 1:1$，相位差 $\phi$ 完全决定图形形状：

- $\phi = 0$：退化为直线段 $y = \pm(A_2/A_1) x$。
- $\phi = \pi/2$：正椭圆（$A_1 = A_2$ 时为圆）。
- $\phi = \pi$：退化为直线段 $y = \mp(A_2/A_1) x$。
- 一般 $\phi$：倾斜椭圆，主轴方向和长短轴比随 $\phi$ 连续变化。

对于 $r \neq 1$ 的情况，相位差同样会引起图形的连续变形，但不改变切点数和交叉点数等拓扑不变量。

### 3.3 曲率与速度

利萨茹曲线上某点的速度为：

$$\mathbf{v}(t) = \left(A_1\omega_1\cos(\omega_1 t + \phi), \; A_2\omega_2\cos(\omega_2 t)\right)$$

速率为：

$$|\mathbf{v}| = \sqrt{A_1^2\omega_1^2\cos^2(\omega_1 t + \phi) + A_2^2\omega_2^2\cos^2(\omega_2 t)}$$

速度为零的点对应曲线的**尖点**（cusps）或**反转点**，出现在 $\cos(\omega_1 t + \phi) = 0$ 和 $\cos(\omega_2 t) = 0$ 同时满足的时刻。

### 3.4 叠加原理的适用边界

利萨茹曲线严格依赖于[[叠加原理]]——即两个方向的振动独立可叠加。这一原理在以下条件下成立：

1. **线性恢复力**：回复力与位移严格线性（如[[胡克定律]]），否则非线性项会导致两个方向运动的耦合。
2. **无耦合**：$x$ 方向和 $y$ 方向的恢复力互不影响，即系统的势能为 $U = \frac{1}{2}k_1 x^2 + \frac{1}{2}k_2 y^2$（可分离变量）。
3. **无阻尼**：阻尼会引入振幅衰减，破坏周期闭合性。

当上述条件不满足时（如非线性耦合振荡器），轨迹将偏离经典利萨茹曲线，可能产生**准周期运动**或[[混沌]]运动。

## 4. 相关验证实验

### 4.1 示波器利萨茹图形

最经典的利萨茹曲线验证方法是使用**示波器**的X-Y模式。将两路正弦信号分别输入示波器的X通道和Y通道，荧光屏上直接显示利萨茹图形。通过调节信号发生器的频率比和相位差，可以实时观察各种形态的利萨茹曲线。这一方法是电子学实验中测量频率比和相位差的标准手段。

### 4.2 双摆简谐近似

在[[弹簧振子_Spring Oscillator]]的小振幅近似下，二维耦合弹簧振子系统（两个正交方向各有一个弹簧振子）的运动即为利萨茹曲线。当两个方向的劲度系数之比为有理数时，振子轨迹形成闭合的利萨茹图形。

### 4.3 光学利萨茹图形

两束正交偏振的激光在非线性介质中传播时，偏振态的演化也可形成利萨茹图形，用于精密相位测量和光学传感。

## 5. 历史上下文

利萨茹曲线以法国物理学家[[朱尔·安托万·利萨茹]]（Jules Antoine Lissajous）的名字命名。他在1857年通过音叉反射光束的偏转首次实验观察并记录了这些曲线，并将成果发表于声学研究文献中。然而，类似的曲线在此前已被美国数学家[[纳撒尼尔·鲍迪奇]]（Nathaniel Bowditch）于1815年研究过，因此在英语文献中亦称**Bowditch曲线**。

利萨茹的研究是19世纪声学实验方法的重要贡献，他将振动从力学现象转化为光学图案，开创了振动可视化的光学方法。这一思路后来被[[恩斯特·克拉尼]]的沙粒图形（[[克拉尼图形_Chladni Patterns]]）以力学方式实现，两者共同构成了振动模式可视化的两大经典范式。

利萨茹曲线在现代科学中仍有广泛应用。在[[弹簧振子_Spring Oscillator]]的二维推广中，利萨茹曲线是分析耦合振动的核心工具；在精密测量中，利萨茹图形的封闭性判据被用于频率标定和相位锁定。从更深层次来看，利萨茹曲线是**准周期运动**（quasiperiodic motion）的几何表示——当频率比为无理数时，轨迹永不闭合却在有限区域内稠密，这一概念在[[哈密顿系统]]的KAM定理和动力系统理论中具有深远意义。
