# 光程极值实验_Optical Path Extremum Experiment

## 1. 实验目的
通过测量光在不均匀介质或折射界面处的实际路径，验证费马原理 "光程取极值"，并将其与力学中的最小作用量原理建立统一联系。

## 2. 实验装置与方法
在透明水槽或浓度梯度糖溶液中引入激光光束，记录光在变折射率介质中的弯曲路径；亦可用棱镜和反射镜测量折射、反射光程。

## 3. 关键结果
实测光线在变折射率介质中遵循 $\delta\int n\,ds = 0$，折射时满足斯涅尔定律 $n_{1}\sin\theta_{1} = n_{2}\sin\theta_{2}$，与费马原理等价；推广到力学即 [[最小作用量原理]] $\delta\int L\,dt = 0$。

## 4. 历史意义
费马原理是 [[最小作用量原理]] 在光学中的原型，奠定了几何光学与分析力学的统一框架，深刻影响了 [[拉格朗日力学]]、[[哈密顿力学]] 与量子力学的路径积分表述。

## 5. 现代复现
现代以梯度折射率光纤、超材料与变换光学实现 "光线弯曲"，并以原子干涉仪直接检验物质波的最小作用量原理。
