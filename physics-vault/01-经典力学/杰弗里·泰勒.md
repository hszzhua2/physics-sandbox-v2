# 杰弗里·泰勒 (Geoffrey Taylor)

## 1. 生平
1886年生于伦敦，在剑桥大学学习并留校。研究流体力学不稳定性与泰勒-库埃特流，二战参与曼哈顿计划。

## 2. 主要贡献
- 研究[[泰勒-库埃特流]]的流体不稳定性
- 发展[[纳维-斯托克斯方程]]的湍流理论
- 研究[[布朗运动]]与涨落

## 3. 关键公式与思想
- $T_a=\dfrac{\Omega^2 R^4}{\nu^2}$（泰勒数）

## 4. 历史影响
泰勒的流体不稳定性研究深化了[[纳维-斯托克斯方程]]的湍流理解。
