# 维塔利·金茨堡 (Vitaly Ginzburg)

## 1. 生平
1916年生于莫斯科，在莫斯科大学学习并任列别捷夫物理研究所教授。与朗道提出超导的GL理论，2003年获诺贝尔物理学奖。

## 2. 主要贡献
- 与朗道建立[[超导]]的金茨堡-朗道理论
- 研究等离子体与宇宙射线

## 3. 关键公式与思想
- $F=\int\left(\alpha|\psi|^2+\dfrac{\beta}{2}|\psi|^4+\dfrac{1}{2m}|(-i\hbar\nabla-2e\mathbf{A})\psi|^2\right)d^3r$

## 4. 历史影响
GL理论是超导唯象理论的杰作，为[[BCS理论]]与拓扑物态研究奠定基础。
