# 阿诺德·索末菲 (Arnold Sommerfeld)

## 1. 生平
1868年生于柯尼斯堡，在哥尼斯堡大学获博士学位，任慕尼黑大学教授。将量子理论引入金属电子论，培养出海森堡、泡利等多位大师。

## 2. 主要贡献
- 发展金属自由电子的[[量子力学]]理论（索末菲模型）
- 提出精细结构常数与椭圆轨道修正

## 3. 关键公式与思想
- $E_F=\dfrac{\hbar^2}{2m}(3\pi^2 n)^{2/3}$（费米能）

## 4. 历史影响
索末菲的金属电子论是[[能带理论]]的先驱，深刻影响[[凝聚态物理]]发展。
