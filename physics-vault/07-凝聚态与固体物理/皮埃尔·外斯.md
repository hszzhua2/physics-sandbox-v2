# 皮埃尔·外斯 (Pierre Weiss)

## 1. 生平
1865年生于米卢斯，在苏黎世联邦理工学院学习并任教，后任斯特拉斯堡大学教授。提出铁磁性的分子场理论。

## 2. 主要贡献
- 提出[[铁磁相变]]的分子场理论与磁畴假说
- 引入外斯温度（居里温度）

## 3. 关键公式与思想
- $M=\dfrac{C}{T-T_c}H$（居里-外斯定律）

## 4. 历史影响
外斯的分子场理论是[[铁磁相变]]唯象理论的先驱，影响了[[相变]]研究。
