# 利昂·库珀 (Leon Cooper)

## 1. 生平
1930年生于纽约，在哥伦比亚大学获博士学位，任布朗大学教授。1956年发现超导中的库珀对机制，1972年获诺贝尔物理学奖。

## 2. 主要贡献
- 发现[[库珀对]]：超导态中电子配对机制
- 与巴丁、施里弗建立[[BCS理论]]

## 3. 关键公式与思想
- $E_k=\sqrt{\xi_k^2+\Delta^2}$（准粒子能谱）

## 4. 历史影响
库珀对概念是[[BCS理论]]的核心，揭示了超导的微观配对机制。
