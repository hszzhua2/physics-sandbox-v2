# 费利克斯·布洛赫 (Felix Bloch)

## 1. 生平
1905年生于苏黎世，在莱比锡大学师从海森堡，后任斯坦福大学教授。1928年提出周期势中电子的布洛赫定理，1952年获诺贝尔物理学奖。

## 2. 主要贡献
- 提出[[布洛赫定理]]描述晶体中电子波函数
- 发展核磁共振方法

## 3. 关键公式与思想
- $\psi_k(r)=u_k(r)e^{ik\cdot r}$（布洛赫波）

## 4. 历史影响
布洛赫定理是[[能带理论]]的基石，解释了固体的导电性与晶体电子结构。
