# 约翰·艾伦 (John Allen)

## 1. 生平
1908年生于加拿大，在多伦多大学与剑桥大学学习，任圣安德鲁斯大学教授。1938年几乎同时发现液氦超流现象。

## 2. 主要贡献
- 几乎同时发现液氦[[超流]]现象
- 研究氦-4的喷泉效应与量子流体输运

## 3. 关键公式与思想
- $\eta\to 0\;(T<2.17\,\mathrm{K})$（超流零黏度）

## 4. 历史影响
艾伦的超流发现与卡皮查并驾齐驱，共同开启了低温[[超流]]研究。
