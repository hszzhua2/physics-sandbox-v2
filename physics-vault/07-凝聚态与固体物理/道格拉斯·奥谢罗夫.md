# 道格拉斯·奥谢罗夫 (Douglas Osheroff)

## 1. 生平
1945年生于华盛顿州，在康奈尔大学师从大卫·李，1972年发现氦-3超流，1996年获诺贝尔物理学奖。

## 2. 主要贡献
- 与李、理查森共同发现[[氦-3超流]]
- 研究低温量子液体相变

## 3. 关键公式与思想
- $T_c(\mathrm{He^3})\approx 2.49\,\mathrm{mK}$

## 4. 历史影响
奥谢罗夫的发现揭示了氦-3中p波超流机制，是[[超流]]研究的里程碑。
