# 神经元网络复杂性实验_Neural Network Complexity Experiment

## 1. 实验目的
研究生物神经网络与人工神经网络的复杂动力学与涌现计算能力，揭示网络拓扑、突触可塑性与认知功能的关系。

## 2. 实验装置与方法
用多电极阵列 (MEA) 记录培养神经元电活动，或 fMRI 测脑区功能连接；人工网络以 Hopfield 网络、深度神经网络为对象，分析临界性、模块化与学习动力学。

## 3. 关键结果
生物皮层网络呈现 "临界雪崩" (neuronal avalanches) 满足幂律 $P(s) \propto s^{-1.5}$；功能网络具有小世界 + 无标度结构；临界态附近信息处理能力最优，与人工神经网络泛化最佳点对应。

## 4. 历史意义
神经元网络研究是 [[复杂性理论]] 在脑科学的核心应用，[[霍普菲尔德]] 1982 年提出 Hopfield 网络奠定神经计算基础，连接复杂网络、临界脑假说与人工智能，催生了现代深度学习。

## 5. 现代复现
现代以类脑芯片、神经形态计算和脑机接口实现大规模仿真，并研究大脑在清醒、睡眠、麻醉状态的临界转变与意识机制。
