# 阿尔文波探测实验_Alfvén Wave Detection Experiment

## 1. 实验目的
在实验室等离子体中产生并探测阿尔文波，验证磁化等离子体中沿磁力线传播的低频剪切波，确立 [[磁流体力学]] (MHD) 的基本波动模式。

## 2. 实验装置与方法
[[阿尔文]] 1942 年理论预言，1949 年 Lundquist 在水银中观测；现代用大型等离子体装置 (如 LAPD、MAGPIE) 或 Q-机器产生等离子体，磁场线圈提供 $B_{0}$，天线激发阿尔文波，磁探针测量。

## 3. 关键结果
测得波速 $v_{A} = B_{0}/\sqrt{\mu_{0}\rho}$ (阿尔文速度)，与理论一致 (典型 $10^{3}-10^{5}$ m/s)；色散关系 $\omega = k_{\parallel}v_{A}$，扰动使磁力线像 "拨动琴弦" 一样振动。

## 4. 历史意义
[[汉内斯·阿尔文]] 因 [[磁流体力学]] 与阿尔文波获 1970 年诺贝尔物理学奖，该波是太阳风、日冕加热、磁层与聚变等离子体能量输运的核心机制，[[托卡马克实验]] 中约束性能受其影响。

## 5. 现代复现
现代以激光干涉、磁探针阵列和卫星 (MMS、Solar Orbiter) 原位观测阿尔文波，研究日冕加热、极光与磁重联。
