# 三体问题 (Three-Body Problem)

## 1. 核心定义

三体问题是研究三个质量质点在相互间[[牛顿万有引力定律]]作用下运动轨迹的动力学问题，其一般情形**不可积**（non-integrable），是经典力学中混沌运动的典型范例。

## 2. 数学表达

### 2.1 运动方程

设三个质点的质量为 $m_1, m_2, m_3$，位置矢量为 $\mathbf{r}_1, \mathbf{r}_2, \mathbf{r}_3$，万有引力常数为 $G$。每个质点受其他两个质点的引力作用，运动方程为：

$$m_i \ddot{\mathbf{r}}_i = -G m_i \sum_{\substack{j=1 \\ j \neq i}}^{3} \frac{m_j (\mathbf{r}_i - \mathbf{r}_j)}{|\mathbf{r}_i - \mathbf{r}_j|^3}, \quad i = 1, 2, 3$$

即：

$$\ddot{\mathbf{r}}_i = -G \sum_{\substack{j=1 \\ j \neq i}}^{3} \frac{m_j (\mathbf{r}_i - \mathbf{r}_j)}{|\mathbf{r}_i - \mathbf{r}_j|^3}$$

这是一组 $18$ 个一阶[[微分方程]]（3个质点 $\times$ 3维空间 $\times$ 位置与速度各3个分量）。

### 2.2 守恒量与可积性

三体系统具有以下守恒量：

| 守恒量 | 表达式 | 守恒定律 |
|--------|--------|---------|
| 总动量 | $\mathbf{P} = \sum_{i=1}^{3} m_i \dot{\mathbf{r}}_i = \text{const}$ | [[动量守恒]] |
| 总角动量 | $\mathbf{L} = \sum_{i=1}^{3} m_i \mathbf{r}_i \times \dot{\mathbf{r}}_i = \text{const}$ | [[角动量守恒]] |
| 总机械能 | $E = \sum_{i=1}^{3} \frac{1}{2}m_i |\dot{\mathbf{r}}_i|^2 - G\sum_{i<j} \frac{m_i m_j}{|\mathbf{r}_i - \mathbf{r}_j|} = \text{const}$ | [[能量守恒定律]] |
| 质心运动 | $\mathbf{R}_{\text{cm}} = \frac{1}{M}\sum_{i=1}^{3} m_i \mathbf{r}_i = \mathbf{V}_0 t + \mathbf{R}_0$ | 质心匀速运动 |

以上共提供 $10$ 个独立积分（动量3 + 角动量3 + 能量1 + 质心运动3），但完整求解 $18$ 阶系统需要 $18$ 个独立积分。[[亨利·庞加莱]]证明了三体问题**不存在足够多的解析积分**（即不可积），剩余的 $8$ 个积分无法用代数函数和初等超越函数表示。

### 2.3 无量纲化

在质心系中消去质心运动后，引入约化变量。定义相对位置矢量 $\mathbf{s}_1 = \mathbf{r}_2 - \mathbf{r}_1$，$\mathbf{s}_2 = \mathbf{r}_3 - \mathbf{r}_1$，系统约化为 $12$ 阶（6个位置分量 + 6个速度分量）。通过特征长度 $a$、特征时间 $T = \sqrt{a^3/(GM)}$（$M = m_1 + m_2 + m_3$）和质量比参数 $\mu_i = m_i / M$ 进行无量纲化，方程仅依赖于质量比参数。

## 3. 物理学意义与推论

### 3.1 庞加莱截面与混沌

[[亨利·庞加莱]]在1889年的研究中引入了**庞加莱截面**（Poincare section）的概念，用以分析三体轨道的长期行为。在相空间中取一个低维截面，记录轨道每次穿过截面时的状态点。如果轨道是周期的，截面上的点集为有限个离散点；如果是准周期的，点集形成光滑闭合曲线；如果轨道是混沌的，点集呈现分形结构的散布区域。

庞加莱发现三体系统在一般情况下，其庞加莱截面上的点集并非离散点或光滑曲线，而是具有复杂分形结构的混沌区域，证明了一般三体运动是**混沌的**（chaotic）。

### 3.2 初值敏感性与Lyapunov指数

三体系统对初始条件具有极端敏感性。两个仅差 $\delta \mathbf{r}_0$ 的初始条件，其轨道偏差随时间指数增长：

$$|\delta \mathbf{r}(t)| \approx |\delta \mathbf{r}_0| \, e^{\lambda t}$$

其中 $\lambda > 0$ 为**最大Lyapunov指数**。$\lambda$ 越大，混沌越强。三体系统的最大Lyapunov指数为正，定量地度量了其不可预测性。实际中，三体系统的轨道预测时间存在**Lyapunov时间**（Lyapunov time）$T_L = 1/\lambda$，超出此时间尺度后预测失效。

### 3.3 限制性三体问题

**限制性三体问题**（Restricted Three-Body Problem）是三体问题的重要简化：第三个质点的质量 $m_3$ 远小于 $m_1$ 和 $m_2$（$m_3 \to 0$），即第三体对前两体的引力反作用可忽略。前两体（主天体）做开普勒二体运动，第三体在主天体的引力场中运动。

在圆型限制性三体问题（CR3BP）中，采用旋转坐标系，引入无量纲参数 $\mu = m_2/(m_1+m_2)$。第三体运动方程为：

$$\ddot{x} - 2\dot{y} = \frac{\partial \Omega}{\partial x}$$

$$\ddot{y} + 2\dot{x} = \frac{\partial \Omega}{\partial y}$$

$$\ddot{z} = \frac{\partial \Omega}{\partial z}$$

其中有效势能为：

$$\Omega(x,y,z) = \frac{1-\mu}{r_1} + \frac{\mu}{r_2} + \frac{1}{2}(x^2 + y^2)$$

$r_1 = \sqrt{(x+\mu)^2 + y^2 + z^2}$，$r_2 = \sqrt{(x-1+\mu)^2 + y^2 + z^2}$。

该系统存在著名的**拉格朗日点**（Lagrange points）$L_1 \sim L_5$——有效势能的平衡点：

- $L_1, L_2, L_3$：共线平衡点（不稳定鞍点）。
- $L_4, L_5$：等边平衡点（当 $\mu < \mu_c \approx 0.0385$ 时稳定）。

Jacobi积分（唯一的运动积分）为：

$$C_J = 2\Omega - (\dot{x}^2 + \dot{y}^2 + \dot{z}^2) = \text{const}$$

Jacobi积分限制了第三体的运动区域，定义了**零速度面**（zero-velocity surfaces），将允许运动区域与禁区分隔。

### 3.4 特解：欧拉共线解与拉格朗日等边解

尽管三体问题一般不可积，但存在几组精确特解：

1. **欧拉共线解**（Euler's collinear solution）：三个质点始终共线，各自绕公共质心做椭圆运动，保持固定的距离比。
2. **拉格朗日等边解**（Lagrange's equilateral solution）：三个质点始终构成等边三角形，各自绕公共质心做椭圆运动。

20世纪后期，更多的精确特解被发现，包括**8字形轨道**（figure-8 orbit，由Chenciner和Montgomery于2000年证明），三个等质量质点沿同一条8字形空间曲线追逐运动。

### 3.5 适用边界

三体问题的牛顿引力框架在以下情形需要修正：

- **强引力场**：当天体接近中子星或黑洞时，需用[[广义相对论]]替代牛顿引力。
- **高速运动**：相对论效应显著时，牛顿力学不再适用。
- **潮汐效应**：有限大小的天体存在内部形变和潮汐耗散，超出质点模型。

## 4. 相关验证实验

### 4.1 数值模拟验证

由于三体问题不可解析求解，数值积分是主要的验证手段。Runge-Kutta方法、辛积分器和Gauchy-Jacobi方法被广泛用于高精度三体轨道计算。数值结果证实了庞加莱关于混沌的预测，轨道对初始条件的指数敏感性在模拟中被直接观测。

### 4.2 系外行星系统

多行星恒星系统（如TRAPPIST-1系统拥有7颗行星）是三体及多体问题的自然实验室。行星轨道的长期稳定性分析直接依赖于限制性多体问题的理论框架。

### 4.3 航天器轨道设计

拉格朗日点 $L_1$ 和 $L_2$ 被广泛用作空间观测平台的驻留位置。太阳-地球系统的 $L_2$ 点是[[引力波探测_Gravitational Wave Detection]]中LISA空间引力波探测器的预定轨道位置。

## 5. 历史上下文

三体问题的历史是现代动力系统理论诞生的历史。[[牛顿]]在《自然哲学的数学原理》（1687年）中精确解决了二体问题（开普勒椭圆），但三体问题始终悬而未决。19世纪末，瑞典国王奥斯卡二世设立奖金征集三体问题的解。[[亨利·庞加莱]]提交的论文虽然未能给出一般解，但发现了三体系统中的混沌行为和**同宿交错**（homoclinic tangles），奠定了现代混沌理论的基础。庞加莱因此获奖，但论文出版后他发现了一个重大错误并自费召回重印——这一错误恰恰是混沌发现的关键契机。

庞加莱的工作揭示了确定性系统中的不可预测性，挑战了拉普拉斯的决定论世界观。这一发现深刻影响了20世纪的科学哲学。在物理学知识库中，三体问题的混沌性质与[[混沌双摆_Chaotic Double Pendulum]]所展现的机械混沌运动一脉相承，其相空间结构的分析方法与[[洛伦兹吸引子_Lorenz Attractor]]所代表的耗散系统混沌既有共性又有差异——三体系统是保守哈密顿系统中的混沌，而洛伦兹系统是耗散系统中的混沌，两者共同构成了混沌动力学的两大范式。三体问题属于"未完全解决问题"——一般三体的精确通解尚未获得，仅存在有限特解和数值方法。
