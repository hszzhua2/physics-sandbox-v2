# 电磁场张量验证实验_Electromagnetic Tensor Verification Experiment

## 1. 实验目的
通过检验电磁场在不同惯性系间的变换规律，验证电磁场张量 $F_{\mu\nu}$ 与狭义相对论协变性，确认电场与磁场是同一物理量的不同分量。

## 2. 实验装置与方法
在不同运动参考系中测量运动电荷的电磁场，或观测相对论性电子在电磁场中的运动；用亥姆霍兹线圈与平板电容产生已知 $\mathbf{E},\mathbf{B}$，配合相对论性粒子束检验洛伦兹力 $f^{\mu} = qF^{\mu\nu}u_{\nu}$。

## 3. 关键结果
测得运动电荷的电场 $E_{\parallel}' = E_{\parallel}$、$E_{\perp}' = \gamma(E_{\perp} + \mathbf{v}\times\mathbf{B})$，磁场 $B_{\perp}' = \gamma(B_{\perp} - \mathbf{v}\times\mathbf{E}/c^{2})$，与 [[闵可夫斯基]] 张量变换一致，验证 $F_{\mu\nu}$ 的协变性。

## 4. 历史意义
电磁场张量是 [[麦克斯韦方程组]] 的相对论协变表述，[[爱因斯坦]] 与 [[闵可夫斯基]] 把电与磁统一为时空张量，是 [[规范场论验证实验]] 与 [[广义相对论]] 张量方法的物理原型。

## 5. 现代复现
现代以相对论性电子加速器、激光等离子体与相对论性离子物理检验电磁场张量的高阶修正，并为量子电动力学 (QED) 提供基础。
