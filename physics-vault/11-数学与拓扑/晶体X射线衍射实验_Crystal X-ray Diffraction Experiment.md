# 晶体X射线衍射实验_Crystal X-ray Diffraction Experiment

## 1. 实验目的
利用 X 射线在晶体中的衍射揭示晶体的空间群对称性与原子结构，验证晶体点阵理论并测定原子间距。

## 2. 实验装置与方法
1912 年 [[劳厄]] 在慕尼黑用 Cu 靶 X 射线照射硫酸铜单晶，底片记录衍射斑点；[[布拉格]] 父子用单色 X 射线与可转晶体测衍射角。

## 3. 关键结果
劳厄观测到规则衍射斑点，证明 X 射线是电磁波且晶体为周期点阵；[[布拉格]] 给出 $2d\sin\theta = n\lambda$，测得 NaCl 离子间距 $d \approx 0.282\,$nm，奠定晶体结构测定基础。

## 4. 历史意义
[[晶体X射线衍射实验]] 把 [[群论]] 中空间群对称性引入结构测定，[[劳厄]] 1914 年、[[布拉格]] 1915 年获诺贝尔奖，开创 X 射线晶体学，是 DNA 双螺旋、蛋白质结构与 [[粒子物理对称性实验]] 的方法论源头。

## 5. 现代复现
现代以同步辐射、自由电子激光与冷冻电镜实现原子分辨与飞秒时间分辨结构测定，并发展电子衍射与中子衍射互补。
