# 拓扑学 (Topology)

## 1. 核心定义

研究在连续变形（拉伸、弯曲、扭转但不撕裂或粘合）下保持不变的几何性质的数学分支，核心概念为拓扑空间与拓扑不变量。

## 2. 数学表达

### 拓扑空间

集合 $X$ 配合开集族 $\tau$ 构成拓扑空间，若：
1. $\emptyset, X \in \tau$
2. 任意并集属于 $\tau$
3. 有限交集属于 $\tau$

### 连续映射与同胚

$f: X \to Y$ 连续当且仅当 $Y$ 中任意开集的原像为 $X$ 中开集。

同胚：$f$ 为双射且 $f$ 与 $f^{-1}$ 均连续。同胚的拓扑空间拓扑等价。

### 同伦

两映射 $f, g: X \to Y$ 同伦（$f \simeq g$），若存在连续族 $H: X \times [0,1] \to Y$ 使 $H(\cdot, 0) = f$，$H(\cdot, 1) = g$。

空间同伦等价：$X \simeq Y$ 当存在 $f: X \to Y$，$g: Y \to X$ 使 $g\circ f \simeq \text{id}_X$，$f\circ g \simeq \text{id}_Y$。

### 基本群

$$\pi_1(X, x_0) = \{\text{基于 } x_0 \text{ 的环路}\}/\text{同伦等价}$$

群运算为环路拼接。$\pi_1 = 0$（平凡群）则空间单连通。

高阶同伦群 $\pi_n(X)$ 定义为 $S^n \to X$ 的同伦类集合。

### 同调群

链复形 $C_n \xrightarrow{\partial_n} C_{n-1} \to \cdots$，$\partial_{n-1}\circ\partial_n = 0$。

$$H_n(X) = \ker(\partial_n)/\text{im}(\partial_{n+1})$$

贝蒂数 $b_n = \text{rank}\,H_n(X)$，度量 $n$ 维"洞"的个数。

### 拓扑不变量

- 连通性（连通分支数）
- 紧致性
- 欧拉示性数 $\chi$，参见[[欧拉示性数_Euler Characteristic]]
- 基本群 $\pi_1$ 与同伦群 $\pi_n$
- 贝蒂数 $b_n$
- 示性类（陈类、Stiefel-Whitney类、Pontryagin类）

### 高斯-博内定理（拓扑-几何联系）

$$\int_M K\,dA = 2\pi\chi(M)$$

将曲率（几何）与欧拉示性数（拓扑）联系起来。

**参数说明**：
- $X$：拓扑空间
- $\tau$：开集族
- $\pi_1(X, x_0)$：基本群
- $H_n(X)$：第 $n$ 阶同调群
- $b_n$：第 $n$ 个贝蒂数
- $\chi(M)$：欧拉示性数

## 3. 物理学意义与推论

拓扑学在现代物理中的影响日益深刻，主要体现在以下方面：

**拓扑相变**：[[戴维·索利斯]]、[[邓肯·霍尔丹]]与[[迈克尔·科斯特利茨]]因发现物质的拓扑相变与拓扑相获2016年诺贝尔物理学奖。拓扑相由拓扑不变量（如陈数、TKNN不变量）标记，具有"全局"特征，不随局部微扰改变。

**拓扑缺陷**：宇宙弦、磁单极子、畴壁与涡旋由同伦群 $\pi_n(G/H)$ 分类（Kibble机制），在相变中自发产生。

**贝里相位**：量子态在参数空间中巡回一周获得的几何相位，其积分给出第一陈数，是[[拓扑绝缘体]]与量子霍尔效应的拓扑根源。

**拓扑量子计算**：利用非阿贝尔任意子（如马约拉纳费米子）的辫子群表示实现容错量子计算（属于假说/未解决问题，实验尚未确证）。

**拓扑量子场论**：[[爱德华·威滕]]将拓扑学引入量子场论，提出拓扑场论（如Chern-Simons理论），关联纽结不变量（Jones多项式）。

**适用边界**：
- 拓扑学关注全局性质，不提供局部几何细节（如曲率）。
- 拓扑不变量是离散的整数或群，难以直接对应连续物理量。
- 拓扑保护效应在有限温度或强退相干下可能被破坏。
- 非平衡拓扑相与含时系统拓扑分类仍在发展中（属于未解决问题）。
- 拓扑量子计算的实用化尚属假说阶段。

## 4. 相关验证实验

- [[整数量子霍尔效应实验_Integer Quantum Hall Effect Experiment]]：[[克劳斯·冯·克利青]]发现霍尔电导平台精确量子化为 $e^2/h$ 的整数倍，由拓扑不变量（第一陈数）解释。
- [[分数量子霍尔效应实验_Fractional Quantum Hall Effect Experiment]]：[[霍斯特·施特默]]、[[崔琦]]与[[罗伯特·劳克林]]发现分数量子化平台，对应拓扑序。
- [[拓扑绝缘体实验_Topological Insulator Experiment]]：实验观测到受拓扑保护的自旋极化表面态。
- [[马约拉纳费米子探测实验_Majorana Fermion Detection Experiment]]：寻找拓扑量子计算的候选任意子（仍属未解决问题）。
- [[宇宙弦探测实验_Cosmic String Detection Experiment]]：拓扑缺陷的宇宙学探测（尚无确证，属假说）。
- [[超流氦涡旋实验_Superfluid Helium Vortex Experiment]]：涡旋数量由拓扑绕数决定。

## 5. 历史上下文

拓扑学起源于[[莱昂哈德·欧拉]]1736年的哥尼斯堡七桥问题与1750年的多面体公式（[[欧拉示性数_Euler Characteristic]]）。1851年，[[黎曼]]在复分析中引入"黎曼面"与连通性的概念。1895年，[[亨利·庞加莱]]发表《位置分析》（*Analysis Situs*），系统建立代数拓扑，引入基本群与同调概念，标志拓扑学成为独立学科。

20世纪，[[所罗门·莱夫谢茨]]、[[奥斯瓦尔德·维布伦]]与[[詹姆斯·亚历山大]]发展同调论；[[维托尔德·胡列维茨]]1935年引入高阶同伦群。1930—1950年代，[[塞缪尔·艾伦伯格]]与[[桑德斯·麦克莱恩]]建立范畴论，[[埃伦费斯特]]与[[勒雷]]发展上同调与谱序列。

1970年代，[[索利斯]]等将拓扑学引入凝聚态物理。1980年代，[[冯·克利青]]发现量子霍尔效应，[[罗伯特·劳克林]]与[[邓肯·霍尔丹]]揭示其拓扑本质。1990年代，[[威滕]]的拓扑场论与[[摩诃·朱玛]]等的高温超导拓扑分类推动拓扑物理蓬勃发展。21世纪，[[拓扑绝缘体]]、[[外尔半金属]]与[[马约拉纳费米子]]的探索成为凝聚态物理的前沿，拓扑学已从纯数学跃升为物理学不可或缺的支柱。
