# 宇宙弦探测实验_Cosmic String Detection Experiment

## 1. 实验目的
寻找早期宇宙对称性破缺产生的拓扑缺陷 (宇宙弦) 留下的观测信号，检验相变宇宙学与拓扑缺陷产生机制。

## 2. 实验装置与方法
用 PTA 脉冲星计时阵列、CMB 各向异性 (Planck)、引力波 (NANOGrav、LIGO) 与强引力透镜巡天搜寻宇宙弦产生的引力波背景与双像。

## 3. 关键结果
目前未直接探测到宇宙弦；Planck 给出弦张力上限 $G\mu/c^{2} < 10^{-7}$，NANOGrav 2023 年引力波背景可解释为宇宙弦贡献但仍有争议。

## 4. 历史意义
宇宙弦是 [[拓扑学]] 在宇宙学的体现，[[基布尔]] 提出 Kibble 机制，[[宇宙弦探测实验]] 检验早期宇宙相变与缺陷形成理论，约束大统一与新物理能标。

## 5. 现代复现
现代以 SKA 脉冲星计时阵列、LISA 空间引力波探测器与 21 cm 巡天进一步约束宇宙弦，并研究其在 [[引力波探测实验]] 中的可能信号。
