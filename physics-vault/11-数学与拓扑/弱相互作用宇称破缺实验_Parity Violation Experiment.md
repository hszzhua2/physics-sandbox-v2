# 弱相互作用宇称破缺实验_Parity Violation Experiment

## 1. 实验目的
检验弱相互作用是否保持宇称 (左右镜像) 对称性，验证 [[杨振宁]] 与 [[李政道]] 1956 年关于弱作用宇称可能破缺的预言。

## 2. 实验装置与方法
理论分析 θ-τ 介子之谜；实验上由 [[吴健雄]] 用 ⁶⁰Co β 衰变验证 (见 [[吴健雄实验]])；后续 Garwin、Friedman、Telegdi 用 π-μ-e 衰变链复核。

## 3. 关键结果
1957 年多个实验一致证明弱相互作用中宇称不守恒：⁶⁰Co 衰变电子优先沿自旋反方向发射 (不对称度 ~40%)；π→μ→e 衰变中 μ 子自旋极化。最大宇称破缺对应 V-A 结构。

## 4. 历史意义
[[杨振宁]] 与 [[李政道]] 因预言弱作用宇称破缺获 1957 年诺贝尔奖，[[弱相互作用宇称破缺实验]] 推翻宇称守恒教条，建立 V-A 弱作用理论，是 [[群论]] 中对称性破缺思想的里程碑，影响 [[CP破缺实验]] 与标准模型。

## 5. 现代复现
现代以原子宇称不守恒实验 (如 APV) 精确测量弱核子流，检验 [[规范场论验证实验]] 与新物理。
