# 马约拉纳费米子探测实验_Majorana Fermion Detection Experiment

## 1. 实验目的
在固体系统中寻找其反粒子即其自身的马约拉纳费米子 (以准粒子形式)，验证拓扑超导预言的零能边缘态，为拓扑量子计算提供容错比特。

## 2. 实验装置与方法
在拓扑绝缘体/超导异质结、铁原子链/铅超导或半导体纳米线-超导耦合系统中，用扫描隧道显微镜 (STM) 测零偏压电导峰，或用库仑阻塞谱学。

## 3. 关键结果
2012 年 [[傅亮]] 等预言，2014 年普林斯顿 [[雅茨比]] 团队在铁原子链/铅超导观测到零能电导峰；2022 年微软在纳米线-超导器件中观测到拓扑超导信号。但仍存在争议。

## 4. 历史意义
[[马约拉纳]] 1937 年预言该粒子，[[傅亮]] 把 [[拓扑学]] 引入超导，[[马约拉纳费米子探测实验]] 是 [[分数量子霍尔效应实验]] 与 [[拓扑绝缘体实验]] 的延伸，是实现容错 [[量子计算复杂性实验]] 的关键。

## 5. 现代复现
现代以更高精度 STM、Majorana 纳米线网络与拓扑超导量子比特推进，并寻找中微子是否为马约拉纳粒子的双 β 衰变证据。
