# 伊戈尔·塔姆 (Igor Tamm)

## 1. 生平
1895年生于海参崴，在莫斯科大学学习并任教。与弗兰克解释切伦科夫辐射，1958年获诺贝尔物理学奖。

## 2. 主要贡献
- 与弗兰克解释[[切伦科夫辐射]]
- 研究[[磁约束聚变]]的托卡马克概念
- 发展量子力学与核物理

## 3. 关键公式与思想
- $\cos\theta=\dfrac{c}{nv}$（切伦科夫角）

## 4. 历史影响
塔姆的切伦科夫辐射理论开辟了粒子探测新途径，并奠基了[[磁约束聚变]]理论。
