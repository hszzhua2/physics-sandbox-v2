# 杰拉德·特·胡夫特 (Gerard 't Hooft)

## 1. 生平
1946年生于登海尔德，在乌得勒支大学师从韦尔特曼，证明杨-米尔斯理论可重整化，1999年获诺贝尔物理学奖。

## 2. 主要贡献
- 证明[[标准模型]]杨-米尔斯规范理论的可重整化
- 研究[[量子色动力学]]与全息原理

## 3. 关键公式与思想
- $\text{Yang-Mills theory is renormalizable}$

## 4. 历史影响
特胡夫特的可重整化证明使[[标准模型]]成为自洽的量子场论，是粒子物理的重大突破。
