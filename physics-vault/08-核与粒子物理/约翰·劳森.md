# 约翰·劳森 (John Lawson)

## 1. 生平
1923年生于利物浦，在剑桥大学学习，任英国原子能管理局研究员。提出聚变等离子体的劳森判据。

## 2. 主要贡献
- 提出[[磁约束聚变]]的劳森判据
- 研究等离子体物理与聚变条件

## 3. 关键公式与思想
- $n\tau_E\gtrsim\dfrac{3k_BT}{\langle\sigma v\rangle E_\alpha}$（劳森判据）

## 4. 历史影响
劳森判据定义了实现聚变点火的关键条件，是[[磁约束聚变]]的核心准则。
