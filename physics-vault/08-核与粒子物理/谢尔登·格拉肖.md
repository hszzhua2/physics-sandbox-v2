# 谢尔登·格拉肖 (Sheldon Glashow)

## 1. 生平
1932年生于纽约，在哈佛大学获博士学位，任波士顿大学教授。与温伯格、萨拉姆共同建立电弱统一理论，1979年获诺贝尔物理学奖。

## 2. 主要贡献
- 与温伯格、萨拉姆建立电弱统一理论
- 提出[[夸克模型]]的粲夸克预言

## 3. 关键公式与思想
- $SU(2)_L\times U(1)_Y$（电弱规范对称性）

## 4. 历史影响
格拉肖的电弱统一理论是[[标准模型]]的核心，统一了两种基本相互作用。
