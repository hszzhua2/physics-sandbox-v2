# 光程测量实验_Optical Path Measurement Experiment

## 1. 实验目的
精确测量光在不同介质中传播的光程 $\int n\,ds$，验证折射率定义并校准干涉测量中的光程差，奠定精密光学测量基础。

## 2. 实验装置与方法
迈克耳孙干涉仪分出一束参考光与一束通过待测介质 (玻璃板、气体室) 的测量光，补偿器保证两路在空气中光程相等；数干涉条纹移动量 $\Delta N$ 反推介质光程。

## 3. 关键结果
插入厚度 $d$、折射率 $n$ 的介质后，光程增加 $\Delta L = (n-1)d$，对应条纹移动 $\Delta N = 2(n-1)d/\lambda$。由此可测得气体折射率精度 $\Delta n \sim 10^{-7}$。

## 4. 历史意义
光程测量是 [[费马原理]] 与干涉学的结合，[[迈克耳孙]] 据此精确测定光速和米原器长度，奠定现代精密计量、光谱学和引力波探测 ([[引力波探测实验]]) 的基础。

## 5. 现代复现
现代以飞秒光频梳和光学频率计实现纳米级光程测量，并用于引力波探测器 LIGO 中 $10^{-19}$ 量级的应变测量。
