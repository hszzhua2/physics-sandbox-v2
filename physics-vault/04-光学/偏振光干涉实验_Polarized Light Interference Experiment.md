# 偏振光干涉实验_Polarized Light Interference Experiment

## 1. 实验目的
研究两束正交偏振光能否发生干涉，揭示偏振方向对干涉条件的影响，确立光波横波性质并验证菲涅耳-阿拉戈定律。

## 2. 实验装置与方法
自然光经起偏器后通过双折射晶片 (如石英) 产生两束正交偏振的相干光，再用检偏器将它们投影到同一偏振方向观察干涉；也可用渥拉斯顿棱镜分光后再复合。

## 3. 关键结果
实验发现：振动方向相互垂直的两束偏振光即使相干也不产生干涉；只有经检偏器投影到同一方向后才会干涉。条纹光强 $I = I_{0}\cos^{2}(\delta/2)$，$\delta$ 为相位差。

## 4. 历史意义
菲涅耳-阿拉戈定律直接证明光是横波，推翻了 [[牛顿]] 的纵波说，为 [[菲涅耳]] 的波动光学理论提供关键证据，奠定了偏振干涉量度、光弹性应力分析和液晶显示的基础。

## 5. 现代复现
现代以偏振光干涉实现应力检测 (光弹性)、晶体取向分析、相衬显微和偏振光学相干断层扫描 (PS-OCT)。
