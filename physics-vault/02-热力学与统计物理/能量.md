# 能量

## 1. 核心定义
能量：做功能力的量度。该概念是热力学与统计物理的重要内容，涉及[[热力学四大定律]]等关联主题。

## 2. 数学表达
核心公式：
$E=T+V$

该公式表达了能量的定量关系，其中各物理量的含义取决于具体物理情境。

## 3. 物理意义
能量揭示了热力学与统计物理中的基本物理规律。做功能力的量度，这一关系深刻反映了自然界的内在结构。该概念是构建物理学理论体系的重要组成部分。

## 4. 历史发展
能量由[[牛顿]]于1687年提出。这一概念的建立是热力学与统计物理发展的重要里程碑，[[牛顿]]通过理论分析和实验验证确立了其正确性。此后该概念被不断完善和推广。

## 5. 应用与拓展
能量在现代物理学和工程技术中有广泛应用。该概念是热力学与统计物理理论分析和实际计算的基础工具。
