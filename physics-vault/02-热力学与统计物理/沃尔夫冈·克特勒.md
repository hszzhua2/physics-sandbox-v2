# 沃尔夫冈·克特勒 (Wolfgang Ketterle)

## 1. 生平
1957年生于海德堡，在慕尼黑工业大学获博士学位，任麻省理工学院教授。实现钠原子BEC，2001年获诺贝尔物理学奖。

## 2. 主要贡献
- 独立实现[[玻色-爱因斯坦凝聚]]并观测原子干涉
- 研究[[超流]]与冷原子

## 3. 关键公式与思想
- $n\lambda_{dB}^3>2.612$（BEC条件）

## 4. 历史影响
克特勒的BEC实验深化了[[玻色-爱因斯坦统计]]与[[超流]]研究。
