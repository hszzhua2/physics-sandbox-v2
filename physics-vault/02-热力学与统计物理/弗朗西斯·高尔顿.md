# 弗朗西斯·高尔顿 (Francis Galton)

## 1. 生平
1822年生于伯明翰，在剑桥大学学习，达尔文表弟。研究统计与遗传，发明高尔顿钉板演示中心极限定理。

## 2. 主要贡献
- 发明[[高尔顿钉板]]演示[[统计物理]]的中心极限定理
- 发展回归与相关统计方法

## 3. 关键公式与思想
- $P(x)\to\dfrac{1}{\sqrt{2\pi}\sigma}e^{-x^2/2\sigma^2}$（高尔顿钉板→正态分布）

## 4. 历史影响
高尔顿钉板直观演示了中心极限定理，是[[统计物理]]与概率论的经典实验。
