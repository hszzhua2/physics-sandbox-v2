# 涨落耗散定理 (Fluctuation-Dissipation Theorem)

## 1. 核心定义

[[涨落耗散定理]]是[[统计物理]]中连接系统在热平衡态下自发涨落与外部微扰引起的耗散响应的基本定理，由[[赫伯特·卡伦]]和[[中野藤男]]于1951年严格建立，是[[线性响应理论]]的核心推论。

## 2. 数学表达

### 经典形式（第一涨落耗散定理）

设可观测量 $A$ 在热平衡态下的自发涨落关联函数为：

$$C_{AA}(t) = \langle \delta A(t)\,\delta A(0) \rangle_{\text{eq}}$$

其中 $\delta A(t) = A(t) - \langle A \rangle_{\text{eq}}$。若对系统施加外场 $F(t)$，哈密顿量变为 $H = H_0 - F(t) A$，则 $A$ 的线性响应为：

$$\delta \langle A(t) \rangle = \int_{-\infty}^{t} \chi_{AA}(t-t')\,F(t')\,dt'$$

其中 $\chi_{AA}(t)$ 为[[响应函数]]。涨落耗散定理给出：

$$\chi_{AA}(t) = -\beta \frac{d}{dt} C_{AA}(t) \cdot \Theta(t), \quad t > 0$$

其中 $\beta = 1/k_B T$，$\Theta(t)$ 为[[亥维赛阶跃函数]]。

### 频域形式

对关联函数进行傅里叶变换得[[谱密度]]（功率谱）：

$$S_{AA}(\omega) = \int_{-\infty}^{\infty} C_{AA}(t)\, e^{i\omega t}\,dt$$

定义复响应率 $\chi_{AA}(\omega) = \chi'(\omega) + i\chi''(\omega)$，则涨落耗散定理在频域为：

$$S_{AA}(\omega) = \frac{2\hbar}{1 - e^{-\beta\hbar\omega}} \,\chi''_{AA}(\omega)$$

经典极限（$\hbar\omega \ll k_BT$）下简化为：

$$S_{AA}(\omega) = \frac{2k_BT}{\omega}\,\chi''_{AA}(\omega)$$

### 量子形式（Kubo形式）

[[久保亮五]]给出的广义量子涨落耗散关系：

$$S_{AA}(\omega) = \hbar \coth\!\left(\frac{\beta\hbar\omega}{2}\right) \chi''_{AA}(\omega)$$

### 约翰逊-奈奎斯特噪声

对电阻 $R$ 在温度 $T$ 下的热噪声（[[约翰逊噪声]]），谱密度为：

$$S_V(\omega) = 2R\,k_BT \quad \text{（单边谱）}$$

即 $V^2 = 4k_BT R \,\Delta f$，直接由涨落耗散定理导出。

## 3. 物理学意义与推论

- 涨落耗散定理揭示了平衡态系统的一个深刻对称性：**自发涨落**与**耗散响应**是同一物理过程的两个侧面。系统对微扰的耗散（如电阻、黏滞、摩擦）完全由其平衡态涨落性质决定。这是[[线性响应理论]]的核心结论。
- 该定理是[[布朗运动_Brownian Motion]]中[[爱因斯坦关系]]（扩散系数 $D$ 与迁移率 $\mu$ 的关系 $D = \mu k_BT$）的一般化推广。[[朗之万方程_Langevin Equation]]中阻尼系数与随机力强度通过[[涨落耗散定理]]的[[第二涨落耗散定理]]精确关联：$\langle \xi(t)\xi(t') \rangle = 2\gamma k_BT\,\delta(t-t')$。
- **约翰逊噪声**（1928）是FDT在电路中的直接体现：电阻的热噪声谱 $S_V = 4k_BT R$ 仅有温度和电阻决定，与材料细节无关。该结果由[[约翰逊]]实验发现并由[[奈奎斯特]]理论推导。
- FDT在[[凝聚态物理]]中有广泛应用：从[[中子散射]]谱的动力学结构因子到[[介观物理]]中的[[量子霍尔效应]]输运，从[[自旋玻璃]]的弛豫到[[玻璃化转变]]中的非谐振响应，FDT均为联系涨落与响应的核心工具。
- FDT在远离平衡态时不再严格成立，其适用范围界定为**线性响应区域**（扰动小到可线性化）。在非线性响应领域（如[[非平衡态热力学]]），需要推广，目前仍是活跃研究方向（属于未解决问题）。
- FDT与[[帕隆多悖论_Parrondo Paradox]]的联系在于：FDT建立了平衡态与线性响应的桥接，而[[帕隆多悖论_Parrondo Paradox]]展示了在非平衡态时两个"亏损"策略可组合产生"盈利"，暗示非平衡统计中涨落与耗散关系可呈现反直觉行为，是FDT框架之外的现象。

## 4. 相关验证实验

- **约翰逊噪声测量**（1928，[[约翰逊]]实验 / [[奈奎斯特]]理论）：测量电阻两端的热噪声电压，直接验证 $S_V = 4k_BT R$，是FDT最早的实验验证之一。
- **[[布朗运动_Brownian Motion]]观测**（1905理论，1908实验[[让·佩兰]]）：[[爱因斯坦]]推导的 $D = \mu k_BT$ 是涨落耗散关系的最早实例，将扩散系数（涨落）与迁移率（耗散）联系起来。[[让·佩兰]]的布朗粒子实验为此提供了精确验证。
- **中子散射谱**：非弹性中子散射测得的动力学结构因子 $S(\vec{q},\omega)$ 与响应函数虚部 $\chi''(\vec{q},\omega)$ 通过FDT直接关联，是研究凝聚态系统元激发的标准工具。
- **介观电导涨落**：在介观导体中，通过测量电导涨落的谱密度与交流阻抗的实部，可在量子区域验证广义量子FDT（Kubo形式）。
- **光镊实验中的微球涨落**：用光阱束缚的微米小球在流体中的位移涨落与拖曳响应满足FDT，可用于标定光阱刚度或介质的复剪切模量（[[流变学]]应用）。

## 5. 历史上下文

- [[涨落耗散定理]]的起源可追溯至[[爱因斯坦]]1905年对[[布朗运动_Brownian Motion]]的研究，其中首次给出涨落（扩散）与耗散（摩擦）的定量关系 $D = k_BT / \gamma$。
- 1928年[[约翰逊]]和[[奈奎斯特]]独立发现电阻热噪声并给出理论解释，是FDT的早期电路实例。
- 1951年，[[赫伯特·卡伦]]和[[中野藤男]]在一般[[统计物理]]框架下严格证明了涨落耗散定理，将其推广至任意可观测量和量子情形。卡伦-中野定理是[[线性响应理论]]的基石。
- 1957年，[[久保亮五]]给出FDT的量子推广形式（Kubo公式），并建立了[[线性响应理论]]的一般框架，成为现代非平衡态统计物理的基础。
- FDT深刻影响了后续发展：[[涨落定理]]（[[埃文斯]]-[[斯波尔]]，1993，[[加利亚沃蒂]]-[[科恩]]，1995）将其推广至远离平衡的有限时间涨落，是21世纪[[非平衡态热力学]]的重要进展。
- 在现代[[量子信息]]与[[量子热力学]]领域，量子FDT用于刻画量子系统的退相干、量子测量反作用等过程，是连接量子噪声与耗散的基本工具。
