# 朗之万方程 (Langevin Equation)

## 1. 核心定义

[[朗之万方程]]是描述随机力作用下粒子运动的[[随机微分方程]]，由[[保罗·朗之万]]于1908年建立，用以描述[[布朗运动_Brownian Motion]]，是[[统计物理]]中处理随机过程的基本工具，揭示了确定性运动与随机涨落的共存。

## 2. 数学表达

### 经典朗之万方程

对质量为 $m$ 的粒子受黏滞阻尼和随机力 $\xi(t)$ 作用，其运动方程为：

$$m\ddot{x}(t) = -\gamma \dot{x}(t) + \xi(t)$$

其中：
- $m$：粒子质量；
- $\gamma$：黏滞阻尼系数（摩擦系数）；
- $-\gamma \dot{x}$：黏滞力，代表与速度成正比的耗散；
- $\xi(t)$：[[随机力]]，表示来自流体分子的随机碰撞。

### 随机力的白噪声假设

[[随机力]] $\xi(t)$ 通常假设为[[高斯白噪声]]，其统计性质为：

$$\langle \xi(t) \rangle = 0$$

$$\langle \xi(t)\,\xi(t') \rangle = 2\gamma k_B T\,\delta(t - t')$$

第二个条件即[[涨落耗散定理_Fluctuation-Dissipation Theorem]]（第二FDT），将随机力强度与阻尼系数和温度关联。强度 $2\gamma k_BT$ 确保粒子在长时间后达到[[麦克斯韦-玻尔兹曼分布_Maxwell-Boltzmann Distribution]]。

### 过阻尼极限

当惯性项可忽略（$m\ddot{x} \ll \gamma\dot{x}$）时，得到过阻尼朗之万方程：

$$\gamma \dot{x}(t) = \xi(t)$$

此时速度直接由随机力决定，运动为纯扩散。

### 福克-普朗克方程对应

朗之万方程可等价转化为[[福克-普朗克方程]]，描述概率密度 $P(x, t)$ 的演化。对一维过阻尼朗之万方程，对应的福克-普朗克方程为：

$$\frac{\partial P}{\partial t} = \frac{\partial}{\partial x}\left[\frac{\gamma}{m} \frac{\partial V}{\partial x} P\right] + D \frac{\partial^2 P}{\partial x^2}$$

其中扩散系数 $D = k_B T / \gamma$ 即[[爱因斯坦关系]]。含势场 $V(x)$ 的情形给出[[斯莫卢霍夫斯基方程]]。

### 广义朗之万方程

考虑记忆效应（[[非马尔可夫过程]]），引入[[记忆核]] $\gamma(t)$：

$$m\ddot{x}(t) = -\int_{-\infty}^{t} \gamma(t-t')\,\dot{x}(t')\,dt' + \xi(t)$$

此时随机力满足广义涨落耗散关系：

$$\langle \xi(t)\,\xi(t') \rangle = k_B T\,\gamma(|t - t'|)$$

## 3. 物理学意义与推论

- 朗之万方程首次将确定性力学与随机过程统一在同一方程中，是[[统计物理]]中处理随机现象的基本范式。它将[[牛顿运动定律_Newton's Laws of Motion]]从决定论推广为概率论框架，为[[随机微分方程]]理论提供了物理原型。
- 方程中阻尼项 $-\gamma\dot{x}$ 与随机力 $\xi(t)$ 通过[[涨落耗散定理_Fluctuation-Dissipation Theorem]]精确关联，是[[涨落耗散定理]]的具体体现。这一关联保证了系统在长时间后达到热平衡（[[细致平衡]]），而非持续偏离平衡。
- 从朗之万方程可导出[[布朗运动_Brownian Motion]]的统计性质：粒子位移方差随时间线性增长 $\langle x^2(t) \rangle = 2Dt$，扩散系数 $D = k_BT/\gamma$，正是[[爱因斯坦]]1905年得到的核心结果。
- **短时间惯性区**：当 $t \ll m/\gamma$ 时，粒子仍接近弹道运动 $\langle x^2 \rangle \sim (k_BT/m) t^2$；长时间后进入扩散区 $\langle x^2 \rangle \sim 2Dt$。两个区域的交叉由弛豫时间 $\tau = m/\gamma$ 标记。
- 朗之万方程在[[凝聚态物理]]、[[化学物理]]、[[生物物理]]中有广泛应用：描述高分子链构象动力学、蛋白质折叠、化学反应随机性、细胞内输运、神经元放电等。
- **量子朗之万方程**：[[光场]]与原子相互作用的量子化推广，[[输入-输出理论]]中用于描述开放量子系统，是[[量子光学]]和[[量子计算]]中处理退相干的核心工具。
- 过阻尼朗之万方程是[[随机过程]]中[[维纳过程]]（Wiener process）的物理来源，与[[伊藤引理]]和[[随机微积分]]（[[伊藤清]]）直接关联。

## 4. 相关验证实验

- **[[布朗运动_Brownian Motion]]观测**（1827发现，1905理论，1908实验[[让·佩兰]]）：[[让·佩兰]]通过显微镜观测胶体粒子的布朗运动，验证了[[爱因斯坦]]的扩散公式 $D = k_BT/\gamma$，从而间接验证了朗之万方程的统计预言。[[让·佩兰]]因此获1926年诺贝尔物理学奖。
- **光阱微球实验**：用[[光镊]]束缚微米球在流体中的位移涨落，通过测量位移功率谱 $S_x(\omega)$ 直接验证朗之万方程预言的洛伦兹型谱：
  $$S_x(\omega) = \frac{2\gamma k_B T / m^2}{(\omega^2 - \omega_0^2)^2 + (\gamma\omega/m)^2}$$
  其中 $\omega_0$ 为光阱频率。
- **单分子实验**：在单分子荧光追踪和磁镊实验中，观测蛋白质和高分子的构象涨落，其动力学由含势场的朗之万方程描述。
- **电噪声测量**：电路中电子的热噪声可由含电阻的朗之万型方程描述，与[[约翰逊噪声]]一致。
- **冷原子实验**：在[[光学晶格]]中囚禁的冷原子在晶格中的扩散，可通过朗之万方程描述其有效温度和阻尼。

## 5. 历史上下文

- 1827年，植物学家[[罗伯特·布朗]]在显微镜下观察到花粉颗粒在液体中的无规则运动，但无法解释其物理机制。
- 1905年，[[阿尔伯特·爱因斯坦]]首次从[[统计物理]]角度定量分析了布朗运动，推导出扩散系数 $D = k_BT/(6\pi\eta a)$（$\eta$ 为黏度，$a$ 为粒子半径），为原子分子的真实存在提供了定量证据。
- 1908年，[[保罗·朗之万]]给出了更直接的运动方程，即朗之万方程，将随机力 $\xi(t)$ 显式纳入[[牛顿运动定律_Newton's Laws of Motion]]。朗之万的方法更直观地展示了"涨落与耗散共存"的物理图像。
- 朗之万方程的严格数学理论由[[维纳]]（1923，[[维纳过程]]）和[[伊藤清]]（1940s，[[随机微积分]]、[[伊藤引理]]）建立，成为[[随机过程]]理论的基础。
- [[吴大猷]]、[[乌伦贝克]]-[[奥恩斯坦]]（1930，[[奥恩斯坦-乌伦贝克过程]]）给出了朗之万方程的完整解析解和速度关联函数，建立了布朗运动的完整理论。
- 量子朗之万方程由[[森肇]]和[[英卢申]]（1960s）及[[加德纳]]-[[兹沃林斯基]]（1969）在[[量子光学]]中发展，成为描述[[开放量子系统]]退相干的标准工具，影响了[[量子退相干]]理论和[[量子测量]]诠释。
