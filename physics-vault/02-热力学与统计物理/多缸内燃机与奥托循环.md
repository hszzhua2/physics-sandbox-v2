# 多缸内燃机与奥托循环 (Internal Combustion Engine & Otto Cycle)

> [🌐 打开交互实验](../exp-engine.html)

## 1. 实验目的
通过三维可视化多缸内燃机模型与实时P-V图，展示四冲程奥托循环的热力学过程，直观呈现热功转换的基本原理。

## 2. 实验装置与方法
基于 Three.js 构建3D机械结构（曲轴/连杆/活塞/气缸/火花塞）与 ECharts 实时P-V图：
- **物理模型**：理想奥托循环，$\gamma = 1.4$，压缩比 $r = 8$
- **多缸相位**：N缸发动机按 $4\pi/N$ 相位增量分配点火顺序
- **控制方式**：自动运行（RPM可调10-600）或手动拖动曲轴角度（0-720°）

## 3. 关键结果
四个冲程的热力学过程：
1. **吸气**（0-180°）：等压过程，$P = P_{\text{atm}}$，工作介质进入气缸
2. **压缩**（180-360°）：绝热压缩，$PV^\gamma = C$，压强非线性上升
3. **做功**（360-540°）：等容加热峰值突跃 $Q_{\text{in}} = C_v \Delta T$，随后绝热膨胀对外做功
4. **排气**（540-720°）：等容放热降压，随后等压排气

峰值压强可达大气压的60倍以上（$P_{\text{peak}} \approx 6.4$ MPa），做功冲程输出有效功。

## 4. 历史意义
- **1854年**：意大利工程师 Eugenio Barsanti 和 Felice Matteucci 制造首台内燃机原型
- **1876年**：[[尼古拉斯·奥托]]发明四冲程内燃机（奥托循环），效率约10%
- **1892年**：[[鲁道夫·狄塞尔]]发明压燃式柴油机，效率更高
- 奥托循环是热力学循环的经典实例，验证了 [[热力学第二定律]]：热不可能完全转化为功，必须向低温热源放热

## 5. 现代复现
现代发动机采用电子燃油喷射、可变气门正时（VVT）、涡轮增压等技术，热效率可达40%以上。发动机仿真软件（如ANSYS Forte、CONVERGE）使用计算流体力学（CFD）模拟缸内燃烧过程，远超本实验的简化模型。
