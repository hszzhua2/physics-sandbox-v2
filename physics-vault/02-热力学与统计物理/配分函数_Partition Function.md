# 配分函数 (Partition Function)

## 1. 核心定义

[[配分函数]]是[[统计物理]]中描述系统在给定温度（及化学势）下微观状态占据统计权重的归一化因子，由[[J.威拉德·吉布斯]]引入，是连接微观能谱与宏观热力学量的核心枢纽，被视为统计力学的"母函数"。

## 2. 数学表达

### 正则配分函数

对温度 $T$ 下的 $N$ 粒子系统（[[正则系综]]），正则配分函数为：

$$Z = \sum_n e^{-\beta E_n} \quad \text{（离散能级）}$$

$$Z = \frac{1}{N! h^{3N}} \int e^{-\beta H(\vec{q},\vec{p})}\,d^{3N}q\,d^{3N}p \quad \text{（经典连续）}$$

其中：
- $\beta = 1/k_BT$，$k_B$ 为[[玻尔兹曼常数]]；
- $E_n$：系统第 $n$ 个量子态的能量本征值；
- $H(\vec{q},\vec{p})$：经典[[哈密顿量]]；
- $h$：[[普朗克常数]]，$h^{3N}/N!$ 为经典相空间[[相格]]体积（[[吉布斯因子]]和[[全同粒子]]修正）。

### 巨正则配分函数

对开放系统（[[巨正则系综]]），温度 $T$ 和化学势 $\mu$ 恒定：

$$\mathcal{Z} = \sum_{N=0}^{\infty} \sum_n e^{-\beta(E_{n,N} - \mu N)} = \sum_{N=0}^{\infty} z^N Z_N$$

其中 $z = e^{\beta\mu}$ 为[[逸度]]。

### 配分函数与热力学量的关系

所有热力学量均可由配分函数的导数给出。对正则系综，[[亥姆霍兹自由能]]为：

$$F = -k_B T \ln Z$$

由此导出：

| 热力学量 | 公式 |
|---------|------|
| 内能 | $U = -\frac{\partial \ln Z}{\partial \beta}$ |
| 熵 | $S = k_B \left(\ln Z - \beta \frac{\partial \ln Z}{\partial \beta}\right)$ |
| 压强 | $P = \frac{1}{\beta}\frac{\partial \ln Z}{\partial V}$ |
| [[亥姆霍兹自由能]] | $F = -k_B T \ln Z$ |
| [[吉布斯自由能]] | $G = F + PV$ |
| 定容热容 | $C_V = k_B \beta^2 \frac{\partial^2 \ln Z}{\partial \beta^2}$ |
| 磁化率 | $\chi = \beta \frac{\partial^2 \ln Z}{\partial B^2}$ |

对巨正则系综，[[巨热力学势]] $\Phi = -k_BT \ln \mathcal{Z}$，平均粒子数 $\langle N \rangle = \frac{1}{\beta}\frac{\partial \ln \mathcal{Z}}{\partial \mu}$。

### 涨落与配分函数

内能涨落：

$$\langle (\Delta E)^2 \rangle = -\frac{\partial U}{\partial \beta} = k_BT^2 C_V$$

粒子数涨落（巨正则系综）：

$$\langle (\Delta N)^2 \rangle = \frac{1}{\beta}\frac{\partial \langle N \rangle}{\partial \mu} = k_BT \left(\frac{\partial \langle N \rangle}{\partial \mu}\right)_T$$

### 因子化性质

对无相互作用系统，$H = \sum_i h_i$，配分函数完全因子化：

$$Z_{\text{total}} = \prod_i Z_i$$

因此 $\ln Z = \sum_i \ln Z_i$，自由能为可加量。这是[[理想气体]]和[[理想顺磁体]]可精确求解的基础。

## 3. 物理学意义与推论

- 配分函数是[[统计物理]]中"母函数"角色：给定 $Z$，所有热力学量均可由其导数生成。这一思想与[[生成函数]]在[[概率论]]和[[组合数学]]中的作用一致，体现了[[统计物理]]的数学统一性。
- 配分函数将**微观能谱**（$E_n$）与**宏观热力学**（$F, U, S, P$等）精确关联，是[[系综理论_Ensemble Theory]]的核心结论。这使得从第一性原理（如[[薛定谔方程_Schrödinger Equation]]）计算热力学量成为可能。
- **玻尔兹曼因子** $e^{-\beta E_n}$：配分函数中的权重因子体现了能量越高的状态被占据的概率越低，是[[热力学第二定律]]（熵增）在微观层面的根源。
- **相变与奇异性**：在[[热力学极限]]（$N,V \to \infty$）下，$\ln Z$ 可产生非解析性（如对数项发散或导数不连续），对应[[相变]]。[[杨振宁]]-[[李政道]]（1952）通过分析巨正则配分函数的零点分布揭示了相变的数学机制（[[杨-李定理]]）。
- 配分函数在量子统计中的推广直接给出[[费米-狄拉克统计_Fermi-Dirac Statistics]]和[[玻色-爱因斯坦统计_Bose-Einstein Statistics]]：对费米子每态最多1个粒子，对玻色子无限制，配分函数分别为 $\prod_i (1+e^{-\beta(\varepsilon_i-\mu)})$ 和 $\prod_i (1-e^{-\beta(\varepsilon_i-\mu)})^{-1}$。
- 配分函数的概念在[[量子场论]]中推广为[[路径积分]] $Z = \int \mathcal{D}[\phi]\, e^{-S[\phi]/\hbar}$，其中 $S$ 为[[欧几里得作用量]]，是连接[[统计物理]]与[[量子场论]]的关键桥梁（[[威克转动]]）。
- 在[[信息论]]中，配分函数与[[自由能]] $F = -k_BT \ln Z$ 对应于[[最大熵原理]]下的[[对数配分函数]]，揭示了统计物理与信息论的深层等价性（[[杰恩斯]]，1957）。

## 4. 相关验证实验

- **理想气体状态方程**：由经典配分函数导出 $PV = Nk_BT$，与实验精确吻合。
- **实际气体状态方程**：通过集团展开（[[迈耶]]集团展开）从配分函数导出[[位力展开]]，修正理想气体方程，与真实气体行为定量一致。
- **固体热容的德拜模型**：[[德拜]]（1912）通过声子配分函数导出低温 $T^3$ 热容律，与实验精确吻合。
- **铁磁相变**：[[二维伊辛模型]]的严格配分函数（[[昂萨格]]，1944）精确预言了比热对数奇异性，与实验定性一致，是[[统计物理]]的里程碑。
- **[[黑体辐射]]**：从光子配分函数导出[[普朗克辐射定律]]，是配分函数方法在[[量子统计物理]]中的首次成功应用（参见[[黑体辐射_Black Body Radiation]]）。

## 5. 历史上下文

- 1860s-1870s，[[路德维希·玻尔兹曼]]在气体动理论中引入[[玻尔兹曼因子]] $e^{-E/k_BT}$，奠定了配分函数的物理基础。玻尔兹曼的 $H$ 定理和熵公式 $S = k_B \ln \Omega$ 是配分函数概念的先驱。
- 1902年，[[J.威拉德·吉布斯]]在《统计力学的基本原理》中正式引入配分函数概念，将其作为从微观力学到宏观热力学的核心数学工具。吉布斯的系统化处理使统计力学成为独立的物理分支。
- [[爱因斯坦]]（1902-1904）独立发展了类似方法，但吉布斯的表述更为系统。[[普朗克]]（1900）的黑体辐射推导隐含使用了配分函数思想（光子的量子统计）。
- 1920s-1930s，[[费米]]、[[狄拉克]]、[[玻色]]、[[爱因斯坦]]将配分函数推广至量子统计，给出F-D和B-E统计的配分函数，开启了[[量子统计物理]]时代。
- [[昂萨格]]（1944）严格求解[[二维伊辛模型]]配分函数，是首次精确求解的相互作用系统相变问题，开创了精确可解模型的数学物理传统。
- 配分函数在20世纪后半叶被推广至[[量子场论]]（[[路径积分]]，[[费曼]]，1948）和[[弦论]]，成为理论物理中跨尺度统一描述的核心工具。[[威克转动]]将量子场论的路径积分与统计配分函数形式上等价，揭示了[[量子力学]]与[[统计物理]]的深层联系。
