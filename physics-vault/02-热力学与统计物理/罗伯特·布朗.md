# 罗伯特·布朗 (Robert Brown)

## 1. 生平
1773年生于蒙特罗斯，在爱丁堡大学学习，任大英博物馆植物部主任。1827年发现花粉颗粒的无规则运动。

## 2. 主要贡献
- 发现[[布朗运动]]现象
- 研究植物学细胞核

## 3. 关键公式与思想
- $\langle x^2\rangle=2Dt$（布朗运动位移方差）

## 4. 历史影响
布朗发现的[[布朗运动]]成为原子论与[[统计物理]]的关键证据。
