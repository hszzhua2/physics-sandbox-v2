# 赫伯特·卡伦 (Herbert Callen)

## 1. 生平
1919年生于费城，在麻省理工学院获博士学位，任宾夕法尼亚大学教授。提出涨落耗散定理与不可逆热力学。

## 2. 主要贡献
- 提出[[涨落耗散定理]]联系涨落与响应
- 发展不可逆热力学与[[系综理论]]

## 3. 关键公式与思想
- $\chi''(\omega)=\dfrac{1-e^{-\beta\hbar\omega}}{2\hbar\omega}S(\omega)$（涨落耗散定理）

## 4. 历史影响
卡伦的涨落耗散定理是[[统计物理]]的核心定理，连接平衡与非平衡现象。
