# 伊辛模型 (Ising Model)

## 1. 核心定义

伊辛模型是描述铁磁体中自旋相互作用的最简单格点模型：每个格点上自旋取离散值 $s_i = \pm 1$，近邻自旋通过交换耦合相互作用。它是 [[相变理论]] 与 [[统计系综]] 的范式模型，以最精简的数学结构揭示了一维无相变、二维存在精确相变的深刻规律，是统计物理中极少数精确可解的相变模型。

## 2. 数学表达

### 哈密顿量与配分函数

无外场下伊辛模型哈密顿量为：

$$\mathcal{H} = -J\sum_{\langle i,j \rangle} s_i s_j$$

其中 $s_i = \pm 1$ 为格点自旋变量，$J > 0$ 为铁磁交换耦合常数，$\langle i,j \rangle$ 表示对最近邻对求和。[[配分函数]] 为：

$$Z = \sum_{\{s_i = \pm 1\}} \exp\!\left(\beta J \sum_{\langle i,j \rangle} s_i s_j\right)$$

其中 $\beta = 1/(k_B T)$，求和遍及所有 $2^N$ 种自旋构型。自由能密度 $f = -k_B T \lim_{N\to\infty} \frac{1}{N}\ln Z$。

### 一维精确解

1925年 [[恩斯特·伊辛]] 证明一维伊辛模型在任意有限温度 $T > 0$ 下无自发磁化、无相变。利用 [[转移矩阵法]]，一维周期链（$N$ 个格点）的配分函数为：

$$Z_N = \lambda_+^N + \lambda_-^N \xrightarrow{N\to\infty} \lambda_+^N$$

其中转移矩阵 $T_{s,s'} = e^{\beta J s s'}$ 的本征值为 $\lambda_\pm = 2\cosh(\beta J) \pm 2\sinh(\beta J)$。自由能密度：

$$f = -k_B T \ln[2\cosh(\beta J)]$$

该函数在所有 $T > 0$ 均解析，无奇点，故无相变。关联长度 $\xi = -[\ln\tanh(\beta J)]^{-1}$ 在 $T \to 0$ 时发散，但任何有限温度下均为有限。

### 二维 Onsager 精确解

1944年 [[拉斯·昂萨格]] 求得零场二维方格子伊辛模型的精确配分函数。自由能密度为：

$$f = -k_B T \ln 2 - \frac{k_B T}{2}\int_0^{\pi}\frac{d\theta}{2\pi}\ln\!\left[\cosh(2\beta J) + \sqrt{1+\sinh^2(2\beta J)\cos^2\theta}\right]$$

由自由能的奇点确定临界温度（Kramers-Wannier 对偶性给出精确关系）：

$$k_B T_c = \frac{2J}{\ln(1+\sqrt{2})} \approx 2.269\,J$$

在 $T_c$ 处比热呈对数发散 $C \sim -\ln|T - T_c|$。自发磁化在 $T < T_c$ 时非零：

$$M(T) = \left[1 - \sinh^{-4}(2\beta J)\right]^{1/8}, \quad T < T_c$$

精确临界指数为 $\beta = 1/8$（磁化指数）、$\gamma = 7/4$（磁化率指数）、$\alpha = 0$（比热对数发散）、$\nu = 1$（关联长度指数）、$\eta = 1/4$（反常维度）。

## 3. 物理学意义与推论

1. **维度与相变**：一维无相变、二维有相变的鲜明对比揭示了空间维度对相变的决定性作用。这体现了 [[梅尔-瓦格纳定理]]（Mermin-Wagner 定理）的精神——连续对称破缺在 $d \leq 2$ 维空间中不存在长程序。伊辛模型具有离散 $Z_2$ 对称性，是二维仍可发生对称破缺的特例。
2. **精确相变的范例**：Onsager 解是统计物理中极少数精确可解的相变模型之一，从数学上严格证明了配分函数的奇点（非解析性）导致相变的理论机制，排除了相变是近似方法人为产物的疑虑。
3. **临界现象与普适性**：临界指数仅依赖于空间维度 $d$ 和序参量分量数 $n$（普适类），伊辛模型属于 $n=1$（即 $O(1)$ 或 $Z_2$）普适类。其临界指数被 [[重整化群]] 理论统一解释——不同微观细节的系统在临界点附近呈现相同的标度行为。
4. **重整化群的验证**：[[肯尼斯·威尔逊]] 的重整化群理论在伊辛模型上得到完美验证。$d = 4 - \epsilon$ 展开给出的临界指数与 Onsager 的 $d=2$ 精确解以及三维数值结果高度一致，确立了重整化群作为相变理论核心工具的地位。

## 4. 相关验证实验

伊辛模型作为理论范式，其统计物理基础通过以下实验节点间接验证：

- [[布朗运动_Brownian Motion]]：统计涨落与热平衡的实验验证，为配分函数的统计诠释提供了微观基础。布朗运动中涨落-耗散的统一性与伊辛模型中涨落驱动的相变同属统计物理核心主题。
- [[高尔顿钉板_Galton Board]]：大数定律与统计涨落的直观演示，验证了统计系综与大数极限的核心假设，是理解伊辛模型热力学极限的基础。
- 铁磁体的比热测量：实验在居里温度 $T_c$ 附近观测到比热临界发散行为，与伊辛模型预言的对数发散定性一致。二维超流薄膜与表面吸附层的相变实验与二维伊辛临界指数 $\beta = 1/8$ 高度吻合。

## 5. 历史上下文

1920年，[[威廉·楞次]] 提出该模型以解释铁磁性，并将其交给博士生 [[恩斯特·伊辛]] 求解。伊辛于1925年证明一维伊辛模型在任意有限温度下无相变后，一度错误地推测所有维度均无相变，几乎使该模型被遗忘。1936年 [[鲁道夫·派尔斯]] 给出物理论证表明二维伊辛应有相变。1944年，[[拉斯·昂萨格]] 以其精妙的转移矩阵对角化方法给出二维精确解，震动了整个物理学界，成为统计力学的里程碑。昂萨格解表明，统计力学的配分函数可以严格产生相变奇点。1971年，[[肯尼斯·威尔逊]] 提出 [[重整化群]] 理论，将伊辛临界现象纳入普适类框架，并因此荣获1982年诺贝尔物理学奖。伊辛模型至今仍是统计物理、凝聚态物理乃至量子信息中相变与临界现象研究的核心范式。
