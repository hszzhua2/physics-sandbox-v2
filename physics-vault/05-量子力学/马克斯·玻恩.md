# 马克斯·玻恩 (Max Born)

## 1. 生平
1882年生于布雷斯劳，在哥廷根大学任教并领导哥廷根学派，1933年流亡英国任爱丁堡大学教授，1954年获诺贝尔物理学奖。

## 2. 主要贡献
- 提出波函数的[[玻恩概率解释]]
- 与海森堡、约尔当共同创立矩阵力学
- 发展晶体动力学与固体比热理论

## 3. 关键公式与思想
- $P=|\Psi(x)|^2\,dx$（玻恩定则）

## 4. 历史影响
玻恩的概率诠释使[[量子力学]]获得统计意义，是哥本哈根诠释的核心。
