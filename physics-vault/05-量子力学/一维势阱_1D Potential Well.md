# 一维势阱 (1D Potential Well)

## 1. 核心定义

一维势阱是粒子被限制在有限空间区间内运动的理想化势场模型，是 [[薛定谔方程]] 最基本的精确可解体系，以最简洁的形式展示了 [[能量量子化]]、波函数驻波结构和零点能等量子力学的本质特征。

## 2. 数学表达

### 无限深方势阱

势能函数定义为：

$$V(x) = \begin{cases} 0, & 0 < x < L \\ \infty, & \text{其他} \end{cases}$$

定态 [[薛定谔方程]] 在阱内简化为自由粒子方程：

$$-\frac{\hbar^2}{2m}\frac{d^2\psi}{dx^2} = E\psi$$

边界条件 $\psi(0) = \psi(L) = 0$（波函数在无穷势壁处必须为零以保证概率有限），解得能量本征值和归一化本征函数：

$$E_n = \frac{n^2\pi^2\hbar^2}{2mL^2}, \quad n = 1, 2, 3, \ldots$$

$$\psi_n(x) = \sqrt{\frac{2}{L}}\sin\!\left(\frac{n\pi x}{L}\right)$$

参数说明：$m$ 为粒子质量，$L$ 为阱宽，$\hbar$ 为约化普朗克常数，$n$ 为主量子数。

**零点能**：基态能量 $E_1 = \frac{\pi^2\hbar^2}{2mL^2} \neq 0$。这是 [[海森堡不确定性原理]] $\Delta x \Delta p \geq \hbar/2$ 的直接后果——粒子被限制在有限空间 $\Delta x \sim L$ 内，必有非零动量涨落 $\Delta p \geq \hbar/(2L)$，因而具有非零最低能量。

### 正交归一性

波函数满足正交归一条件：

$$\int_0^L \psi_m^*(x)\psi_n(x)\,dx = \delta_{mn}$$

任意量子态可展开为本征态的线性叠加：

$$\Psi(x,t) = \sum_{n=1}^{\infty} c_n \psi_n(x)\, e^{-iE_n t/\hbar}$$

展开系数 $c_n = \int_0^L \psi_n^*(x)\Psi(x,0)\,dx$，测量能量得到 $E_n$ 的概率为 $|c_n|^2$。

### 有限深方势阱

势能定义为 $V(x) = -V_0$（$|x| < a$），$V(x) = 0$（$|x| \geq a$）。在阱外（经典禁戒区），波函数指数衰减为倏逝波 $\psi \propto e^{-\kappa|x|}$，其中 $\kappa = \sqrt{2m(V_0 - |E|)}/\hbar$。束缚态能级由边界处波函数及导数连续的匹配条件确定，仅存在有限个离散束缚态。

## 3. 物理学意义与推论

1. **量子化的起源**：能量量子化源于波函数的边界条件与驻波条件 $k_n = n\pi/L$。波函数必须"恰好容纳"在阱内形成驻波，类似于弦上的驻波模式，与经典连续能谱截然不同。
2. **不确定性原理的体现**：零点能是位置-动量不确定性关系的必然结果。阱越窄（$L$ 越小），零点能越高，体现了"限制越强、涨落越大"的量子特征。
3. **量子隧穿**：在有限势阱或势垒情形下，波函数在经典禁戒区非零，形成量子隧穿效应——粒子有一定概率穿越高于其总能量的势垒。隧穿系数近似为 $T \propto e^{-2\kappa a}$，其中 $\kappa$ 依赖势垒高度与粒子能量之差。
4. **半导体物理应用**：量子阱、量子线和量子点是一维势阱的低维推广。半导体异质结中形成的量子阱具有离散能级 $E_n \propto n^2$，是现代半导体激光器、量子级联激光器和红外探测器的物理基础。

## 4. 相关验证实验

一维势阱的理论预言在多个实验中得到验证：

- [[双缝干涉_Double-Slit Interference]]：波函数叠加与量子概率诠释的实验核心，验证了量子态叠加的物理实在性，与势阱中波函数的叠加展开原理一致。
- 半导体量子阱的光学跃迁实验：通过光致发光光谱观测到量子阱中 $E_n \propto n^2$ 的离散能级间距，与无限深势阱模型高度吻合，是势阱理论最直接的实验验证。
- 扫描隧道显微镜（STM）：利用量子隧穿效应实现原子级成像，直接验证了有限势垒中波函数指数衰减 $\psi \propto e^{-\kappa x}$ 的预言。

## 5. 历史上下文

1926年，[[埃尔温·薛定谔]] 在提出以他命名的波动方程后，立即将其应用于势阱、谐振子和氢原子等可精确求解的体系，证明量子化可以自然涌现于波方程的边界条件之中，无需玻尔-索末菲理论中人为引入的量子化规则。一维势阱作为量子力学最简单的精确可解模型，成为量子力学教学与理解量子效应的基石。它以最少的数学复杂度揭示了量子世界的核心特征——离散能级、零点能和量子隧穿，为理解更复杂的量子体系提供了概念模板。
