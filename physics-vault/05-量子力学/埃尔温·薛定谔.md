# 埃尔温·薛定谔 (Erwin Schrödinger)

## 1. 生平
1887年生于维也纳，在维也纳大学获博士学位。1926年在苏黎世提出波动力学与薛定谔方程，1933年获诺贝尔物理学奖。晚年任都柏林高等研究院院长。

## 2. 主要贡献
- 建立波动力学，提出[[薛定谔方程]]
- 证明波动力学与矩阵力学等价
- 提出[[薛定谔猫]]思想实验讨论量子测量

## 3. 关键公式与思想
- $i\hbar\dfrac{\partial}{\partial t}\Psi=\hat{H}\Psi$（薛定谔方程）
- $\hat{H}\psi=E\psi$（定态薛定谔方程）

## 4. 历史影响
薛定谔方程是[[量子力学]]的核心方程，波动力学成为微观物理的标准框架。
