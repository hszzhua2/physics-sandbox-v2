# 戴维·玻姆 (David Bohm)

## 1. 生平
1917年生于宾夕法尼亚，在加州大学伯克利分校师从奥本海默，后因麦卡锡主义流亡巴西、英国。提出量子力学的隐变量诠释。

## 2. 主要贡献
- 提出[[玻姆力学]]（量子势的隐变量理论）
- 发展[[不确定性原理]]的因果诠释
- 研究等离子体与全息宇宙观

## 3. 关键公式与思想
- $Q=-\dfrac{\hbar^2}{2m}\dfrac{\nabla^2 R}{R}$（量子势）

## 4. 历史影响
玻姆诠释为[[量子力学]]提供了决定论替代方案，引发对测量与实在的持续讨论。
