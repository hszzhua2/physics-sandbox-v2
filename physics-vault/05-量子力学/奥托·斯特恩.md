# 奥托·斯特恩 (Otto Stern)

## 1. 生平
1888年生于索劳，在布雷斯劳大学获博士学位，先后在法兰克福、汉堡任教，1933年流亡美国任卡内基理工学院教授，1943年获诺贝尔物理学奖。

## 2. 主要贡献
- 与盖拉赫完成[[斯特恩-盖拉赫实验]]，证实空间量子化
- 发展分子束方法测量[[质子磁矩]]

## 3. 关键公式与思想
- $\Delta z=\dfrac{\mu_z L^2}{4E_k d}$（斯特恩-盖拉赫偏转）

## 4. 历史影响
斯特恩-盖拉赫实验直接证实了[[自旋]]与空间量子化，是[[量子力学]]的决定性实验。
