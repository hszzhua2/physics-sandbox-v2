# 双缝干涉 (Double-Slit Interference)

## 1. 实验目的与背景

1801年，[[托马斯·杨]] 设计了双缝干涉实验，首次以可控方式展示了光的干涉现象，为光的波动说提供了关键实验证据，动摇了牛顿光微粒说长达百年的统治地位。此后该实验被提升至量子层面：将光源替换为单电子或单光子发射源，揭示了微观粒子的波粒二象性。[[理查德·费曼]] 将双缝干涉称为"量子力学中唯一的奥秘"，认为它是理解量子力学全部奇异性的核心入口。

双缝干涉是 [[态叠加原理]] 和 [[波函数]] 概念最简洁、最深刻的实验体现，也是从经典波动光学过渡到量子力学的关键桥梁。

## 2. 实验装置与运行机制

**经典光学版本**：相干单色光通过两条间距为 $d$ 的狭缝（缝宽 $a$），在距离 $L$（$L \gg d$）的远场屏幕上形成干涉条纹。单缝宽度产生衍射包络，双缝间距产生干涉调制，两者相乘给出完整的光强分布。

**量子版本**：将光源减弱至每次仅有一个光子（或电子）通过装置。在屏幕处放置高灵敏度探测器阵列（如CCD或荧光屏），逐个记录粒子到达位置。按经典粒子图像，粒子应仅通过一条缝，长时间累积后屏幕应呈现两条亮带。但实验表明：即使粒子逐一发射、彼此之间不可能发生经典波动叠加，长时间累积后屏幕上仍出现清晰的干涉条纹。

装置核心参数：双缝间距 $d$，缝到屏距离 $L$，光或粒子的德布罗意波长 $\lambda$。

## 3. 观测数据与数学模型

### 经典波动干涉

两束相干波叠加，屏幕上光强分布为：

$$I(x) = I_0 \cos^2\!\left(\frac{\pi d x}{\lambda L}\right) \cdot \text{sinc}^2\!\left(\frac{\pi a x}{\lambda L}\right)$$

亮纹位置（相长干涉）：

$$x_m = m\frac{\lambda L}{d}, \quad m = 0, \pm 1, \pm 2, \ldots$$

条纹间距 $\Delta x = \lambda L / d$，与波长成正比，与缝间距成反比。

### 量子概率诠释

根据 [[玻恩定则]]，单粒子在屏幕位置 $x$ 处的探测概率密度正比于波函数的模平方：

$$P(x) \propto |\psi_1(x) + \psi_2(x)|^2 = |\psi_1|^2 + |\psi_2|^2 + 2\,\text{Re}[\psi_1^*(x)\psi_2(x)]$$

其中 $\psi_1(x)$、$\psi_2(x)$ 分别为粒子通过缝1和缝2的概率幅。交叉项 $2\,\text{Re}[\psi_1^*\psi_2]$ 即干涉项，是 [[态叠加原理]] 的直接体现。即使单粒子每次只通过一条缝，其量子态仍以概率幅形式同时"探索"两条路径，叠加后产生干涉。

### 观察者效应

若在缝处放置探测器进行 which-path 测量，以确定粒子通过哪条缝，则干涉条纹消失，屏幕呈现经典单缝衍射包络：

$$P(x) \propto |\psi_1(x)|^2 + |\psi_2(x)|^2$$

这体现了量子测量中的 [[波函数坍缩]]，即 [[尼尔斯·玻尔]] 互补原理——路径信息与干涉图样互斥，不可同时完美获得。

### 费曼路径积分视角

[[理查德·费曼]] 提出粒子从源到屏经过所有可能路径，每条路径贡献一个相位因子 $e^{iS/\hbar}$（$S$ 为沿该路径的经典作用量），概率幅为所有路径相位的叠加：

$$\psi(x) = \sum_{\text{paths}} \mathcal{D}[x(t)]\, e^{iS[x(t)]/\hbar}$$

经典路径附近相位变化平缓、贡献相干加强；远离经典路径的相位剧烈振荡、贡献相消。双缝实验中两条路径的相干叠加正是路径积分最简洁的体现，也是 [[费曼路径积分]] 的物理直觉来源。

## 4. 实验结论与影响

1. **波动说的确立**：杨氏实验彻底动摇了牛顿光微粒说，确立了光的波动本质，详见 [[波的干涉_Wave Interference]]。
2. **波粒二象性的核心展示**：单粒子双缝干涉是波粒二象性最简洁的实验证明。粒子的个体探测行为表现为粒子性，但其统计分布呈现波动干涉图样。
3. **量子测量与互补性**：which-path 实验验证了 [[尼尔斯·玻尔]] 互补原理——路径信息与干涉图样互斥。该原理是量子测量理论的基石。
4. **路径积分的物理直觉**：双缝实验为 [[费曼路径积分]] 提供了最直观的物理图像，成为量子电动力学（QED）的基础之一。
5. **现代量子信息**：双缝干涉是量子干涉、量子计算和量子信息处理的物理原型。量子擦除实验和延迟选择实验进一步深化了对量子测量本质的理解，属于量子力学基础研究中持续活跃的前沿课题。
