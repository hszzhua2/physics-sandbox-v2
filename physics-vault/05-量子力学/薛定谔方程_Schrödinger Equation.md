# 薛定谔方程 (Schrödinger Equation)

## 1. 核心定义

[[薛定谔方程]]是[[量子力学]]中描述微观粒子[[波函数]]随时间和空间演化的基本动力学方程，由[[埃尔温·薛定谔]]于1926年建立，是量子力学的核心公理之一，地位相当于经典力学中的[[牛顿运动定律_Newton's Laws of Motion]]。

## 2. 数学表达

### 含时薛定谔方程

$$i\hbar \frac{\partial \Psi(\vec{r}, t)}{\partial t} = \hat{H} \Psi(\vec{r}, t)$$

其中：
- $\Psi(\vec{r}, t)$：[[波函数]]，描述系统的量子态；
- $\hat{H}$：[[哈密顿算符]]，通常为 $\hat{H} = -\frac{\hbar^2}{2m}\nabla^2 + V(\vec{r}, t)$；
- $\hbar = h/(2\pi)$：[[约化普朗克常数]]，$\hbar \approx 1.055 \times 10^{-34}\;\text{J·s}$；
- $i$：虚数单位。

对单粒子系统，哈密顿算符为动能加势能：

$$\hat{H} = -\frac{\hbar^2}{2m}\nabla^2 + V(\vec{r}, t)$$

### 定态薛定谔方程

当势能 $V(\vec{r})$ 不显含时间时，波函数可分离变量 $\Psi(\vec{r},t) = \psi(\vec{r}) e^{-iEt/\hbar}$，得到[[定态薛定谔方程]]（[[本征值问题]]）：

$$\hat{H}\psi_n(\vec{r}) = E_n \psi_n(\vec{r})$$

即：

$$\left[-\frac{\hbar^2}{2m}\nabla^2 + V(\vec{r})\right]\psi_n(\vec{r}) = E_n \psi_n(\vec{r})$$

$\{E_n\}$ 为能量本征值谱，$\{\psi_n\}$ 为对应的能量本征态。一般量子态可展开为：

$$\Psi(\vec{r},t) = \sum_n c_n \psi_n(\vec{r}) e^{-iE_n t/\hbar}$$

### 几率流与连续性方程

由薛定谔方程可导出[[几率守恒]]（[[连续性方程]]）：

$$\frac{\partial \rho}{\partial t} + \nabla \cdot \vec{j} = 0$$

其中概率密度 $\rho = |\Psi|^2$，[[几率流密度]]：

$$\vec{j} = \frac{\hbar}{2mi}\left(\Psi^* \nabla \Psi - \Psi \nabla \Psi^*\right)$$

### 多粒子系统

对 $N$ 粒子系统：

$$i\hbar \frac{\partial \Psi}{\partial t} = \left[\sum_{i=1}^{N} -\frac{\hbar^2}{2m_i}\nabla_i^2 + V(\vec{r}_1, \ldots, \vec{r}_N, t)\right] \Psi$$

### 路径积分形式

[[费曼]]（1948）给出等价的[[路径积分]]表述：

$$K(\vec{r}_f, t_f; \vec{r}_i, t_i) = \int \mathcal{D}[\vec{r}(t)]\, e^{iS[\vec{r}]/\hbar}$$

其中 $S$ 为经典[[作用量]]。这一形式与[[统计物理]]的[[配分函数_Partition Function]]形式上相似（[[威克转动]]关联）。

## 3. 物理学意义与推论

- 薛定谔方程是[[量子力学]]的基本公设之一，不能从更基本的原理推导。它将经典[[哈密顿力学_Hamiltonian Mechanics]]中的能量关系 $E = H$ 提升为算符方程 $i\hbar\partial_t \to \hat{H}$，体现了[[对应原理]]。
- **[[波函数]]的概率诠释**（[[玻恩概率解释_Born Rule]]）赋予 $\Psi$ 以物理意义：$|\Psi(\vec{r},t)|^2$ 是在时刻 $t$、位置 $\vec{r}$ 处找到粒子的概率密度。薛定谔方程本身只决定 $\Psi$ 的演化，其物理诠释由[[玻恩]]定则补充。
- 薛定谔方程是**线性**方程，保证了[[态叠加原理_Superposition Principle]]的成立：若 $\Psi_1$ 和 $\Psi_2$ 是解，则 $c_1\Psi_1 + c_2\Psi_2$ 也是解。这一线性性是量子干涉（如[[双缝干涉_Double-Slit Interference]]）的数学基础。
- **[[一维势阱_1D Potential Well]]** 是薛定谔方程的最简单精确可解模型之一，给出分立能级 $E_n = n^2 \pi^2 \hbar^2 / (2mL^2)$ 和[[零点能]] $E_1 = \pi^2\hbar^2/(2mL^2) \neq 0$，揭示了量子效应的本质。
- **[[不确定性原理_Uncertainty Principle]]** $\Delta x \Delta p \geq \hbar/2$ 可从薛定谔方程的波函数结构中导出，是量子态内在的统计性质，与测量精度无关。
- **谐振子**精确解给出等间距能级 $E_n = (n + 1/2)\hbar\omega$，零点能 $E_0 = \hbar\omega/2$ 是量子涨落的直接体现。这一结果在[[量子场论]]中推广为每个振动模式的零点能。
- **氢原子**精确解给出能级 $E_n = -13.6\;\text{eV}/n^2$，精确解释了[[玻尔模型]]的结果并预言了更多精细结构，是薛定谔方程的标志性成就。
- 薛定谔方程在[[相对论]]框架下需推广为[[克莱因-戈尔登方程]]（标量场）和[[狄拉克方程]]（旋量场），后者预言了[[自旋_Spin]]和[[反物质]]。

## 4. 相关验证实验

- **氢原子光谱**（[[巴耳末系]]，1885；[[玻尔模型]]，1913）：薛定谔方程对氢原子的精确解完美解释了氢光谱的所有谱线，包括[[巴耳末系]]、[[赖曼系]]等，是方程最早的也是最精确的验证之一。
- **[[一维势阱_1D Potential Well]]与量子阱实验**：半导体[[量子阱]]中的电子能级测量精确验证了无限深势阱的 $E_n \propto n^2$ 关系。
- **[[双缝干涉_Double-Slit Interference]]实验**：电子双缝实验直接展示了薛定谔方程预言的波函数干涉图样，是量子力学波动性的核心证据（参见[[双缝干涉_Double-Slit Interference]]）。
- **[[斯特恩-盖拉赫实验_Stern-Gerlach Experiment]]**：空间量子化的发现要求薛定谔方程纳入[[自旋_Spin]]自由度，推动薛定谔方程向[[泡利方程]]和[[狄拉克方程]]的推广。
- **[[量子隧穿效应_Quantum Tunneling]]实验**：$\alpha$ 衰变（[[伽莫夫]]-[[格尼]]-[[康登]]，1928）和[[扫描隧道显微镜]]（[[宾尼希]]-[[罗雷尔]]，1981，获1986年诺贝尔物理学奖）直接验证了薛定谔方程对势垒穿透的预言。
- **[[氢原子精细结构]]与[[兰姆位移]]**：薛定谔方程的非相对论近似不足以解释精细结构，需[[狄拉克方程]]和[[量子电动力学]]修正。[[兰姆位移]]的测量推动了QED的发展。

## 5. 历史上下文

- 1924年[[路易·德布罗意]]提出[[德布罗意波]]假说：实物粒子具有波动性，波长 $\lambda = h/p$。这一假说由[[戴维森]]-[[革末]]的电子衍射实验（1927）直接验证。
- 1925年[[海森堡]]建立[[矩阵力学]]，是量子力学的第一种形式。[[玻恩]]、[[约旦]]和[[海森堡]]共同完善了矩阵力学的数学结构。
- 1926年初，[[埃尔温·薛定谔]]在研究[[德布罗意波]]的过程中，受[[德拜]]的建议（"既然有波，就应该有波动方程"），建立了以他命名的波动方程。薛定谔最初从[[哈密顿-雅可比方程]]出发，通过类比光学-力学对应（[[哈密顿]]的光学类比）导出方程。
- 1926年薛定谔证明[[矩阵力学]]和[[波动力学]]在数学上等价（[[狄拉克]]也独立证明）。两种形式的等价性统一了量子力学的表述。
- [[玻恩]]在1926年给出波函数的概率诠释（[[玻恩概率解释_Born Rule]]），将薛定谔的"物质波"解释为概率幅。这一诠释引发了长达一个世纪的[[量子力学诠释]]争论（[[哥本哈根诠释]]、[[多世界诠释]]、[[隐变量理论]]等），至今未完全解决（属于未解决问题）。
- [[费曼]]于1948年提出[[路径积分]]表述，给出了薛定谔方程的第三种等价形式。这一形式在[[量子场论]]和[[统计物理]]中有巨大优势，成为现代理论物理的标准工具。
- 薛定谔方程在现代物理中有广泛应用：从[[凝聚态物理]]（能带计算、[[超导理论]]）到[[化学]]（[[量子化学]]计算分子轨道），从[[量子计算]]（量子门操作）到[[宇宙学]]（[[惠勒-德维特方程]]），是描述微观世界的基本语言。
