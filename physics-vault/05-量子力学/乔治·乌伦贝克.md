# 乔治·乌伦贝克 (George Uhlenbeck)

## 1. 生平
1900年生于雅加达，在莱顿大学师从埃伦费斯特，1925年与古德斯密特提出电子自旋。后任密歇根大学与洛克菲勒大学教授。

## 2. 主要贡献
- 与古德斯密特共同提出电子[[自旋]]假说
- 发展统计物理与分子运动论

## 3. 关键公式与思想
- $\mu_s=-g_s\dfrac{e}{2m_e}\mathbf{S}$（自旋磁矩）

## 4. 历史影响
乌伦贝克的自旋假说完善了[[量子力学]]的电子图像，解释了[[原子光谱]]精细结构。
