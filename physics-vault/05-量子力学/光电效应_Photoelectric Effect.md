# 光电效应 (Photoelectric Effect)

## 1. 实验目的与背景

[[光电效应]]是指金属表面在光照射下发射电子（[[光电子]]）的现象，最早由[[赫兹]]于1887年偶然发现，[[哈尔瓦克斯]]于1888年系统研究。19世纪末的经典[[电动力学]]（[[麦克斯韦方程组_Maxwell's Equations]]）预言光为连续[[电磁波]]，其能量应与光强成正比、与频率无关。然而实验观测到三个经典物理无法解释的反常现象：

1. **存在截止频率**：仅当入射光频率 $\nu$ 超过某金属特征阈值 $\nu_0$（[[截止频率]]/[[遏止频率]]）时才发射光电子，与光强无关。
2. **光电子动能依赖频率而非光强**：光电子最大动能 $K_{\max}$ 随频率线性增加，与光强无关；增大光强仅增加光电子数目，不增加单个电子的动能。
3. **无时间延迟**：即使光强极弱，只要频率超过 $\nu_0$，光电子几乎瞬间发射（延迟 $< 10^{-9}\;\text{s}$），与经典波动理论预言的能量积累时间（可达数秒）严重矛盾。

1905年[[阿尔伯特·爱因斯坦]]提出[[光量子假说]]以解释上述反常，是[[量子力学]]诞生的关键事件。

## 2. 实验装置与运行机制

### 经典光电效应实验装置

光电效应实验的核心装置包括：
- **真空光电管**：内含金属阴极（[[光阴极]]，如钠、钾、铯等碱金属或锌、铜）和阳极，密封在高真空玻璃管中，避免电子与气体分子碰撞。
- **单色光源**：通过[[棱镜]]或[[光栅]]产生特定频率 $\nu$ 的单色光，照射阴极。
- **可调电压源与检流计**：在阴阳极间施加可调电压 $V$，测量光电流 $I$。

### 实验测量流程

1. 固定入射光频率 $\nu$ 和光强 $I_{\text{light}}$，扫描电压 $V$，测量光电流 $I(V)$。
2. 当 $V > 0$（阳极正），光电子全部被收集，$I$ 达饱和值 $I_{\text{sat}} \propto I_{\text{light}}$。
3. 当 $V < 0$（阳极负），光电子被减速。当 $V = -V_s$ 时，$I = 0$，$V_s$ 为[[遏止电压]]。

### 遏止电压与最大动能

遏止电压 $V_s$ 直接给出光电子的最大动能：

$$K_{\max} = e V_s$$

实验测得 $V_s$ 随频率 $\nu$ 线性变化：

$$e V_s = h\nu - W$$

### 爱因斯坦光量子假说的物理图像

[[爱因斯坦]]假设光本身由离散的能量量子（[[光量子]]/[[光子]]）组成，每个光子携带能量：

$$E_{\text{photon}} = h\nu$$

其中 $h$ 为[[普朗克常数]]。光子被金属中的电子一次性吸收：若 $h\nu$ 超过[[逸出功]] $W$（也称[[功函数]]），电子逸出，多余能量转化为动能：

$$K_{\max} = h\nu - W$$

[[逸出功]] $W$ 是电子从金属表面逸出所需的最小能量，是金属的特征参数。

## 3. 观测数据与数学模型

### 爱因斯坦光电方程

$$K_{\max} = h\nu - W$$

或用遏止电压表示：

$$e V_s = h\nu - W$$

即：

$$V_s = \frac{h}{e}\nu - \frac{W}{e}$$

### 关键实验数据

[[密立根]]（1916）的精密测量数据（以钠为例）：

| 频率 $\nu$（$10^{14}$ Hz） | 遏止电压 $V_s$（V） |
|---|---|
| 5.18 | 0.52 |
| 6.21 | 0.96 |
| 6.91 | 1.24 |
| 7.39 | 1.45 |
| 8.20 | 1.80 |

$V_s$-$\nu$ 图为直线，斜率 $h/e$ 给出[[普朗克常数]] $h$，截距 $-W/e$ 给出逸出功。

### 普朗克常数的独立测定

[[密立根]]从光电效应测得的 $h$ 与[[黑体辐射]]独立测定的 $h$ 一致：

$$h \approx 6.626 \times 10^{-34}\;\text{J·s}$$

这一一致性是光量子假说的有力证据。

### 逸出功的典型值

| 金属 | 逸出功 $W$（eV） | 截止波长 $\lambda_0$（nm） |
|------|------|------|
| 铯 Cs | 2.1 | 590 |
| 钠 Na | 2.3 | 540 |
| 锌 Zn | 4.3 | 288 |
| 铜 Cu | 4.7 | 264 |
| 银 Ag | 4.3 | 288 |

碱金属逸出功低，在可见光区即可发生光电效应，故常用于光阴极。

### 光电流与光强的关系

饱和光电流 $I_{\text{sat}}$ 与光强 $I_{\text{light}}$ 成正比：

$$I_{\text{sat}} = \eta \frac{I_{\text{light}}}{h\nu} e$$

其中 $\eta$ 为[[量子效率]]，即每个入射光子产生的光电子比例。这一关系验证了光子图像：光强增大意味着光子数增多，但每个光子能量 $h\nu$ 不变。

## 4. 实验结论与影响

- **光量子假说的确立**：[[爱因斯坦]]1905年的光电效应理论首次将[[普朗克]]的量子假说从[[黑体辐射]]的数学技巧提升为光的物理本质，提出光本身由量子（[[光子]]）构成。这一假说直接给出了光的[[粒子性]]，与[[麦克斯韦方程组_Maxwell's Equations]]的[[波动性]]形成对立，催生了[[波粒二象性]]概念。
- **[[密立根]]的精密验证**：[[罗伯特·密立根]]（1916）经过十年精密实验，确认了[[爱因斯坦]]光电方程 $eV_s = h\nu - W$ 的精确性，独立测定了[[普朗克常数]] $h$。尽管[[密立根]]本人最初反对光量子假说，但其实验结果无可辩驳地支持了该假说。[[密立根]]获1923年诺贝尔物理学奖。
- **[[量子力学]]的诞生**：[[爱因斯坦]]因光电效应（而非[[相对论]]）获1921年诺贝尔物理学奖。光电效应是[[量子力学]]的两大起源之一（另一为[[黑体辐射]]），直接催生了[[光子]]概念和[[波粒二象性]]，为[[德布罗意波]]假说（1924）和[[量子力学]]的建立铺平道路。
- **光电效应的应用**：[[光电倍增管]]、[[太阳能电池]]、[[CCD传感器]]、[[夜视仪]]等均基于光电效应。在[[光电倍增管]]中，光电子级联倍增可实现单光子探测。
- **光子动量的验证**：[[康普顿散射_Compton Scattering]]（1923）进一步验证了光子的粒子性（动量 $p = h/\lambda$），与光电效应共同确立了[[光子]]的物理地位。
- **多光子光电效应**：激光发明后观测到[[多光子光电效应]]（1960s），电子可同时吸收多个光子逸出，此时 $K_{\max} = nh\nu - W$（$n$ 为吸收光子数）。这是非线性光学和[[强场物理]]的基础。
- **[[光电效应]]在[[量子电动力学]]中的地位**：QED将光子描述为电磁场的量子激发，光电效应中光子-电子相互作用由[[费曼图]]中的吸收顶点 $e\gamma^\mu A_\mu$ 描述。[[兰姆位移]]和[[反常磁矩]]的精确测量进一步验证了QED框架的普适性，将光电效应的物理基础提升至量子场论层面。

### 历史意义总结

光电效应实验是物理学史上最具有范式转换意义的实验之一。它从经典[[电动力学]]的"反常"出发，推翻了光的纯波动图像，引入了[[光子]]概念，是[[量子力学]]的起点。[[爱因斯坦]]1905年的论文（"关于光的产生和转化的一个启发性观点"）被评为"唯一一篇使[[爱因斯坦]]值得获诺贝尔奖的论文"（[[诺贝尔奖委员会]]语），标志着物理学从经典连续图景向量子离散图景的根本转变。
