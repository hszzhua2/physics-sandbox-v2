# 自旋 (Spin)

## 1. 核心定义

[[自旋]]是微观粒子的内禀[[角动量]]，与空间轨道运动无关，由[[乔治·乌伦贝克]]和[[塞缪尔·古德斯密特]]于1925年提出以解释[[斯特恩-盖拉赫实验_Stern-Gerlach Experiment]]和原子光谱精细结构，是[[量子力学]]中粒子内禀自由度的基本属性。

## 2. 数学表达

### 自旋算符

自旋满足[[角动量]]的一般对易关系：

$$[\hat{S}_i, \hat{S}_j] = i\hbar \varepsilon_{ijk} \hat{S}_k$$

总自旋平方 $\hat{S}^2 = \hat{S}_x^2 + \hat{S}_y^2 + \hat{S}_z^2$ 与分量 $\hat{S}_z$ 对易：

$$\hat{S}^2 |s, m_s\rangle = s(s+1)\hbar^2 |s, m_s\rangle$$

$$\hat{S}_z |s, m_s\rangle = m_s \hbar |s, m_s\rangle$$

其中 $s$ 为自旋量子数（整数或半整数），$m_s = -s, -s+1, \ldots, s$ 共 $2s+1$ 个取值。

### 自旋1/2与泡利矩阵

对自旋 $s = 1/2$ 的粒子（如电子、质子、中子），$m_s = \pm 1/2$，自旋算符用[[泡利矩阵]]表示：

$$\hat{S}_i = \frac{\hbar}{2} \sigma_i$$

[[泡利矩阵]]：

$$\sigma_x = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix}, \quad \sigma_y = \begin{pmatrix} 0 & -i \\ i & 0 \end{pmatrix}, \quad \sigma_z = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$$

满足：

$$\sigma_i \sigma_j = \delta_{ij} I + i \varepsilon_{ijk} \sigma_k$$

$$[\sigma_i, \sigma_j] = 2i\varepsilon_{ijk}\sigma_k, \quad \{\sigma_i, \sigma_j\} = 2\delta_{ij}I$$

自旋向上和向下态：

$$|\uparrow\rangle = \begin{pmatrix} 1 \\ 0 \end{pmatrix}, \quad |\downarrow\rangle = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$$

### 升降算符

$$\hat{S}_+ = \hat{S}_x + i\hat{S}_y, \quad \hat{S}_- = \hat{S}_x - i\hat{S}_y$$

$$\hat{S}_\pm |s, m_s\rangle = \hbar\sqrt{s(s+1) - m_s(m_s \pm 1)}\,|s, m_s \pm 1\rangle$$

### 自旋磁矩

电子自旋产生磁矩：

$$\vec{\mu}_s = -g_s \frac{e}{2m_e} \vec{S}$$

其中 $g_s \approx 2.002319$ 为电子[[自旋g因子]]，$g_s \approx 2$ 是[[狄拉克方程]]的预言，$g_s - 2$ 的微小偏差由[[量子电动力学]]的[[反常磁矩]]解释。

### 自旋-轨道耦合

电子轨道角动量 $\vec{L}$ 与自旋 $\vec{S}$ 之间的相互作用（[[自旋轨道耦合]]）：

$$\hat{H}_{SO} = \frac{1}{2m_e^2 c^2} \frac{1}{r} \frac{dV}{dr} \vec{L} \cdot \vec{S}$$

由[[托马斯进动]]修正因子 $1/2$。总角动量 $\vec{J} = \vec{L} + \vec{S}$，$[\hat{J}^2, \hat{H}] = 0$ 使 $\vec{L}$ 和 $\vec{S}$ 分别不守恒而 $\vec{J}$ 守恒。

### 自旋-统计定理

[[自旋-统计定理]]（[[泡利]]，1940）：自旋为整数的粒子（[[玻色子]]）服从[[玻色-爱因斯坦统计_Bose-Einstein Statistics]]，自旋为半整数的粒子（[[费米子]]）服从[[费米-狄拉克统计_Fermi-Dirac Statistics]]。这一对应在[[量子场论]]中由[[局域性]]和[[洛伦兹不变性]]严格证明。

## 3. 物理学意义与推论

- 自旋是粒子的内禀性质，不能像轨道角动量那样通过坐标变换消去。它没有经典对应——经典角动量 $\vec{r} \times \vec{p}$ 总可通过旋转坐标系改变，而自旋是粒子的固有"标签"。
- **自旋1/2**是最基本的费米子自旋。电子、质子、中子、夸克和中微子均为自旋1/2粒子。自旋1/2粒子在旋转 $2\pi$ 后波函数变号（$|\psi\rangle \to -|\psi\rangle$），需旋转 $4\pi$ 才复原，这是[[SU(2)]]双值表示的体现，与经典旋转[[SO(3)]]有本质区别。
- **泡利不相容原理_Pauli Exclusion Principle**直接源于电子自旋1/2的费米子性质。自旋-统计定理将统计行为与自旋严格关联，是[[量子场论]]的深刻结论。[[费米-狄拉克统计_Fermi-Dirac Statistics]]的整个理论框架建立在自旋概念之上。
- **[[斯特恩-盖拉赫实验_Stern-Gerlach Experiment]]**是自旋的直接实验证据。银原子束在不均匀磁场中分裂为两束，对应 $m_s = \pm 1/2$。这一结果无法用轨道角动量解释（轨道角动量量子数为整数，应分裂为奇数束），直接揭示了自旋1/2的存在。
- **精细结构**：氢原子光谱的精细结构由自旋-轨道耦合和相对论修正引起。[[狄拉克方程]]的精确解自然包含自旋1/2和精细结构，是[[相对论量子力学]]的标志性成就。
- **反常塞曼效应**：磁场中原子光谱的复杂分裂模式需引入自旋才能解释。正常塞曼效应（$S=0$）和反常塞曼效应（$S \neq 0$）的区别直接体现了自旋的作用。
- **自旋是[[标准模型]]的基本量子数**：所有基本粒子均由自旋标记——[[费米子]]（自旋1/2的夸克和轻子）、[[规范玻色子]]（自旋1的光子、$W^\pm$、$Z^0$、胶子）和[[希格斯玻色子]]（自旋0）。[[引力子]]若存在则为自旋2（属于假说/未解决问题）。
- **自旋在[[量子信息]]中是核心资源**：自旋1/2天然构成量子比特（[[qubit]]），其二维[[希尔伯特空间]] $|\uparrow\rangle, |\downarrow\rangle$ 是[[量子计算]]的标准实现。[[核磁共振]]（NMR）、[[离子阱]]和[[NV色心]]均利用自旋作为量子比特。

## 4. 相关验证实验

- **[[斯特恩-盖拉赫实验_Stern-Gerlach Experiment]]**（1922，[[奥托·斯特恩]]、[[瓦尔特·盖拉赫]]）：银原子束在不均匀磁场中分裂为两束，首次直接观测到空间量子化和自旋1/2（尽管当时自旋概念尚未提出）。这是自旋存在的最直接实验证据。
- **反常塞曼效应观测**（1896-1897，[[彼得·塞曼]]）：磁场中原子光谱的复杂分裂模式促使[[乌伦贝克]]和[[古德斯密特]]提出自旋假说。
- **精细结构光谱**（1887，[[迈克尔逊]]-[[莫雷]]）：氢原子光谱的精细分裂由自旋-轨道耦合解释。
- **电子反常磁矩测量**（1947，[[库什]]-[[弗利]]）：$g_s \neq 2$ 的微小偏差（[[反常磁矩]]）由[[量子电动力学]]精确解释，是自旋与QED的联合验证。$g_s - 2$ 的实验与理论吻合至 $10^{-12}$，是物理学最精确的检验之一。
- **[[电子自旋共振]]（ESR）**（1944，[[扎沃伊斯基]]）：利用自旋在磁场中的塞曼分裂，通过微波共振直接测量 $g$ 因子，是自旋的精密实验技术。
- **[[量子自旋霍尔效应]]**（2007，[[克尼希]]等；[[Haldane]]理论预言）：在二维拓扑绝缘体中观测到无外加磁场的量子化自旋输运，是自旋在[[拓扑物态]]中的现代体现。
- **自旋纠缠实验**：利用光子或电子的自旋纠缠态检验[[贝尔不等式]]（[[阿斯佩]]，1982等），验证了自旋量子纠缠的非局域性。

## 5. 历史上下文

- 1922年[[斯特恩-盖拉赫实验_Stern-Gerlach Experiment]]观测到银原子束分裂为两束，但当时被解释为轨道角动量的空间量子化（[[索末菲]]的理论）。实际上银原子基态 $L=0$，分裂完全来自电子自旋，但这一点直到1925年才被认识。
- 1925年1月，[[乔治·乌伦贝克]]和[[塞缪尔·古德斯密特]]（均为[[埃伦费斯特]]的学生）提出电子具有内禀自旋 $s = 1/2$ 的假说。[[埃伦费斯特]]建议发表，但指出"年轻人太冒失"。[[洛伦兹]]的计算显示若自旋是机械旋转，电子表面速度需超光速，故自旋不可能是经典旋转。[[乌伦贝克]]欲撤稿但已付印。
- [[乌伦贝克]]-[[古德斯密特]]的假说最初受到质疑，但迅速解释了[[碱金属光谱]]的双线结构和[[反常塞曼效应]]，被广泛接受。[[海森堡]]和[[约旦]]给出了自旋的完整数学形式。
- 1927年[[泡利]]将自旋纳入[[量子力学]]的[[泡利矩阵]]形式（非相对论），建立了[[泡利方程]]——含自旋的[[薛定谔方程_Schrödinger Equation]]推广。[[泡利矩阵]]成为[[量子计算]]中量子比特操作的基本工具。
- 1928年[[狄拉克]]建立相对论性电子方程（[[狄拉克方程]]），自旋1/2自然从方程的结构中涌现，无需额外假设。这是自旋概念的理论顶峰——内禀角动量是[[相对论]]与[[量子力学]]结合的必然结果。
- 1940年[[泡利]]证明[[自旋-统计定理]]，将自旋与统计行为严格关联，是[[量子场论]]的基本结论。该定理排除了"整数自旋费米子"或"半整数自旋玻色子"的可能性。
- 1947年电子[[反常磁矩]] $g_s - 2$ 的发现（[[库什]]-[[弗利]]实验）推动了[[量子电动力学]]的发展。[[施温格]]计算了 $\alpha/(2\pi) \approx 0.00116$ 的一阶辐射修正，精确解释了反常磁矩。这是QED的里程碑成就之一。
- 21世纪以来，自旋在[[自旋电子学]]（[[Fert]]、[[Grünberg]]，1988发现[[巨磁阻效应]]，获2007年诺贝尔物理学奖）和[[量子计算]]中成为核心技术资源，推动了信息技术的革新。
