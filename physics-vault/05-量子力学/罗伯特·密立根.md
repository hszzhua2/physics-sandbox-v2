# 罗伯特·密立根 (Robert Millikan)

## 1. 生平
1868年生于伊利诺伊，在哥伦比亚大学获博士学位，任加州理工学院校长。以油滴实验精确测量电子电荷，1923年获诺贝尔物理学奖。

## 2. 主要贡献
- 用油滴实验精确测定[[电子电荷]]
- 验证[[光电效应]]的量子解释
- 研究宇宙射线

## 3. 关键公式与思想
- $e=1.602\times10^{-19}\,\mathrm{C}$（电子电荷）
- $E_k=h\nu-W$（光电效应方程验证）

## 4. 历史影响
密立根的实验精确测定基本电荷，并为[[光电效应]]的量子理论提供了关键验证。
