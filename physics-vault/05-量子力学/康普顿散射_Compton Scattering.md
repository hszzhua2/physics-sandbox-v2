# 康普顿散射 (Compton Scattering)

## 1. 实验目的与背景

[[康普顿散射]]是X射线被自由（或弱束缚）电子散射后波长增大的现象，由[[阿瑟·康普顿]]于1923年在华盛顿大学观测到。该实验直接验证了[[爱因斯坦]]1905年提出的[[光量子假说]]，提供了光子具有动量 $p = h/\lambda$ 的直接证据，是[[波粒二象性]]的确立性实验，与[[光电效应_Photoelectric Effect]]共同奠定了[[光子]]概念的物理地位。

实验背景：尽管[[爱因斯坦]]1905年的[[光量子假说]]成功解释了[[光电效应]]，但光子是否真正具有动量（而不仅是能量）尚存争议。经典[[电动力学]]将X射线散射视为[[汤姆孙散射]]——散射波频率不变，无法解释波长偏移。[[康普顿]]的实验精确测量了散射X射线的波长变化，直接揭示了光子作为粒子的弹性碰撞行为。

## 2. 实验装置与运行机制

### 实验装置

康普顿散射实验的装置包括：

1. **X射线源**：[[钼]]靶X射线管，产生特征X射线（如 $K_\alpha$ 线，$\lambda_0 \approx 0.71\;\text{Å}$）。
2. **散射体**：低原子序数材料（如石墨、石蜡、铝），其中外层电子可近似为自由电子。
3. **[[布拉格晶体光谱仪]]**：使用单晶（如[[方解石]]或[[岩盐]]）作为[[布拉格衍射]]光栅，精确测量散射X射线的波长。根据[[布拉格定律]] $n\lambda = 2d\sin\theta$，通过旋转晶体改变 $\theta$ 可扫描不同波长。
4. **探测器**：[[电离室]]或[[盖革计数器]]，测量各角度的X射线强度。

### 实验流程

1. X射线源发射单色特征X射线，经准直后照射散射体。
2. 在不同散射角 $\theta$（$0°$ 至 $180°$）放置布拉格光谱仪，测量散射X射线的波长谱。
3. 对比入射波长 $\lambda_0$ 和散射波长 $\lambda$，分析波长偏移 $\Delta\lambda = \lambda - \lambda_0$ 与散射角 $\theta$ 的关系。

### 物理机制：光子-电子弹性碰撞

[[康普顿]]将散射过程建模为**光子与自由电子的弹性碰撞**：

- 入射光子能量 $E = h\nu$，动量 $p = h/\lambda$。
- 电子初始静止（$E_e = m_e c^2$，$p_e = 0$）。
- 碰撞后：光子以角度 $\theta$ 散射，能量降低（波长增大）；电子获得反冲动量。

这一过程遵循[[能量守恒]]和[[动量守恒]]，但需使用[[狭义相对论]]的能量-动量关系 $E^2 = (pc)^2 + (m_0 c^2)^2$。

## 3. 观测数据与数学模型

### 康普顿波长偏移公式

由[[能量守恒]]和[[动量守恒]]，结合[[狭义相对论]]能量-动量关系：

$$\Delta\lambda = \lambda - \lambda_0 = \frac{h}{m_e c}(1 - \cos\theta)$$

即：

$$\lambda' - \lambda = \lambda_C (1 - \cos\theta)$$

其中[[康普顿波长]]：

$$\lambda_C = \frac{h}{m_e c} \approx 2.426 \times 10^{-12}\;\text{m} = 0.02426\;\text{Å}$$

### 康普顿波长的物理意义

$\lambda_C = h/(m_e c)$ 是电子的"自然波长"——当光子波长接近 $\lambda_C$ 时，量子效应显著。

### 散射光子能量

由 $\lambda' = \lambda + \lambda_C(1-\cos\theta)$，散射光子能量：

$$E' = \frac{E}{1 + \frac{E}{m_e c^2}(1 - \cos\theta)}$$

其中 $E = h\nu$ 为入射光子能量。

### 电子反冲动能

电子获得的反冲动能：

$$K_e = E - E' = E \cdot \frac{\frac{E}{m_e c^2}(1-\cos\theta)}{1 + \frac{E}{m_e c^2}(1-\cos\theta)}$$

### 微分散射截面（Klein-Nishina公式）

[[克莱因]]-[[仁科]]（1929）从[[狄拉克方程]]和[[量子电动力学]]导出光子-电子散射的微分散射截面：

$$\frac{d\sigma}{d\Omega} = \frac{r_e^2}{2} \left(\frac{\lambda}{\lambda'}\right)^2 \left[\frac{\lambda}{\lambda'} + \frac{\lambda'}{\lambda} - \sin^2\theta\right]$$

其中 $r_e = e^2/(4\pi\varepsilon_0 m_e c^2) \approx 2.818\;\text{fm}$ 为[[经典电子半径]]。

**低能极限**（$E \ll m_e c^2$，即 $\lambda \gg \lambda_C$）：$\lambda' \approx \lambda$，$\Delta\lambda \to 0$，恢复[[汤姆孙散射]]截面：

$$\frac{d\sigma}{d\Omega}\bigg|_{\text{Thomson}} = \frac{r_e^2}{2}(1 + \cos^2\theta)$$

### 实验观测的关键数据

[[康普顿]]1923年原始实验数据（钼 $K_\alpha$ 线，$\lambda_0 = 0.707\;\text{Å}$，石墨散射体）：

| 散射角 $\theta$ | 散射波长 $\lambda'$（Å） | 波长偏移 $\Delta\lambda$（Å） | 理论 $\lambda_C(1-\cos\theta)$（Å） |
|---|---|---|---|
| 0° | 0.707 | 0.000 | 0.000 |
| 45° | 0.714 | 0.007 | 0.0071 |
| 90° | 0.731 | 0.024 | 0.0243 |
| 135° | 0.749 | 0.042 | 0.0412 |
| 180° | 0.760 | 0.053 | 0.0485 |

实验数据与理论公式 $\Delta\lambda = \lambda_C(1-\cos\theta)$ 在误差范围内精确吻合。

### 两条散射峰的解释

实验观测到散射X射线谱中含**两条峰**：
1. **偏移峰**（波长增大）：来自光子与自由电子的康普顿散射。
2. **未偏移峰**（波长不变）：来自光子与原子内层紧束缚电子的散射（电子与原子核共同反冲，有效质量远大于 $m_e$，$\Delta\lambda \approx 0$）。

这一双峰结构进一步支持了光子-电子碰撞的物理图像。

## 4. 实验结论与影响

- **光子动量的直接验证**：[[康普顿散射]]是[[光子]]具有动量 $p = h/\lambda$ 的直接实验证据。[[光电效应_Photoelectric Effect]]验证了光子能量 $E = h\nu$，而康普顿散射进一步验证了光子动量。两者共同确立了光子的完整粒子性描述：$E = h\nu$, $p = h/\lambda$，以及[[狭义相对论]]关系 $E = pc$。
- **[[波粒二象性]]的确立**：康普顿散射与[[光电效应]]、[[黑体辐射]]共同构成了光的粒子性的三大实验证据，推翻了19世纪[[麦克斯韦方程组_Maxwell's Equations]]建立的纯波动图像。1924年[[德布罗意]]将波粒二象性推广至实物粒子（[[德布罗意波]] $\lambda = h/p$），1927年[[戴维森]]-[[革末]]的电子衍射实验验证了电子的波动性，最终确立了波粒二象性为所有微观粒子的普遍性质。
- **[[狭义相对论]]的验证**：康普顿散射的分析需使用相对论能量-动量关系 $E^2 = (pc)^2 + (m_0c^2)^2$，是量子力学与[[狭义相对论]]结合的早期实例。Klein-Nishina公式的精确推导更需[[狄拉克方程]]（相对论量子力学），将康普顿散射提升为相对论量子力学的验证实验。
- **[[量子电动力学]]的奠基**：Klein-Nishina公式（1929）是[[狄拉克方程]]和早期量子电动力学的首批定量预言之一，与实验精确吻合。后续[[量子电动力学]]的完整理论（[[费曼]]、[[施温格]]、[[朝永振一郎]]，1940s）将康普顿散射纳入费曼图框架（[[康普顿散射费曼图]]含两个顶点），给出了更精确的辐射修正。
- **[[康普顿]]获1927年诺贝尔物理学奖**：[[阿瑟·康普顿]]因发现康普顿效应获奖，与[[查尔斯·威尔逊]]（[[云室]]发明者）共享。[[康普顿]]的实验被认为是"光子假说的最终判决性证据"。
- **天体物理应用**：[[逆康普顿散射]]（高能电子将能量转移给低能光子）是[[X射线天文学]]和[[伽马射线天文学]]中的重要辐射机制，解释了[[活动星系核]]和[[脉冲星]]的高能辐射。[[宇宙微波背景辐射]]光子与高能电子的康普顿散射（[[Sunyaev-Zel'dovich效应]]）是宇宙学的重要观测手段。
- **医学与工业应用**：康普顿散射是X射线和伽马射线与物质相互作用的主要机制之一，是[[辐射防护]]、[[辐射剂量学]]和[[医学影像]]（如[[康普顿成像]]）的基础。
- **[[光子]]概念的最终确立**：康普顿散射后，光子作为真实的物理粒子被物理学界普遍接受。[[光子]]不再是[[爱因斯坦]]1905年的"启发性观点"，而是具有确定能量和动量的基本粒子。这一转变为[[量子场论]]中光子作为电磁场的量子激发奠定了概念基础。

### 历史意义总结

康普顿散射实验是物理学史上最重要的判决性实验之一。它从X射线波长偏移这一"细节"出发，直接揭示了光子的粒子性（能量和动量），与[[光电效应]]、[[黑体辐射]]共同构成了量子力学诞生的三大实验支柱。康普顿散射不仅是光子假说的"最后证据"，也是[[量子力学]]与[[狭义相对论]]结合的早期实例，更是[[量子电动力学]]的奠基实验之一。从1923年[[康普顿]]的原始测量到21世纪[[高能物理]]中的精密散射实验，康普顿散射始终是检验量子理论的基本工具。
