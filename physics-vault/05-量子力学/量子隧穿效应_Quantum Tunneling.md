# 量子隧穿效应 (Quantum Tunneling)

## 1. 核心定义

[[量子隧穿效应]]是[[量子力学]]中粒子以一定概率穿过经典禁戒的[[势垒]]（粒子能量低于势垒高度）的现象，由[[乔治·伽莫夫]]等人于1928年用于解释 $\alpha$ 衰变，是量子力学非经典行为的标志性体现，在固体电子学、核物理和宇宙学中有广泛应用。

## 2. 数学表达

### 一维矩形势垒

考虑能量为 $E$ 的粒子遇到高度为 $V_0$、宽度为 $a$ 的矩形势垒（$E < V_0$）：

$$V(x) = \begin{cases} 0 & x < 0 \text{ 或 } x > a \\ V_0 & 0 \leq x \leq a \end{cases}$$

在势垒区域（$0 \leq x \leq a$），波函数指数衰减：

$$\psi(x) = C e^{-\kappa x} + D e^{\kappa x}, \quad \kappa = \sqrt{\frac{2m(V_0 - E)}{\hbar^2}}$$

**透射系数**（$E < V_0$，宽势垒近似 $\kappa a \gg 1$）：

$$T \approx \frac{16E(V_0 - E)}{V_0^2} e^{-2\kappa a}$$

在 $\kappa a \gg 1$ 极限下，指数项主导：

$$T \propto e^{-2\kappa a} = \exp\!\left[-\frac{2a}{\hbar}\sqrt{2m(V_0 - E)}\right]$$

### WKB近似

对任意形状势垒 $V(x)$（$E < V(x)$），[[WKB近似]]给出透射系数：

$$T \approx \exp\!\left[-\frac{2}{\hbar} \int_{x_1}^{x_2} \sqrt{2m(V(x) - E)}\, dx\right]$$

其中 $x_1, x_2$ 为经典转折点（$V(x_1) = V(x_2) = E$）。这一公式由[[伽莫夫]]、[[格尼]]和[[康登]]独立导出，用于解释 $\alpha$ 衰变。

### Gamow 因子

$$G = \exp\!\left[-\frac{2}{\hbar} \int_{r_1}^{r_2} \sqrt{2\mu(V(r) - E)}\, dr\right]$$

其中 $\mu$ 为约化质量，$r_1, r_2$ 为原子核- $\alpha$ 粒子系统的转折点。这是[[伽莫夫]]在 $\alpha$ 衰变理论中引入的隧穿概率因子。

### 隧穿电流（隧道结）

金属-绝缘体-金属结中的隧穿电流（[[Simmons公式]]）：

$$J \propto \int_0^{eV} \rho_1(E_F - eV + \varepsilon) \rho_2(E_F + \varepsilon) T(\varepsilon)\, d\varepsilon$$

其中 $\rho_1, \rho_2$ 为两侧态密度，$T(\varepsilon)$ 为能量 $\varepsilon$ 的透射系数。

### 隧穿二极管公式

[[Esaki]]隧道二极管的伏安特性在负阻区由隧穿主导：

$$I \propto \int [f(E) - f(E + eV)]\, \rho_c(E)\, \rho_v(E + eV)\, T(E)\, dE$$

## 3. 物理学意义与推论

- 量子隧穿是[[量子力学]]最反直觉的预言之一：粒子能量低于势垒高度时，在经典力学中不可能穿过，但在量子力学中概率不为零。这一现象直接源于[[波函数]]的波动性和[[不确定性原理_Uncertainty Principle]]——动量的不确定使粒子"借"到额外能量穿越势垒（[[能量-时间不确定性]]的定性解释）。
- **透射系数** $T \propto e^{-2\kappa a}$ 对势垒宽度 $a$ 和高度 $V_0$ 极其敏感。每增加 $1\;\text{nm}$ 的势垒宽度，透射率可下降数个量级。这种指数敏感性使隧穿效应在纳米尺度显著而在宏观尺度可忽略，是量子-经典过渡的典型体现。
- **$\alpha$ 衰变**：[[伽莫夫]]、[[格尼]]和[[康登]]（1928）用量子隧穿解释了 $\alpha$ 粒子如何逃离原子核的库仑势垒。他们推导出衰变常数 $\lambda$ 与 $\alpha$ 粒子能量 $E$ 的关系（[[Geiger-Nuttall定律]]）：

  $$\ln \lambda = A - B/\sqrt{E}$$

  与实验数据精确吻合，是量子隧穿的最早定量验证。
- **扫描隧道显微镜（STM）**（[[宾尼希]]-[[罗雷尔]]，1981）：利用电子在针尖与样品间的量子隧穿，测量 $T \propto e^{-2\kappa d}$ 随距离 $d$ 的变化，实现原子级分辨率成像。STM是量子隧穿在技术中的里程碑应用，获1986年诺贝尔物理学奖。
- **[[隧穿二极管]]**（[[江崎玲于奈]]，1957）：利用半导体p-n结中的量子隧穿产生负阻效应，是首个被利用的量子电子器件。[[江崎]]因此获1973年诺贝尔物理学奖。
- **化学中的隧穿**：氢原子转移反应中量子隧穿起重要作用，影响低温化学反应速率（如星际化学）。[[基元同位素效应]]是隧穿的直接表现——氢比氘更容易隧穿。
- **生物中的隧穿**：[[酶催化]]中[[电子隧穿]]被认为是某些氧化还原酶的关键机制（如[[细胞色素c氧化酶]]中的电子传递链）。
- **宇宙学中的隧穿**：[[宇宙暴胀]]理论中假真空的衰变通过[[气泡成核]]（[[Coleman]]-[[De Luccia]]隧穿）实现，是[[早期宇宙学]]的重要机制（属于假说/未解决问题）。[[霍金辐射]]中黑洞的粒子辐射也涉及量子隧穿。
- **[[超导]]中的[[约瑟夫森效应]]**：超导电子对（库珀对）通过绝缘势垒的隧穿产生零电压超流电流（[[直流约瑟夫森效应]]），是宏观量子隧穿的典型。[[约瑟夫森]]因此获1973年诺贝尔物理学奖。
- 在[[一维势阱_1D Potential Well]]中的延伸：有限深势阱中粒子波函数在阱外指数衰减但不为零，与隧穿效应密切关联。[[共振隧穿]]（量子阱双势垒）可实现选择性高透射，是[[量子级联激光器]]的基础。

## 4. 相关验证实验

- **[[α衰变]]理论验证**（1928，[[伽莫夫]]-[[格尼]]-[[康登]]）：从量子隧穿理论导出的[[Geiger-Nuttall定律]] $\ln\lambda = A - B/\sqrt{E}$ 与实验精确吻合，是隧穿效应的首次定量验证。该理论成功解释了为何不同核素的 $\alpha$ 衰变半衰期跨越20多个数量级。
- **[[扫描隧道显微镜]]实验**（1981，[[宾尼希]]-[[罗雷尔]]）：STM直接利用隧穿电流随针尖-样品距离的指数变化实现原子成像。STM图像清晰展示了单原子排列，是隧穿效应的精确技术化验证。获1986年诺贝尔物理学奖。
- **[[隧穿二极管]]实验**（1957，[[江崎玲于奈]]）：半导体p-n结中观测到负阻伏安特性，与隧穿理论一致。[[江崎]]因此获1973年诺贝尔物理学奖。
- **[[约瑟夫森效应]]实验**（1962理论，1963实验[[安德森]]-[[罗威尔]]）：超导结中库珀对的宏观隧穿直接验证了[[约瑟夫森]]的理论预言。[[约瑟夫森]]获1973年诺贝尔物理学奖。
- **场发射显微镜**（[[米勒]]，1936）：利用强电场下电子从金属表面隧穿至真空，直接成像表面原子结构。
- **分子隧穿实验**：在[[分子结]]中测量单分子隧穿电流，可探测分子轨道和电子输运机制，是[[分子电子学]]的基础。
- **冷原子隧穿实验**：在[[光学晶格]]中囚禁的冷原子可通过势垒隧穿，实验可直接观测[[玻色-哈伯德模型]]中的隧穿-相互作用竞争导致的超流-Mott绝缘体相变（[[Greiner]]等，2002）。

## 5. 历史上下文

- 量子隧穿的概念在[[薛定谔方程_Schrödinger Equation]]建立后不久即被认识。1926年[[薛定谔]]本人已注意到波函数可穿透经典禁戒区（指数衰减而非严格为零）。
- 1928年，[[乔治·伽莫夫]]（苏联）和[[格尼]]-[[康登]]（美英）独立用量子隧穿解释 $\alpha$ 衰变。这是量子力学对核物理的第一个重大应用，使半衰期跨越20+数量级的实验事实获得统一解释。[[伽莫夫]]的原始论文最初被 *Nature* 拒稿，后发表于 *Zeitschrift für Physik*。
- 1928年[[奥本海默]]将隧穿用于解释金属-真空界面在场致发射中的电子逸出，[[福勒]]-[[诺德海姆]]给出定量公式（[[Fowler-Nordheim隧穿]]），是场发射显微镜的理论基础。
- 1930s-1950s，隧穿效应在[[半导体物理]]中逐渐被利用。1957年[[江崎玲于奈]]发现[[隧穿二极管]]（[[Esaki二极管]]），是首个量子电子学器件，标志着量子隧穿从理论走向技术。
- 1962年[[约瑟夫森]]预言[[约瑟夫森效应]]，将隧穿推广至超导库珀对。这一发现催生了[[超导量子干涉仪]]（SQUID）等精密测量技术，并将隧穿与[[宏观量子现象]]联系起来。
- 1981年[[宾尼希]]和[[罗雷尔]]在IBM苏黎世发明[[扫描隧道显微镜]]（STM），将隧穿效应的指数敏感性转化为原子级成像能力。STM的革命性在于首次实现了对单原子的直接成像和操控，开启了[[纳米科技]]时代。
- 21世纪以来，量子隧穿在[[量子计算]]（[[超导量子比特]]中的电荷隧穿）、[[分子电子学]]（单分子结输运）、[[冷原子物理]]（光学晶格中的原子隧穿）等前沿领域持续发挥核心作用。隧穿效应也从单纯的量子力学现象演变为精密测量和量子技术的基础工具。
- 在[[宇宙学]]中，[[暴胀理论]]和[[假真空衰变]]假说涉及量子隧穿在早期宇宙中的作用（[[Coleman]]-[[De Luccia]]瞬子方法），是将量子隧穿推广至引力系统的尝试（属于假说/未解决问题）。
