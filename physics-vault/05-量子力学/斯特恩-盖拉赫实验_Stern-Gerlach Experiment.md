# 斯特恩-盖拉赫实验 (Stern-Gerlach Experiment)

## 1. 实验目的与背景

[[斯特恩-盖拉赫实验]]由[[奥托·斯特恩]]和[[瓦尔特·盖拉赫]]于1922年在法兰克福完成，旨在检验原子角动量的[[空间量子化]]假说。[[旧量子论]]（[[玻尔]]-[[索末菲]]模型）预言轨道角动量分量只能取分立值 $m_l\hbar$（$m_l = -l, -l+1, \ldots, l$），而非经典力学中的连续取值。若空间量子化成立，原子束在不均匀磁场中应分裂为分立的 $2l+1$ 条束。

实验的核心动机是检验[[旧量子论]]的[[空间量子化]]是否为真实的物理效应，而非纯数学假设。同时，该实验也检验了[[玻尔]]-[[索末菲]]模型的角动量量子数预言。

## 2. 实验装置与运行机制

### 实验装置

斯特恩-盖拉赫实验装置包括：

1. **银原子源**：在高温炉中蒸发银（Ag），银原子通过小孔形成细束。选择银是因为银原子基态为 $^2S_{1/2}$（$L=0$），总角动量完全来自电子自旋（当时自旋概念尚未提出）。
2. **准直狭缝**：两道狭缝将原子束准直为极窄的平行束。
3. **不均匀磁场**：一对特殊设计的磁极（上极凹、下极凸），产生 $z$ 方向梯度 $\partial B_z/\partial z$ 很大的不均匀磁场。
4. **检测板**：玻璃板或金属板置于磁场下游，银原子沉积形成可观测的条纹。

### 运行机制

**磁矩与力的关系**：银原子具有磁矩 $\vec{\mu}$（由电子角动量贡献），在不均匀磁场中受力：

$$F_z = \mu_z \frac{\partial B_z}{\partial z}$$

**经典预期**：若角动量可任意取向（经典力学），$\mu_z$ 连续分布，原子束应在检测板上形成连续展宽的带。

**量子力学预言**：若[[空间量子化]]成立，$\mu_z$ 只取分立值 $g_J m_J \mu_B$（$\mu_B$ 为[[玻尔磁子]]），原子束应分裂为 $2J+1$ 条分立束。

**轨道角动量预言**：若仅考虑轨道角动量（$L=1$），应分裂为 $2L+1 = 3$ 条束（$m_l = -1, 0, +1$）。

### 历史关键：为何选择银

[[斯特恩]]选择银原子是实验成功的关键。银原子基态电子组态为 $[\text{Kr}] 4d^{10} 5s^1$，最外层单 $s$ 电子的轨道角动量 $L=0$。因此银原子的总角动量完全来自电子自旋 $S=1/2$（当时未知）。若选择其它原子（如氢的 $2p$ 态，$L=1$），结果将更复杂，自旋效应可能被掩盖。

## 3. 观测数据与数学模型

### 观测结果

实验观测到银原子束在检测板上分裂为**两条分立条纹**，而非连续带。这一结果具有决定性意义：

- 分裂为分立条纹**证实了空间量子化**——角动量分量确实只取分立值。
- 但分裂为**两**条（而非三条）与轨道角动量 $L=1$ 的预言（$2L+1=3$）**不符**。

### 数学解释（事后理解）

银原子基态 $L=0$，轨道角动量为零。总角动量 $J = L + S = 0 + 1/2 = 1/2$，故 $m_J = \pm 1/2$，对应**两条**束：

$$\mu_z = -g_J m_J \mu_B = \mp \frac{g_s}{2}\mu_B$$

其中 $g_s \approx 2$（电子[[自旋g因子]]），$g_J \approx 2$。

原子受力：

$$F_z = \mu_z \frac{\partial B_z}{\partial z} = \mp \frac{g_s \mu_B}{2} \frac{\partial B_z}{\partial z}$$

两条束的间距：

$$\Delta z = \frac{\mu_B}{m_{\text{Ag}} v^2} \frac{\partial B_z}{\partial z} \cdot L^2$$

其中 $m_{\text{Ag}}$ 为银原子质量，$v$ 为原子速度，$L$ 为磁场区长度。

### 条纹间距的定量检验

1922年原始实验数据：
- 炉温约 $1000\;\text{K}$，银原子平均速度 $v \sim 500\;\text{m/s}$
- 磁场梯度 $\partial B_z/\partial z \sim 10^2\;\text{T/m}$
- 磁场区长度 $L \sim 3.5\;\text{cm}$
- 观测到两条纹间距约 $0.2\;\text{mm}$

这一间距与理论预言定量吻合。

### 串接Stern-Gerlach实验

将两个S-G装置串联（[[Weyl]]-[[ reinterpretation]]）可展示量子测量的本质：

1. **第一个S-G**（$z$方向）：束分裂为 $|\uparrow_z\rangle$ 和 $|\downarrow_z\rangle$。
2. **选择 $|\uparrow_z\rangle$ 进入第二个S-G**（$z$方向）：只有 $|\uparrow_z\rangle$ 输出（确定性）。
3. **选择 $|\uparrow_z\rangle$ 进入第二个S-G**（$x$方向）：分裂为 $|\uparrow_x\rangle$ 和 $|\downarrow_x\rangle$（各 $50\%$）。

这表明 $z$ 方向自旋确定后，$x$ 方向自旋完全不确定，是[[不确定性原理_Uncertainty Principle]]的直接体现（$[\hat{S}_z, \hat{S}_x] = i\hbar\hat{S}_y \neq 0$）。

## 4. 实验结论与影响

- **空间量子化的直接验证**：斯特恩-盖拉赫实验首次直接、实验性地证实了[[空间量子化]]——角动量在空间中的投影只取分立值。此前[[旧量子论]]的空间量子化仅是理论假设，无直接实验证据。这一验证是[[量子力学]]发展的里程碑。
- **自旋的发现**：实验结果"分裂为两条"与轨道角动量预言（$2L+1=3$）不符，这是困扰物理学家的"反常"。1925年[[乌伦贝克]]和[[古德斯密特]]提出电子[[自旋_Spin]] $s=1/2$ 后，"分裂为两条"的谜团才被解开——银原子基态 $L=0$，磁矩完全来自电子自旋。S-G实验成为自旋1/2存在的最直接实验证据。
- **自旋的后续验证**：S-G实验直接催生了[[自旋_Spin]]概念。1927年[[泡利]]将自旋纳入[[量子力学]]（[[泡利矩阵]]），1928年[[狄拉克]]从[[狄拉克方程]]自然导出自旋1/2，完成了自旋的理论化。
- **量子测量的范式**：S-G实验是[[量子测量]]的典型范例。串接S-G实验展示了测量对量子态的"制备"和"破坏"——测量 $S_z$ 后态坍缩为 $|\uparrow_z\rangle$ 或 $|\downarrow_z\rangle$，再测 $S_x$ 得到不确定结果。这是[[玻恩概率解释_Born Rule]]和[[态叠加原理_Superposition Principle]]的实验体现。
- **[[斯特恩]]获1943年诺贝尔物理学奖**：尽管主要因分子束方法获奖，S-G实验是其核心贡献之一。[[盖拉赫]]未获奖。
- **量子信息的基础**：S-G装置是制备和测量自旋量子比特的标准工具。在[[量子计算]]中，自旋1/2是最基本的量子比特实现，其操作（旋转、测量）均与S-G实验的原理直接关联。
- **原子束与分子束方法**：S-G实验开创了[[原子分子束实验]]方法，后续发展包括[[核磁共振]]（NMR）、[[原子钟]]、[[玻色-爱因斯坦凝聚]]实验等。[[斯特恩]]和[[拉比]]等人发展的分子束磁共振方法是现代精密测量物理的基础。
- **对[[量子力学诠释]]的影响**：S-G实验中原子"选择"某一本征值的过程，是[[量子测量问题]]的典型实例。波函数在通过磁场时是否"真实地坍缩"（[[哥本哈根诠释]]）或"分裂为多个分支"（[[多世界诠释]]），至今仍是诠释争议的核心（属于未解决问题）。

### 历史意义总结

斯特恩-盖拉赫实验是物理学史上最精巧的基础实验之一。它以极简的装置——不均匀磁场和原子束——直接展示了量子力学的两个核心特征：[[空间量子化]]和[[自旋]]。该实验的意义远超最初的检验目的，成为自旋1/2的发现实验和量子测量的范式实验。其后续发展（串接S-G、S-G-自旋旋转-S-G序列）奠定了[[量子态制备]]和[[量子测量]]的现代框架，是连接量子力学基础理论与实验技术的桥梁。
