# 态叠加原理 (Superposition Principle)

## 1. 核心定义

[[态叠加原理]]是[[量子力学]]的基本公理之一，指出若 $|\psi_1\rangle$ 和 $|\psi_2\rangle$ 是系统的可能量子态，则其线性组合 $|\psi\rangle = c_1|\psi_1\rangle + c_2|\psi_2\rangle$ 也是系统可能的量子态，由[[保罗·狄拉克]]在[[狄拉克符号]]体系中系统表述。

## 2. 数学表达

### 量子态的线性叠加

在[[希尔伯特空间]]中，量子态 $|\psi\rangle$ 可展开为某组基矢 $\{|n\rangle\}$ 的线性组合：

$$|\psi\rangle = \sum_n c_n |n\rangle, \quad \sum_n |c_n|^2 = 1$$

其中 $c_n = \langle n | \psi \rangle$ 为[[概率幅]]，$|c_n|^2$ 为测量得到本征态 $|n\rangle$ 的概率。

### 连续谱展开

对位置表象：

$$|\psi\rangle = \int |\vec{r}\rangle \psi(\vec{r})\, d^3r$$

其中 $\psi(\vec{r}) = \langle \vec{r} | \psi \rangle$ 即通常的[[波函数]]。

### 态叠加与干涉

叠加态的测量概率不仅含各分量概率之和，还含交叉项（[[干涉项]]）：

$$|\langle \phi | \psi \rangle|^2 = |c_1 \langle \phi | 1 \rangle + c_2 \langle \phi | 2 \rangle|^2 = |c_1|^2 |\langle\phi|1\rangle|^2 + |c_2|^2 |\langle\phi|2\rangle|^2 + 2\,\text{Re}\!\left(c_1^* c_2 \langle\phi|1\rangle^* \langle\phi|2\rangle\right)$$

最后一项即[[量子干涉]]项，其符号取决于概率幅的相位关系。若该项为零（退相干），则退化为经典概率叠加。

### 测量坍缩

对可观测量 $\hat{A}$ 的本征态 $\{|a_n\rangle\}$ 展开后，测量 $\hat{A}$ 得到本征值 $a_n$ 的概率为 $|c_n|^2$，测量后态"坍缩"为：

$$|\psi\rangle \xrightarrow{\text{测量得 } a_n} |a_n\rangle$$

这一过程即[[波函数坍缩]]，是[[量子测量问题]]的核心。

### 薛定谔猫态

最著名的叠加态是[[薛定谔猫]]态——宏观死活态的叠加：

$$|\text{猫}\rangle = \frac{1}{\sqrt{2}}\left(|\text{活}\rangle + |\text{死}\rangle\right)$$

这一思想实验揭示了宏观叠加态的悖论性（属于量子力学诠释争议/未解决问题）。

## 3. 物理学意义与推论

- 态叠加原理是量子力学区别于经典物理的根本特征之一。经典概率叠加遵循 $P = P_1 + P_2$（概率相加），而量子叠加遵循概率幅相加 $|\psi\rangle = c_1|\psi_1\rangle + c_2|\psi_2\rangle$，导致干涉效应。这一区别是[[双缝干涉_Double-Slit Interference]]中电子"同时通过两条缝"的数学根源。
- **[[双缝干涉_Double-Slit Interference]]**是态叠加原理最直观的体现：电子通过双缝后的态为 $|\psi\rangle = |缝1\rangle + |缝2\rangle$，屏幕上的干涉图样来自两个概率幅的相干叠加。若探测电子通过哪条缝（路径信息），叠加退相干，干涉消失——这是[[量子互补性]]的体现。
- 态叠加原理直接源于[[薛定谔方程_Schrödinger Equation]]的**线性性**：若 $\Psi_1$ 和 $\Psi_2$ 是解，则 $c_1\Psi_1 + c_2\Psi_2$ 也是解。因此态叠加与薛定谔方程的线性演化是自洽的。
- **测量坍缩**与线性演化之间的矛盾是[[量子测量问题]]的核心：薛定谔方程给出幺正演化（可逆），而测量过程给出非幺正的坍缩（不可逆）。这一矛盾在[[哥本哈根诠释]]中被视为公理，在[[多世界诠释]]中被消解为"全部分支均实现"，在[[隐变量理论]]（如[[玻姆力学]]）中被归因于隐变量的非局域性。该问题至今未完全解决（属于未解决问题）。
- **量子纠缠**是态叠加原理在多粒子系统中的直接推论：复合系统的态可为不可分离的叠加 $|\Psi\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$，即[[贝尔态]]。纠缠态无法分解为子系统的直积，是[[量子信息]]的核心资源。
- **量子并行性**是[[量子计算]]的基础：量子计算机的 $n$ 个量子比特可同时处于 $2^n$ 个基态的叠加，理论上实现对所有基态的"并行"处理。这一能力由[[Shor算法]]（大数分解）和[[Grover算法]]（搜索）具体体现。
- **退相干**（[[Zurek]]，1980s）解释了为何宏观物体不呈现叠加态：与环境纠缠导致叠加态的干涉项在极短时间内衰减（退相干时间 $\tau_D \sim \hbar/(k_BT \cdot \Delta x^2/\lambda_{\text{th}}^2)$），使宏观系统表现为经典混合态。退相干理论部分回答了[[薛定谔猫]]悖论，但测量问题仍未完全解决。

## 4. 相关验证实验

- **[[双缝干涉_Double-Slit Interference]]实验**（电子版，1961，[[约恩松]]；单电子版，1989，[[外村彰]]）：单电子双缝实验直接观测到干涉图样的逐步形成，是态叠加原理最直接的实验验证。[[外村彰]]实验证明即使每次只有一个电子通过，干涉图样仍会出现，排除了电子间相互作用的解释。
- **[[贝尔不等式]]实验**（1972，[[弗里德曼]]-[[克劳泽]]；1982，[[阿斯佩]]；2015，[[亨泽]]-[[罗伊德]]等无漏洞实验）：验证了量子纠缠态违反贝尔不等式，排除了局域隐变量理论，间接支持态叠加原理在多粒子系统中的有效性。[[阿斯佩]]、[[克劳泽]]、[[塞林格]]获2022年诺贝尔物理学奖。
- **量子叠加宏观化实验**：在超导量子比特（[[超导库珀对盒]]）、离子阱（[[ Monroe]] -[[Wineland]]）、光子干涉等系统中实现了越来越大的量子叠加态。[[Wineland]]和[[Haroche]]获2012年诺贝尔物理学奖。
- **[[量子计算]]实验**：[[IBM]]、[[Google]]、[[中科大潘建伟组]]等在超导和离子阱量子比特中实现了多比特叠加态和量子算法，验证了量子并行性的物理可行性。
- **分子干涉实验**：[[分子干涉实验]]（如 $C_{60}$ 富勒烯双缝干涉，[[阿恩特]]-[[蔡林格]]，1999）将量子叠加推广到大分子，探索了量子-经典边界。

## 5. 历史上下文

- 1925年[[海森堡]]建立[[矩阵力学]]时已隐含态叠加的线性结构，但表述不直观。1926年[[薛定谔]]的波动力学通过[[波函数]]的线性叠加展示了叠加原理的物理图景。
- 1926年[[保罗·狄拉克]]在论文中首次系统使用了[[狄拉克符号]] $|\psi\rangle$ 和 $\langle\phi|$，明确阐述了态叠加原理，将其提升为量子力学的基本公理。狄拉克的《量子力学原理》（1930）成为标准教材。
- 1935年[[爱因斯坦]]-[[波多尔斯基]]-[[罗森]]提出[[EPR佯谬]]，质疑量子力学的完备性，指出纠缠态的非局域性似乎违反定域实在论。1964年[[贝尔]]提出[[贝尔不等式]]使EPR争论可实验检验。
- 1935年[[薛定谔]]提出"[[薛定谔猫]]"思想实验，以宏观猫的死活叠加揭示态叠加原理在宏观尺度的悖论性，推动了量子测量问题和量子-经典边界的长期讨论。
- 1957年[[埃弗里特]]提出[[多世界诠释]]，将测量坍缩解释为宇宙波函数的全局幺正演化，所有可能结果在"平行宇宙"中均实现。该诠释避免了坍缩假设，但代价是接受多重实在（属于诠释争议）。
- 1970s-1980s，[[Zurek]]、[[祖雷克]]等发展[[退相干理论]]，定量解释了宏观叠加态的衰减机制，为量子-经典过渡提供了部分物理基础。
- 21世纪以来，态叠加原理在[[量子信息]]和[[量子计算]]中的技术应用呈爆发式增长。[[量子纠错]]、[[量子隐形传态]]、[[量子密钥分发]]等技术均以态叠加和量子纠缠为基础。
