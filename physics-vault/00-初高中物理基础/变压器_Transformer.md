# 变压器 (Transformer)

## 1. 核心定义

**变压器**（Transformer）是利用[[电磁感应现象_Electromagnetic Induction]]原理改变[[交变电流_Alternating Current]]电压的电气设备。它由一个闭合铁芯和绕在铁芯上的两组线圈（原线圈/初级线圈和副线圈/次级线圈）组成，通过互感效应实现电能从原线圈到副线圈的传输，同时改变电压和电流的大小。

变压器的工作基础是法拉第电磁感应定律：当原线圈中通入交变电流时，铁芯中产生交变磁通量，该交变磁通量同时穿过原线圈和副线圈，在副线圈中感应出电动势。由于原、副线圈的匝数不同，感应电动势的电压也不同，从而实现升压或降压。

变压器是现代电力系统的关键设备，使得高压远距离输电和低压安全用电成为可能。其概念最早由[[迈克尔·法拉第]]的电磁感应发现所奠基，实用变压器的设计则由[[尼古拉·特斯拉]]等人在交流电系统发展中完善。

## 2. 数学表达

### 2.1 理想变压器电压比

对于**理想变压器**（忽略铁芯磁漏、线圈铜损和铁芯铁损），原线圈和副线圈的电压比等于匝数比：

$$\frac{U_1}{U_2} = \frac{N_1}{N_2}$$

其中：
- $U_1$、$U_2$ 为原、副线圈电压（有效值）
- $N_1$、$N_2$ 为原、副线圈匝数

**推导**：交变磁通量 $\Phi(t)$ 同时穿过原、副线圈，感应电动势分别为 $\varepsilon_1 = N_1 \frac{d\Phi}{dt}$ 和 $\varepsilon_2 = N_2 \frac{d\Phi}{dt}$，因此 $\varepsilon_1 / \varepsilon_2 = N_1 / N_2$。理想情况下 $U_1 \approx \varepsilon_1$，$U_2 \approx \varepsilon_2$。

- 当 $N_2 > N_1$ 时，$U_2 > U_1$，为**升压变压器**。
- 当 $N_2 < N_1$ 时，$U_2 < U_1$，为**降压变压器**。

### 2.2 理想变压器功率关系

理想变压器无能量损耗，输入功率等于输出功率：

$$P_1 = P_2$$

即：

$$U_1 I_1 = U_2 I_2$$

由此可得电流比：

$$\frac{I_1}{I_2} = \frac{N_2}{N_1} = \frac{U_2}{U_1}$$

即电流与匝数成反比。升压变压器在升高电压的同时减小电流，降压变压器在降低电压的同时增大电流。

### 2.3 阻抗变换

变压器还可实现阻抗变换。设副线圈接负载电阻 $R_L$，则从原线圈看入的等效电阻（反射阻抗）为：

$$R_{\text{eq}} = \frac{U_1}{I_1} = \frac{\frac{N_1}{N_2} U_2}{\frac{N_2}{N_1} I_2} = \left(\frac{N_1}{N_2}\right)^2 R_L$$

这一特性在电子电路的阻抗匹配中极为重要。

### 2.4 实际变压器的损耗与效率

实际变压器存在两类主要损耗：

**铜损**（Copper Loss）：线圈导线电阻 $R_{\text{铜}}$ 引起的焦耳热损耗：

$$P_{\text{铜}} = I_1^2 R_1 + I_2^2 R_2$$

铜损随负载电流增大而增大，属于可变损耗。

**铁损**（Iron Loss）：铁芯中的损耗，包括**磁滞损耗**（磁畴反复翻转摩擦生热）和**涡流损耗**（交变磁通在铁芯中感应涡电流生热）。铁损在空载时即存在，主要取决于电压和频率，属于不变损耗。

$$P_{\text{铁}} = P_{\text{磁滞}} + P_{\text{涡流}}$$

为减小涡流损耗，变压器铁芯通常采用相互绝缘的硅钢片叠压而成。

变压器效率为：

$$\eta = \frac{P_{\text{out}}}{P_{\text{in}}} = \frac{U_2 I_2 \cos\varphi_2}{U_2 I_2 \cos\varphi_2 + P_{\text{铜}} + P_{\text{铁}}} \times 100\%$$

大型电力变压器的效率可达 $98\%$ 以上。

### 2.5 远距离输电模型

设发电厂输出功率 $P$，输电线路总电阻 $R_{\text{线}}$，输电电压 $U_{\text{输}}$，则：

输电电流：

$$I_{\text{输}} = \frac{P}{U_{\text{输}}}$$

线路损耗功率：

$$P_{\text{损}} = I_{\text{输}}^2 R_{\text{线}} = \frac{P^2 R_{\text{线}}}{U_{\text{输}}^2}$$

线路电压损失：

$$\Delta U = I_{\text{输}} R_{\text{线}} = \frac{P \, R_{\text{线}}}{U_{\text{输}}}$$

可见，输电电压越高，线路损耗越小。将输电电压提高 $n$ 倍，线路损耗降为原来的 $1/n^2$。这就是远距离输电采用高压输电的根本原因。

## 3. 物理学意义与推论

### 3.1 互感原理的工程应用

变压器是[[电磁感应现象_Electromagnetic Induction]]中互感现象的最重要工程应用。铁芯的作用是集中和引导磁通量，使原、副线圈之间的磁耦合尽可能紧密（耦合系数接近1），从而实现高效的能量传输。

### 3.2 高压输电的经济与物理意义

远距离输电中，线路电阻 $R_{\text{线}} = \rho L / S$ 不可避免。提高输电电压是减少线损最有效的方法：$P_{\text{损}} \propto 1/U_{\text{输}}^2$。中国采用"特高压"输电技术（$\pm 800 \, \text{kV}$ 直流、$1000 \, \text{kV}$ 交流），可将线损降至极低水平，实现跨区域能源调配。

### 3.3 变压器不能改变直流电压

变压器仅对交变电流有效，因为直流电流产生恒定磁通量，$d\Phi/dt = 0$，副线圈中无感应电动势。这一限制正是交变电流在电力传输中取代直流电的关键原因之一。

### 3.4 与大学物理的衔接

变压器在大学物理和电气工程中有更深入的分析：

- **耦合电感模型**：用自感 $L_1$、$L_2$ 和互感 $M$ 描述，耦合系数 $k = M / \sqrt{L_1 L_2}$，理想变压器 $k = 1$。
- **等效电路**：大学课程中用T型或$\Pi$型等效电路分析实际变压器的漏感、激磁电感和损耗。
- **三相变压器**：电力系统采用三相变压器，有Y/Y、Y/$\Delta$、$\Delta$/Y等多种接法，涉及相位移动和零序阻抗。
- **高频变压器**：在开关电源和电力电子中，采用铁氧体磁芯的高频变压器工作在kHz至MHz范围，体积小、效率高，是现代电力电子技术的核心组件。
- 与[[RLC谐振_RLC Resonance]]的衔接：变压器的漏感和分布电容可在某些频率下引发谐振，需要特别设计以避免。

## 4. 相关验证实验

### 4.1 变压器电压比和电流比验证

**实验装置**：可拆式变压器（铁芯、原线圈、副线圈可更换），交流电压表，交流电流表，低压交流电源。

**实验步骤**：
1. 固定原线圈匝数 $N_1$ 和输入电压 $U_1$，更换不同匝数 $N_2$ 的副线圈，测量 $U_2$，验证 $U_1/U_2 = N_1/N_2$。
2. 在副线圈接负载，测量原、副线圈电流 $I_1$、$I_2$，验证 $I_1/I_2 = N_2/N_1$。
3. 测量输入功率 $P_1 = U_1 I_1$ 和输出功率 $P_2 = U_2 I_2$，计算效率 $\eta = P_2/P_1$。

### 4.2 远距离输电模拟实验

**实验装置**：用长电阻丝模拟输电线路，用升压变压器和降压变压器模拟变电站。

**实验步骤**：
1. 低压直接输电：测量负载端电压和线路损耗。
2. 升压后输电再降压：测量负载端电压和线路损耗。
3. 对比两种方案的输电效率，验证高压输电的优越性。

### 4.3 涡流现象演示

将整块金属板和叠片铁芯分别置于交变磁场中，用温度计测量温升，展示涡流损耗的存在及叠片铁芯的减损效果。

## 5. 历史上下文

变压器的原理根植于[[迈克尔·法拉第]]1831年发现的[[电磁感应现象_Electromagnetic Induction]]。法拉第的互感实验实际上就是世界上第一台变压器的雏形。

1885年，匈牙利工程师[[奥托·布拉西]]（Ottó Bláthy）、[[米克萨·德利]]（Miksa Déri）和[[卡罗利·齐佩尔诺夫斯基]]（Károly Zipernowsky）发明了第一台实用的闭合磁路变压器（ZBD变压器），使交流配电系统成为可能。

1880年代末，[[尼古拉·特斯拉]]设计了完整的多相交流电系统，包括发电机、变压器和电动机。[[乔治·威斯汀豪斯]]购买了特斯拉的专利，并建造了基于变压器的交流配电系统。变压器的成熟使交流电在"电流之战"中击败直流电，成为电力传输的标准。

变压器的发明使得电力可以在发电厂以高电压远距离传输，再在用户端降压使用，从根本上解决了电力传输的效率问题。这一技术突破催生了现代电力工业，也使电气化成为第二次工业革命的核心动力。
