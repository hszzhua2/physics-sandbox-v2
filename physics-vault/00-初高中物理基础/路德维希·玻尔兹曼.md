# 路德维希·玻尔兹曼 (Ludwig Boltzmann)

## 1. 生平
1844年生于维也纳，在维也纳大学学习，任维也纳大学与莱比锡大学教授。提出统计力学与熵的微观解释，1906年逝于的里雅斯特。

## 2. 主要贡献
- 提出[[分子动理论]]与统计力学
- 给出[[比热容]]的熵微观解释
- 发展[[分子动理论]]的H定理

## 3. 关键公式与思想
- $S=k_B\ln W$（玻尔兹曼熵公式）

## 4. 历史影响
玻尔兹曼的统计力学是[[分子动理论]]与[[比热容]]的理论基础。
