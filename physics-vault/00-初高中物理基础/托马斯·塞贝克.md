# 托马斯·塞贝克 (Thomas Seebeck)

## 1. 生平
1770年生于雷瓦尔，在格廷根大学学习，任柏林科学院研究员。发现塞贝克效应（温差电），研究[[欧姆定律]]前身，1831年逝于柏林。

## 2. 主要贡献
- 发现[[欧姆定律]]前身的塞贝克效应
- 研究[[欧姆定律]]与温差电
- 发展[[欧姆定律]]与热电学

## 3. 关键公式与思想
- $V=S\Delta T$（塞贝克效应/温差电动势）

## 4. 历史影响
塞贝克效应是[[欧姆定律]]与热电学的重要发现。
