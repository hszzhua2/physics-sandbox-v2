# 电流与电路 (Current and Circuits)

## 1. 核心定义

电流是[[电荷]]的定向移动，大小等于单位时间内通过导体横截面的电荷量 $I = Q/t$；电路是由电源、导线和用电器组成的电荷流动路径。

## 2. 数学表达

### 2.1 电流定义

$$I = \frac{Q}{t}$$

| 参数 | 物理量 | 单位 |
|------|--------|------|
| $I$ | 电流 | $\text{A}$（安培） |
| $Q$ | 通过横截面的电荷量 | $\text{C}$（库仑） |
| $t$ | 时间 | $\text{s}$ |

$1\,\text{A} = 1\,\text{C/s}$。常用单位还有毫安（$\text{mA}$）和微安（$\mu\text{A}$）：$1\,\text{A} = 10^3\,\text{mA} = 10^6\,\mu\text{A}$。

### 2.2 电流方向的规定

电流方向规定为**正电荷定向移动的方向**。在金属导体中，实际移动的是自由电子（负电荷），其运动方向与电流方向相反。在电解液或气体放电中，正离子和负离子分别向相反方向移动，电流方向由正离子运动方向决定。

### 2.3 串并联电路特性

| 特性 | 串联电路 | 并联电路 |
|------|---------|---------|
| 电流 | $I = I_1 = I_2 = \cdots$（处处相等） | $I = I_1 + I_2 + \cdots$（干路等于各支路之和） |
| 电压 | $U = U_1 + U_2 + \cdots$（总电压等于各部分之和） | $U = U_1 = U_2 = \cdots$（各支路电压相等） |
| 电阻 | $R = R_1 + R_2 + \cdots$ | $\frac{1}{R} = \frac{1}{R_1} + \frac{1}{R_2} + \cdots$ |

### 2.4 电荷量的微观表述

$$I = nqSv_d$$

| 参数 | 物理量 | 说明 |
|------|--------|------|
| $n$ | 自由电荷数密度 | 单位体积内自由电荷数（$\text{m}^{-3}$） |
| $q$ | 单个电荷的电荷量 | 电子 $e = 1.6 \times 10^{-19}\,\text{C}$ |
| $S$ | 导体横截面积 | $\text{m}^2$ |
| $v_d$ | 漂移速度 | $\text{m/s}$ |

## 3. 物理学意义与推论

### 3.1 电路的三要素

一个完整的电路必须包含三个基本部分：

| 要素 | 功能 | 说明 |
|------|------|------|
| 电源 | 提供[[电势差]]（电压），驱动电荷定向移动 | 电池、发电机 |
| 导线 | 连接电路各部分，传输电流 | 铜导线、铝导线 |
| 用电器 | 消耗电能，实现能量转化 | 灯泡、电动机、电阻 |

此外，电路通常还包含**开关**（控制电路通断）和**保护器件**（如保险丝，防止电流过大）。

### 3.2 电路状态

| 状态 | 特征 | 说明 |
|------|------|------|
| 通路 | 电路处处接通，有电流通过 | 正常工作状态 |
| 断路（开路） | 电路某处断开，无电流 | 开关断开或导线断裂 |
| 短路 | 电流不经过用电器直接从电源正极回到负极 | 电流极大，可能烧毁电源或引发火灾 |

**短路的危害**：短路时电路电阻极小，由 $I = U/R$（见[[欧姆定律_Ohm's Law]]）可知电流极大，产生大量热量（$Q = I^2Rt$，见[[电功与电功率_Electric Work and Power]]），可能烧毁导线绝缘层、引发火灾或损坏电源。

### 3.3 电流表与电压表

| 仪器 | 测量对象 | 接入方式 | 内阻特点 | 量程选择 |
|------|---------|---------|---------|---------|
| 电流表 | 电流 $I$ | 串联接入待测电路 | 内阻很小（接近导线） | 选择大于待测电流的最小量程 |
| 电压表 | 电压 $U$ | 并联接入待测元件两端 | 内阻很大（接近断路） | 选择大于待测电压的最小量程 |

**使用规则**：
1. 电流必须从"+"接线柱流入，"−"接线柱流出。
2. 选择合适的量程（先用大量程试触，再换小量程精确测量）。
3. 电流表绝对不能直接接在电源两极上（会短路）。
4. 电压表可以直接接在电源两极上（测量电源电压）。

### 3.4 串联与并联电路分析

**串联电路**特点：
- 只有一条电流路径，一处断开则全路不通（如圣诞树旧式串联灯——一个坏全灭）。
- 开关控制整个电路。
- 适用于需要分压的场合。

**并联电路**特点：
- 有多条电流路径（支路），各支路独立工作，互不影响（如家庭电路——各用电器并联）。
- 干路开关控制全部，支路开关控制该支路。
- 适用于各用电器独立工作的场合。

### 3.5 等效电阻分析

**串联等效电阻**：

$$R_{\text{总}} = R_1 + R_2 + \cdots + R_n$$

串联总电阻大于任何一个分电阻（串联越多总电阻越大）。

**并联等效电阻**：

$$\frac{1}{R_{\text{总}}} = \frac{1}{R_1} + \frac{1}{R_2} + \cdots + \frac{1}{R_n}$$

并联总电阻小于任何一个分电阻（并联越多总电阻越小）。两个电阻并联时：

$$R_{\text{总}} = \frac{R_1 R_2}{R_1 + R_2}$$

### 3.6 适用边界与知识衔接

电流与电路在初中阶段以直流电路的串并联分析和电表使用为核心。在高中物理中，将引入电源电动势和内阻，建立[[闭合电路欧姆定律_Ohm's Law for Complete Circuit]]（$I = \varepsilon/(R+r)$），分析含电源的复杂电路。基尔霍夫定律（节点电流定律 $\sum I = 0$ 和回路电压定律 $\sum U = 0$）将提供分析任意复杂电路的系统方法。在大学物理中，电流的微观理论建立在[[麦克斯韦方程组_Maxwell's Equations]]和固体物理基础上——金属中自由电子的漂移速度 $v_d$ 由电场力 $eE$ 和散射（弛豫时间 $\tau$）的平衡决定（德鲁德模型），$j = \sigma E$ 是[[欧姆定律_Ohm's Law]]的微观形式。交流电路将引入复阻抗、相量分析法和[[RLC谐振_RLC Resonance]]等概念。

## 4. 相关验证实验

### 4.1 串联电路电流规律实验

在串联电路中串联两个灯泡，在电路中 $A$、$B$、$C$ 三点分别串联接入电流表：
1. 分别测量各点电流 $I_A$、$I_B$、$I_C$。
2. 验证 $I_A = I_B = I_C$（串联电路电流处处相等）。

### 4.2 并联电路电流规律实验

在并联电路中接入两个灯泡，在干路和两个支路分别串联电流表：
1. 测量干路电流 $I$ 和两支路电流 $I_1$、$I_2$。
2. 验证 $I = I_1 + I_2$（干路电流等于各支路电流之和）。

### 4.3 串并联电压测量实验

- **串联**：用电压表分别测量各灯泡两端电压和总电压，验证 $U = U_1 + U_2$。
- **并联**：用电压表分别测量各支路电压，验证 $U = U_1 = U_2$。

### 4.4 短路危害演示实验

用一个灯泡和一根导线组成简单电路，正常工作后用另一根导线将灯泡两端短接（并联在灯泡两端）：
1. 观察灯泡熄灭（电流从导线走，不经过灯泡）。
2. 用电流表测量短路电流，发现远大于正常工作电流。
3. 注意：此实验需串联保护电阻或使用低压电源，防止过大的短路电流损坏器材。

### 4.5 电路设计与故障排查实验

给出若干灯泡、开关、导线，要求学生设计电路：
- 两个开关控制一个灯泡（楼梯灯电路）。
- 多个灯泡独立控制（并联电路）。
通过断开导线模拟故障，用电流表和电压表排查故障位置（断路时，电压表跨接断点处有电压示数）。

## 5. 历史上下文

### 5.1 伏打与电池的发明

[[亚历山德罗·伏打]]（Alessandro Volta, 1745—1827）是意大利物理学家。1799年，伏打发明了"伏打电堆"（Voltaic Pile）——将锌片、铜片和浸有盐水的纸板交替叠放，首次实现了持续稳定的电流输出。这一发明是电学史上划时代的里程碑：在此之前，电学研究局限于静电（[[莱顿瓶]]放电、摩擦起电），无法获得持续电流。伏打电堆使持续电流成为可能，直接催生了电化学（[[汉弗里·戴维]]电解实验）、电磁学（[[汉斯·奥斯特]]发现电流磁效应，见[[电流的磁效应_Magnetic Effect of Current]]）等新领域的快速发展。电压单位"伏特"（$\text{V}$）即以伏打命名。

### 5.2 从静电到动电

电学研究始于古希腊人对琥珀摩擦起电的观察（"电子"electron一词源于希腊语 elektron，意为琥珀）。18世纪，[[本杰明·富兰克林]]通过风筝实验研究了大气电，并提出了正负电荷的概念。[[查尔斯·库仑]]于1785年用扭秤实验精确测量了电荷间的作用力（库仑定律）。但这些研究都是静电学范畴。伏打电堆的出现使"动电"（电流）的研究成为可能，电学从静电学跃迁到电流学和电磁学，开启了19世纪电磁学的黄金时代。

### 5.3 电流概念的建立

"电流"（electric current）的概念在伏打电堆发明后才逐渐明确。早期研究者曾误以为电流是电流体在导体中的流动（类似于水管的类比），这一"水模型"虽然形象但在微观上不准确。[[詹姆斯·麦克斯韦]]和[[开尔文勋爵]]等人发展了电磁场理论后，电流被理解为电场驱动下的电荷运动。1897年，[[约瑟夫·约翰·汤姆孙]]发现了电子，揭示了金属导体中电流的载体是自由电子的定向运动。20世纪的固体物理进一步阐明了金属导电的能带理论机制。

### 5.4 电路理论的发展

电路理论在19世纪迅速发展。[[格奥尔格·欧姆]]于1827年发现了[[欧姆定律_Ohm's Law]]，建立了电压、电流和电阻的定量关系。[[古斯塔夫·基尔霍夫]]于1845年提出了电路网络分析的两大定律（基尔霍夫电流定律和电压定律），使任意复杂电路的分析成为可能。此后，交流电路理论（[[查尔斯·普罗透斯·斯坦梅茨]]引入复数方法）、电路网络理论（[[莱昂·查尔斯·戴维南]]等效定理、[[诺顿定理]]）等逐步建立。今天，电路理论已从简单的直流电路发展为涵盖模拟电路、数字电路和集成电路的庞大工程学科，但其根基仍建立在电流、电压和电路基本定律之上。
