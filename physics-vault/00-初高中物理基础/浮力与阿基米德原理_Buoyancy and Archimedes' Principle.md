# 浮力与阿基米德原理 (Buoyancy and Archimedes' Principle)

## 1. 核心定义

浸在液体（或气体）中的物体受到液体向上的托力称为浮力；阿基米德原理指出浮力等于物体排开流体所受的重力：$F_{\text{浮}} = \rho_{\text{液}} g V_{\text{排}}$。

## 2. 数学表达

### 2.1 阿基米德原理

$$F_{\text{浮}} = G_{\text{排}} = \rho_{\text{液}} g V_{\text{排}}$$

| 参数 | 物理量 | 单位 |
|------|--------|------|
| $F_{\text{浮}}$ | 浮力 | $\text{N}$ |
| $G_{\text{排}}$ | 排开流体所受重力 | $\text{N}$ |
| $\rho_{\text{液}}$ | 液体（或气体）密度 | $\text{kg/m}^3$ |
| $g$ | 重力加速度 | $9.8\,\text{N/kg}$ |
| $V_{\text{排}}$ | 排开流体的体积 | $\text{m}^3$ |

### 2.2 浮力产生的微观解释

浮力的本质是液体对物体上下表面的压力差。设物体为横截面积 $S$、高为 $h$ 的柱体，上表面深度 $h_1$、下表面深度 $h_2 = h_1 + h$：

$$F_{\text{向上}} = p_{\text{下}} \cdot S = \rho_{\text{液}} g h_2 \cdot S$$

$$F_{\text{向下}} = p_{\text{上}} \cdot S = \rho_{\text{液}} g h_1 \cdot S$$

$$F_{\text{浮}} = F_{\text{向上}} - F_{\text{向下}} = \rho_{\text{液}} g (h_2 - h_1) S = \rho_{\text{液}} g V_{\text{物}}$$

此推导用液体压强公式 $p = \rho g h$（见[[压强_Pressure]]）从微观压力差角度导出了阿基米德原理。

### 2.3 浮沉条件

物体在液体中的运动状态由浮力 $F_{\text{浮}}$ 与重力 $G$ 的关系决定：

| 条件 | 浮沉状态 | 密度关系 |
|------|---------|---------|
| $F_{\text{浮}} > G$ | 上浮（最终漂浮） | $\rho_{\text{物}} < \rho_{\text{液}}$ |
| $F_{\text{浮}} = G$ | 悬浮 | $\rho_{\text{物}} = \rho_{\text{液}}$ |
| $F_{\text{浮}} < G$ | 下沉 | $\rho_{\text{物}} > \rho_{\text{液}}$ |

**漂浮时**：$F_{\text{浮}} = G$，但 $V_{\text{排}} < V_{\text{物}}$（部分浸入）。

### 2.4 漂浮物体的排水体积

$$\rho_{\text{物}} V_{\text{物}} g = \rho_{\text{液}} g V_{\text{排}}$$

$$\frac{V_{\text{排}}}{V_{\text{物}}} = \frac{\rho_{\text{物}}}{\rho_{\text{液}}}$$

例如冰漂浮在水面：$\rho_{\text{冰}} = 0.9 \times 10^3\,\text{kg/m}^3$，$\rho_{\text{水}} = 1.0 \times 10^3\,\text{kg/m}^3$，故 $V_{\text{排}}/V_{\text{物}} = 0.9$，即冰的 $90\%$ 浸在水中、$10\%$ 露出水面。

## 3. 物理学意义与推论

### 3.1 阿基米德原理的普适性

阿基米德原理适用于所有流体（液体和气体）。气体中的浮力同样遵循 $F_{\text{浮}} = \rho_{\text{气}} g V_{\text{排}}$。热气球升空即是利用加热使球内空气密度降低，浮力大于总重力而上升；氢气球和氦气球利用内部气体密度远小于空气密度产生净浮力。

### 3.2 浮力与浸入深度的关系

- **完全浸没的物体**：$V_{\text{排}} = V_{\text{物}}$ 为常数，故 $F_{\text{浮}}$ 与浸入深度无关（只要完全浸没）。
- **部分浸入的物体**：$V_{\text{排}}$ 随浸入深度增大而增大，故 $F_{\text{浮}}$ 随深度增大而增大。

### 3.3 浮力与物体形状无关

只要完全浸没且 $V_{\text{排}}$ 相同，不论物体形状如何，浮力相同。这由阿基米德原理直接保证——浮力仅取决于排开流体的体积和流体密度。

### 3.4 船的浮力原理

钢的密度（$7.9 \times 10^3\,\text{kg/m}^3$）远大于水（$1.0 \times 10^3\,\text{kg/m}^3$），实心钢块会沉入水底。但将钢板制成空心船体后，船体整体体积远大于钢板体积，即 $V_{\text{排}}$ 大幅增大，使得 $F_{\text{浮}} = \rho_{\text{水}} g V_{\text{排}}$ 可以等于甚至大于船的重力 $G$，从而实现漂浮。

船的排水量（满载时排开水的质量）等于船和货物的总质量：$m_{\text{排水}} = m_{\text{船}} + m_{\text{货}}$。

### 3.5 潜水艇原理

潜水艇通过改变自身重力实现浮沉：
- **下潜**：向水舱注水，$G$ 增大，当 $G > F_{\text{浮}}$ 时下沉。
- **上浮**：向水舱充压缩空气排出水，$G$ 减小，当 $G < F_{\text{浮}}$ 时上浮。
- **悬浮**：调整水量使 $G = F_{\text{浮}}$，可在任意深度停留。

潜水艇的浮力 $F_{\text{浮}}$ 不变（完全浸没时 $V_{\text{排}}$ 恒定），通过改变 $G$ 控制浮沉。

### 3.6 适用边界与知识衔接

阿基米德原理在静力学条件下严格成立——流体静止、物体静止或匀速运动。若流体有加速度或物体在流体中高速运动，需考虑流体的非静力学效应。在高中物理中，浮力问题将与力的平衡、牛顿运动定律结合，作为力学综合应用的重点内容。在大学物理中，阿基米德原理可从流体静力学方程（$\nabla p = \rho \vec{g}$）积分严格推导，且可推广到加速参考系中的"等效重力"情形。物体在流体中运动时还受到粘滞阻力（斯托克斯定律 $F = 6\pi\eta r v$），与浮力共同决定物体的最终速度（收尾速度）。这些内容与[[伯努利原理_Bernoulli Principle]]和[[纳维-斯托克斯方程_Navier-Stokes Equations]]构成完整的流体力学体系。

## 4. 相关验证实验

### 4.1 阿基米德原理验证实验

使用弹簧测力计、物体、溢水杯、小桶：
1. 用弹簧测力计测出物体在空气中的重力 $G$。
2. 在溢水杯中装满水，将空小桶挂在弹簧测力计下，记录示数 $G_1$（小桶重力）。
3. 将物体缓慢浸入溢水杯中，水从溢水口流入小桶。物体完全浸没后，记录弹簧测力计示数 $F'$（物体在水中视重）。
4. 浮力 $F_{\text{浮}} = G - F'$。
5. 记录小桶和排水的总重力 $G_2$，排开水重 $G_{\text{排}} = G_2 - G_1$。
6. 验证 $F_{\text{浮}} = G_{\text{排}}$。

### 4.2 浮沉条件实验

准备三组盐水浓度不同的容器（密度分别大于、等于、小于鸡蛋密度），将鸡蛋分别放入：
- 盐水浓度低（$\rho_{\text{盐}} < \rho_{\text{蛋}}$）：鸡蛋下沉。
- 盐水浓度适中（$\rho_{\text{盐}} = \rho_{\text{蛋}}$）：鸡蛋悬浮。
- 盐水浓度高（$\rho_{\text{盐}} > \rho_{\text{蛋}}$）：鸡蛋漂浮。

通过逐步加盐搅拌，可观察鸡蛋从下沉到悬浮再到漂浮的全过程。

### 4.3 潜水艇模型实验

用带有进出水口的浮沉子（或小型塑料瓶）模拟潜水艇。用注射器向瓶内注水增加重力（下沉），用注射器抽水减小重力（上浮），调整水量实现悬浮。

### 4.4 密度计使用实验

将密度计放入纯水、盐水、酒精中，观察密度计浸入深度变化。液体密度越大，密度计露出部分越多。读出密度计在各液体中的刻度值，验证漂浮条件 $F_{\text{浮}} = G$ 下 $V_{\text{排}}$ 随 $\rho_{\text{液}}$ 变化。

## 5. 历史上下文

### 5.1 阿基米德与王冠故事

[[阿基米德]]（Archimedes, 约公元前287—前212）是古希腊最伟大的数学家和物理学家之一。据罗马建筑师[[维特鲁威]]记载，叙拉古国王希伦二世命工匠制作纯金王冠，但怀疑工匠掺银。他请[[阿基米德]]在不损坏王冠的前提下鉴定。阿基米德苦思不得其解，一日入浴时，身体浸入水中使水溢出，他意识到可以用排水法测王冠体积，再与同质量纯金的体积比较。他兴奋地跃出浴盆高呼"Eureka!"（希腊语"我找到了"）。据记载，他发现王冠排出的水比同质量纯金多，证实了掺假。这一发现直接催生了浮力原理。尽管故事的具体细节可能被后人加工，但[[阿基米德]]确实在《论浮体》中给出了严格的浮力原理表述。

### 5.2 《论浮体》

[[阿基米德]]的《论浮体》（*On Floating Bodies*）是流体静力学的奠基之作。书中以公理化方法严格证明了浮力原理：
1. 任何流体的静止表面是以地球中心为圆心的球面。
2. 浸入流体的物体受到等于排开流体重的向上浮力。
3. 漂浮物体的重量等于排开流体的重量。

这部著作的严密程度在古代科学中无出其右，其方法直到16世纪才被[[伽利略]]和[[西蒙·斯蒂文]]重新发展。

### 5.3 阿基米德原理的传播

阿基米德原理在古代主要在地中海地区传播。中世纪伊斯兰学者如[[阿布·拉伊汗·比鲁尼]]利用浮力原理精确测量了不规则物体和金属的密度。文艺复兴时期，[[伽利略]]于1586年写了《小天平》（*La Bilancetta*），设计了一种流体静力秤来精确鉴别金中掺假，改进了阿基米德的方法。

### 5.4 现代应用

阿基米德原理至今是船舶设计、潜水艇工程、气球和飞艇设计的理论基础。船体的排水量、吃水深度、稳定性分析均基于浮力原理。在航天工程中，微重力环境下的流体行为需考虑表面张力主导的毛细现象，但浮力原理在地面模拟实验中仍是基础工具。阿基米德原理的简洁与普适，使之成为物理学中最古老而历久弥新的定律之一。
