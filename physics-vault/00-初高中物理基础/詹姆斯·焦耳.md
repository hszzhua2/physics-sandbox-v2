# 詹姆斯·焦耳 (James Joule)

## 1. 生平
1818年生于索尔福德，在曼彻斯特学习。通过实验确立热功当量，奠基能量守恒，1889年逝于塞尔。

## 2. 主要贡献
- 确立[[分子动理论]]的热功当量
- 发展[[功和机械能]]与能量守恒
- 研究[[比热容]]与焦耳定律

## 3. 关键公式与思想
- $W=JH,\;J=4.186\,\text{J/cal}$（热功当量）

## 4. 历史影响
焦耳的热功当量实验是[[功和机械能]]守恒与热力学第一定律的实验基础。
