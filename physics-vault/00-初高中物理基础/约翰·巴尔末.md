# 约翰·巴尔末 (Johann Balmer)

## 1. 生平
1825年生于洛森，在巴塞尔大学学习，任巴塞尔女子中学教师。发现氢原子光谱的巴尔末公式，1898年逝于巴塞尔。

## 2. 主要贡献
- 发现[[原子光谱]]的巴尔末线系
- 提出氢原子[[原子光谱]]的经验公式

## 3. 关键公式与思想
- $\lambda=B\dfrac{n^2}{n^2-4},\;n=3,4,5,\ldots$（巴尔末公式）

## 4. 历史影响
巴尔末公式是[[原子光谱]]研究的起点，启发了[[玻尔模型]]。
