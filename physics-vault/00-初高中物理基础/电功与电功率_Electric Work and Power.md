# 电功与电功率 (Electric Work and Power)

## 1. 核心定义

电功是电流通过用电器时所做的功 $W = UIt$；电功率是单位时间内电流做的功 $P = UI$；焦耳定律描述电流通过电阻产生的热量 $Q = I^2Rt$。

## 2. 数学表达

### 2.1 电功

$$W = UIt$$

| 参数 | 物理量 | 单位 |
|------|--------|------|
| $W$ | 电功 | $\text{J}$（焦耳） |
| $U$ | [[电势差]]（电压） | $\text{V}$（伏特） |
| $I$ | 电流 | $\text{A}$（安培） |
| $t$ | 通电时间 | $\text{s}$ |

常用单位：$1\,\text{kW}\cdot\text{h} = 3.6 \times 10^6\,\text{J}$（1度电），电能表（电度表）以此为单位计量。

### 2.2 电功率

$$P = \frac{W}{t} = UI$$

| 参数 | 物理量 | 单位 |
|------|--------|------|
| $P$ | 电功率 | $\text{W}$（瓦特），$1\,\text{W} = 1\,\text{J/s}$ |
| $U$ | 电压 | $\text{V}$ |
| $I$ | 电流 | $\text{A}$ |

结合[[欧姆定律_Ohm's Law]]（$U = IR$），纯电阻电路中：

$$P = UI = I^2 R = \frac{U^2}{R}$$

### 2.3 焦耳定律（电热）

$$Q = I^2 R t$$

| 参数 | 物理量 | 单位 |
|------|--------|------|
| $Q$ | 电流产生的热量 | $\text{J}$ |
| $I$ | 电流 | $\text{A}$ |
| $R$ | [[电阻]] | $\Omega$ |
| $t$ | 通电时间 | $\text{s}$ |

### 2.4 额定功率与实际功率

| 概念 | 定义 | 说明 |
|------|------|------|
| 额定电压 $U_{\text{额}}$ | 用电器正常工作时所需的电压 | 标在铭牌上 |
| 额定功率 $P_{\text{额}}$ | 在额定电压下工作的功率 | $P_{\text{额}} = U_{\text{额}} I_{\text{额}}$ |
| 实际电压 $U_{\text{实}}$ | 实际加在用电器两端的电压 | 可能偏离额定值 |
| 实际功率 $P_{\text{实}}$ | 实际工作时的功率 | $P_{\text{实}} = U_{\text{实}} I_{\text{实}}$ |

当 $U_{\text{实}} = U_{\text{额}}$ 时，$P_{\text{实}} = P_{\text{额}}$（正常工作）；当 $U_{\text{实}} < U_{\text{额}}$ 时，$P_{\text{实}} < P_{\text{额}}$（亮度暗、功率不足）；当 $U_{\text{实}} > U_{\text{额}}$ 时，$P_{\text{实}} > P_{\text{额}}$（过亮、可能烧毁）。

### 2.5 纯电阻与非纯电阻电路

| 类型 | 能量转化 | 电功与电热关系 | 公式适用 |
|------|---------|-------------|---------|
| 纯电阻电路 | 电能 $\to$ 热能 | $W = Q$（电功等于电热） | $W = UIt = I^2Rt = \frac{U^2}{R}t$ 全部适用 |
| 非纯电阻电路 | 电能 $\to$ 热能 + 其他能 | $W > Q$（电功大于电热） | $W = UIt$，$Q = I^2Rt$ 分别使用 |

典型非纯电阻电路：电动机（电能 $\to$ 机械能 + 热能）、蓄电池充电（电能 $\to$ 化学能 + 热能）。

## 3. 物理学意义与推论

### 3.1 电功的物理意义

电功是电能转化为其他形式能量的量度。电流通过用电器做功的过程，本质上是电场力对电荷做功的过程：

$$W = qU = (It)U = UIt$$

其中 $q = It$ 为时间 $t$ 内通过的电荷量，$U$ 为电压（电场对单位电荷做的功）。电功反映了电能消耗的总量。

### 3.2 电功率的意义

电功率描述电流做功的快慢——单位时间内电能转化为其他形式能量的速率：
- 大功率用电器（如电热水器 $2000\,\text{W}$）电能消耗快。
- 小功率用电器（如 LED 灯 $5\,\text{W}$）电能消耗慢。

**家庭电路的功率限制**：家庭电路总功率受限于电能表的额定最大电流和总开关的额定电流。例如 $220\,\text{V}$、$10\,\text{A}$ 的电路最大总功率 $P = UI = 220 \times 10 = 2200\,\text{W}$，超过此功率会跳闸。

### 3.3 焦耳定律的物理意义

焦耳定律描述电流通过电阻时产生热量的定量规律。其核心是：热量与电流的**平方**成正比，而非与电流的一次方成正比。这意味着电流增大 $2$ 倍时，发热量增大 $4$ 倍。

**焦耳定律的普遍性**：无论电路是否为纯电阻，$Q = I^2Rt$ 始终成立。这是因为在任何导体中，电流通过电阻时总有一部分电能因碰撞散射而转化为热能（焦耳热），这一过程不可逆。

**焦耳热的微观机制**：自由电子在电场加速获得动能，与晶格离子碰撞时将动能传递给晶格，使晶格振动加剧——宏观表现为温度升高。电场持续加速电子，电子持续碰撞晶格，形成稳态的热产生过程。

### 3.4 电热的应用与危害

| 方面 | 实例 | 原理 |
|------|------|------|
| 有益应用 | 电热水器、电饭锅、电烙铁、电热毯 | 利用焦耳热加热 |
| 有益应用 | 白炽灯 | 灯丝高温发光（热辐射） |
| 有害影响 | 导线过热引发火灾 | 电流过大时 $Q = I^2Rt$ 急剧增大 |
| 有害影响 | 电动机过热损坏 | 部分电能转化为焦耳热 |
| 有害影响 | 电子元件散热需求 | CPU等需散热器降低焦耳热 |

### 3.5 纯电阻与非纯电阻电路的分析

**纯电阻电路**（如电热器、白炽灯）：
- 电能全部转化为热能（或通过热辐射转化为光能）。
- $W = Q$，即 $UIt = I^2Rt$，故 $U = IR$（[[欧姆定律_Ohm's Law]]成立）。
- $P = UI = I^2R = U^2/R$。

**非纯电阻电路**（如电动机正常运转时）：
- 电能大部分转化为机械能，少部分转化为热能。
- $W = W_{\text{机械}} + Q$，即 $UIt = W_{\text{机械}} + I^2Rt$。
- $U \neq IR$（欧姆定律不直接适用），$U > IR$。
- 电动机的机械功率 $P_{\text{机械}} = UI - I^2R$。
- 电动机效率 $\eta = P_{\text{机械}}/P_{\text{电}} = (UI - I^2R)/(UI) = 1 - IR/U$。

### 3.6 适用边界与知识衔接

电功与电功率在初中阶段以直流电路分析为主。在高中物理中，将扩展到[[闭合电路欧姆定律_Ohm's Law for Complete Circuit]]框架下的功率分析（电源输出功率 $P_{\text{出}} = I^2R$，最大输出功率条件 $R = r$），以及交流电中的有效值和功率因数概念。在交流电路中，$P = UI\cos\varphi$（$\cos\varphi$ 为功率因数），这涉及[[RLC谐振_RLC Resonance]]中的阻抗和相位关系。在大学物理中，电功率的微观分析建立在[[麦克斯韦方程组_Maxwell's Equations]]和固体物理之上，焦耳热损耗最终归结为电磁场能量在导体中的耗散过程（坡印廷定理）。在工程领域，电力系统分析、电机设计和集成电路功耗优化均以电功率理论为基础。

## 4. 相关验证实验

### 4.1 电功测量实验（电能表法）

用电能表（电度表）测量家用电器的电功：
1. 关闭其他所有用电器，只接入待测电器。
2. 记录电能表转盘转 $N$ 圈所用时间 $t$（或闪烁 $N$ 次所用时间）。
3. 由电能表常数 $C$（每 $\text{kW}\cdot\text{h}$ 的转数）计算电功：
   $$W = \frac{N}{C} \times 3.6 \times 10^6\,\text{J}$$
4. 计算功率 $P = W/t$。

### 4.2 电功率测量实验（伏安法）

用电压表和电流表测量小灯泡的电功率：
1. 按电路图连接：电源、开关、滑动变阻器、小灯泡、电流表（串联）、电压表（并联在灯泡两端）。
2. 调节滑动变阻器使灯泡两端电压等于额定电压，读出 $U_{\text{额}}$、$I_{\text{额}}$，计算额定功率 $P_{\text{额}} = U_{\text{额}} I_{\text{额}}$。
3. 调节使电压低于额定值，计算实际功率 $P_{\text{实}}$，观察灯泡变暗。
4. 调节使电压略高于额定值，计算实际功率，观察灯泡变亮（注意不要超过太多，防止烧毁）。

### 4.3 焦耳定律验证实验

用控制变量法研究影响电热的因素：
1. **电流的影响**：在两个相同的烧瓶中装入等量煤油和电阻丝（$R$ 相同），串联接入电路使电流相同——不对，需改变电流。将两个电阻丝并联在不同电路中，一个串联变阻器使电流为 $I$，另一个使电流为 $2I$，相同时间后比较温度升高，验证 $Q \propto I^2$。
2. **电阻的影响**：两烧瓶中电阻丝 $R_1 \neq R_2$，串联接入电路（电流相同），相同时间后比较温升，验证 $Q \propto R$。
3. **时间的影响**：同一电阻丝通电不同时间，比较温升，验证 $Q \propto t$。

### 4.4 额定功率与实际功率实验

用滑动变阻器调节小灯泡两端电压：
1. 记录灯泡铭牌上的额定电压和额定功率。
2. 调节到额定电压，观察亮度，测量 $U$、$I$，验证 $P = UI$ 等于额定功率。
3. 降至额定电压的 $80\%$，观察灯变暗，计算实际功率 $P_{\text{实}} = U_{\text{实}}^2/R$。
4. 验证 $P_{\text{实}}/P_{\text{额}} = (U_{\text{实}}/U_{\text{额}})^2$（电阻近似不变时）。

## 5. 历史上下文

### 5.1 詹姆斯·焦耳

[[詹姆斯·焦耳]]（James Prescott Joule, 1818—1889）是英国物理学家。他在1840年代进行了大量精确的热功当量实验，奠定了能量守恒定律的实验基础。焦耳在1841年发表了关于电流热效应的研究——即后来的焦耳定律 $Q = I^2Rt$。他发现电流通过电阻产生的热量与电流的平方成正比，这一规律后来被称为"焦耳定律"或"焦耳-楞次定律"（[[海因里希·楞次]]于1842年独立发现了相同规律）。

### 5.2 热功当量实验

焦耳最著名的实验是测量热功当量——精确测定机械功与热量之间的等价关系。他设计了多种实验装置，其中最著名的是"搅拌器实验"：用下落的重物驱动叶片搅拌水，测量重物做的机械功 $W = mgh$ 与水温升高 $\Delta T$，由 $Q = cm\Delta T$ 计算热量，得到 $J = W/Q \approx 4.18\,\text{J/cal}$。

这一实验推翻了热质说，证明了热是能量的一种形式，为热力学第一定律（[[热力学四大定律_Four Laws of Thermodynamics]]中的第一定律）奠定了实验基础。能量和功的单位"焦耳"（$\text{J}$）即以他命名。

### 5.3 电能计量的历史

电能的计量始于19世纪末。1881年，[[都柏林]]的爱迪生公司安装了第一台电能表。最早的电能表基于电解原理，后来发展为感应式电能表（利用电磁感应驱动铝盘旋转），现代则广泛使用电子式电能表（数字计量）。$1\,\text{kW}\cdot\text{h}$（1度电）$= 3.6 \times 10^6\,\text{J}$ 成为电力计费的基本单位，这一单位的选取使家用电器的电功率（数百至数千瓦）与使用时间（小时）的乘积为易于计量的数值。

### 5.4 现代意义

电功和电功率的理论至今是电力工程和电子工程的基础。从发电厂的兆瓦级功率管理到手机芯片的微瓦级功耗优化，电功率分析贯穿了整个电气和电子工程领域。在集成电路设计中，焦耳热是芯片散热和可靠性设计的核心约束——芯片功耗密度已逼近物理极限，推动了低功耗设计和三维集成等新技术的发展。在交流电路中，功率因数校正、谐波分析和电能质量监测等均建立在电功率理论之上，并涉及[[RLC谐振_RLC Resonance]]中的阻抗匹配和能量传输优化。
