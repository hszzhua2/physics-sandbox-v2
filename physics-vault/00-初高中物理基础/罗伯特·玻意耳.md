# 罗伯特·玻意耳 (Robert Boyle)

## 1. 生平
1627年生于利斯莫尔，即罗伯特·波义耳，在牛津大学学习，任牛津大学实验员。提出波义耳定律，研究[[理想气体状态方程]]，1691年逝于伦敦。

## 2. 主要贡献
- 提出[[理想气体状态方程]]的波义耳定律
- 研究[[理想气体状态方程]]与气体
- 发展[[理想气体状态方程]]与化学

## 3. 关键公式与思想
- $PV=\text{const}$（波义耳定律）

## 4. 历史影响
波义耳定律是[[理想气体状态方程]]的基础。
