# 放射性衰变 (Radioactive Decay)

## 1. 核心定义

**放射性衰变**（Radioactive Decay）是不稳定原子核自发地放出粒子或电磁辐射，转变为另一种原子核的物理过程。具有放射性的原子核称为放射性核素（Radionuclide），其衰变过程是随机的、不可预测的，但大量原子核的统计行为遵循确定的指数衰变规律。

放射性现象由法国物理学家[[亨利·贝克勒尔]]于1896年发现，随后[[玛丽·居里]]和[[皮埃尔·居里]]深入研究了放射性元素，发现了钋和镭。放射性衰变的研究揭示了[[原子核的组成_Composition of Nucleus]]的可变性，为核物理学和粒子物理学的发展奠定了基础。

放射性衰变有三种主要类型：**α衰变**、**β衰变**和**γ衰变**，分别对应放出α粒子（氦核）、β粒子（电子或正电子）和γ射线（高能光子）。不同衰变类型由不同的基本相互作用驱动——α衰变由强相互作用和电磁相互作用的竞争决定，β衰变由[[弱相互作用_Weak Interaction]]驱动，γ衰变由电磁相互作用驱动。

## 2. 数学表达

### 2.1 三种衰变类型

**α衰变**：原子核放出一个α粒子（$\ce{^4_2He}$核），质量数减4，电荷数减2：

$$\ce{_{Z}^{A}X -> _{Z-2}^{A-4}Y + _{2}^{4}He}$$

例如：$\ce{_{92}^{238}U -> _{90}^{234}Th + _{2}^{4}He}$

α衰变的条件：衰变能 $Q > 0$，即母核质量大于子核质量与α粒子质量之和。

**β衰变**：包括三种形式：

- **β⁻衰变**：中子转变为质子，放出电子和反中微子：

$$\ce{_{Z}^{A}X -> _{Z+1}^{A}Y + e^{-} + \bar{\nu}_e}$$

微观过程：$\ce{n -> p + e^{-} + \bar{\nu}_e}$

- **β⁺衰变**：质子转变为中子，放出正电子和中微子：

$$\ce{_{Z}^{A}X -> _{Z-1}^{A}Y + e^{+} + \nu_e}$$

微观过程：$\ce{p -> n + e^{+} + \nu_e}$

- **电子俘获（EC）**：原子核俘获一个轨道电子，质子转变为中子：

$$\ce{_{Z}^{A}X + e^{-} -> _{Z-1}^{A}Y + \nu_e}$$

β衰变由[[弱相互作用_Weak Interaction]]驱动，涉及中微子的发射。[[沃尔夫冈·泡利]]于1930年提出中微子假说以解释β衰变中能量和动量看似不守恒的困难，[[恩里科·费米]]于1933年建立了β衰变的定量理论。

**γ衰变**：处于激发态的原子核放出γ光子跃迁到低能态：

$$\ce{_{Z}^{A}X^{*} -> _{Z}^{A}X + \gamma}$$

γ衰变不改变质量数和电荷数，仅降低核的能态。γ射线是高能电磁波，波长极短（$\lambda < 0.01 \, \text{nm}$），穿透力极强。

### 2.2 衰变定律

放射性衰变是一个统计过程。设 $t$ 时刻有 $N(t)$ 个放射性原子核，在 $dt$ 时间内衰变的核数 $dN$ 与 $N$ 成正比：

$$-dN = \lambda N \, dt$$

其中 $\lambda$ 为**衰变常数**（Decay Constant），反映核素衰变的快慢。解此微分方程得：

$$\boxed{N(t) = N_0 \, e^{-\lambda t}}$$

其中 $N_0$ 为 $t = 0$ 时的原子核数。这就是**放射性衰变定律**，表明放射性核数随时间按指数规律衰减。

### 2.3 半衰期

**半衰期**（Half-Life）$T_{1/2}$ 是放射性核数衰减到一半所需的时间：

$$N(T_{1/2}) = \frac{N_0}{2} = N_0 \, e^{-\lambda T_{1/2}}$$

解得：

$$\boxed{T_{1/2} = \frac{\ln 2}{\lambda} = \frac{0.693}{\lambda}}$$

不同核素的半衰期差异极大：
- $\ce{^{238}U}$：$T_{1/2} = 4.47 \times 10^9$ 年（与地球年龄同量级）
- $\ce{^{14}C}$：$T_{1/2} = 5730$ 年（碳-14测年法的基础）
- $\ce{^{60}Co}$：$T_{1/2} = 5.27$ 年（医用放射源）
- $\ce{^{212}Po}$：$T_{1/2} = 3.0 \times 10^{-7}$ 秒（极短）

### 2.4 平均寿命

每个原子核的**平均寿命**（Mean Lifetime）为：

$$\tau = \frac{1}{\lambda} = \frac{T_{1/2}}{\ln 2} \approx 1.443 \, T_{1/2}$$

### 2.5 放射性活度

**放射性活度**（Activity）为单位时间内衰变的次数：

$$A(t) = -\frac{dN}{dt} = \lambda N(t) = \lambda N_0 \, e^{-\lambda t} = A_0 \, e^{-\lambda t}$$

活度的单位为**贝克勒尔**（Bq），$1 \, \text{Bq} = 1$ 次衰变/秒。历史上常用的单位为**居里**（Ci），$1 \, \text{Ci} = 3.7 \times 10^{10} \, \text{Bq}$（约等于1克镭-226的活度）。

### 2.6 碳-14测年法

大气中 $\ce{^{14}C}$ 由宇宙射线轰击氮原子产生（$\ce{^{14}N + n -> ^{14}C + p}$），与 $\ce{^{12}C}$ 保持恒定比例。生物体在生命过程中不断交换碳元素，体内 $\ce{^{14}C}$ 含量与大气一致。生物死亡后停止碳交换，$\ce{^{14}C}$ 按半衰期 $5730$ 年衰变减少。测量样品中 $\ce{^{14}C}$ 与 $\ce{^{12}C}$ 的比例，即可推算死亡年代：

$$t = \frac{1}{\lambda}\ln\frac{N_0}{N(t)} = \frac{T_{1/2}}{\ln 2}\ln\frac{N_0}{N(t)}$$

该方法适用于约5万年以内的考古样品测年。

## 3. 物理学意义与推论

### 3.1 衰变的统计本质

放射性衰变是量子力学不确定性的宏观体现。单个原子核何时衰变完全不可预测，但大量原子核的统计行为遵循精确的指数规律 $N = N_0 e^{-\lambda t}$。这一统计性是量子力学基本特征的直接表现——衰变概率由量子隧穿效应（α衰变）或弱相互作用耦合（β衰变）决定。

### 3.2 质能守恒与衰变能

每种衰变都遵守质能守恒、电荷守恒、动量守恒和角动量守恒。衰变能 $Q$ 来自母核与子核及粒子之间的质量差：

$$Q = (m_{\text{母}} - m_{\text{子}} - m_{\text{粒子}})c^2$$

$Q > 0$ 是衰变自发进行的条件。衰变能以子核和粒子的动能形式释放。

### 3.3 位移定则

衰变前后质量数和电荷数守恒的规律称为**位移定则**：
- α衰变：子核在元素周期表中前移2位（$Z \to Z - 2$）。
- β⁻衰变：子核后移1位（$Z \to Z + 1$）。
- β⁺衰变/电子俘获：子核前移1位（$Z \to Z - 1$）。

### 3.4 与大学物理的衔接

放射性衰变在大学核物理中有更深入的理论：

- 与[[弱相互作用_Weak Interaction]]的衔接：β衰变是弱相互作用的典型过程。[[恩里科·费米]]的四费米子相互作用理论是弱相互作用理论的起点，后发展为电弱统一理论（[[格拉肖-温伯格-萨拉姆模型]]），W和Z玻色子的发现验证了该理论。
- **α衰变的量子隧穿**：[[乔治·伽莫夫]]于1928年用量子隧穿效应解释了α衰变，衰变常数 $\lambda$ 与隧穿概率相关，成功解释了半衰期跨越数十个数量级的差异。
- **放射系**：重放射性核素经过多次α和β衰变形成衰变链（铀系、钍系、锕系），最终稳定于铅同位素。大学课程中详细研究这些衰变链。
- **核反应与激活**：稳定核素在中子辐照下可被"激活"为放射性核素，这是核反应堆和放射性同位素生产的原理。
- 与[[核裂变与核聚变_Nuclear Fission and Fusion]]的衔接：放射性衰变是自发核转变，而裂变和聚变是诱发的核反应，但都遵循相同的核物理基本规律。

## 4. 相关验证实验

### 4.1 贝克勒尔的发现实验（1896年）

**实验者**：[[亨利·贝克勒尔]]。

**实验**：将铀盐包在黑纸中放在照相底片上，发现底片被感光，表明铀盐发出了穿透性射线。进一步实验发现这种射线与荧光无关，是铀元素本身的性质。

**意义**：放射性现象的首次发现。

### 4.2 居里夫妇的放射性研究

**实验者**：[[玛丽·居里]]和[[皮埃尔·居里]]。

**实验**：从沥青铀矿中分离出新的放射性元素钋（Po，以玛丽的祖国波兰命名）和镭（Ra），并测量了各种元素的放射性强度。

**意义**：证明放射性是某些元素的原子核性质，而非分子性质。[[玛丽·居里]]因此获得1903年物理学奖和1911年化学奖，是唯一在两个不同学科获得诺贝尔奖的科学家。

### 4.3 三种射线的电场偏转实验

**实验**：将放射性源置于电场（或磁场）中，用照相底片记录射线偏转。

**结果**：
- α射线：向负极偏转，偏转小，质量大，带正电。
- β射线：向正极偏转，偏转大，质量小，带负电。
- γ射线：不偏转，不带电。

**意义**：确定了三种射线的带电性质和荷质比，为理解衰变机制提供了关键证据。

### 4.4 半衰期测量

**实验**：用盖革-米勒计数器测量放射性样品的计数率随时间的变化。

**验证**：计数率 $A(t)$ 随时间按 $A_0 e^{-\lambda t}$ 衰减，作 $\ln A$-$t$ 图为直线，斜率为 $-\lambda$，由此可测量半衰期。

## 5. 历史上下文

**1895年**：[[威廉·伦琴]]发现X射线，激发了[[亨利·贝克勒尔]]研究荧光物质是否也能发射类似射线的兴趣。

**1896年**：贝克勒尔在研究铀盐荧光时，偶然发现铀盐在不受光照的情况下也能使照相底片感光，这是放射性现象的首次发现。贝克勒尔因此与居里夫妇共同获得1903年诺贝尔物理学奖。

**1898年**：[[玛丽·居里]]和[[皮埃尔·居里]]系统研究了各种元素的放射性，发现钋和镭，提出了"放射性"（Radioactivity）一词。玛丽·居里在极其简陋的条件下，从数吨沥青铀矿中提取了0.1克纯镭。

**1899-1903年**：[[欧内斯特·卢瑟福]]和[[弗雷德里克·索迪]]研究了放射性衰变，提出了衰变定律 $N = N_0 e^{-\lambda t}$ 和原子嬗变理论（放射性衰变是一种元素转变为另一种元素的过程），颠覆了"原子不可变"的传统观念。索迪因此获得1921年诺贝尔化学奖。

**1913年**：[[弗雷德里克·索迪]]和[[卡西米尔·法扬斯]]独立提出位移定则。

**1930年**：[[沃尔夫冈·泡利]]提出中微子假说解释β衰变的连续能谱。

**1934年**：[[伊雷娜·约里奥-居里]]和[[弗雷德里克·约里奥-居里]]发现人工放射性，开创了放射性同位素的人工制备。

放射性衰变的发现彻底改变了人类对物质本质的认识，证明了原子核的可变性，为核能利用和核医学的发展开辟了道路。
