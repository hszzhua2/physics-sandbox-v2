# 纪尧姆·阿蒙东 (Guillaume Amontons)

## 1. 生平
1663年生于巴黎，在巴黎学习，任法国科学院研究员。研究[[摩擦力]]与温度计，1705年逝于巴黎。

## 2. 主要贡献
- 研究[[摩擦力]]的阿蒙东摩擦定律
- 发展[[摩擦力]]与干摩擦
- 研究[[摩擦力]]与气体温度

## 3. 关键公式与思想
- $F_f=\mu N$（阿蒙东摩擦定律）

## 4. 历史影响
阿蒙东的[[摩擦力]]定律是干摩擦研究的基础。
