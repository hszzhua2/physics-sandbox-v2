# 尤利乌斯·迈尔 (Julius von Mayer)

## 1. 生平
1814年生于海尔布隆，在图宾根大学学习，任船医。独立提出能量守恒定律，1878年逝于海尔布隆。

## 2. 主要贡献
- 提出[[功和机械能]]的能量守恒定律
- 发展[[机械能守恒定律]]与热功当量
- 研究[[功和机械能]]与生理热

## 3. 关键公式与思想
- $\Delta U=Q-W$（热力学第一定律）

## 4. 历史影响
迈尔的能量守恒是[[功和机械能]]与[[机械能守恒定律]]的独立发现。
