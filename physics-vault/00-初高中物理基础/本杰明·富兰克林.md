# 本杰明·富兰克林 (Benjamin Franklin)

## 1. 生平
1706年生于波士顿，自学成才，任美国驻法大使。研究[[电流与电路]]与闪电，发明避雷针，1790年逝于费城。

## 2. 主要贡献
- 研究[[电流与电路]]与闪电电学
- 发明[[电流与电路]]的避雷针
- 发展[[电流与电路]]与单流理论

## 3. 关键公式与思想
- $\text{lightning is electricity}$（风筝实验）

## 4. 历史影响
富兰克林的[[电流与电路]]研究是电学早期重要贡献。
