# 克里斯蒂安·惠更斯 (Christiaan Huygens)

## 1. 生平
1629年生于海牙，在莱顿大学与布雷达大学学习。提出光的波动说与惠更斯原理，发明摆钟，1695年逝于海牙。

## 2. 主要贡献
- 提出[[光的直线传播]]的波动说与惠更斯原理
- 研究[[光的色散]]与双折射
- 发明摆钟与[[光的衍射]]理论

## 3. 关键公式与思想
- $\text{every point on a wavefront is a source of new wavelets}$（惠更斯原理）

## 4. 历史影响
惠更斯的波动说是[[光的色散]]与[[光的衍射]]的理论基础。
