# 力的合成与分解 (Composition and Decomposition of Forces)

## 1. 核心定义

求几个力的合力称为力的合成；求一个力的分力称为力的分解。力的合成与分解均遵循平行四边形定则。

## 2. 数学表达

### 平行四边形定则

两个共点力 $\vec{F}_1$ 和 $\vec{F}_2$ 的合力 $\vec{F}$，是以这两个力为邻边所作平行四边形的对角线。

合力大小：

$$F = \sqrt{F_1^2 + F_2^2 + 2F_1 F_2 \cos\theta}$$

合力方向（与 $\vec{F}_1$ 的夹角）：

$$\tan\alpha = \frac{F_2 \sin\theta}{F_1 + F_2 \cos\theta}$$

其中 $\theta$ 为 $\vec{F}_1$ 与 $\vec{F}_2$ 之间的夹角。

**参数说明：**

| 符号 | 物理量 | 单位 |
|------|--------|------|
| $F_1$、$F_2$ | 两个分力的大小 | $\text{N}$ |
| $F$ | 合力的大小 | $\text{N}$ |
| $\theta$ | 两分力之间的夹角 | $\text{rad}$ |
| $\alpha$ | 合力与 $F_1$ 的夹角 | $\text{rad}$ |

### 合力范围

$$|F_1 - F_2| \leq F \leq F_1 + F_2$$

- 当 $\theta = 0^\circ$（同向）时，$F_{\max} = F_1 + F_2$。
- 当 $\theta = 180^\circ$（反向）时，$F_{\min} = |F_1 - F_2|$。
- 当 $\theta = 90^\circ$（垂直）时，$F = \sqrt{F_1^2 + F_2^2}$。

### 正交分解法

将各力沿两个互相垂直的方向（通常取 $x$ 轴和 $y$ 轴）分解：

$$F_x = \sum F_{ix}, \quad F_y = \sum F_{iy}$$

$$F = \sqrt{F_x^2 + F_y^2}, \quad \tan\alpha = \frac{F_y}{F_x}$$

### 三角形定则

将两个力首尾相接，从第一个力的起点指向第二个力的终点的有向线段即为合力。这是平行四边形定则的简化形式，本质相同。

### 多个力的合成

多个共点力的合力为各分力的矢量和：

$$\vec{F} = \sum_{i=1}^{n} \vec{F}_i$$

## 3. 物理学意义与推论

### 物理意义

力的合成与分解揭示了力作为**矢量**的合成规律。平行四边形定则是所有矢量（力、速度、加速度、位移、电场强度等）合成的普遍法则，其物理基础在于力的作用效果可以叠加。力的分解不是唯一的——同一个力可以沿不同方向分解，具体选择取决于问题的需要。

### 正交分解的优越性

正交分解法将矢量运算转化为标量运算，是处理多力问题的最有效方法。选择坐标轴方向时应使尽可能多的力落在坐标轴上，以简化计算。

### 适用边界

- 平行四边形定则仅适用于**共点力**（作用线交于一点的力）。
- 对于非共点力（平行力），需引入力矩的概念，用力矩的合成来处理。
- 力的分解中，已知合力求两分力时，若无附加条件则有无穷多组解；需给定一个分力的大小和方向，或两个分力的方向，才能唯一确定。

### 与大学物理的衔接

在大学物理中，力的合成与分解被纳入更一般的**矢量分析**框架：

1. 在三维空间中，力用矢量 $\vec{F} = F_x\hat{i} + F_y\hat{j} + F_z\hat{k}$ 表示，合成遵循矢量加法。
2. 在[[分析力学_Analytical Mechanics]]中，力的概念被推广为**广义力**，力的合成与分解在广义坐标下进行，这构成了[[拉格朗日力学_Lagrangian Mechanics]]的基础。
3. 在连续介质力学中，力的合成推广为应力张量的概念，力的分解涉及到法向应力和切向应力的分析。

## 4. 相关验证实验

### 互成角度的两个力的合成实验

**实验器材**：方木板、白纸、弹簧测力计两只、橡皮条、细绳套、三角板、刻度尺、图钉。

**实验过程**：
1. 用两只弹簧测力计互成角度地拉橡皮条，使结点到达某位置 $O$，记录两拉力的大小 $F_1$、$F_2$ 和方向。
2. 用一只弹簧测力计单独拉橡皮条使结点到达同一位置 $O$，记录拉力的大小 $F$ 和方向。
3. 在白纸上按相同标度作出 $F_1$、$F_2$ 和 $F$ 的图示。
4. 以 $F_1$、$F_2$ 为邻边作平行四边形，其对角线记为 $F'$，比较 $F'$ 与 $F$ 的大小和方向。

**实验结论**：在实验误差范围内，$F'$ 与 $F$ 重合，验证了平行四边形定则。

**注意事项**：
- 弹簧测力计使用前需调零。
- 拉力方向应与木板平行。
- 结点位置必须严格相同，以保证力的效果等效。

## 5. 历史上下文

力的平行四边形定则的历史可追溯到16世纪。[[西蒙·斯特文]]（Simon Stevin, 1548–1620）在1586年发表的《静力学原理》中，通过分析斜面上物体的平衡条件，首次隐含地使用了力的分解思想。斯特文通过著名的"永动机不可能"论证（一串小球绕在三角形棱柱上不可能自行运动），推导出斜面上力的分解关系。

[[伽利略·伽利雷]]在研究斜面问题和抛体运动时也运用了力的分解方法。[[艾萨克·牛顿]]在《自然哲学的数学原理》（1687年）中明确表述了力的平行四边形定则，将其作为力的合成的基本原理。此后，平行四边形定则成为[[矢量运算]]的基础法则。

力的正交分解法作为一种计算技巧，在19世纪矢量分析体系建立后得到了系统化。[[约西亚·吉布斯]]（Josiah Willard Gibbs）和[[奥利弗·亥维赛]]（Oliver Heaviside）独立发展的矢量分析为力的合成与分解提供了现代数学工具。
