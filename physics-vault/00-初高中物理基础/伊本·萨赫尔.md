# 伊本·萨赫尔 (Ibn Sahl)

## 1. 生平
约940年生于波斯，在巴格达学习。研究[[光的折射与斯涅尔定律]]，比斯涅尔早数百年发现折射定律。

## 2. 主要贡献
- 发现[[光的折射与斯涅尔定律]]的折射定律
- 研究[[光的折射与斯涅尔定律]]与透镜设计

## 3. 关键公式与思想
- $n_1\sin\theta_1=n_2\sin\theta_2$（折射定律/萨赫尔）

## 4. 历史影响
伊本·萨赫尔的折射定律是[[光的折射与斯涅尔定律]]的最早发现。
