# 罗伯特·本生 (Robert Bunsen)

## 1. 生平
1811年生于格廷根，在格廷根大学学习，任海德堡大学教授。发明本生灯与光谱分析，研究[[光的色散]]，1899年逝于海德堡。

## 2. 主要贡献
- 发明[[光的色散]]的光谱分析方法
- 研究[[原子光谱]]与本生灯
- 发展[[光的色散]]与化学分析

## 3. 关键公式与思想
- $\text{each element has unique emission spectrum}$（光谱分析）

## 4. 历史影响
本生的光谱分析是[[光的色散]]与[[原子光谱]]化学分析的基础。
