# 让·达朗贝尔 (Jean le Rond d'Alembert)

## 1. 生平
1717年生于巴黎，在巴黎大学学习，任法兰西学院秘书。提出达朗贝尔原理与波动方程，1783年逝于巴黎。

## 2. 主要贡献
- 提出[[共点力平衡]]的达朗贝尔原理
- 发展[[动能定理]]与动力学
- 研究[[共点力平衡]]与偏微分方程

## 3. 关键公式与思想
- $F-ma=0$（达朗贝尔原理）
- $\nabla^2u=\dfrac{1}{c^2}\dfrac{\partial^2u}{\partial t^2}$（波动方程）

## 4. 历史影响
达朗贝尔原理是[[共点力平衡]]与[[动能定理]]的虚功方法基础。
