# 机械波 (Mechanical Waves)

## 1. 核心定义

机械振动在弹性介质中的传播过程。介质各质点在平衡位置附近振动，不随波迁移，传播的是振动形式和能量。

## 2. 数学表达

### 波速、波长与频率的关系

$$v = \lambda f = \frac{\lambda}{T}$$

**参数说明：**

| 符号 | 物理量 | 单位 |
|------|--------|------|
| $v$ | 波速 | $\text{m/s}$ |
| $\lambda$ | 波长 | $\text{m}$ |
| $f$ | 频率 | $\text{Hz}$ |
| $T$ | 周期 | $\text{s}$ |

### 简谐波方程

沿 $x$ 轴正方向传播的平面简谐波：

$$y(x, t) = A\cos\left(\omega t - \frac{2\pi}{\lambda}x + \varphi\right)$$

或写成：

$$y(x, t) = A\cos\left[\omega\left(t - \frac{x}{v}\right) + \varphi\right]$$

### 波的分类

**横波**：质点振动方向与波传播方向垂直（如绳波、电磁波）。

**纵波**：质点振动方向与波传播方向平行（如声波、弹簧纵波）。

### 波的能量

波传播过程中，介质质点既有动能又有势能，二者同相变化（与孤立的[[简谐运动_Simple Harmonic Motion]]不同）。波的能流密度（波强）：

$$I = \frac{1}{2}\rho v \omega^2 A^2$$

| 符号 | 物理量 | 单位 |
|------|--------|------|
| $\rho$ | 介质密度 | $\text{kg/m}^3$ |
| $A$ | 振幅 | $\text{m}$ |
| $I$ | 波强 | $\text{W/m}^2$ |

### 波的叠加原理

多列波相遇时，各自保持独立传播，相遇区域质点的位移为各波单独引起位移的矢量和：

$$y = y_1 + y_2 + \cdots$$

### 干涉条件

两列波产生稳定干涉的条件（相干条件）：
1. 频率相同
2. 振动方向相同
3. 相位差恒定

干涉加强条件（相位差为 $2\pi$ 的整数倍）：

$$\Delta\varphi = 2n\pi \quad (n = 0, \pm 1, \pm 2, \ldots)$$

$$\Delta x = n\lambda \quad (\text{波程差})$$

干涉减弱条件（相位差为 $\pi$ 的奇数倍）：

$$\Delta\varphi = (2n+1)\pi \quad (n = 0, \pm 1, \pm 2, \ldots)$$

$$\Delta x = (2n+1)\frac{\lambda}{2} \quad (\text{波程差})$$

### 驻波

两列振幅相同、频率相同、传播方向相反的波叠加形成驻波：

$$y = 2A\cos\frac{2\pi x}{\lambda}\cos\omega t$$

- **波节**：振幅为零的点，位于 $x = (2n+1)\frac{\lambda}{4}$。
- **波腹**：振幅最大的点，位于 $x = n\frac{\lambda}{2}$。
- 相邻波节（或波腹）间距为 $\frac{\lambda}{2}$。

### 多普勒效应

波源或观察者相对介质运动时，观察者接收到的频率与波源频率不同：

$$f' = \frac{v \pm v_o}{v \mp v_s} f$$

| 符号 | 物理量 | 说明 |
|------|--------|------|
| $f$ | 波源频率 | |
| $f'$ | 接收频率 | |
| $v$ | 波速 | 介质中的波速 |
| $v_o$ | 观察者速度 | 接近波源取 $+$，远离取 $-$ |
| $v_s$ | 波源速度 | 接近观察者取 $-$，远离取 $+$ |

## 3. 物理学意义与推论

### 物理意义

机械波是振动在介质中的传播过程，其本质是**能量和信息的传播**而非物质的宏观迁移。介质中每个质点只在自己平衡位置附近振动，但振动状态（相位）以波速向前传播。波速由介质的性质（弹性和惯性）决定，与波的频率无关（无色散介质中）。

### 横波与纵波的传播条件

- **横波**只能在固体中传播（需要剪切弹性），不能在液体和气体中传播。
- **纵波**可以在固体、液体和气体中传播（需要容变弹性或长变弹性）。
- 地震波中，P波（纵波）可穿过地球核部，S波（横波）不能穿过液态外核——这是推断地球外核为液态的依据。

### 波速与介质的关系

| 介质 | 波速公式 |
|------|----------|
| 绳上横波 | $v = \sqrt{\frac{T}{\mu}}$（$T$ 为张力，$\mu$ 为线密度） |
| 固体中纵波 | $v = \sqrt{\frac{Y}{\rho}}$（$Y$ 为杨氏模量，$\rho$ 为密度） |
| 气体中声速 | $v = \sqrt{\frac{\gamma RT}{M}}$（$\gamma$ 为绝热指数） |

### 适用边界

- 机械波必须有介质，不能在真空中传播（与电磁波不同）。
- 简谐波方程假设波在无吸收、无色散的均匀介质中传播。
- 多普勒效应公式仅适用于 $v_s < v$（波源速度小于波速）的情形；当 $v_s > v$ 时产生**冲击波**（[[马赫锥_Mach Cone]]）。
- 波的叠加原理仅适用于**线性介质**（小振幅情形）；大振幅时非线性效应显著。

### 与大学物理的衔接

1. **波动方程**：机械波满足经典的波动方程 $\frac{\partial^2 y}{\partial t^2} = v^2 \frac{\partial^2 y}{\partial x^2}$，这是物理学中最重要的偏微分方程之一。[[电磁波_Electromagnetic Waves]]也满足相同形式的波动方程，但不需要介质。

2. **惠更斯-菲涅耳原理**：[[惠更斯-菲涅耳原理_Huygens-Fresnel Principle]]将波前上每一点视为次级波源，可以解释波的传播、反射、折射和[[波的衍射_Diffraction]]。这是[[波动光学_Wave Optics]]的基础。

3. **波的干涉**：[[波的干涉_Wave Interference]]是波动理论的核心内容。[[托马斯·杨]]的双缝实验展示了光的干涉现象，为光的波动说提供了关键证据。

4. **克拉尼图形**：[[克拉尼图形_Chladni Patterns]]是金属板上声波驻波形成的可视化图案，揭示了二维振动模式的复杂性，与[[简正模_Normal Mode]]理论密切相关。

5. **傅里叶分析**：任意形状的波都可以用[[傅里叶分析_Fourier Analysis]]分解为简谐波的叠加。这是信号处理、通信技术和量子力学的基础工具。

6. **声学与非破坏检测**：超声波在材料中的传播特性是[[声学_Acoustics]]和[[非破坏性检测_Nondestructive Testing]]的基础，涉及波的反射、折射、散射和衰减。

7. 在[[量子力学_Quantum Mechanics]]中，物质具有波粒二象性，[[德布罗意波_De Broglie Wave]]描述了粒子的波动性质，[[薛定谔方程_Schrödinger Equation]]本质上是量子版本的波动方程。

## 4. 相关验证实验

### 绳波演示实验

**实验过程**：固定绳的一端，手持另一端上下振动，观察横波沿绳传播。改变张力 $T$ 和线密度 $\mu$，观察波速变化，验证 $v = \sqrt{T/\mu}$。

### 水波干涉实验

**实验器材**：水波槽、双振源（或单振源加双缝）、频闪灯。

**实验过程**：
1. 在水波槽中用两个振源产生频率相同的水波。
2. 观察水面上形成的干涉图样（加强区和减弱区交替排列）。
3. 用频闪灯"冻结"水面波形，测量干涉条纹间距。
4. 验证干涉加强与减弱的条件。

### 驻波实验

**实验器材**：弦振动实验仪、信号发生器、示波器。

**实验过程**：
1. 将弦的一端固定，另一端连接振动源。
2. 调节振动频率，在特定频率下弦上形成稳定的驻波（可观察到波节和波腹）。
3. 测量相邻波节间距，计算波长 $\lambda = 2 \times \text{波节间距}$。
4. 验证 $v = f\lambda$。

### 声速测量实验

**方法一：共振管法**

在玻璃管中调节空气柱长度，找到声波共振（驻波形成）的位置，由 $v = f\lambda$ 计算声速。

**方法二：双传声器法**

用两个间距已知的传声器接收声波，通过相位差计算波长和声速。

### 多普勒效应演示

让发声器（如蜂鸣器）在绳上旋转运动，观察者听到音调随声源运动方向变化，验证多普勒效应。

## 5. 历史上下文

[[克里斯蒂安·惠更斯]]（Christiaan Huygens, 1629–1695）是波动理论的奠基者。在1678年完成的《光论》（1690年出版）中，惠更斯提出了[[惠更斯原理_Huygens' Principle]]：波前上每一点都是新的次级波源，这些次级波的包络面构成新的波前。惠更斯用这一原理解释了波的传播、反射和折射，并支持光的波动说。

然而，[[艾萨克·牛顿]]的光的微粒说在18世纪占据主导地位。直到19世纪初，[[托马斯·杨]]（Thomas Young）在1801年的双缝干涉实验和[[奥古斯丁·菲涅耳]]（Augustin-Jean Fresnel）的衍射理论才使光的波动说重新获得承认。菲涅耳补充了惠更斯原理中次级波的相位关系，形成了[[惠更斯-菲涅耳原理_Huygens-Fresnel Principle]]。

[[恩斯特·克拉尼]]（Ernst Chladni, 1756–1827）在1787年通过在金属板上撒沙并振动，观察到沙粒聚集在波节线上形成[[克拉尼图形_Chladni Patterns]]，首次使振动模式可视化。

[[克里斯蒂安·多普勒]]（Christian Doppler, 1803–1853）在1842年提出了[[多普勒效应_Doppler Effect]]，预言波源运动时频率会改变。这一效应在后来被广泛应用于天文学（测量星体退行速度）、医学（超声波多普勒成像）和雷达测速等领域。

19世纪，[[约瑟夫·傅里叶]]的[[傅里叶分析_Fourier Analysis]]为波的研究提供了强大的数学工具。[[詹姆斯·麦克斯韦]]（James Clerk Maxwell）在1865年证明光是[[电磁波_Electromagnetic Waves]]，将机械波的理论框架推广到电磁现象。20世纪，[[德布罗意波_De Broglie Wave]]的提出和[[薛定谔方程_Schrödinger Equation]]的建立将波动概念引入了量子物理的领域。
