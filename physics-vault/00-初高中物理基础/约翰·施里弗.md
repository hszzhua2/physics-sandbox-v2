# 约翰·施里弗 (John Schrieffer)

## 1. 生平
1931年生于奥克帕克，在伊利诺伊大学学习，任佛罗里达州立大学教授。与巴丁、库珀提出BCS超导理论，1972年获诺贝尔物理学奖，2019年逝于塔拉哈西。

## 2. 主要贡献
- 研究[[欧姆定律]]与BCS超导理论
- 发展[[欧姆定律]]与超导物理
- 提出[[欧姆定律]]的库珀对概念

## 3. 关键公式与思想
- $\Psi=\prod_k(\upsilon_k+c_k^\dagger c_{-k}^\dagger)|0\rangle$（BCS基态波函数）

## 4. 历史影响
施里弗的BCS理论推动了[[欧姆定律]]与超导物理的发展。
