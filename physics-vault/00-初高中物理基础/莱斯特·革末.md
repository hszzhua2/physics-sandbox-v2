# 莱斯特·革末 (Lester Germer)

## 1. 生平
1896年生于芝加哥，在哥伦比亚大学学习，任贝尔实验室研究员。与戴维森完成电子衍射实验，验证物质波，1971年逝于纽约。

## 2. 主要贡献
- 完成[[波粒二象性]]的电子衍射实验
- 验证[[波粒二象性]]与德布罗意假说
- 发展[[波粒二象性]]实验

## 3. 关键公式与思想
- $n\lambda=2d\sin\theta$（电子衍射条件）

## 4. 历史影响
革末的电子衍射是[[波粒二象性]]物质波的实验验证。
