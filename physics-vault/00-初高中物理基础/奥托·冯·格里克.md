# 奥托·冯·格里克 (Otto von Guericke)

## 1. 生平
1602年生于马格德堡，在莱比锡大学学习，任马格德堡市长。发明真空泵并完成马格德堡半球实验，研究[[压强]]，1686年逝于汉堡。

## 2. 主要贡献
- 发明真空泵研究[[压强]]
- 完成马格德堡半球实验
- 研究[[声音的产生与传播]]与真空

## 3. 关键公式与思想
- $P=\rho gh$（大气压强）

## 4. 历史影响
冯·格里克的真空实验是[[压强]]与[[声音的产生与传播]]研究的基础。
