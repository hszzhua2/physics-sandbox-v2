# 戈特弗里德·莱布尼茨 (Gottfried Leibniz)

## 1. 生平
1646年生于莱比锡，在莱比锡大学学习，任汉诺威宫廷图书馆长。与牛顿各自独立发明微积分，研究动能与动量，1716年逝于汉诺威。

## 2. 主要贡献
- 提出[[动能定理]]的活力(vis viva)概念
- 发展[[动量守恒定律]]与[[机械能守恒定律]]
- 发明微积分与[[动能定理]]

## 3. 关键公式与思想
- $E=\dfrac{1}{2}mv^2$（动能/活力）

## 4. 历史影响
莱布尼茨的活力概念是[[动能定理]]与[[机械能守恒定律]]的早期形式。
