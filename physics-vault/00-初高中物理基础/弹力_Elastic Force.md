# 弹力 (Elastic Force)

## 1. 核心定义

发生弹性形变的物体由于要恢复原状而对与它接触的物体产生的力。方向始终与形变方向相反，指向恢复原状的方向。

## 2. 数学表达

### 胡克定律

在弹性限度内，弹簧的弹力与形变量成正比：

$$F = -kx$$

**参数说明：**

| 符号 | 物理量 | 单位 | 说明 |
|------|--------|------|------|
| $F$ | 弹力 | $\text{N}$ | 方向指向平衡位置 |
| $k$ | 劲度系数（弹簧常数） | $\text{N/m}$ | 由弹簧材料、粗细、长度决定 |
| $x$ | 形变量 | $\text{m}$ | 偏离平衡位置的位移 |
| 负号 | — | — | 表示弹力方向与形变方向相反（恢复力） |

### 常见弹力

| 弹力类型 | 产生原因 | 方向 |
|----------|----------|------|
| 弹簧弹力 | 弹簧拉伸或压缩 | 沿弹簧轴线指向恢复方向 |
| 绳的拉力 | 绳被拉伸 | 沿绳指向绳收缩方向 |
| 接触面支持力 | 面被压缩 | 垂直于接触面指向被支持物体 |
| 杆的弹力 | 杆被拉伸或压缩 | 沿杆方向（可拉可推） |

### 弹簧的串并联

**串联**：$n$ 根劲度系数分别为 $k_1, k_2, \ldots, k_n$ 的弹簧串联：

$$\frac{1}{k_{\text{串}}} = \frac{1}{k_1} + \frac{1}{k_2} + \cdots + \frac{1}{k_n}$$

特例：$n$ 根相同的弹簧（劲度系数均为 $k_0$）串联：

$$k_{\text{串}} = \frac{k_0}{n}$$

**并联**：$n$ 根弹簧并联：

$$k_{\text{并}} = k_1 + k_2 + \cdots + k_n$$

特例：$n$ 根相同的弹簧并联：

$$k_{\text{并}} = nk_0$$

### 弹性势能

弹簧存储的弹性势能：

$$E_p = \frac{1}{2}kx^2$$

## 3. 物理学意义与推论

### 物理意义

弹力的本质是构成物体的分子间**电磁相互作用**在宏观上的表现。当物体发生弹性形变时，分子间距偏离平衡位置，分子力（表现为恢复力）使物体趋向恢复原状。弹力是一种**接触力**，必须有形变才会产生。

### 弹性形变与塑性形变

- **弹性形变**：撤去外力后物体能完全恢复原状的形变。胡克定律仅适用于弹性形变。
- **塑性形变**（范性形变）：撤去外力后物体不能完全恢复原状的形变，此时外力做的功转化为内能（发热）。

### 胡克定律的适用范围

- 仅在**弹性限度内**成立。当形变超过弹性限度（屈服点），$F$-$x$ 关系偏离线性。
- 弹性限度与材料性质有关：橡胶的弹性范围大，金属的弹性范围相对较小。
- 胡克定律是**线性近似**，在微小形变条件下成立。

### 弹力的方向判断

- 弹力方向始终与接触面**垂直**（法向），或沿绳/弹簧方向。
- 对于点接触，弹力方向垂直于过接触点的公切面。
- 对于绳索，绳中各处拉力大小相等（不计绳的质量和摩擦时）。

### 与大学物理的衔接

1. **弹性力学**：在大学物理中，胡克定律推广为三维的**广义胡克定律**，用应力-应变张量描述：

$$\sigma_{ij} = C_{ijkl}\varepsilon_{kl}$$

其中 $C_{ijkl}$ 为弹性张量（四阶张量）。[[弹性力学_Theory of Elasticity]]研究固体在各种载荷下的应力和变形分布。

2. **弹性势能与振动**：弹簧弹力是[[简谐运动_Simple Harmonic Motion]]的恢复力，弹性势能与动能的周期性交换构成[[弹簧振子_Spring Oscillator]]的振动基础。在[[哈密顿力学_Hamiltonian Mechanics]]中，弹簧振子的哈密顿量为 $H = \frac{p^2}{2m} + \frac{1}{2}kx^2$。

3. **声波与弹性波**：固体中弹性力的传播形成声波（弹性波），[[机械波_Mechanical Waves]]的传播速度由介质的弹性模量和密度决定。

4. 在[[量子力学_Quantum Mechanics]]中，弹簧振子模型推广为[[量子谐振子_Quantum Harmonic Oscillator]]，是量子场论中粒子激发的基础模型。

## 4. 相关验证实验

### 胡克定律验证实验

**实验器材**：弹簧、钩码若干、铁架台、刻度尺。

**实验过程**：
1. 将弹簧悬挂在铁架台上，记录弹簧自然长度 $l_0$。
2. 逐个增加钩码（每个质量已知），记录每次弹簧的长度 $l_i$，计算形变量 $x_i = l_i - l_0$。
3. 以弹力 $F = mg$（钩码重力）为纵轴，形变量 $x$ 为横轴，作 $F$-$x$ 图像。
4. 在弹性限度内，$F$-$x$ 图为一条过原点的直线，斜率等于劲度系数 $k$。

**注意事项**：
- 所挂钩码总重力不能超过弹簧的弹性限度。
- 待弹簧静止后再读数。
- 实验前应检查弹簧是否已有永久形变。

### 弹簧串并联实验

分别测量单根弹簧的劲度系数 $k_0$，再测量串联和并联后的等效劲度系数，验证 $k_{\text{串}} = \frac{k_0}{2}$ 和 $k_{\text{并}} = 2k_0$（两根相同弹簧的情况）。

## 5. 历史上下文

[[罗伯特·胡克]]（Robert Hooke, 1635–1703）在1660年发现了弹力与形变成正比的规律，并于1678年公开发表。胡克最初用拉丁文将其表述为"ut tensio, sic vis"（"如伸长，则力"），即力与伸长成正比。

胡克与[[艾萨克·牛顿]]之间有着著名的学术争端。胡克曾声称牛顿的万有引力反平方定律部分源于他的启发，牛顿因此在《自然哲学的数学原理》中未提及胡克的贡献。两人的矛盾持续了终生，牛顿在胡克去世后才愿意发表光学著作。

胡克定律的数学表达 $F = -kx$ 虽然形式简单，但意义深远。它奠定了[[弹性力学_Theory of Elasticity]]的基础，也是[[简谐运动_Simple Harmonic Motion]]和波动理论的出发点。[[托马斯·杨]]（Thomas Young）在1807年引入了**杨氏模量**的概念，将胡克定律推广到一般弹性体。19世纪，[[柯西]]（Augustin-Louis Cauchy）建立了应力-应变张量理论，将胡克定律推广为三维各向异性弹性体的广义胡克定律，成为现代弹性力学的基石。
