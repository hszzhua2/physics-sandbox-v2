# 詹姆斯·查德威克 (James Chadwick)

## 1. 生平
1891年生于伯灵顿，在曼彻斯特大学与剑桥大学学习，任卡文迪什实验室教授。发现中子，1935年获诺贝尔物理学奖，1974年逝于剑桥。

## 2. 主要贡献
- 发现[[原子核的组成]]的中子
- 研究[[原子核的组成]]与核反应
- 参与曼哈顿计划

## 3. 关键公式与思想
- $\text{neutron: neutral nucleon, mass}\approx1.675\times10^{-27}\,\text{kg}$（中子）

## 4. 历史影响
查德威克发现的中子完善了[[原子核的组成]]理论。
