# 海因里希·楞次 (Heinrich Lenz)

## 1. 生平
1804年生于多尔帕特，在多尔帕特大学学习，任圣彼得堡大学教授。提出楞次定律，研究[[电功与电功率]]，1865年逝于罗马。

## 2. 主要贡献
- 提出[[电功与电功率]]的楞次定律
- 研究[[电功与电功率]]与电磁感应
- 发展[[电功与电功率]]与焦耳热

## 3. 关键公式与思想
- $\mathcal{E}=-\dfrac{d\Phi_B}{dt}$（楞次定律/负号）

## 4. 历史影响
楞次定律是[[电功与电功率]]与电磁感应的方向判定规则。
