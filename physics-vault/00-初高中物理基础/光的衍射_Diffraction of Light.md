# 光的衍射 (Diffraction of Light)

## 1. 核心定义

**光的衍射**（Diffraction of Light）是指光波在传播过程中遇到障碍物或孔隙时，偏离直线传播方向，绕过障碍物边缘进入几何阴影区域的现象。衍射是波动的本质特征之一，直接证明了光的波动性。

光的衍射现象由意大利物理学家[[弗朗西斯科·格里马尔迪]]于17世纪首次观察到并命名。然而，衍射现象的系统理论解释直到19世纪初才由[[奥古斯丁·菲涅耳]]基于[[惠更斯-菲涅耳原理_Huygens-Fresnel Principle]]完成，成为光的波动理论的关键证据。

衍射现象的物理本质是：当光波波阵面的一部分被障碍物遮挡时，被遮挡的边缘处各点作为次级波源发出球面子波，这些子波的叠加使得光不仅出现在直线传播的几何照明区，也出现在几何阴影区内，形成衍射图样。

根据障碍物或孔隙的尺寸与光波波长的关系，衍射可分为**菲涅耳衍射**（近场衍射）和**夫琅禾费衍射**（远场衍射）两大类。

## 2. 数学表达

### 2.1 单缝夫琅禾费衍射

当波长为 $\lambda$ 的单色平行光垂直照射宽度为 $a$ 的狭缝时，在焦距为 $f$ 的透镜后焦面上形成衍射图样。

**暗纹条件**（衍射极小）：

$$a\sin\theta = k\lambda, \quad k = \pm 1, \pm 2, \pm 3, \ldots$$

**中央亮纹角宽度**：

$$\Delta\theta_0 = \frac{2\lambda}{a}$$

中央亮纹线宽度（在屏上）：

$$\Delta x_0 = \frac{2\lambda f}{a}$$

**次级亮纹位置**（近似）：

$$a\sin\theta \approx \left(k + \frac{1}{2}\right)\lambda, \quad k = \pm 1, \pm 2, \ldots$$

中央亮纹最亮最宽，两侧次级亮纹强度迅速衰减。单缝衍射的光强分布为：

$$I(\theta) = I_0 \left[\frac{\sin\alpha}{\alpha}\right]^2, \quad \alpha = \frac{\pi a \sin\theta}{\lambda}$$

此函数称为**sinc²函数**，其中 $\text{sinc}(\alpha) = \sin\alpha/\alpha$。

### 2.2 圆孔衍射与艾里斑

当平行光通过直径为 $D$ 的圆孔时，衍射图样为中心亮斑（**艾里斑**，Airy Disk）和同心圆环。

艾里斑角半径（第一暗环对应的衍射角）为：

$$\theta_{\text{艾里}} = 1.22\frac{\lambda}{D}$$

艾里斑线半径（在焦面上）：

$$r_{\text{艾里}} = 1.22\frac{\lambda f}{D}$$

### 2.3 瑞利判据与光学仪器分辨率

**瑞利判据**（Rayleigh Criterion）：当一个物点的艾里斑中心恰好落在另一个物点的艾里斑第一暗环上时，两物点恰好可被分辨。此时两物点对透镜的张角为**最小分辨角**：

$$\boxed{\theta_{\min} = 1.22\frac{\lambda}{D}}$$

其中 $D$ 为透镜有效孔径（直径）。

光学仪器的**分辨本领**（Resolving Power）定义为最小分辨角的倒数：

$$R = \frac{1}{\theta_{\min}} = \frac{D}{1.22\lambda}$$

提高分辨本领的途径：
- 增大孔径 $D$（如天文望远镜的大口径物镜）。
- 减小波长 $\lambda$（如电子显微镜利用短波长电子束替代可见光）。

### 2.4 衍射光栅

衍射光栅由大量等间距的狭缝组成，缝宽为 $a$，缝间距（光栅常数）为 $d = a + b$（$b$ 为不透光部分宽度），缝数为 $N$。

**光栅方程**（主极大条件）：

$$d\sin\theta = k\lambda, \quad k = 0, \pm 1, \pm 2, \ldots$$

其中 $k$ 为衍射级次。

**主极大半角宽度**（谱线锐度）：

$$\Delta\theta = \frac{\lambda}{Nd\cos\theta}$$

缝数 $N$ 越多，主极大越窄越锐，分辨率越高。

**光栅分辨本领**：

$$R = \frac{\lambda}{\Delta\lambda} = kN$$

即分辨本领正比于衍射级次 $k$ 和总缝数 $N$。这是光栅光谱仪优于棱镜光谱仪的关键。

### 2.5 菲涅耳衍射与夫琅禾费衍射的区别

| 特征 | 菲涅耳衍射 | 夫琅禾费衍射 |
|------|-----------|-------------|
| 入射波 | 球面波（发散）或非平行光 | 平面波（平行光） |
| 观察距离 | 近场（有限远） | 远场（无限远，或透镜后焦面） |
| 数学处理 | 菲涅耳积分（复杂） | 傅里叶变换（较简洁） |
| 典型图样 | 同心圆环（圆孔） | sinc²函数（单缝） |

夫琅禾费衍射在实验中通常通过在狭缝前后各放置一个透镜实现：前透镜将入射光变为平行光，后透镜将衍射光会聚到焦面上观察。

## 3. 物理学意义与推论

### 3.1 光波动性的决定性证据

衍射现象是光具有波动性的最直接证据。[[艾萨克·牛顿]]的微粒说无法解释光绕过障碍物的衍射现象。1801年[[托马斯·杨]]的双缝实验和1818年菲涅耳的衍射理论最终确立了光的波动说。特别是1818年巴黎科学院的"衍射征文"竞赛中，菲涅耳的论文遭到了泊松的质疑——泊松指出，若波动说正确，则圆盘阴影中心应出现亮斑（泊松斑），这看似荒谬。但实验随即证实了泊松斑的存在，反而成为波动说的有力证据。

### 3.2 衍射极限与光学仪器设计

瑞利判据 $\theta_{\min} = 1.22\lambda/D$ 表明，任何光学仪器都存在衍射极限，无法无限提高分辨率。这一定律制约了望远镜、显微镜、照相机等所有成像系统的分辨能力。要突破衍射极限，需要采用超分辨技术（如受激发射损耗显微术 STED、结构光照明 SIM 等，2014年诺贝尔化学奖）。

### 3.3 波粒二象性的伏笔

光的衍射证明了光的波动性，而[[光电效应]]证明了光的粒子性。这两种看似矛盾的性质在[[波粒二象性_Wave-Particle Duality]]中得到统一。衍射现象在电子、中子等物质粒子实验中同样被观测到，是量子力学波粒二象性的实验基础之一。

### 3.4 与大学物理的衔接

光的衍射在大学物理中有更严格的数学处理：

- 与[[惠更斯-菲涅耳原理_Huygens-Fresnel Principle]]的衔接：菲涅耳-基尔霍夫衍射积分公式严格地给出了衍射光场的计算方法，是标量衍射理论的核心。
- **傅里叶光学**：夫琅禾费衍射图样是孔径函数的傅里叶变换，透镜可实现光学傅里叶变换，这是光学信息处理的理论基础。
- **巴比涅原理**（Babinet's Principle）：互补屏（孔与障碍物互换）的夫琅禾费衍射图样除中心外完全相同。
- **X射线衍射**：晶体中原子间距与X射线波长同量级（$\sim 10^{-10} \, \text{m}$），晶体可作为三维衍射光栅（布拉格衍射 $2d\sin\theta = k\lambda$），是结构生物学和材料科学的核心分析手段。
- **全息术**：利用干涉和衍射原理记录和重建光波的全部信息（振幅和相位），由[[丹尼斯·盖伯]]（Dennis Gabor）于1948年提出。

## 4. 相关验证实验

### 4.1 单缝衍射实验

**实验装置**：激光（如He-Ne激光 $\lambda = 632.8 \, \text{nm}$）垂直照射可调宽度的狭缝，衍射屏置于远处（或透镜后焦面）。

**观测内容**：
- 中央亮纹最宽最亮，两侧对称分布次级亮纹，强度递减。
- 改变缝宽 $a$：缝越窄，衍射图样越宽（衍射效应越明显）；缝越宽，衍射图样越窄（趋近几何光学极限）。
- 改变波长 $\lambda$：红光衍射图样比蓝光宽。
- 验证暗纹条件 $a\sin\theta = k\lambda$。

### 4.2 圆孔衍射与艾里斑观察

**实验装置**：激光通过小圆孔（直径约 $0.1 \sim 0.5 \, \text{mm}$），观察屏上的衍射图样。

**现象**：中心为明亮的艾里斑，外围环绕明暗交替的同心圆环。测量艾里斑直径，验证 $\theta = 1.22\lambda/D$。

### 4.3 光栅衍射实验

**实验装置**：激光通过透射式衍射光栅（如每毫米300线的光栅），观察各级主极大。

**观测内容**：
- 中央零级亮纹最亮，两侧对称分布各级主极大，间距相等。
- 测量各级衍射角，由 $d\sin\theta = k\lambda$ 计算光栅常数或光波长。
- 白光入射时，零级仍为白色，其他各级展开为连续光谱（光栅光谱）。

### 4.4 泊松斑演示

**实验装置**：激光照射一个直径约 $2 \sim 3 \, \text{mm}$ 的圆形挡板（如钢珠），观察挡板阴影中心。

**现象**：阴影正中心出现一个亮斑（泊松斑），周围环绕衍射圆环，验证波动理论。

## 5. 历史上下文

光的衍射现象最早由意大利耶稣会士[[弗朗西斯科·格里马尔迪]]在1660年代观察到。他在1665年出版的《光的物理数学》中记录了光通过小孔后展宽并产生彩色条纹的现象，并将其命名为"衍射"（diffraction，源自拉丁语"打破"）。

[[克里斯蒂安·惠更斯]]于1678年提出惠更斯原理，将波阵面上每一点视为次级波源，为衍射现象提供了初步的波动解释。1818年，[[奥古斯丁·菲涅耳]]在惠更斯原理基础上补充了次级波的干涉条件，形成了[[惠更斯-菲涅耳原理_Huygens-Fresnel Principle]]，能够定量计算衍射图样。

1818年巴黎科学院衍射征文竞赛中，菲涅耳的论文遭到了评委[[西梅翁·泊松]]的质疑。泊松根据菲涅耳理论推导出一个看似荒谬的结论：若在光束中放置一个不透明圆盘，则圆盘阴影中心应出现一个亮斑。[[弗朗索瓦·阿拉戈]]随即用实验证实了这一亮斑的存在（后被称为"泊松斑"或"阿拉戈斑"），菲涅耳因此获得论文奖。这一事件被视为光的波动理论的决定性胜利。

衍射光栅的理论由[[约瑟夫·冯·夫琅禾费]]于1821年系统研究，他制造了最早的衍射光栅并研究了光栅光谱。夫琅禾费还在太阳光谱中发现了数百条暗线（[[夫琅禾费线]] Fraunhofer lines），为后来的光谱分析奠定了基础。
