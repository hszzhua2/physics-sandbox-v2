# 欧姆定律 (Ohm's Law)

## 1. 核心定义

通过导体的电流与导体两端的电压成正比，与导体的电阻成反比：$I = U/R$；电阻是导体对电流的阻碍作用，电阻率 $\rho = RS/L$ 反映材料的导电特性。

## 2. 数学表达

### 2.1 欧姆定律

$$I = \frac{U}{R}$$

| 参数 | 物理量 | 单位 |
|------|--------|------|
| $I$ | 电流 | $\text{A}$（安培） |
| $U$ | [[电势差]]（电压） | $\text{V}$（伏特） |
| $R$ | [[电阻]] | $\Omega$（欧姆） |

$1\,\Omega = 1\,\text{V/A}$。

### 2.2 变形公式

$$U = IR$$

$$R = \frac{U}{I}$$

**注意**：$R = U/I$ 是电阻的**定义式**（测量式），而非决定式。电阻 $R$ 由导体本身的性质（材料、长度、横截面积、温度）决定，与加在它上面的电压和电流无关。

### 2.3 电阻定律

$$R = \rho \frac{L}{S}$$

| 参数 | 物理量 | 单位 | 说明 |
|------|--------|------|------|
| $R$ | 电阻 | $\Omega$ | 导体电阻 |
| $\rho$ | 电阻率 | $\Omega \cdot \text{m}$ | 材料特性，与温度有关 |
| $L$ | 导体长度 | $\text{m}$ | 电流方向的长度 |
| $S$ | 横截面积 | $\text{m}^2$ | 垂直于电流方向的面积 |

### 2.4 串并联电阻

**串联**：

$$R_{\text{总}} = R_1 + R_2 + \cdots + R_n$$

**并联**：

$$\frac{1}{R_{\text{总}}} = \frac{1}{R_1} + \frac{1}{R_2} + \cdots + \frac{1}{R_n}$$

两个电阻并联：$R_{\text{总}} = \frac{R_1 R_2}{R_1 + R_2}$。

### 2.5 电阻率与温度的关系

$$\rho = \rho_0 [1 + \alpha (T - T_0)]$$

| 参数 | 物理量 | 说明 |
|------|--------|------|
| $\rho_0$ | 参考温度 $T_0$ 时的电阻率 | 材料特性 |
| $\alpha$ | 电阻温度系数 | 金属为正，半导体为负 |
| $T$ | 当前温度 | $^{\circ}\text{C}$ 或 $\text{K}$ |

常见材料电阻率（$20\,^{\circ}\text{C}$）：

| 材料 | 电阻率 $\rho$（$\Omega \cdot \text{m}$） | 类型 |
|------|----------------------------------------|------|
| 银 | $1.6 \times 10^{-8}$ | 导体 |
| 铜 | $1.7 \times 10^{-8}$ | 导体 |
| 铝 | $2.9 \times 10^{-8}$ | 导体 |
| 钨 | $5.3 \times 10^{-8}$ | 导体 |
| 铁 | $1.0 \times 10^{-7}$ | 导体 |
| 碳（石墨） | $5 \times 10^{-6} \sim 5 \times 10^{-5}$ | 半导体 |
| 硅 | $\sim 10^3$ | 半导体 |
| 橡胶 | $10^{13} \sim 10^{16}$ | 绝缘体 |
| 玻璃 | $10^{10} \sim 10^{14}$ | 绝缘体 |

## 3. 物理学意义与推论

### 3.1 欧姆定律的物理意义

欧姆定律揭示了电流、电压和电阻三者的定量关系：
- **电压 $U$ 是原因**：电压（电势差）驱动电荷定向移动，是产生电流的"推力"。
- **电流 $I$ 是结果**：电流是电压驱动下电荷流动的表现。
- **电阻 $R$ 是阻碍**：电阻限制了电流的大小，$R$ 越大同样电压下电流越小。

**线性关系**：欧姆定律表明 $I$ 与 $U$ 成正比（$R$ 一定时），$I$ 与 $R$ 成反比（$U$ 一定时），这是线性关系。伏安特性曲线（$I$-$U$ 图）为过原点的直线，斜率 $= 1/R$。

### 3.2 电阻的物理本质

电阻的微观机制是载流子（自由电子）在导体中运动时与晶格离子碰撞散射导致的能量损耗。从微观角度看：

$$\rho = \frac{m}{n e^2 \tau}$$

其中 $m$ 为电子质量，$n$ 为自由电子数密度，$e$ 为电子电荷量，$\tau$ 为弛豫时间（电子两次碰撞之间的平均时间）。

温度升高时，晶格振动加剧，电子与晶格碰撞更频繁（$\tau$ 减小），电阻增大——这解释了金属导体电阻随温度升高而增大的现象（$\alpha > 0$）。

### 3.3 电阻率与材料分类

| 类型 | 电阻率范围 | 导电机制 | 典型材料 |
|------|-----------|---------|---------|
| 导体 | $\rho < 10^{-6}\,\Omega \cdot \text{m}$ | 自由电子导电 | 金属、石墨 |
| 半导体 | $10^{-6} \sim 10^6\,\Omega \cdot \text{m}$ | 电子和空穴导电 | 硅、锗、砷化镓 |
| 绝缘体 | $\rho > 10^6\,\Omega \cdot \text{m}$ | 几乎无自由载流子 | 橡胶、玻璃、陶瓷 |

**半导体特性**：半导体的电阻率随温度升高而减小（$\alpha < 0$），因为温度升高时更多电子被激发到导带，载流子数增加的效果超过了碰撞频率增大的影响。这一性质使半导体可用于制造热敏电阻、光敏器件等。掺杂后的半导体（P型和N型）是现代电子器件的基础。

### 3.4 超导现象

某些金属和合金在温度降低到临界温度 $T_c$ 以下时，电阻突然降为零，这种现象称为超导现象，由[[海克·卡末林·昂内斯]]于1911年发现。超导状态下电流可以无损耗地持续流动，电阻率严格为零。超导现象无法用经典电子理论解释，需用量子力学（BCS理论，[[约翰·巴丁]]、[[莱昂·库珀]]、[[约翰·施里弗]]，1957年）来理解——电子通过晶格振动媒介形成库珀对，不受散射。

### 3.5 欧姆定律的适用范围

欧姆定律并非对所有导体都成立。满足欧姆定律的导体称为**线性元件**（如金属导体、电解液），其 $I$-$U$ 关系为过原点的直线。不满足欧姆定律的元件称为**非线性元件**：

| 元件 | $I$-$U$ 特性 | 说明 |
|------|-------------|------|
| 金属导体 | 线性（过原点的直线） | 遵循欧姆定律 |
| 气体导体 | 非线性 | 击穿后导电，不遵循欧姆定律 |
| 二极管 | 非线性（单向导电） | 正向导通、反向截止 |
| 热敏电阻 | 非线性 | 电阻随温度变化 |

### 3.6 适用边界与知识衔接

欧姆定律在初中阶段限于纯电阻电路的直流分析。在高中物理中，将引入电源电动势 $\varepsilon$ 和内阻 $r$，建立[[闭合电路欧姆定律_Ohm's Law for Complete Circuit]]：

$$I = \frac{\varepsilon}{R + r}$$

其中 $\varepsilon = U_{\text{外}} + U_{\text{内}} = IR + Ir$。路端电压 $U = \varepsilon - Ir$ 随外电路电阻增大而增大。同时将分析含二极管等非线性元件的电路。在大学物理中，欧姆定律的微观形式 $j = \sigma E$（电流密度正比于电场）可从[[麦克斯韦方程组_Maxwell's Equations]]和德鲁德模型推导，但深层理解需固体物理的能带理论和[[量子力学]]。交流电路中，欧姆定律推广为复数形式 $\tilde{I} = \tilde{U}/\tilde{Z}$，其中 $\tilde{Z}$ 为复阻抗，涉及电阻、感抗和容抗的综合，在[[RLC谐振_RLC Resonance]]中进一步展开。

## 4. 相关验证实验

### 4.1 欧姆定律验证实验

在电路中串联定值电阻 $R$、电流表，并联电压表在 $R$ 两端：
1. 滑动变阻器改变 $R$ 两端电压。
2. 每次记录电压 $U$ 和电流 $I$。
3. 多次测量，绘制 $I$-$U$ 图像。
4. 验证图像为过原点的直线，斜率 $= 1/R$，即 $I = U/R$。

### 4.2 电阻测量实验（伏安法）

利用电压表和电流表测量未知电阻：
1. 按电路图连接：电源、开关、滑动变阻器、待测电阻 $R_x$、电流表（串联）、电压表（并联在 $R_x$ 两端）。
2. 调节滑动变阻器，记录多组 $U$、$I$ 值。
3. 计算各次 $R = U/I$，取平均值。
4. 也可用 $I$-$U$ 图像斜率求 $R$。

**两种接法的误差分析**：
- **电流表内接法**：电压表读数偏大（含电流表分压），$R_{\text{测}} = R_x + R_A > R_x$，适用于 $R_x \gg R_A$ 的大电阻测量。
- **电流表外接法**：电流表读数偏大（含电压表分流），$R_{\text{测}} = R_x \| R_V < R_x$，适用于 $R_x \ll R_V$ 的小电阻测量。

### 4.3 电阻定律验证实验

使用电阻丝（如镍铬合金丝），改变以下条件：
1. **长度**：保持材料和横截面积不变，改变长度 $L$，测量电阻 $R$，验证 $R \propto L$。
2. **横截面积**：保持材料和长度不变，用不同粗细的电阻丝，验证 $R \propto 1/S$。
3. **材料**：保持长度和横截面积不变，换不同材料的电阻丝，测量电阻率 $\rho$ 的差异。
4. **温度**：对电阻丝加热，观察电阻变化（金属电阻增大），验证 $\rho$ 与温度有关。

### 4.4 滑动变阻器实验

滑动变阻器通过改变接入电路的电阻丝长度来改变电阻：
- **限流式接法**：变阻器与用电器串联，调节电流大小。
- **分压式接法**：变阻器两端接电源，滑片分出可变电压，调节电压从零连续变化。

## 5. 历史上下文

### 5.1 格奥尔格·欧姆

[[格奥尔格·欧姆]]（Georg Ohm, 1789—1854）是德国物理学家。1826年，欧姆通过大量精确实验发现了导体中电流与电压的正比关系。他使用[[托马斯·塞贝克]]于1821年发明的温差电池作为稳定电源（当时伏打电堆电压不稳定），用检流计测量电流，系统研究了不同材料、不同长度和粗细导线的导电特性。

欧姆于1827年在《电路的数学研究》（*Die galvanische Kette, mathematisch bearbeitet*）中发表了完整的定律表述，将电流、电压和电阻的关系以数学形式 $I = U/R$ 精确表达。他还提出了电阻与导体长度成正比、与横截面积成反比的电阻定律。

### 5.2 欧姆的困境

欧姆的工作在发表之初并未得到承认，反而受到了德国教育界的冷遇甚至批评。当时德国物理学界受[[黑格尔]]自然哲学影响，重思辨轻实验，欧姆的纯实验-数学方法被认为"缺乏哲学深度"。他因此失去了科隆大学的教职。直到1830年代，欧姆定律逐渐被英国和法国物理学界接受并验证，欧姆才获得应有的学术地位。1841年，英国皇家学会授予他[[科普利奖章]]。电阻单位"欧姆"（$\Omega$）于1881年国际电学大会正式以他命名。

### 5.3 从欧姆定律到电路理论

欧姆定律为电路理论奠定了定量基础。[[古斯塔夫·基尔霍夫]]在此基础上提出了基尔霍夫定律（1845年），使任意复杂电路的分析成为可能。[[开尔文勋爵]]和[[詹姆斯·麦克斯韦]]发展了电网络理论。19世纪末，交流电技术发展后，[[查尔斯·普罗透斯·斯坦梅茨]]将欧姆定律推广到交流电路，引入复阻抗概念，$\tilde{Z} = R + jX$（$R$ 为电阻，$X$ 为电抗）。

### 5.4 现代意义

欧姆定律至今是电学教育和工程实践的基础。从简单的电路设计到超大规模集成电路（VLSI）的仿真分析，欧姆定律无处不在。在大学物理和材料科学中，欧姆定律的微观理论（德鲁德模型、能带理论）揭示了导电性的本质——载流子在周期性晶格中的量子行为。超导体（电阻为零）和拓扑绝缘体（表面导电、内部绝缘）等新型材料的研究不断拓展着人们对"电阻"概念的理解边界。
