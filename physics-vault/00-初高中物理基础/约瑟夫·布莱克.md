# 约瑟夫·布莱克 (Joseph Black)

## 1. 生平
1728年生于波尔多，在格拉斯哥大学与爱丁堡大学学习，任爱丁堡大学教授。提出比热容与潜热概念，1799年逝于爱丁堡。

## 2. 主要贡献
- 提出[[比热容]]的概念
- 发现[[物态变化]]的潜热
- 研究[[比热容]]与量热学

## 3. 关键公式与思想
- $Q=mc\Delta T$（比热容公式）
- $Q=mL$（潜热）

## 4. 历史影响
布莱克的比热容与潜热概念是[[比热容]]与[[物态变化]]的基础。
