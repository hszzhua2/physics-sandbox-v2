# 卡诺循环 (Carnot Cycle)

## 1. 核心定义
卡诺循环：最大热机效率。该概念是基础物理的重要内容，涉及[[理想气体状态方程]]等关联主题。

## 2. 数学表达
核心公式：
$\eta=1-T_c/T_h$

该公式表达了卡诺循环的定量关系，其中各物理量的含义取决于具体物理情境。

## 3. 物理意义
卡诺循环揭示了基础物理中的基本物理规律。最大热机效率，这一关系深刻反映了自然界的内在结构。该概念是构建物理学理论体系的重要组成部分。

## 4. 历史发展
卡诺循环由[[卡诺]]于1824年提出。这一概念的建立是基础物理发展的重要里程碑，[[卡诺]]通过理论分析和实验验证确立了其正确性。此后该概念被不断完善和推广。

## 5. 应用与拓展
卡诺循环在现代物理学和工程技术中有广泛应用。该概念是基础物理理论分析和实际计算的基础工具。
