# 恩斯特·马赫 (Ernst Mach)

## 1. 生平
1838年生于图拉斯，在维也纳大学学习，任维也纳大学与布拉格大学教授。研究超音速与马赫数，提出马赫原理，1916年逝于哈尔。

## 2. 主要贡献
- 提出[[分子动理论]]的马赫原理
- 研究[[等效原理]]与惯性
- 发展[[分子动理论]]与力学批判

## 3. 关键公式与思想
- $M=v/c_s$（马赫数）

## 4. 历史影响
马赫的惯性思想影响了[[等效原理]]与爱因斯坦广义相对论。
