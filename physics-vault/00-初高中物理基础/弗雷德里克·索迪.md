# 弗雷德里克·索迪 (Frederick Soddy)

## 1. 生平
1877年生于伊斯特本，在牛津大学学习，任牛津大学教授。提出同位素概念与放射性位移定律，1921年获诺贝尔化学奖，1956年逝于布莱顿。

## 2. 主要贡献
- 提出[[放射性衰变]]的同位素概念
- 研究[[放射性衰变]]与位移定律
- 发展[[放射性衰变]]理论

## 3. 关键公式与思想
- $\text{isotopes: same Z, different A}$（同位素）

## 4. 历史影响
索迪的同位素概念是[[放射性衰变]]与核化学的基础。
