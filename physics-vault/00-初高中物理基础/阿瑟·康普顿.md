# 阿瑟·康普顿 (Arthur Compton)

## 1. 生平
1892年生于伍斯特，在伍斯特学院与普林斯顿大学学习，任华盛顿大学与芝加哥大学教授。发现康普顿效应，1927年获诺贝尔物理学奖，1962年逝于伯克利。

## 2. 主要贡献
- 发现[[波粒二象性]]的康普顿散射
- 研究[[核裂变与核聚变]]与[[波粒二象性]]
- 发展[[波粒二象性]]与X射线散射

## 3. 关键公式与思想
- $\lambda'-\lambda=\dfrac{h}{m_ec}(1-\cos\theta)$（康普顿散射）

## 4. 历史影响
康普顿效应是[[波粒二象性]]的光子粒子性的关键证据。
