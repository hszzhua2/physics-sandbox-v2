# 质量与密度 (Mass and Density)

## 1. 核心定义

质量是物体所含物质的多少，密度是某种物质单位体积的质量，定义为 $\rho = m/V$，是物质的特征物理量。

## 2. 数学表达

### 2.1 密度定义式

$$\rho = \frac{m}{V}$$

| 参数 | 物理量 | 单位 |
|------|--------|------|
| $\rho$ | 密度 | $\text{kg/m}^3$（或 $\text{g/cm}^3$） |
| $m$ | [[质量]] | $\text{kg}$（或 $\text{g}$） |
| $V$ | [[体积]] | $\text{m}^3$（或 $\text{cm}^3$） |

单位换算：$1\,\text{g/cm}^3 = 1 \times 10^3\,\text{kg/m}^3$。

### 2.2 密度的变形公式

$$m = \rho V$$

$$V = \frac{m}{\rho}$$

### 2.3 常见物质密度（常温常压）

| 物质 | 密度 $\rho$（$\text{kg/m}^3$） | 状态 |
|------|-------------------------------|------|
| 金 | $19.3 \times 10^3$ | 固体 |
| 铅 | $11.3 \times 10^3$ | 固体 |
| 银 | $10.5 \times 10^3$ | 固体 |
| 铜 | $8.9 \times 10^3$ | 固体 |
| 铁/钢 | $7.9 \times 10^3$ | 固体 |
| 铝 | $2.7 \times 10^3$ | 固体 |
| 冰 | $0.9 \times 10^3$ | 固体 |
| 水银 | $13.6 \times 10^3$ | 液体 |
| 纯水 | $1.0 \times 10^3$ | 液体 |
| 煤油/酒精 | $0.8 \times 10^3$ | 液体 |
| 汽油 | $0.71 \times 10^3$ | 液体 |
| 空气 | $1.29$ | 气体 |

## 3. 物理学意义与推论

### 3.1 密度是物质的特征量

密度反映了物质"致密程度"的物理本质。在相同条件下（温度、压强），同种物质的密度为常数，与物体的质量、体积大小无关。因此密度可用于鉴别物质：

- 金 vs 黄铜：金的密度 $19.3 \times 10^3\,\text{kg/m}^3$ 远大于黄铜（约 $8.5 \times 10^3\,\text{kg/m}^3$），可用于鉴别。
- 酒精 vs 纯水：酒精密度 $0.8 \times 10^3\,\text{kg/m}^3$ 小于水，可用密度计区分。

### 3.2 密度与温度的关系

物质的密度随温度变化：

- **一般固体和液体**：温度升高时体积膨胀，密度减小。但变化幅度通常很小。
- **水的反常膨胀**：水在 $4\,^{\circ}\text{C}$ 时密度最大（$\rho = 1.0 \times 10^3\,\text{kg/m}^3$）。$0\,^{\circ}\text{C}$ 到 $4\,^{\circ}\text{C}$ 之间，水升温时体积缩小、密度增大（反常）；$4\,^{\circ}\text{C}$ 以上恢复正常膨胀。这一反常性质使冬季湖水从表面先结冰，底层保持 $4\,^{\circ}\text{C}$，保护了水生生物的生存。
- **冰的密度**小于水（$\rho_{\text{冰}} = 0.9 \times 10^3 < \rho_{\text{水}} = 1.0 \times 10^3$），故冰浮于水面。
- **气体**：密度受温度和压强影响显著，温度升高或压强降低均使密度减小。

### 3.3 密度与物态的关系

同种物质在不同物态下密度不同：

| 物质 | 固态 $\rho$（$\text{kg/m}^3$） | 液态 $\rho$（$\text{kg/m}^3$） | 变化 |
|------|-------------------------------|-------------------------------|------|
| 水 | $0.9 \times 10^3$（冰） | $1.0 \times 10^3$（水） | 固态密度小于液态（反常） |
| 铁 | $7.9 \times 10^3$ | $7.0 \times 10^3$（铁水） | 固态密度大于液态（一般规律） |

### 3.4 浮沉条件的初步分析

根据物体密度 $\rho_{\text{物}}$ 与液体密度 $\rho_{\text{液}}$ 的关系，可判断物体浮沉：

| 条件 | 浮沉状态 | 原因 |
|------|---------|------|
| $\rho_{\text{物}} < \rho_{\text{液}}$ | 漂浮（浮在液面） | $F_{\text{浮}} > G$，最终漂浮时 $F_{\text{浮}} = G$ |
| $\rho_{\text{物}} = \rho_{\text{液}}$ | 悬浮（停在液体内部任意深度） | $F_{\text{浮}} = G$ |
| $\rho_{\text{物}} > \rho_{\text{液}}$ | 下沉（沉到容器底部） | $F_{\text{浮}} < G$ |

此部分的详细推导见[[浮力与阿基米德原理_Buoyancy and Archimedes' Principle]]。

### 3.5 适用边界与知识衔接

密度的概念在初中阶段以宏观测量和应用为主。在高中物理中，密度将联系到物质的微观结构：固体和液体分子排列紧密、密度大，气体分子间距大、密度小。密度的微观表达式为 $\rho = \frac{nm_0}{1}$，其中 $n$ 为分子数密度，$m_0$ 为分子质量。在大学物理中，密度进一步扩展为连续介质力学中的场量 $\rho(\vec{r}, t)$，并在相对论中引入"相对论质量密度"和"能量密度"概念（$E = mc^2$）。流体力学中密度是描述流体运动的基本参量之一，与[[压强_Pressure]]和[[浮力与阿基米德原理_Buoyancy and Archimedes' Principle]]密切相关。

## 4. 相关验证实验

### 4.1 固体密度测量（天平+量筒法）

1. 用托盘天平称量固体的质量 $m$。
2. 在量筒中倒入适量水，读出体积 $V_1$。
3. 用细线系住固体，缓缓浸入量筒水中（完全浸没），读出总体积 $V_2$。
4. 计算固体体积 $V = V_2 - V_1$。
5. 计算密度 $\rho = m/V$。

**注意事项**：固体不溶于水且不与水反应；量筒读数时视线与液面凹面最低处相平。

### 4.2 液体密度测量（天平+量筒法）

1. 用天平称量烧杯和液体的总质量 $m_1$。
2. 将部分液体倒入量筒，读出液体体积 $V$。
3. 称量剩余液体和烧杯的质量 $m_2$。
4. 倒出液体的质量 $m = m_1 - m_2$。
5. 计算密度 $\rho = m/V = (m_1 - m_2)/V$。

**此方法减小了量筒内壁残留液体带来的误差**（先测总质量，倒出后再测剩余质量，避免量筒残留影响）。

### 4.3 测不规则固体密度（漂浮法）

对于密度小于水的不规则固体（如石蜡、木块），可用"针压法"或"沉坠法"：
- **针压法**：用细针将漂浮固体压入水中使其完全浸没，读出体积变化即为固体体积。
- **沉坠法**：在固体下方系一重物（如铁块），先测重物浸没时的体积，再测固体和重物同时浸没的总体积，差值即为固体体积。

### 4.4 密度计原理演示

密度计（比重计）利用漂浮原理（$F_{\text{浮}} = G$）测量液体密度。将密度计放入不同密度液体中，浸入深度不同：液体密度越大，密度计露出越多。可在不同浓度的盐水中观察同一密度计的浸没深度变化。

## 5. 历史上下文

### 5.1 阿基米德与密度

[[阿基米德]]（Archimedes, 约公元前287—前212）是古希腊数学家和物理学家。据传，叙拉古国王希伦二世怀疑金匠在王冠中掺银，命[[阿基米德]]鉴定。阿基米德苦思冥想，在浴盆中洗澡时注意到身体浸入水中时水溢出，灵感顿生——通过排水法测体积、再与质量结合即可求密度，从而鉴别王冠成分。他兴奋地跳出浴盆裸奔喊道"Eureka!"（"我找到了！"）。这一故事虽可能是传说，但它准确反映了密度概念与排水法测体积的早期应用，并直接引出了[[浮力与阿基米德原理_Buoyancy and Archimedes' Principle]]。

### 5.2 密度概念的建立

密度作为明确的物理量定义，通常归功于[[阿基米德]]。他在《论浮体》中通过杠杆和浮力实验间接比较了不同物质的密度。中世纪伊斯兰学者[[阿布·拉伊汗·比鲁尼]]（Al-Biruni, 973—1048）发明了用流体静力秤精确测量固体和液体密度的方法，编制了多种物质的密度表。17世纪，[[伽利略]]改进了流体静力秤，使其成为精密测量密度的工具。

### 5.3 密度的微观理解

19世纪原子分子论确立后，密度的宏观量获得了微观解释：$\rho = n \cdot m_0$，其中 $n$ 为单位体积内的分子数，$m_0$ 为单个分子质量。[[阿伏伽德罗]]于1811年提出的分子假说为这一解释奠定了基础。20世纪以来，通过 X射线衍射实验测定晶体中原子的排列和间距，人们可从微观结构直接计算宏观密度，验证了宏观与微观的一致性。

### 5.4 密度在现代科学中的地位

密度不仅是日常物质鉴别的基础，更是天体物理学和宇宙学中的关键参数。白矮星密度可达 $10^9\,\text{kg/m}^3$，中子星密度更高达 $10^{17}\,\text{kg/m}^3$，这些极端密度只能用量子力学和广义相对论来理解。在地球科学中，通过地震波速度变化推断地球内部各层密度分布，是研究地球内部结构的基本方法。
