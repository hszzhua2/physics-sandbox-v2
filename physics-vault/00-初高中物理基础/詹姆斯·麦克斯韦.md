# 詹姆斯·麦克斯韦 (James Clerk Maxwell)

## 1. 生平
1831年生于爱丁堡，在爱丁堡大学与剑桥大学学习，任剑桥大学卡文迪什实验室首任教授。统一电磁理论，1879年逝于剑桥。

## 2. 主要贡献
- 统一[[麦克斯韦方程组]]电磁理论
- 提出[[光的直线传播]]的电磁波理论
- 发展[[分子动理论]]与统计物理

## 3. 关键公式与思想
- $\nabla\times\mathbf{E}=-\dfrac{\partial\mathbf{B}}{\partial t}$（法拉第定律）
- $c=1/\sqrt{\mu_0\epsilon_0}$（光速）

## 4. 历史影响
麦克斯韦的电磁统一是物理学第二次大统一，预言了[[光的直线传播]]本质为电磁波。
