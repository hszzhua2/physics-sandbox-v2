# 尼尔斯·玻尔 (Niels Bohr)

## 1. 生平
1885年生于哥本哈根，在哥本哈根大学学习并留校，任哥本哈根理论物理研究所所长。提出玻尔原子模型，1922年获诺贝尔物理学奖，1962年逝于哥本哈根。

## 2. 主要贡献
- 提出[[原子光谱]]的玻尔原子模型
- 发展[[原子光谱]]与量子跃迁
- 提出互补原理与哥本哈根诠释

## 3. 关键公式与思想
- $E_n=-\dfrac{13.6\,\text{eV}}{n^2}$（玻尔能级）
- $L=n\hbar$（角动量量子化）

## 4. 历史影响
玻尔的原子模型解释了[[原子光谱]]的线状结构，是[[原子结构模型]]的里程碑。
