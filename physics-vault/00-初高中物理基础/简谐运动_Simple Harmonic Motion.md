# 简谐运动 (Simple Harmonic Motion)

## 1. 核心定义

物体在回复力 $F = -kx$ 作用下所做的周期性运动，其位移随时间按余弦（或正弦）规律变化。是最基本、最简单的机械振动。

## 2. 数学表达

### 回复力

$$F = -kx$$

负号表示回复力方向始终与位移方向相反，指向平衡位置。

### 运动方程

由[[牛顿运动定律_Newton's Laws of Motion]]第二定律 $F = ma = m\ddot{x}$：

$$m\ddot{x} = -kx$$

$$\ddot{x} + \omega^2 x = 0$$

其中 $\omega = \sqrt{\frac{k}{m}}$ 为圆频率。

### 位移方程

$$x = A\cos(\omega t + \varphi)$$

### 速度与加速度

$$v = -A\omega\sin(\omega t + \varphi)$$

$$a = -A\omega^2\cos(\omega t + \varphi) = -\omega^2 x$$

**参数说明：**

| 符号 | 物理量 | 单位 | 说明 |
|------|--------|------|------|
| $A$ | 振幅 | $\text{m}$ | 最大位移，$A > 0$ |
| $\omega$ | 圆频率 | $\text{rad/s}$ | $\omega = \sqrt{k/m}$ |
| $t$ | 时间 | $\text{s}$ | |
| $\varphi$ | 初相 | $\text{rad}$ | $t=0$ 时的相位 |
| $\omega t + \varphi$ | 相位 | $\text{rad}$ | 描述振动状态 |

### 周期与频率

$$T = \frac{2\pi}{\omega} = 2\pi\sqrt{\frac{m}{k}}$$

$$f = \frac{1}{T} = \frac{\omega}{2\pi} = \frac{1}{2\pi}\sqrt{\frac{k}{m}}$$

### 能量

动能：

$$E_k = \frac{1}{2}mv^2 = \frac{1}{2}mA^2\omega^2\sin^2(\omega t + \varphi)$$

势能：

$$E_p = \frac{1}{2}kx^2 = \frac{1}{2}kA^2\cos^2(\omega t + \varphi)$$

总机械能（利用 $k = m\omega^2$）：

$$E = E_k + E_p = \frac{1}{2}kA^2 = \frac{1}{2}mA^2\omega^2$$

总能量守恒，动能与势能周期性交换。

### 单摆

小角度摆动（$\theta < 5^\circ$）时，单摆近似为简谐运动：

$$T = 2\pi\sqrt{\frac{L}{g}}$$

| 符号 | 物理量 | 单位 |
|------|--------|------|
| $L$ | 摆长 | $\text{m}$ |
| $g$ | 重力加速度 | $\text{m/s}^2$ |

## 3. 物理学意义与推论

### 物理意义

简谐运动是振动理论的基本模型。任何复杂的周期性振动都可以通过[[傅里叶分析_Fourier Analysis]]分解为一系列简谐运动的叠加（傅里叶级数），因此简谐运动被称为"振动的原子"。回复力 $F = -kx$ 是线性恢复力，它使物体在平衡位置附近做周期性往返运动。

### 简谐运动的特征

- **周期性**：运动每经过 $T$ 时间重复一次。
- **对称性**：平衡位置两侧运动对称，速度大小相等方向相反。
- **能量守恒**：只有保守力做功，机械能守恒。
- **相位**：相位 $(\omega t + \varphi)$ 完整描述了振动的瞬时状态。

### 各物理量在特殊位置的特征

| 位置 | 位移 $x$ | 速度 $v$ | 加速度 $a$ | 动能 $E_k$ | 势能 $E_p$ |
|------|----------|----------|------------|------------|------------|
| 平衡位置 | $0$ | 最大 $A\omega$ | $0$ | 最大 $\frac{1}{2}kA^2$ | $0$ |
| 最大位移 | $\pm A$ | $0$ | 最大 $\omega^2 A$ | $0$ | 最大 $\frac{1}{2}kA^2$ |

### 单摆的近似条件

单摆做简谐运动的条件是摆角 $\theta$ 很小（$\theta < 5^\circ$），此时 $\sin\theta \approx \theta$。当摆角较大时，单摆周期增大，需用椭圆积分求解，不再严格满足简谐运动规律。

### 适用边界

- 简谐运动要求回复力严格线性（$F \propto x$），实际弹簧在大形变时偏离线性。
- 忽略阻尼——实际振动总有能量损耗（阻尼振动）。
- 单摆近似要求小角度，大角度时为非线性振动。
- 简谐运动是一维振动，多维振动需要更一般的分析。

### 与大学物理的衔接

1. **阻尼振动与受迫振动**：实际振动总有阻尼，运动方程变为 $\ddot{x} + 2\beta\dot{x} + \omega_0^2 x = 0$。在周期性外力驱动下发生受迫振动，当驱动频率等于固有频率时发生**共振**——[[共振_Resonance]]现象在工程中极为重要。

2. **利萨茹图形**：两个相互垂直的简谐运动合成时，当频率比为简单整数比时形成稳定的闭合曲线，即[[利萨茹曲线_Lissajous Curve]]。这是示波器测量频率和相位差的原理。

3. **量子谐振子**：在[[量子力学_Quantum Mechanics]]中，[[量子谐振子_Quantum Harmonic Oscillator]]是最重要的精确可解模型之一。其能级为等间距的 $E_n = (n + \frac{1}{2})\hbar\omega$，零点能 $\frac{1}{2}\hbar\omega$ 是量子效应的体现。量子谐振子是量子场论中粒子产生和湮灭的基础模型。

4. **耦合振子与简正模**：多个弹簧振子耦合时，系统可分解为独立的简正模式（简正模），每个模式都是一个独立的简谐运动。这是[[固体物理学_Solid State Physics]]中晶格振动（声子）的基础。

5. 在[[拉格朗日力学_Lagrangian Mechanics]]中，简谐运动对应于势能在平衡点附近的二次展开 $V(x) \approx V_0 + \frac{1}{2}V''(x_0)(x-x_0)^2$，任何系统在稳定平衡点附近的小振动都近似为简谐运动。

## 4. 相关验证实验

### 弹簧振子实验

**实验器材**：气垫导轨、弹簧、滑块、光电门、电子天平。

**实验过程**：
1. 将滑块用弹簧连接在气垫导轨上，使滑块在水平方向做简谐运动。
2. 用光电门测量滑块经过平衡位置的速度（最大速度 $v_{\max} = A\omega$）。
3. 测量振幅 $A$ 和周期 $T$。
4. 验证 $T = 2\pi\sqrt{m/k}$：改变滑块质量 $m$ 或更换不同 $k$ 的弹簧，测量周期变化。
5. 验证能量守恒：$\frac{1}{2}mv_{\max}^2 = \frac{1}{2}kA^2$。

### 单摆实验

**实验器材**：细线、小球、铁架台、秒表、游标卡尺、米尺。

**实验过程**：
1. 测量摆长 $L$（悬点到球心的距离）和小球质量 $m$。
2. 使摆球在小角度（$\theta < 5^\circ$）下摆动。
3. 测量 $n$ 次全振动的时间 $t$，计算周期 $T = t/n$。
4. 验证 $T = 2\pi\sqrt{L/g}$，并利用该公式测量重力加速度 $g = \frac{4\pi^2 L}{T^2}$。
5. 改变摆长 $L$ 和小球质量 $m$，验证周期与质量无关、与 $\sqrt{L}$ 成正比。

**注意事项**：
- 摆角要小（$< 5^\circ$）。
- 摆球应在同一竖直面内摆动，不发生圆锥摆动。
- 计时应从平衡位置开始（速度最大处计时误差最小）。

### 演示能量转化的实验

在弹簧振子运动过程中，用传感器实时记录位移和速度，分别计算动能和势能，验证两者之和为常数。

## 5. 历史上下文

[[伽利略·伽利雷]]（Galileo Galilei, 1564–1642）是最早系统观察摆的等时性的科学家。据传年轻的伽利略在比萨大教堂中观察到吊灯的摆动，用自己的脉搏计时，发现摆的周期与振幅无关（小角度近似下），这就是单摆等时性原理。伽利略后来设计了摆钟的雏形。

[[克里斯蒂安·惠更斯]]（Christiaan Huygens, 1629–1695）在1656年制造了第一台实用的摆钟，并在1673年出版的《摆钟论》中给出了单摆周期的精确公式 $T = 2\pi\sqrt{L/g}$。惠更斯还发现大摆角下单摆不等时，并设计了等时摆（摆线摆）使周期与振幅严格无关。

[[罗伯特·胡克]]发现的[[胡克定律_Hooke's Law]] $F = -kx$ 为弹簧振子的简谐运动提供了理论基础。[[艾萨克·牛顿]]在《自然哲学的数学原理》中用力学原理分析了摆的运动。

19世纪，[[约瑟夫·傅里叶]]（Joseph Fourier）提出的[[傅里叶分析_Fourier Analysis]]表明，任何周期性振动都可以分解为简谐运动的叠加，确立了简谐运动在振动理论中的基础地位。[[利萨茹曲线_Lissajous Curve]]由[[朱尔·利萨茹]]（Jules Antoine Lissajous）在1857年系统研究，成为振动合成理论的重要内容。

20世纪，[[量子谐振子_Quantum Harmonic Oscillator]]的求解是[[量子力学_Quantum Mechanics]]发展的重要里程碑，简谐运动模型从经典力学延伸到了量子物理的最前沿。
