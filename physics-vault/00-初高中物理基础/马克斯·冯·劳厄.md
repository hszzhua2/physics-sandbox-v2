# 马克斯·冯·劳厄 (Max von Laue)

## 1. 生平
1879年生于普芬多夫，在斯特拉斯堡大学学习，任柏林大学与法兰克福大学教授。发现X射线衍射，1914年获诺贝尔物理学奖，1960年逝于柏林。

## 2. 主要贡献
- 发现X射线衍射研究[[物态变化]]
- 研究[[物态变化]]与晶体结构
- 发展[[物态变化]]与X射线结晶学

## 3. 关键公式与思想
- $n\lambda=2d\sin\theta$（布拉格-冯·劳厄衍射条件）

## 4. 历史影响
冯·劳厄的X射线衍射是[[物态变化]]与晶体结构分析的基础。
