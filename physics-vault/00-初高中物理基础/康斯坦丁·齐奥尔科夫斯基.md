# 康斯坦丁·齐奥尔科夫斯基 (Konstantin Tsiolkovsky)

## 1. 生平
1857年生于伊热夫斯科，自学成才，任卡卢加中学教师。提出火箭方程与航天理论，研究[[动量守恒定律]]，1935年逝于卡卢加。

## 2. 主要贡献
- 提出[[动量守恒定律]]的火箭方程
- 研究[[动量守恒定律]]与航天
- 发展[[动量守恒定律]]与宇宙航行

## 3. 关键公式与思想
- $\Delta v=v_e\ln(m_0/m_f)$（齐奥尔科夫斯基火箭方程）

## 4. 历史影响
齐奥尔科夫斯基的火箭方程是[[动量守恒定律]]在航天中的应用。
