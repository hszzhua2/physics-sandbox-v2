# 格奥尔格·欧姆 (Georg Ohm)

## 1. 生平
1789年生于埃尔朗根，在埃尔朗根大学学习，任慕尼黑大学教授。发现欧姆定律，1854年逝于慕尼黑。

## 2. 主要贡献
- 发现[[欧姆定律]]
- 研究[[电流与电路]]与电阻
- 发展[[欧姆定律]]与[[闭合电路欧姆定律]]

## 3. 关键公式与思想
- $V=IR$（欧姆定律）

## 4. 历史影响
欧姆定律是[[欧姆定律]]与[[电流与电路]]的基础定律。
