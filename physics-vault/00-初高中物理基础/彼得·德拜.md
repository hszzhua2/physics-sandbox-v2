# 彼得·德拜 (Peter Debye)

## 1. 生平
1884年生于马斯特里赫特，在亚琛工业大学与慕尼黑大学学习，任康奈尔大学教授。提出德拜模型与[[比热容]]理论，1936年获诺贝尔化学奖，1966年逝于伊萨卡。

## 2. 主要贡献
- 提出[[比热容]]的德拜模型
- 研究[[比热容]]与低温热容
- 发展[[比热容]]与固体物理

## 3. 关键公式与思想
- $C_V\propto T^3$（德拜低温热容$T^3$定律）

## 4. 历史影响
德拜模型是[[比热容]]与固体物理的重要理论。
