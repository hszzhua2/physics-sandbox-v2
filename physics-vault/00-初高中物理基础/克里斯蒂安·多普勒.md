# 克里斯蒂安·多普勒 (Christian Doppler)

## 1. 生平
1803年生于萨尔茨堡，在维也纳工业大学学习，任维也纳大学教授。提出多普勒效应，研究[[机械波]]，1853年逝于威尼斯。

## 2. 主要贡献
- 提出[[机械波]]的多普勒效应
- 研究[[机械波]]与波动
- 发展[[机械波]]与声学

## 3. 关键公式与思想
- $f'=f\dfrac{v\pm v_o}{v\mp v_s}$（多普勒效应）

## 4. 历史影响
多普勒效应是[[机械波]]与波动现象的重要定律。
