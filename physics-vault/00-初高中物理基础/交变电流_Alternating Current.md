# 交变电流 (Alternating Current)

## 1. 核心定义

**交变电流**（Alternating Current，简称 AC）是指大小和方向随时间周期性变化的电流。最常见的交变电流是**正弦交流电**（Sinusoidal AC），其电流和电压随时间按正弦（或余弦）函数规律变化。交变电流由[[尼古拉·特斯拉]]系统发展并推广，是现代电力系统的基础。

交变电流的产生基于[[电磁感应现象_Electromagnetic Induction]]：当矩形线圈在匀强磁场中以角速度 $\omega$ 匀速转动时，穿过线圈的磁通量随时间按余弦规律变化，根据法拉第电磁感应定律，线圈中产生正弦交变电动势。

与直流电相比，交变电流具有易于升降压（通过[[变压器]]）、便于远距离传输、发电机结构简单等优势，因此成为全球电力供应的标准形式。

## 2. 数学表达

### 2.1 瞬时值

正弦交变电流的电压和电流瞬时值分别为：

$$u(t) = U_m \sin(\omega t + \varphi_u)$$

$$i(t) = I_m \sin(\omega t + \varphi_i)$$

其中：
- $U_m$、$I_m$ 为电压和电流的峰值（最大值）
- $\omega = 2\pi f = \frac{2\pi}{T}$ 为角频率（单位：rad/s）
- $f$ 为频率（单位：Hz），中国市电频率为 $50 \, \text{Hz}$
- $T$ 为周期（单位：s）
- $\varphi_u$、$\varphi_i$ 为初相位

### 2.2 有效值（方均根值）

交变电流的**有效值**（RMS，Root Mean Square）定义为：让交变电流和直流电通过同样阻值的电阻，若在相同时间内产生的热量相等，则该直流电的值即为交变电流的有效值。

对于正弦交流电：

$$U = \frac{U_m}{\sqrt{2}} \approx 0.707 \, U_m$$

$$I = \frac{I_m}{\sqrt{2}} \approx 0.707 \, I_m$$

推导过程：

$$U = \sqrt{\frac{1}{T}\int_0^T u^2(t) \, dt} = \sqrt{\frac{1}{T}\int_0^T U_m^2 \sin^2(\omega t) \, dt} = \frac{U_m}{\sqrt{2}}$$

中国市电电压有效值为 $U = 220 \, \text{V}$，对应峰值 $U_m = 220\sqrt{2} \approx 311 \, \text{V}$。

### 2.3 平均值

正弦交流电在半个周期内的平均值为：

$$\bar{U} = \frac{2}{\pi} U_m \approx 0.637 \, U_m$$

$$\bar{I} = \frac{2}{\pi} I_m \approx 0.637 \, I_m$$

平均值在计算通过电路的电荷量时有用：$q = \bar{I} \cdot \Delta t$。

### 2.4 感抗与容抗

**感抗**（Inductive Reactance）：纯电感线圈对交流电的阻碍作用为：

$$X_L = \omega L = 2\pi f L$$

其中 $L$ 为自感系数（单位：H）。感抗随频率增大而增大，故电感具有"通直流、阻交流"和"通低频、阻高频"的特性。在纯电感电路中，电压超前电流 $\pi/2$。

**容抗**（Capacitive Reactance）：纯电容器对交流电的阻碍作用为：

$$X_C = \frac{1}{\omega C} = \frac{1}{2\pi f C}$$

其中 $C$ 为电容（单位：F）。容抗随频率增大而减小，故电容具有"通交流、隔直流"和"通高频、阻低频"的特性。在纯电容电路中，电流超前电压 $\pi/2$。

### 2.5 阻抗

在包含电阻 $R$、电感 $L$ 和电容 $C$ 的串联电路中，总阻抗为：

$$Z = \sqrt{R^2 + \left(X_L - X_C\right)^2}$$

电流有效值为：

$$I = \frac{U}{Z} = \frac{U}{\sqrt{R^2 + \left(\omega L - \frac{1}{\omega C}\right)^2}}$$

电压与电流的相位差为：

$$\varphi = \arctan\frac{X_L - X_C}{R}$$

### 2.6 交流电功率

**有功功率**（Active Power）：电路实际消耗的功率：

$$P = UI\cos\varphi$$

其中 $\cos\varphi$ 为**功率因数**（Power Factor），$\varphi$ 为电压与电流的相位差。

**视在功率**（Apparent Power）：

$$S = UI$$

**无功功率**（Reactive Power）：

$$Q = UI\sin\varphi$$

三者关系为：

$$S^2 = P^2 + Q^2$$

对于纯电阻电路 $\cos\varphi = 1$，$P = UI$；对于纯电感或纯电容电路 $\cos\varphi = 0$，$P = 0$（不消耗能量，仅与电源交换能量）。

## 3. 物理学意义与推论

### 3.1 有效值的物理意义

有效值是描述交流电热效应的等效量。日常生活中所说的交流电压"220 V"指的是有效值。交流电表测量的也是有效值。有效值的平方平均定义 $U_{\text{rms}} = \sqrt{\langle u^2 \rangle}$ 适用于任意波形的周期性电流，不仅限于正弦波。

### 3.2 功率因数与电能传输

功率因数 $\cos\varphi$ 反映了电能利用效率。当 $\cos\varphi < 1$ 时，虽然用户实际消耗的有功功率 $P = UI\cos\varphi$ 较小，但供电线路中仍需传输大小为 $I$ 的电流，导致线路损耗 $I^2R_{\text{线}}$ 增加。因此，电力部门要求用户提高功率因数（通常通过并联电容器补偿感性负载），以减少无功电流传输，降低线损。

### 3.3 频率特性与滤波

感抗 $X_L = \omega L$ 和容抗 $X_C = 1/(\omega C)$ 的频率依赖性是交流电路滤波功能的基础。电感通低频、阻高频；电容通高频、阻低频。利用这一特性可构建低通滤波器、高通滤波器、带通滤波器等，在电子工程中应用广泛。

### 3.4 与大学物理的衔接

交变电流在大学物理中用**相量法**（Phasor Method）和**复阻抗**进行系统分析：

- **复阻抗**：$Z = R + j\left(\omega L - \frac{1}{\omega C}\right)$，将电阻、感抗、容抗统一在复数框架中，大大简化交流电路的计算。
- 与[[RLC谐振_RLC Resonance]]的衔接：当 $X_L = X_C$ 时，即 $\omega L = 1/(\omega C)$，电路发生串联谐振，此时 $Z = R$ 最小，电流最大，谐振频率为 $\omega_0 = 1/\sqrt{LC}$。谐振电路在无线电调谐、振荡器、滤波器中有核心应用。
- **交流电的相量表示**：$u(t) = \text{Re}[\tilde{U} e^{j\omega t}]$，其中 $\tilde{U} = U_m e^{j\varphi}$ 为电压相量，将微分方程转化为代数方程。
- **三相交流电**：大学电工学中进一步研究三相交流电的星形和三角形接法，这是实际电力系统的标准配置。
- **傅里叶分析**：任意周期性信号可分解为一系列正弦波的叠加（傅里叶级数），正弦交流电的重要性正在于此。

## 4. 相关验证实验

### 4.1 交流电的产生

**实验装置**：矩形线圈在蹄形磁铁（或电磁铁）产生的匀强磁场中绕垂直于磁场方向的轴匀速转动，线圈两端通过滑环和电刷连接到示波器。

**现象**：示波器显示正弦波形，转动越快（角速度 $\omega$ 越大），频率越高，峰值不变（若磁场和线圈面积不变）。

### 4.2 有效值验证

**实验**：将直流电和交流电分别通过相同阻值的电阻，调节直流电压使两者在相同时间内产生的热量相同（可通过温度计测量水温升高），则该直流电压值即为交流电的有效值。验证 $U = U_m / \sqrt{2}$。

### 4.3 感抗和容抗的频率特性

**实验**：将电感（或电容）与交流电流表串联，接入信号发生器。改变信号频率，观察电流表示数变化：
- 电感电路：频率升高，电流减小（感抗增大）。
- 电容电路：频率升高，电流增大（容抗减小）。

### 4.4 示波器观察交流电波形

使用示波器观察市电或信号发生器输出的交流波形，测量峰值、周期、频率，验证 $U_m = \sqrt{2} U$ 和 $f = 1/T$。

## 5. 历史上下文

19世纪末，电力工业面临一个重大抉择：采用直流电还是交流电。[[托马斯·爱迪生]]大力推广直流电系统，而[[尼古拉·特斯拉]]和[[乔治·威斯汀豪斯]]则主张交流电系统。

1888年，特斯拉获得了多相交流电系统的专利，包括交流发电机、变压器和电动机。交流电的核心优势在于可以通过变压器方便地升压和降压：高压传输可大幅减少线路损耗，低压使用则安全便捷。而直流电在当时难以高效变压。

1893年，威斯汀豪斯公司赢得了尼亚加拉瀑布水电站项目，采用特斯拉设计的交流电系统，标志着"电流之战"（War of Currents）以交流电的胜利告终。此后，交流电成为全球电力传输的标准，直至今日。

特斯拉不仅发明了交流发电机和电动机，还设计了多相交流电系统，为现代电力工业奠定了全部技术基础。为了纪念他的贡献，国际单位制中磁感应强度的单位"特斯拉"（T）以他的名字命名。
