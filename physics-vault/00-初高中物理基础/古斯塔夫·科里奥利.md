# 古斯塔夫·科里奥利 (Gaspard-Gustave Coriolis)

## 1. 生平
1792年生于巴黎，在巴黎综合理工学院学习并留校任教。提出科里奥利力与功的概念，1843年逝于巴黎。

## 2. 主要贡献
- 提出[[功和机械能]]的科里奥利力
- 定义[[功和机械能]]中功的概念
- 研究[[动能定理]]与旋转参照系

## 3. 关键公式与思想
- $\mathbf{F}_c=-2m\boldsymbol{\omega}\times\mathbf{v}$（科里奥利力）

## 4. 历史影响
科里奥利力是旋转系统中[[功和机械能]]与[[动能定理]]的重要概念。
