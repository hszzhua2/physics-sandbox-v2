# 平抛运动 (Projectile Motion)

## 1. 核心定义

物体以一定初速度水平抛出后，仅受重力作用的曲线运动。可分解为水平方向的匀速直线运动和竖直方向的自由落体运动。

## 2. 数学表达

### 运动分解

以抛出点为坐标原点，水平方向为 $x$ 轴，竖直向下为 $y$ 轴，初速度 $v_0$ 沿 $x$ 方向。

**水平方向**（匀速直线运动）：

$$v_x = v_0$$

$$x = v_0 t$$

**竖直方向**（自由落体运动）：

$$v_y = gt$$

$$y = \frac{1}{2}gt^2$$

### 合速度与合位移

合速度大小：

$$v = \sqrt{v_x^2 + v_y^2} = \sqrt{v_0^2 + g^2t^2}$$

合速度方向（与水平方向的夹角）：

$$\tan\theta = \frac{v_y}{v_x} = \frac{gt}{v_0}$$

合位移大小：

$$s = \sqrt{x^2 + y^2} = \sqrt{(v_0 t)^2 + \left(\frac{1}{2}gt^2\right)^2}$$

### 轨迹方程

由 $x = v_0 t$ 得 $t = \frac{x}{v_0}$，代入 $y = \frac{1}{2}gt^2$：

$$y = \frac{g}{2v_0^2}x^2$$

该轨迹为一条**抛物线**（开口向下，此处 $y$ 轴向下为正），顶点在原点。

**参数说明：**

| 符号 | 物理量 | 单位 |
|------|--------|------|
| $v_0$ | 初速度（水平方向） | $\text{m/s}$ |
| $v_x$ | 水平分速度 | $\text{m/s}$ |
| $v_y$ | 竖直分速度 | $\text{m/s}$ |
| $x$ | 水平位移 | $\text{m}$ |
| $y$ | 竖直位移 | $\text{m}$ |
| $t$ | 运动时间 | $\text{s}$ |
| $\theta$ | 速度与水平方向夹角 | $\text{rad}$ |

### 飞行时间与射程

从高度 $h$ 处水平抛出：

飞行时间（由竖直方向决定）：

$$t = \sqrt{\frac{2h}{g}}$$

水平射程：

$$R = v_0 \sqrt{\frac{2h}{g}}$$

### 重要推论

任意时刻速度方向的反向延长线与水平位移的交点，平分水平位移。即速度方向反向延长线交 $x$ 轴于 $\frac{x}{2}$ 处。

证明：速度方向所在直线方程为 $y - y_0 = \frac{v_y}{v_x}(x - x_0)$，其中 $(x_0, y_0) = (v_0 t, \frac{1}{2}gt^2)$，令 $y = 0$，解得 $x = \frac{x_0}{2}$。

## 3. 物理学意义与推论

### 物理意义

平抛运动是[[运动的合成与分解]]的经典范例，体现了运动的**独立性原理**：水平方向和竖直方向的运动互不影响，各自独立遵循相应的运动规律。这一原理是处理复杂运动的基本方法。

### 适用边界

- 仅适用于初速度**水平**且仅受**重力**的情形（忽略空气阻力）。
- 若初速度有竖直分量（斜上抛或斜下抛），运动仍可分解为水平和竖直方向，但竖直方向初速度不为零。
- 在高度不大、速度不高时，$g$ 视为常数；高速度或大尺度时需考虑空气阻力和 $g$ 的变化。

### 与大学物理的衔接

在大学物理中，平抛运动的研究扩展到以下方向：

1. **斜抛运动的一般理论**：抛射角 $\alpha$ 任意时，最大射程公式 $R_{\max} = \frac{v_0^2}{g}$（$\alpha = 45^\circ$），抛物线轨道方程的完整推导。
2. **空气阻力的影响**：引入与速度相关的阻力项后，轨道不再是抛物线，需用数值方法求解微分方程。
3. **科里奥利力效应**：在大尺度抛射体问题中（如远程炮弹），需考虑地球自转引起的[[科里奥利力_Coriolis Force]]修正。
4. 在[[拉格朗日力学_Lagrangian Mechanics]]中，平抛运动可由拉格朗日量 $L = \frac{1}{2}m(\dot{x}^2 + \dot{y}^2) - mgy$ 导出，是分析力学中最简单的二维运动实例之一。

## 4. 相关验证实验

### 平抛运动演示实验

**实验器材**：平抛运动实验仪（含两个相同的小球、电磁铁、碰撞开关、水平轨道）、白纸、复写纸。

**实验过程**：
1. 将一个小球置于水平轨道末端，另一个小球用电磁铁吸在同一高度的正上方。
2. 切断电磁铁电源的同时，水平轨道上的小球被弹出做平抛运动，上方小球开始自由下落。
3. 两球在空中**相碰**，证明平抛运动在竖直方向的分运动是自由落体运动。

### 频闪照相法

对平抛运动的小球进行频闪照相，照片上同时记录等时间间隔的小球位置。将各位置的水平间距和竖直间距分别测量，验证：
- 水平方向：等间距（匀速运动）
- 竖直方向：间距成等差数列，公差为 $gT^2$（匀加速运动）

### 用描迹法研究轨迹

在竖直板上固定白纸和复写纸，使小球从斜槽上同一位置滚下做平抛运动，在白纸上描出轨迹，验证其形状为抛物线 $y = \frac{g}{2v_0^2}x^2$。

## 5. 历史上下文

[[伽利略·伽利雷]]是第一个正确分析抛体运动的科学家。在《两门新科学》（1638年）中，伽利略运用[[运动的合成与分解]]思想，将抛体运动分解为水平方向的惯性运动和竖直方向的加速运动，推导出抛物线轨迹。

伽利略的这一成就在当时具有革命性意义。此前，人们受[[亚里士多德]]物理学的影响，认为抛体在离开抛掷者后会沿直线运动一段距离，然后竖直落下。伽利略的分析表明，水平运动和竖直运动是同时进行的、独立的，合运动的轨迹是一条光滑的抛物线。

这一工作直接启发了[[艾萨克·牛顿]]的力学体系。在[[牛顿运动定律_Newton's Laws of Motion]]框架下，平抛运动被理解为初速度水平、合外力恒定（等于重力）的匀变速曲线运动。当初速度足够大时，抛体不再落回地面而成为绕地球运行的卫星——这正是[[牛顿万有引力定律_Newton's Law of Universal Gravitation]]中"牛顿大炮"思想实验的核心。
