# 伽利略·伽利雷 (Galileo Galilei)

## 1. 生平
1564年生于比萨，在比萨大学学习，任帕多瓦大学教授。开创实验物理与望远镜天文观测，奠基运动学，1642年逝于阿切特里。

## 2. 主要贡献
- 建立[[匀变速直线运动]]与自由落体定律
- 发明望远镜并观测木星卫星
- 研究[[共点力平衡]]与惯性质理

## 3. 关键公式与思想
- $s=\dfrac{1}{2}at^2$（匀加速运动）
- $\text{inertia: object resists change in motion}$（惯性原理）

## 4. 历史影响
伽利略的实验方法开创了近代物理学，奠基了[[匀变速直线运动]]与[[共点力平衡]]。
