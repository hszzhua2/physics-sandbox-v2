# 闭合电路欧姆定律 (Ohm's Law for Complete Circuit)

## 1. 核心定义

**闭合电路欧姆定律**（Ohm's Law for Complete Circuit）是描述包含电源的完整闭合电路中电流、电动势与总电阻之间关系的基本定律。它指出，闭合电路中的电流 $I$ 与电源[[电动势]] $\varepsilon$ 成正比，与电路的总电阻（外电阻 $R$ 与[[内阻]] $r$ 之和）成反比。

在一个完整的闭合电路中，电源不仅提供电动势 $\varepsilon$ 驱动电荷定向移动，本身还具有内阻 $r$。外电路的电阻 $R$ 称为负载电阻或外电阻。电流流经内阻时会产生内电压降 $Ir$，导致电源两端的实际输出电压（路端电压）小于电动势。

该定律是[[欧姆定律_Ohm's Law]]从部分电路到完整电路的自然推广，由[[古斯塔夫·基尔霍夫]]在完善电路理论时系统阐述，是分析一切直流电路的基石。

## 2. 数学表达

### 2.1 电流公式

闭合电路中的电流为：

$$I = \frac{\varepsilon}{R + r}$$

其中：
- $\varepsilon$ 为电源电动势（单位：V）
- $R$ 为外电路总电阻（单位：$\Omega$）
- $r$ 为电源内阻（单位：$\Omega$）
- $I$ 为电路电流（单位：A）

### 2.2 路端电压

电源两端的输出电压（路端电压）为：

$$U = \varepsilon - Ir$$

将电流代入可得：

$$U = \varepsilon - \frac{\varepsilon \, r}{R + r} = \frac{\varepsilon \, R}{R + r}$$

由此可见，路端电压 $U$ 随外电阻 $R$ 的增大而增大：
- 当 $R \to \infty$（断路）时，$U \to \varepsilon$，即断路时路端电压等于电动势。
- 当 $R \to 0$（短路）时，$U \to 0$，此时短路电流 $I_{\text{短路}} = \varepsilon / r$，可能极大，具有危险性。

### 2.3 电源功率关系

电源的总功率（消耗的化学能转化为电能的功率）为：

$$P_{\text{总}} = \varepsilon \, I = \frac{\varepsilon^2}{R + r}$$

电源内阻消耗的功率（热损耗）为：

$$P_{\text{内}} = I^2 r$$

电源对外电路输出的功率为：

$$P_{\text{out}} = I^2 R = \frac{\varepsilon^2 R}{(R + r)^2}$$

三者满足能量守恒：

$$P_{\text{总}} = P_{\text{out}} + P_{\text{内}}$$

### 2.4 最大功率传输条件

对 $P_{\text{out}}$ 关于 $R$ 求导并令其为零：

$$\frac{dP_{\text{out}}}{dR} = \varepsilon^2 \cdot \frac{(R+r)^2 - 2R(R+r)}{(R+r)^4} = 0$$

解得：

$$\boxed{R = r}$$

即当外电阻等于内阻时，电源输出功率达到最大值：

$$P_{\max} = \frac{\varepsilon^2}{4r}$$

此为**最大功率匹配条件**。

### 2.5 电源效率

电源效率定义为输出功率与总功率之比：

$$\eta = \frac{P_{\text{out}}}{P_{\text{总}}} = \frac{I^2 R}{\varepsilon \, I} = \frac{R}{R + r}$$

当 $R = r$ 时效率仅为 $\eta = 50\%$，说明最大功率传输与高效率不可兼得。

## 3. 物理学意义与推论

### 3.1 内阻的物理本质

电源内阻 $r$ 反映了电源内部电荷迁移过程中受到的阻碍作用。在化学电池中，内阻来源于电解液的离子导电阻力及电极极化效应；在发电机中，内阻来源于线圈导线的欧姆电阻。内阻的存在使得电源不可能将全部电能输送给外电路。

### 3.2 路端电压与负载的关系

路端电压 $U = \varepsilon - Ir$ 揭示了一个重要事实：电源输出电压并非恒定，而是随负载电流增大而线性下降。这一特性在实际电路设计中至关重要。例如，在电力传输中需考虑电源的输出阻抗对电压稳定性的影响；在电子仪器中，信号源的内阻会影响测量精度。

### 3.3 功率匹配的工程意义

最大功率传输条件 $R = r$ 在电子工程中有广泛应用，如音频放大器的阻抗匹配、射频电路的天线匹配等。但在电力传输系统中，追求的是高效率而非最大功率传输，因此输电线路的等效负载电阻远大于电源内阻。

### 3.4 与大学物理的衔接

闭合电路欧姆定律是大学物理中[[基尔霍夫定律]]（Kirchhoff's Laws）的特殊情形：

- **基尔霍夫电压定律（KVL）**：沿任意闭合回路，电动势代数和等于电压降代数和，即 $\sum \varepsilon = \sum IR$，这是闭合电路欧姆定律在多回路网络中的推广。
- **戴维南定理**（Thévenin's Theorem）：任何含源线性二端网络可等效为一个电压源 $\varepsilon_{\text{eq}}$ 串联一个内阻 $R_{\text{eq}}$，直接对应本节中 $\varepsilon$ 和 $r$ 的概念。
- **诺顿定理**（Norton's Theorem）：等效为电流源并联内阻，是闭合电路欧姆定律的对偶形式。
- 与[[RLC谐振_RLC Resonance]]的衔接：在交流电路中，闭合电路欧姆定律推广为交流欧姆定律 $I = \varepsilon / Z$，其中 $Z$ 为复阻抗，包含电阻、感抗和容抗的共同贡献，是分析 RLC 谐振电路的基础。

## 4. 相关验证实验

### 4.1 伏安法测电源电动势和内阻

**实验原理**：改变外电阻 $R$，测量多组路端电压 $U$ 和电流 $I$。由 $U = \varepsilon - Ir$ 可知 $U$-$I$ 图像为一条直线，纵截距为 $\varepsilon$，斜率绝对值为 $r$。

**实验器材**：待测电源（干电池）、电压表、电流表、滑动变阻器、开关、导线。

**电路设计**：采用电流表外接法（相对于电源），电压表并联在电源两端测量路端电压，电流表测量干路电流。

**数据处理**：作 $U$-$I$ 图像，通过线性拟合求得 $\varepsilon$ 和 $r$。

**注意事项**：
- 实验中电流不宜过大，以免电源极化导致内阻变化。
- 测量应快速完成，避免长时间通电导致电动势下降。
- 电压表内阻非无穷大，会引入系统误差，使测得的 $\varepsilon_{\text{测}} < \varepsilon_{\text{真}}$，$r_{\text{测}} < r_{\text{真}}$。

### 4.2 验证最大功率传输条件

通过改变外电阻 $R$，测量不同 $R$ 值下的输出功率 $P = I^2 R$，绘制 $P$-$R$ 曲线，验证当 $R = r$ 时输出功率最大。

## 5. 历史上下文

[[乔治·西蒙·欧姆]]于1827年在《电路的数学研究》（*Die galvanische Kette, mathematisch bearbeitet*）中发表了[[欧姆定律_Ohm's Law]]，最初仅描述部分电路中电流与电压、电阻的关系。[[古斯塔夫·基尔霍夫]]于1845年提出基尔霍夫电路定律，将欧姆定律推广到包含电源的完整闭合电路和复杂网络，奠定了电路分析的理论基础。

闭合电路欧姆定律的提出使得工程师能够精确计算含源电路中的电流分布和功率传输，为后来的电力工程和电子学的发展提供了理论工具。最大功率传输条件的发现则深刻影响了通信工程和射频电子学的发展，在阻抗匹配理论中至今仍发挥核心作用。
