# 马克斯·普朗克 (Max Planck)

## 1. 生平
1858年生于基尔，在慕尼黑大学与柏林大学学习，任柏林大学教授。提出能量量子化假说，1900年开创量子论，1918年获诺贝尔物理学奖，1947年逝于格廷根。

## 2. 主要贡献
- 提出[[原子结构模型]]的能量量子化假说
- 研究[[波粒二象性]]与黑体辐射
- 发展[[热传递]]与热力学

## 3. 关键公式与思想
- $E=h\nu$（普朗克能量量子化）
- $\rho(\nu)=\dfrac{8\pi h\nu^3}{c^3}\dfrac{1}{e^{h\nu/kT}-1}$（普朗克黑体辐射）

## 4. 历史影响
普朗克的量子假说开创了量子物理学，是[[波粒二象性]]与[[原子结构模型]]的基础。
