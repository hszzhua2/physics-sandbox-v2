# 玻尔模型 (Bohr Model)

## 1. 核心定义

**玻尔模型**（Bohr Model）是[[尼尔斯·玻尔]]于1913年提出的氢原子结构理论。它在[[原子结构模型_Atomic Structure Models]]的卢瑟福核式结构基础上引入了[[量子化]]条件，成功解释了氢原子光谱的线状结构和[[原子光谱]]的实验规律。

玻尔模型是物理学史上第一个将量子概念成功应用于原子结构的理论，是经典物理向量子力学过渡的关键桥梁。虽然玻尔模型在数学上是半经典半量子的混合体（保留了电子的轨道概念，但人为引入了量子化条件），但它对氢原子和类氢离子的光谱给出了精确的定量解释，奠定了原子物理学的理论基础。

玻尔模型的核心是三个基本假设：**轨道量子化**、**定态假设**和**频率条件（跃迁假设）**。

## 2. 数学表达

### 2.1 玻尔三假设

**假设一：轨道量子化**

电子只能在满足特定条件的轨道上运动，这些轨道称为**定态轨道**。定态轨道的角动量必须满足量子化条件：

$$L = m_e v r = n\hbar = n\frac{h}{2\pi}, \quad n = 1, 2, 3, \ldots$$

其中 $n$ 为主量子数，$\hbar = h/(2\pi)$ 为约化普朗克常数。

**假设二：定态假设**

电子在定态轨道上运动时不辐射电磁波，原子处于稳定状态。每个定态对应一个确定的能量值 $E_n$。

**假设三：跃迁假设（频率条件）**

电子从一个定态 $E_i$ 跃迁到另一个定态 $E_f$ 时，发射（$E_i > E_f$）或吸收（$E_i < E_f$）一个光子，光子频率满足：

$$h\nu = |E_i - E_f|$$

### 2.2 氢原子轨道半径

由库仑力提供向心力（圆轨道）：

$$\frac{ke^2}{r^2} = \frac{m_e v^2}{r}$$

结合角动量量子化 $m_e v r = n\hbar$，消去 $v$：

$$r_n = \frac{n^2 \hbar^2}{m_e ke^2} = n^2 a_0$$

其中 $a_0$ 为**玻尔半径**：

$$a_0 = \frac{\hbar^2}{m_e ke^2} = \frac{4\pi\varepsilon_0 \hbar^2}{m_e e^2} \approx 0.529 \times 10^{-10} \, \text{m} = 0.529 \, \text{\AA}$$

$n = 1$ 对应基态轨道半径 $r_1 = a_0$，即玻尔半径。

### 2.3 氢原子能级

电子的总能量为动能与势能之和：

$$E = \frac{1}{2}m_e v^2 - \frac{ke^2}{r}$$

由 $\frac{ke^2}{r^2} = \frac{m_e v^2}{r}$ 得 $m_e v^2 = \frac{ke^2}{r}$，故：

$$E_n = \frac{ke^2}{2r_n} - \frac{ke^2}{r_n} = -\frac{ke^2}{2r_n}$$

代入 $r_n = n^2 a_0$：

$$\boxed{E_n = -\frac{13.6}{n^2} \, \text{eV}, \quad n = 1, 2, 3, \ldots}$$

- 基态（$n = 1$）：$E_1 = -13.6 \, \text{eV}$
- 第一激发态（$n = 2$）：$E_2 = -3.4 \, \text{eV}$
- 电离能（基态到 $n \to \infty$）：$\Delta E = 13.6 \, \text{eV}$

### 2.4 类氢离子推广

对于核电荷数为 $Z$ 的类氢离子（如 He⁺, Li²⁺），轨道半径和能级为：

$$r_n = \frac{n^2 a_0}{Z}$$

$$E_n = -\frac{13.6 \, Z^2}{n^2} \, \text{eV}$$

### 2.5 氢原子光谱系

由跃迁假设 $h\nu = E_i - E_f$ 和 $\nu = c/\lambda$，可得光谱线波长：

$$\frac{1}{\lambda} = R\left(\frac{1}{n_f^2} - \frac{1}{n_i^2}\right), \quad n_i > n_f$$

其中 $R = \frac{m_e e^4}{8\varepsilon_0^2 h^3 c} \approx 1.097 \times 10^7 \, \text{m}^{-1}$ 为里德伯常数。

| 光谱系 | $n_f$ | $n_i$ | 光谱区域 | 发现者 |
|--------|-------|-------|---------|--------|
| 莱曼系 | 1 | 2, 3, 4, ... | 紫外 | 莱曼 (1906) |
| 巴尔末系 | 2 | 3, 4, 5, ... | 可见光 | [[约翰·巴尔末]] (1885) |
| 帕邢系 | 3 | 4, 5, 6, ... | 红外 | 帕邢 (1908) |
| 布拉开系 | 4 | 5, 6, 7, ... | 远红外 | 布拉开 (1922) |
| 普丰德系 | 5 | 6, 7, 8, ... | 远红外 | 普丰德 (1924) |

玻尔模型从理论上推导出了[[约翰·巴尔末]]凭经验发现的公式，并预言了当时尚未观测到的莱曼系和帕邢系，后均被实验证实。

## 3. 物理学意义与推论

### 3.1 量子化概念的引入

玻尔模型最重要的贡献是将[[量子化]]概念引入原子结构。角动量量子化条件 $L = n\hbar$ 意味着电子不能在任意轨道上运动，只能在分立的轨道上运行。这一"量子化"思想虽然最初是人为假设，但后来被证明是[[波粒二象性_Wave-Particle Duality]]的自然推论——电子的德布罗意波在轨道上形成驻波时，自然要求 $2\pi r = n\lambda$，即 $m_evr = n\hbar$。

### 3.2 原子稳定性的解释

玻尔模型的定态假设解决了卢瑟福模型的根本困难：电子在定态轨道上不辐射电磁波，原子是稳定的。虽然这一假设在经典框架内不可理解，但它成功解释了实验事实，并推动了量子力学的发展。

### 3.3 玻尔模型的局限性

玻尔模型虽然成功解释了氢原子光谱，但存在严重局限：

1. **多电子原子失效**：无法解释氦原子及更复杂原子的光谱。
2. **精细结构无法解释**：不能解释光谱线的精细结构（需要引入电子[[自旋_Spin]]和相对论修正）。
3. **谱线强度问题**：无法计算光谱线的相对强度和跃迁概率。
4. **塞曼效应**：无法解释磁场中光谱线的分裂现象。
5. **本质上不自洽**：混合了经典力学（轨道运动）和量子概念（量子化条件），缺乏统一的理论基础。

### 3.4 与大学物理的衔接

玻尔模型在大学物理中作为量子力学的入门被学习，其局限由量子力学克服：

- 与[[薛定谔方程_Schrödinger Equation]]的衔接：量子力学中，电子不再有确定的圆轨道，而是由波函数 $\Psi_{nlm}$ 描述的概率云。氢原子薛定谔方程的严格解自然给出量子数 $n$（主量子数）、$l$（角量子数）、$m$（磁量子数），能量 $E_n = -13.6/n^2 \, \text{eV}$ 与玻尔模型一致，但轨道概念被概率分布取代。
- 与[[自旋_Spin]]的衔接：电子自旋 $s = 1/2$ 是玻尔模型完全未涉及的量子自由度，引入自旋后可解释光谱精细结构、塞曼效应和元素周期律。
- **对应原理**：玻尔提出的对应原理——量子理论在大量子数极限下应回到经典理论——成为量子力学建立的重要指导原则。
- **量子力学修正**：狄拉克的相对论量子力学进一步给出氢原子能级的精细结构公式，量子电动力学（QED）解释了兰姆移位，这些远超玻尔模型的能力范围。

## 4. 相关验证实验

### 4.1 氢原子光谱实验

**实验**：用光谱仪观测氢气放电管的发射光谱，测量巴尔末系各谱线的波长。

**验证**：实验测得的波长与玻尔模型公式 $1/\lambda = R(1/n_f^2 - 1/n_i^2)$ 精确吻合，里德伯常数的理论值与实验值误差小于 $0.01\%$。

### 4.2 弗兰克-赫兹实验（1914年）

**实验**：电子轰击汞原子蒸气，测量电子能量损失。

**结果**：原子只在特定能量（$4.9 \, \text{eV}$ 的整数倍）处吸收能量，证明原子能级是量子化的，直接验证了玻尔的定态假设和跃迁假设。[[詹姆斯·弗兰克]]和[[古斯塔夫·赫兹]]获1925年诺贝尔物理学奖。

### 4.3 电离能测量

**实验**：测量氢原子从基态电离所需的最小能量。

**验证**：实验测得氢原子电离能为 $13.6 \, \text{eV}$，与玻尔模型 $E_1 = -13.6 \, \text{eV}$ 的预言完全一致。

## 5. 历史上下文

1911年，[[欧内斯特·卢瑟福]]提出原子核式结构模型后，面临电子绕核辐射坠落的根本困难。1912年，年轻的丹麦物理学家[[尼尔斯·玻尔]]来到曼彻斯特卢瑟福实验室工作，接触到了原子结构问题。

玻尔了解到[[约翰·巴尔末]]于1885年发现的氢光谱经验公式和[[原子光谱]]的实验规律后，意识到光谱线的分立性与[[量子化]]概念之间可能存在深层联系。1913年，玻尔在《哲学杂志》上分三部分发表了题为《论原子和分子的构成》的论文，提出了后来被称为"玻尔模型"的氢原子理论。

玻尔模型的成功令人震惊：它不仅精确解释了已知的巴尔末系，还预言了莱曼系（紫外）和帕邢系（红外）的存在，这些预言很快被实验证实。1922年，玻尔因"研究原子结构及原子辐射"获得诺贝尔物理学奖。

然而，玻尔本人深知模型的局限性。他提出的**对应原理**（Correspondence Principle）——量子理论在大量子数极限下应回到经典理论——为后来量子力学的建立提供了重要指导。1925-1926年，[[维尔纳·海森堡]]的矩阵力学和[[埃尔温·薛定谔]]的波动力学相继建立，量子力学最终取代了玻尔模型，给出了原子结构的完整描述。但玻尔模型作为量子力学的先驱，其历史地位和教学价值不可替代。
