# 肯尼思·威尔逊 (Kenneth Wilson)

## 1. 生平
1936年生于沃尔本，在哈佛大学与加州理工学院学习，任康奈尔大学与俄亥俄州立大学教授。提出重正化群与[[物态变化]]相变理论，1982年获诺贝尔物理学奖，2013年逝于锡伯克。

## 2. 主要贡献
- 提出[[物态变化]]的重正化群理论
- 研究[[物态变化]]与临界现象
- 发展[[物态变化]]与相变

## 3. 关键公式与思想
- $\xi\sim|T-T_c|^{-\nu}$（关联长度临界指数）

## 4. 历史影响
威尔逊的重正化群是[[物态变化]]与相变理论的核心。
