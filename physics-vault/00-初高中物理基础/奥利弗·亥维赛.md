# 奥利弗·亥维赛 (Oliver Heaviside)

## 1. 生平
1850年生于伦敦，自学成才，任英国电报工程师。简化麦克斯韦方程组，研究[[力的合成与分解]]，1925年逝于托基。

## 2. 主要贡献
- 简化[[麦克斯韦方程组]]为现代形式
- 研究[[力的合成与分解]]与矢量分析
- 发展[[麦克斯韦方程组]]与运算微积

## 3. 关键公式与思想
- $\nabla\cdot\mathbf{E}=\rho/\epsilon_0$（亥维赛形式的麦克斯韦方程）

## 4. 历史影响
亥维赛简化的[[麦克斯韦方程组]]是现代电磁学的基础形式。
