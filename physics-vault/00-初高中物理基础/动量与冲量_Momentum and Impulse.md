# 动量与冲量 (Momentum and Impulse)

## 1. 核心定义

动量是物体质量与速度的乘积 $p = mv$，描述物体的运动状态；冲量是力对时间的累积效应 $I = Ft$，描述力对物体运动状态的改变。

## 2. 数学表达

### 动量

$$\vec{p} = m\vec{v}$$

动量是**矢量**，方向与速度方向相同。

**参数说明：**

| 符号 | 物理量 | 单位 |
|------|--------|------|
| $\vec{p}$ | 动量 | $\text{kg}\cdot\text{m/s}$ |
| $m$ | 质量 | $\text{kg}$ |
| $\vec{v}$ | 速度 | $\text{m/s}$ |

### 冲量

恒力的冲量：

$$\vec{I} = \vec{F}\Delta t$$

变力的冲量（用积分表示）：

$$\vec{I} = \int_{t_1}^{t_2} \vec{F}(t)\,dt$$

冲量是**矢量**，方向与力的方向相同（恒力情形）。冲量在 $F$-$t$ 图像中等于力曲线下的面积。

**参数说明：**

| 符号 | 物理量 | 单位 |
|------|--------|------|
| $\vec{I}$ | 冲量 | $\text{N}\cdot\text{s}$ |
| $\vec{F}$ | 力 | $\text{N}$ |
| $\Delta t$ | 力作用时间 | $\text{s}$ |

### 动量定理

合外力的冲量等于物体动量的变化量：

$$\vec{I}_{\text{合}} = \Delta\vec{p} = m\vec{v}_2 - m\vec{v}_1$$

对于恒力：

$$\vec{F}_{\text{合}} \Delta t = m\vec{v}_2 - m\vec{v}_1$$

分量形式：

$$F_x \Delta t = mv_{2x} - mv_{1x}$$

$$F_y \Delta t = mv_{2y} - mv_{1y}$$

### 平均力

在碰撞等短时间内力变化剧烈时，平均冲力为：

$$\bar{F} = \frac{\Delta p}{\Delta t}$$

## 3. 物理学意义与推论

### 物理意义

动量定理揭示了力对时间的累积效应——冲量引起动量的改变。这与[[功和机械能_Work and Mechanical Energy]]中力对空间的累积效应（功引起动能的改变）形成对偶关系。动量定理是[[牛顿运动定律_Newton's Laws of Motion]]第二定律的另一种表述形式，由 $\vec{F} = \frac{d\vec{p}}{dt}$ 积分直接得到。

### 动量的矢量性

动量和冲量都是矢量。在应用动量定理时，必须注意各物理量的方向，通常采用正交分解法分别处理各方向上的动量变化。一维问题中需规定正方向，用正负号表示方向。

### 动量定理的应用

- **缓冲原理**：增大作用时间 $\Delta t$ 可减小冲击力 $\bar{F}$。如安全气囊、跳高垫、汽车保险杠等。
- **打击与碰撞**：作用时间极短，冲力很大。碰撞过程中重力等常规力远小于冲力，通常可忽略。
- **流体问题**：水流或气流冲击物体时，由 $\bar{F} = \frac{\Delta p}{\Delta t}$ 计算冲击力，其中 $\Delta p$ 为单位时间内流体动量的变化量。

### 适用边界

- 动量定理适用于**恒力**和**变力**，适用于**直线运动**和**曲线运动**。
- 动量定理中的 $\vec{F}$ 是合外力，不是某一个力。
- 动量定理中各动量应对应同一惯性参考系。
- 在相对论速度下，经典动量公式 $p = mv$ 不再适用，需修正为 $p = \gamma mv$。

### 与大学物理的衔接

1. **广义动量**：在[[拉格朗日力学_Lagrangian Mechanics]]中，动量推广为**广义动量** $p_i = \frac{\partial L}{\partial \dot{q}_i}$。当广义坐标不是笛卡尔坐标时，广义动量可以具有角动量的量纲（如 $L = mr^2\dot{\theta}$ 对应角动量）。这使动量的概念超越了"质量乘速度"的经典定义。

2. **守恒量与对称性**：在[[哈密顿力学_Hamiltonian Mechanics]]中，动量守恒与空间平移对称性相关，这是[[诺特定理_Noether's Theorem]]的核心结论——每一种连续对称性对应一个守恒量。

3. **相对论动量**：在[[狭义相对论_Special Relativity]]中，动量修正为 $p = \gamma m_0 v$，其中 $\gamma = \frac{1}{\sqrt{1-v^2/c^2}}$。动量-能量关系 $E^2 = p^2c^2 + m_0^2c^4$ 统一了动量和能量。

4. 在[[量子力学_Quantum Mechanics]]中，动量对应于动量算符 $\hat{p} = -i\hbar\nabla$，动量是波函数在动量表象中的本征值。[[德布罗意波_De Broglie Wave]]将动量与波长联系起来：$\lambda = \frac{h}{p}$。

## 4. 相关验证实验

### 碰撞实验中的动量测量

**实验器材**：气垫导轨、滑块两个（带挡光片）、光电门两个、电子天平。

**实验过程**：
1. 测量两滑块质量 $m_1$、$m_2$。
2. 在气垫导轨上使滑块碰撞，用光电门记录碰撞前后的速度。
3. 计算碰撞前后的总动量，验证动量定理和动量守恒。

### 冲力测量实验

用力传感器记录碰撞过程中的力-时间曲线，通过积分 $\int F\,dt$ 计算冲量，验证冲量等于动量变化。

### 缓冲效果对比实验

将鸡蛋从同一高度分别落到硬地面和海绵垫上，比较冲击力大小。由 $\bar{F} = \frac{\Delta p}{\Delta t}$ 可知，海绵垫增大了 $\Delta t$，从而减小了 $\bar{F}$，鸡蛋不破。

## 5. 历史上下文

"动量"概念的历史可追溯到17世纪。[[勒内·笛卡尔]]（René Descartes, 1596–1650）最早提出宇宙中"运动量"守恒的思想，但他将运动量定义为质量与速率的乘积（标量），忽略了方向性。

[[艾萨克·牛顿]]（Isaac Newton, 1643–1727）在《自然哲学的数学原理》（1687年）中将动量（"运动的量"）明确定义为质量与速度的乘积，并指出其矢量性质。牛顿第二定律最初的表述正是"力的作用等于动量的变化率"，即 $\vec{F} = \frac{d\vec{p}}{dt}$，而非高中教材中常见的 $\vec{F} = m\vec{a}$。后者仅适用于质量不变的情形，而前者更普遍。

[[克里斯蒂安·惠更斯]]（Christiaan Huygens）在研究碰撞问题时发现动量守恒定律，[[约翰·沃利斯]]（John Wallis）和[[克里斯托弗·雷恩]]（Christopher Wren）也独立地对碰撞问题做出了贡献。[[莱昂哈德·欧拉]]（Leonhard Euler）后来将动量概念系统化，并推广到连续介质。

19世纪，[[古斯塔夫·科里奥利]]（Gaspard-Gustave de Coriolis）引入了"功"和"动能"的概念，使动量和能量成为描述机械运动的两个并列的物理量。在20世纪，[[诺特定理_Noether's Theorem]]揭示了动量守恒与空间平移对称性的深刻联系，将动量提升为理论物理的基本概念。
