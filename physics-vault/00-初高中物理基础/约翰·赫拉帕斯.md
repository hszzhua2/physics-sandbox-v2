# 约翰·赫拉帕斯 (John Herapath)

## 1. 生平
1790年生于布里斯托尔，自学成才，任铁路工程师。独立提出[[分子动理论]]的气体动理论，1868年逝于英格兰。

## 2. 主要贡献
- 提出[[分子动理论]]的气体动理论
- 研究[[分子动理论]]与气体压强
- 发展[[分子动理论]]与统计力学

## 3. 关键公式与思想
- $P=\dfrac{1}{3}\rho\bar{v^2}$（气体压强公式）

## 4. 历史影响
赫拉帕斯的[[分子动理论]]是统计力学的早期独立发展。
