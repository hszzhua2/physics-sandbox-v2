# 乔治·汤姆逊 (George Thomson)

## 1. 生平
1892年生于剑桥，在剑桥大学学习，任伦敦大学帝国理工学院教授。通过电子衍射实验验证物质波，1937年获诺贝尔物理学奖，1975年逝于剑桥。

## 2. 主要贡献
- 验证[[波粒二象性]]的电子衍射
- 研究[[波粒二象性]]与[[偏振]]
- 发展[[波粒二象性]]实验

## 3. 关键公式与思想
- $\lambda=h/p$（电子德布罗意波长）

## 4. 历史影响
汤姆逊的电子衍射是[[波粒二象性]]物质波的实验验证。
