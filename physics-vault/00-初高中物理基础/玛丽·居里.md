# 玛丽·居里 (Marie Curie)

## 1. 生平
1867年生于华沙，在索邦大学学习，任索邦大学教授。发现钋与镭，研究放射性，1903年与1911年两获诺贝尔奖，1934年逝于桑塞勒穆兹。

## 2. 主要贡献
- 发现[[放射性衰变]]的钋与镭
- 研究[[放射性衰变]]与放射性元素
- 发展[[放射性衰变]]理论

## 3. 关键公式与思想
- $\dfrac{dN}{dt}=-\lambda N$（放射性衰变定律）

## 4. 历史影响
居里夫人的放射性研究开创了[[放射性衰变]]与放射性化学。
