# 比热容 (Specific Heat Capacity)

## 1. 核心定义

比热容是单位质量的某种物质温度升高（或降低）$1\,^{\circ}\text{C}$ 所吸收（或放出）的热量，定义为 $c = Q/(m\Delta T)$，反映物质吸放热的能力。

## 2. 数学表达

### 2.1 比热容定义式

$$c = \frac{Q}{m \cdot \Delta T}$$

| 参数 | 物理量 | 单位 |
|------|--------|------|
| $c$ | 比热容 | $\text{J/(kg}\cdot^{\circ}\text{C)}$ |
| $Q$ | 吸收或放出的热量 | $\text{J}$ |
| $m$ | 物质[[质量]] | $\text{kg}$ |
| $\Delta T$ | [[温度]]变化量 | $^{\circ}\text{C}$（或 $\text{K}$） |

### 2.2 热量计算公式

$$Q = c \cdot m \cdot \Delta T$$

吸热时 $\Delta T = T_{\text{末}} - T_{\text{初}} > 0$，$Q > 0$；放热时 $\Delta T < 0$，$Q < 0$。

### 2.3 常见物质比热容

| 物质 | 比热容 $c$（$\text{J/(kg}\cdot^{\circ}\text{C)}$） |
|------|-------------------------------------------------|
| 水 | $4.2 \times 10^3$ |
| 酒精 | $2.4 \times 10^3$ |
| 煤油 | $2.1 \times 10^3$ |
| 冰 | $2.1 \times 10^3$ |
| 铝 | $0.88 \times 10^3$ |
| 铁/钢 | $0.46 \times 10^3$ |
| 铜 | $0.39 \times 10^3$ |
| 铅 | $0.13 \times 10^3$ |
| 砂石 | $0.92 \times 10^3$ |

水的比热容在常见物质中最大（$4.2 \times 10^3\,\text{J/(kg}\cdot^{\circ}\text{C)}$），这一特性具有重要的物理和生态意义。

### 2.4 热平衡方程

当两个温度不同的物体接触达到热平衡时，低温物体吸收的热量等于高温物体放出的热量（无热量损失）：

$$Q_{\text{吸}} = Q_{\text{放}}$$

$$c_1 m_1 (T_{\text{末}} - T_1) = c_2 m_2 (T_2 - T_{\text{末}})$$

其中 $T_1 < T_{\text{末}} < T_2$，$T_{\text{末}}$ 为热平衡温度。

## 3. 物理学意义与推论

### 3.1 比热容的物理意义

比热容反映了物质"储热能力"的大小。比热容越大，使相同质量的物质升高相同温度所需的热量越多，即该物质越"难升温也难降温"。

以水和砂石为例：
- 水的比热容 $c_{\text{水}} = 4.2 \times 10^3\,\text{J/(kg}\cdot^{\circ}\text{C)}$，砂石 $c_{\text{砂}} \approx 0.92 \times 10^3\,\text{J/(kg}\cdot^{\circ}\text{C)}$。
- 同等质量和日照条件下，砂石升温约为水的 $4.6$ 倍。
- 因此沿海地区昼夜温差小（海水热容量大、调节温度），内陆沙漠地区昼夜温差大（砂石热容量小、升降温快）。

### 3.2 水的比热容大的应用

水的比热容大这一特性在自然界和工程中有广泛应用：

| 应用领域 | 原理 | 具体实例 |
|---------|------|---------|
| 气候调节 | 水吸放热多，温度变化慢 | 沿海地区气候温和，昼夜温差小 |
| 散热介质 | 水吸收大量热量而温升不大 | 汽车发动机水冷系统、核电站冷却 |
| 供暖 | 水放热多，降温慢 | 暖气用水作为传热介质 |
| 调温 | 水的吸放热缓冲 | 海洋是地球的"恒温器"，调节全球气候 |
| 农业 | 水防霜冻 | 寒潮来临前灌水，利用水结冰放热保护秧苗 |

### 3.3 比热容与物质结构的关系

比热容的微观本质是物质内部微观粒子（分子、原子）热运动能量的存储能力。从能量均分定理看：

- **固体**：原子在晶格中振动，每个自由度分配 $\frac{1}{2}k_BT$ 能量（$k_B$ 为[[玻尔兹曼常数]]，$T$ 为绝对温度）。经典理论预言固体摩尔热容为 $3R$（杜隆-珀蒂定律），但对低温下固体不成立——需[[量子力学]]修正（[[爱因斯坦模型_Einstein Model]]和[[德拜模型_Debeye Model]]）。
- **液体**：分子既有振动又有平动自由度，比热容通常介于固体和气体之间。水的比热容特别大，与水分子间的氢键有关——加热水时需额外能量破坏氢键。
- **气体**：理想气体定容比热容 $c_V$ 由自由度决定（单原子 $c_V = \frac{3}{2}R$，双原子 $c_V = \frac{5}{2}R$），定压比热容 $c_p = c_V + R$（[[迈耶关系_Mayer's Relation]]）。

### 3.4 热平衡与热传递

热传递有三种方式：
1. **热传导**：热量通过物体内部或接触面从高温区传向低温区，介质不发生宏观运动。金属是良好的热导体。
2. **对流**：流体中温度不同的部分由于密度差异发生宏观流动而传热。如烧水时底部热水上升、冷水下降形成循环。
3. **热辐射**：物体以电磁波形式向外辐射热量，不需介质，可在真空中传播。太阳热就是通过热辐射传到地球。

热平衡是热传递的最终结果：当各物体温度相同时，净热传递为零。热平衡时满足热平衡方程 $Q_{\text{吸}} = Q_{\text{放}}$。

### 3.5 比热容与潜热的区别

| 比较项 | 比热容（显热） | 潜热 |
|--------|-------------|------|
| 温度变化 | 有温度变化 | 温度不变（相变中） |
| 公式 | $Q = cm\Delta T$ | $Q = \lambda m$ 或 $Q = Lm$ |
| 物理意义 | 温度变化时吸放的热量 | 相变时吸放的热量 |
| 详见 | 本节 | [[物态变化_Phase Change]] |

### 3.6 适用边界与知识衔接

比热容在初中阶段以宏观测量和实际应用为主。在高中物理中，比热容将联系到气体状态方程和[[热力学四大定律_Four Laws of Thermodynamics]]中的热力学第一定律 $\Delta U = Q + W$。在大学物理中，比热容从统计力学严格推导：[[配分函数_Partition Function]] $Z$ 可计算系统的内能 $U$ 和热容 $C = \partial U/\partial T$。[[爱因斯坦模型_Einstein Model]]和[[德拜模型_Debeye Model]]分别解释了固体低温下热容的量子行为——$T \to 0$ 时 $C \to 0$（热力学第三定律），这是经典理论无法预言的。水的反常大比热容则需用量子化学方法分析氢键网络。此外，比热容在工程热力学中是传热设计和能量系统分析的基本参数。

## 4. 相关验证实验

### 4.1 比较不同物质吸热能力实验

取质量相同的水和煤油（或砂），分别装入两个相同烧杯中，用相同的电加热器加热：
1. 记录初始温度（确保相同）。
2. 同时开始加热，每隔相同时间记录两者温度。
3. 相同时间内（吸收相同热量 $Q$），煤油升温比水快。
4. 使两者升高相同温度，水需要更长的加热时间（吸收更多热量）。
5. 计算 $c = Q/(m\Delta T)$，验证 $c_{\text{水}} > c_{\text{煤油}}$。

**控制变量**：质量相同、加热器功率相同（保证吸收热量相同）、初始温度相同。

### 4.2 比热容测量实验（混合法）

1. 用天平称量金属块质量 $m_1$，用细线系住。
2. 将金属块放入沸水中加热至沸腾，记录水温 $T_2$（即金属块初温）。
3. 用量筒量取冷水质量 $m_2$，倒入量热器，测初温 $T_1$。
4. 迅速将热金属块放入量热器冷水中，搅拌至温度稳定，记录末温 $T_{\text{末}}$。
5. 由热平衡 $Q_{\text{放}} = Q_{\text{吸}}$：
   $$c_1 m_1 (T_2 - T_{\text{末}}) = c_2 m_2 (T_{\text{末}} - T_1)$$
6. 已知水的比热容 $c_2$，计算金属比热容 $c_1$。

**注意事项**：使用量热器减少热量散失；快速转移减少热量损失；搅拌使温度均匀。

### 4.3 水的比热容大验证实验

将等质量的水和铁砂放入相同容器中，同时置于阳光下或用相同热源加热：
1. 每隔 $2\,\text{min}$ 记录两者温度。
2. 观察铁砂温度上升比水快得多。
3. 移去热源后，铁砂降温也比水快。
4. 说明水吸热多、放热也多，比热容大。

### 4.4 热传递方式演示实验

- **热传导**：将金属棒一端放入热水中，另一端固定蜡球，观察蜡球熔化脱落。
- **对流**：在烧杯水中加入木屑或高锰酸钾晶粒，加热底部一侧，观察水流循环。
- **热辐射**：用手靠近但不接触热源（如电炉丝），感受热量——真空中热辐射仍可传播。

## 5. 历史上下文

### 5.1 约瑟夫·布莱克与比热容

[[约瑟夫·布莱克]]（Joseph Black, 1728—1799）在发现潜热（见[[物态变化_Phase Change]]）的同时，也发现了比热容的概念。他观察到同等质量的不同物质升高相同温度所需的热量不同——等量的水和水银升高同样的温度，水所需热量约为水银的 $30$ 倍。布莱克将这一性质称为物质的"热容量"（heat capacity），后来拉瓦锡和拉普拉斯将其发展为"比热"的概念。布莱克使用冰量热器精确测量了多种物质的比热容。

### 5.2 热质说与热的运动说

18世纪，布莱克和[[安托万·拉瓦锡]]等学者信奉"热质说"（caloric theory）——将热视为一种不可称量的流体（热质），比热容被解释为不同物质对热质的"容量"。虽然热质说后来被热的运动说（[[詹姆斯·焦耳]]的热功当量实验）取代，但布莱克的比热容和潜热概念本身是正确的实验发现，被完整保留至今。

### 5.3 热功当量与能量守恒

[[詹姆斯·焦耳]]（James Joule, 1818—1889）于1840年代通过精确的实验确定了热与功的定量关系——$1\,\text{cal} = 4.18\,\text{J}$。他用搅拌器搅拌水，测量做功 $W = mgh$（重物下落）与水温升高 $\Delta T$ 的关系，验证 $W = cm\Delta T$。这一"热功当量"实验推翻了热质说，确立了热是能量的一种形式，为[[热力学四大定律_Four Laws of Thermodynamics]]中的第一定律（能量守恒定律）奠定了实验基础。

### 5.4 统计力学的深层解释

19世纪后期，[[詹姆斯·麦克斯韦]]和[[路德维希·玻尔兹曼]]建立了统计力学，从分子运动的角度解释了比热容的来源——比热容正比于分子运动自由度。能量均分定理预言了理想气体的比热容，但对固体低温行为的失败导致了[[量子力学]]的诞生。[[爱因斯坦]]于1907年用量子化振子模型（[[爱因斯坦模型_Einstein Model]]）成功解释了固体低温下比热容随温度降低而减小的现象，这是量子理论在固体物理中的最早应用之一。[[彼得·德拜]]于1912年进一步改进为[[德拜模型_Debeye Model]]，精确描述了固体比热容在整个温度范围内的行为。这些发展使比热容从单纯的宏观测量量成为探索物质微观结构的重要窗口。
