# 匀变速直线运动 (Uniformly Accelerated Linear Motion)

## 1. 核心定义

加速度恒定的直线运动，即速度随时间均匀变化的运动。

## 2. 数学表达

### 基本公式

速度公式：

$$v = v_0 + at$$

位移公式：

$$x = v_0 t + \frac{1}{2}at^2$$

速度-位移关系式：

$$v^2 - v_0^2 = 2ax$$

平均速度公式：

$$\bar{v} = \frac{v_0 + v}{2} = \frac{x}{t}$$

**参数说明：**

| 符号 | 物理量 | 单位 |
|------|--------|------|
| $v_0$ | 初速度 | $\text{m/s}$ |
| $v$ | 末速度 | $\text{m/s}$ |
| $a$ | 加速度（恒量） | $\text{m/s}^2$ |
| $t$ | 时间 | $\text{s}$ |
| $x$ | 位移 | $\text{m}$ |

### 重要推论

在连续相等时间间隔 $T$ 内的位移差为常数：

$$\Delta x = x_{n+1} - x_n = aT^2$$

中间时刻速度 $v_{t/2}$ 与中间位置速度 $v_{x/2}$ 的关系：

$$v_{t/2} = \bar{v} = \frac{v_0 + v}{2}, \quad v_{x/2} = \sqrt{\frac{v_0^2 + v^2}{2}}$$

$$v_{x/2} > v_{t/2}$$

### 图像表示

**$v$-$t$ 图像**：匀变速直线运动的 $v$-$t$ 图为一条倾斜直线，其斜率等于加速度 $a$，直线与时间轴围成的面积等于位移 $x$。

**$x$-$t$ 图像**：匀变速直线运动的 $x$-$t$ 图为一条抛物线，其切线斜率等于瞬时速度。

## 3. 物理学意义与推论

### 物理意义

匀变速直线运动是最简单的变速运动模型，描述了加速度恒定条件下物体位置和速度随时间演化的规律。三个基本公式之间存在内在关联，其中两个是独立的，第三个可由前两个推导得出。

### 适用边界

- 仅适用于**加速度恒定**的情形，即合外力恒定且方向不变。
- 仅适用于**直线运动**，即速度方向与加速度方向在同一直线上。
- 当 $a$ 与 $v_0$ 同向时为匀加速；当 $a$ 与 $v_0$ 反向时为匀减速。
- 实际物理过程中，空气阻力等因素会使加速度变化，此模型为理想化近似。

### 与大学物理的衔接

在大学物理中，匀变速直线运动是**运动学一般理论**的特例。当加速度 $a$ 不再是常数而是时间或位置的函数 $a(t)$ 或 $a(x)$ 时，需通过**积分**求解：

$$v(t) = v_0 + \int_0^t a(\tau)\,d\tau$$

$$x(t) = x_0 + \int_0^t v(\tau)\,d\tau$$

在[[理论力学_Theoretical Mechanics]]框架下，运动学被纳入[[拉格朗日力学_Lagrangian Mechanics]]和[[哈密顿力学_Hamiltonian Mechanics]]的更一般化体系中，匀变速运动对应于势能为线性函数的系统。

## 4. 相关验证实验

### 打点计时器实验

**实验器材**：电磁打点计时器（或电火花计时器）、纸带、交流电源（$6\,\text{V}$～$8\,\text{V}$，$50\,\text{Hz}$）、小车、长木板、砝码。

**实验原理**：打点计时器每隔 $T = 0.02\,\text{s}$ 在纸带上打一个点，通过测量纸带上各计数点之间的距离来验证匀变速直线运动的规律。

**数据处理**：

1. 选取计数点，每$5$个点取一个计数点，时间间隔 $T = 0.1\,\text{s}$。
2. 用逐差法计算加速度：

$$a = \frac{(x_4 + x_5 + x_6) - (x_1 + x_2 + x_3)}{9T^2}$$

3. 验证 $\Delta x = aT^2$ 是否为常数。
4. 利用 $v_n = \frac{x_n + x_{n+1}}{2T}$ 计算各点瞬时速度，作 $v$-$t$ 图求斜率得到加速度。

### 频闪照相实验

利用频闪照相机记录物体在不同时刻的位置，通过照片测量验证匀变速直线运动的位移-时间关系。

## 5. 历史上下文

[[伽利略·伽利雷]]（Galileo Galilei, 1564–1642）是系统研究匀变速运动的先驱。在17世纪初，伽利略面临的最大困难是无法精确测量极短时间内的速度和位移。他创造性地采用了**斜面实验**：通过减小斜面倾角来"稀释"重力，使铜球在斜面上的运动速度减慢到可测量的程度。

伽利略利用水钟（通过称量流出的水量来计时）测量铜球在不同距离上滚过的时间，发现位移与时间的平方成正比，即 $x \propto t^2$，由此确立了匀变速运动的规律。这一实验方法标志着**实验物理学**的开端，是近代科学方法论的奠基性贡献。

伽利略的工作为[[艾萨克·牛顿]]建立[[牛顿运动定律_Newton's Laws of Motion]]奠定了运动学基础，其斜面实验的推理方法（理想实验法）也成为物理学研究的重要范式。
