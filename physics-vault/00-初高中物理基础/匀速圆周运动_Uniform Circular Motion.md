# 匀速圆周运动 (Uniform Circular Motion)

## 1. 核心定义

质点沿圆周运动且速率不变的运动，其速度大小恒定但方向时刻改变，加速度始终指向圆心。

## 2. 数学表达

### 基本物理量

**线速度**（切向速度）：

$$v = \frac{\Delta s}{\Delta t} = \frac{2\pi r}{T} = \omega r$$

**角速度**：

$$\omega = \frac{\Delta\theta}{\Delta t} = \frac{2\pi}{T} = 2\pi f$$

**周期与频率**：

$$T = \frac{2\pi}{\omega}, \quad f = \frac{1}{T} = \frac{\omega}{2\pi}$$

**转速**（单位：$\text{r/min}$）：

$$n = 60f$$

### 向心加速度

$$a_n = \frac{v^2}{r} = \omega^2 r = \frac{4\pi^2 r}{T^2} = v\omega$$

方向始终指向圆心，与速度方向垂直。

### 向心力

$$F_n = ma_n = \frac{mv^2}{r} = m\omega^2 r$$

方向始终指向圆心，由合外力提供。

**参数说明：**

| 符号 | 物理量 | 单位 | 说明 |
|------|--------|------|------|
| $v$ | 线速度 | $\text{m/s}$ | 方向沿切线方向 |
| $\omega$ | 角速度 | $\text{rad/s}$ | 标量 |
| $r$ | 圆半径 | $\text{m}$ | |
| $T$ | 周期 | $\text{s}$ | 转一圈所用时间 |
| $f$ | 频率 | $\text{Hz}$ | 单位时间转数 |
| $a_n$ | 向心加速度 | $\text{m/s}^2$ | 指向圆心 |
| $F_n$ | 向心力 | $\text{N}$ | 指向圆心 |

### 同轴与同带传动

- **同轴传动**：各点 $\omega$ 相同，$v \propto r$。
- **皮带（齿轮）传动**：接触点 $v$ 相同，$\omega \propto \frac{1}{r}$。

## 3. 物理学意义与推论

### 物理意义

匀速圆周运动的核心特征是**速度大小不变而方向时刻变化**，因此存在加速度（向心加速度）。向心加速度不改变速度的大小，只改变速度的方向。向心力是产生向心加速度的原因，它可以是某一个力，也可以是几个力的合力。

### 常见向心力来源

| 运动情景 | 向心力来源 |
|----------|------------|
| 水平圆盘上的物体 | 静摩擦力 |
| 圆锥摆 | 重力与拉力的合力 |
| 汽车在水平弯道转弯 | 静摩擦力 |
| 汽车在倾斜弯道转弯 | 重力与支持力的合力 |
| 绳系小球在竖直面内运动（最低点） | 拉力与重力的合力（$F - mg$） |
| 绳系小球在竖直面内运动（最高点） | 拉力与重力的合力（$F + mg$） |
| 卫星绕地球运动 | [[万有引力]] |

### 适用边界

- "匀速"指**速率不变**（线速度大小不变），但速度方向时刻变化，因此匀速圆周运动是**变速运动**。
- 向心加速度和向心力的大小恒定，但方向时刻改变，因此它们是**变力**（变矢量）。
- 严格意义上的匀速圆周运动要求合外力大小恒定且始终垂直于速度方向指向圆心。

### 与大学物理的衔接

在大学物理中，圆周运动的研究扩展到以下方向：

1. **变速圆周运动**：当速率也变化时，加速度不仅有法向（向心）分量 $a_n = \frac{v^2}{r}$，还有切向分量 $a_\tau = \frac{dv}{dt}$。总加速度为：

$$\vec{a} = a_n\hat{n} + a_\tau\hat{\tau}, \quad |\vec{a}| = \sqrt{a_n^2 + a_\tau^2}$$

2. **一般曲线运动**：将曲线在某点的邻域视为"曲率圆"，向心加速度推广为**法向加速度** $a_n = \frac{v^2}{\rho}$，其中 $\rho$ 为曲率半径。

3. 在[[理论力学_Theoretical Mechanics]]中，圆周运动用**极坐标**描述更为自然，引入径向和横向分量。在[[哈密顿力学_Hamiltonian Mechanics]]中，匀速圆周运动对应于角动量守恒的系统。

4. 在电磁学中，[[洛伦兹力_Lorentz Force]]使带电粒子在均匀磁场中做匀速圆周运动，回旋半径 $r = \frac{mv}{qB}$，这是[[回旋加速器_Cyclotron]]的理论基础。

## 4. 相关验证实验

### 向心力演示实验

**实验器材**：向心力演示仪（弹簧、标尺、转台、不同质量的小球）。

**实验过程**：通过控制变量法研究向心力与各物理量的关系：
1. 保持 $r$、$\omega$ 不变，改变 $m$，验证 $F \propto m$。
2. 保持 $m$、$\omega$ 不变，改变 $r$，验证 $F \propto r$。
3. 保持 $m$、$r$ 不变，改变 $\omega$，验证 $F \propto \omega^2$。

### 圆锥摆实验

用细线悬挂小球，使小球在水平面内做匀速圆周运动形成圆锥摆。测量摆长 $L$、偏角 $\theta$、周期 $T$，验证：

$$mg\tan\theta = m\omega^2 r = m\frac{4\pi^2}{T^2}L\sin\theta$$

### 离心现象演示

在转台上放置不同质量的物体，逐渐增大转速，观察物体在静摩擦力不足以提供向心力时发生离心运动，验证向心力的临界条件。

## 5. 历史上下文

圆周运动的系统研究始于17世纪。[[克里斯蒂安·惠更斯]]（Christiaan Huygens, 1629–1695）在1673年出版的《摆钟论》中首次给出了向心加速度的正确公式 $a = \frac{v^2}{r}$，这是运动学的重要里程碑。惠更斯推导这一公式的方法基于几何论证，展现了极高的数学技巧。

[[艾萨克·牛顿]]在惠更斯工作的基础上，将向心加速度与[[牛顿运动定律_Newton's Laws of Motion]]结合，提出了向心力 $F = \frac{mv^2}{r}$。牛顿进一步将地球上的圆周运动（如月球绕地球的运动）与地面上的重力统一为同一种力——[[万有引力]]，这是物理学史上最伟大的统一之一。

牛顿在《自然哲学的数学原理》（1687年）中通过"牛顿大炮"思想实验论证：当平抛物体的初速度足够大时，其抛物线轨道将弯曲到与地球表面曲率一致，从而成为绕地球运行的卫星——这正是匀速圆周运动在天体力学中的体现。这一思想为后人造卫星的发射奠定了理论基础。
