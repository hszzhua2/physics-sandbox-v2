# 约瑟夫·冯·夫琅禾费 (Joseph von Fraunhofer)

## 1. 生平
1787年生于斯特劳宾，在慕尼黑大学学习，任慕尼黑大学教授。发现夫琅禾费线，发展光栅光谱学，1826年逝于慕尼黑。

## 2. 主要贡献
- 发现[[光的色散]]的夫琅禾费吸收线
- 发展[[光的衍射]]与衍射光栅
- 研究[[原子光谱]]与精密光学

## 3. 关键公式与思想
- $d\sin\theta=n\lambda$（光栅方程）

## 4. 历史影响
夫琅禾费线是[[光的色散]]与[[原子光谱]]分析的关键发现。
