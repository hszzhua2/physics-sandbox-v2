# 古斯塔夫·赫兹 (Gustav Hertz)

## 1. 生平
1887年生于汉堡，在柏林大学学习，任莱比锡大学教授。与弗兰克完成弗兰克-赫兹实验，1925年获诺贝尔物理学奖，1975年逝于柏林。

## 2. 主要贡献
- 完成[[原子光谱]]的弗兰克-赫兹实验
- 验证[[原子结构模型]]的原子能级
- 研究[[原子结构模型]]与量子物理

## 3. 关键公式与思想
- $eV_0=4.9\,\text{eV}$（汞原子第一激发电势）

## 4. 历史影响
赫兹的弗兰克-赫兹实验验证了[[原子光谱]]与[[原子结构模型]]的能级量子化。
