# 肯特·福特 (Kent Ford)

## 1. 生平
1931年生于美国，在凯斯西储大学与加州理工学院学习，任卡内基科学研究所研究员。发展像增强器技术并与鲁宾测量星系旋转曲线。

## 2. 主要贡献
- 发展像增强器用于天文光谱观测
- 与鲁宾测量星系旋转曲线，发现[[暗物质]]证据

## 3. 关键公式与思想
- $v(r)\approx\text{const}$（平坦旋转曲线）

## 4. 历史影响
福特的技术使鲁宾的[[暗物质]]观测成为可能，间接改变宇宙学认知。
