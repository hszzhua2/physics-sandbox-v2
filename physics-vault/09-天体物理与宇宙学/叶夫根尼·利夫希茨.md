# 叶夫根尼·利夫希茨 (Evgeny Lifshitz)

## 1. 生平
1915年生于哈尔科夫，在哈尔科夫大学师从朗道，任莫斯科物理问题研究所教授。与朗道合著《理论物理教程》，研究宇宙学微扰。

## 2. 主要贡献
- 与朗道合著《理论物理教程》
- 研究[[宇宙学微扰理论]]的相对论微扰演化
- 发展凝聚态与弹性理论

## 3. 关键公式与思想
- $\delta_k\propto t^{2/3}$（物质主导期增长）

## 4. 历史影响
利夫希茨的宇宙学微扰理论是结构形成的理论基础，与朗道合著的教材影响深远。
