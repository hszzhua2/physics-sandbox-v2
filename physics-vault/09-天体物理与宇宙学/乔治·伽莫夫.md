# 乔治·伽莫夫 (George Gamow)

## 1. 生平
1904年生于敖德萨，在列宁格勒大学学习后游学欧洲，1934年定居美国任乔治·华盛顿大学与科罗拉多大学教授。提出大爆炸核合成与量子隧穿解释α衰变。

## 2. 主要贡献
- 用[[量子隧穿效应]]解释α衰变
- 提出[[大爆炸理论]]的核合成机制，预言宇宙氦丰度
- 与阿尔弗、赫尔曼预言[[宇宙微波背景辐射]]

## 3. 关键公式与思想
- $P\propto e^{-2\pi Z_1Z_2e^2/\hbar v}$（伽莫夫因子）

## 4. 历史影响
伽莫夫的大爆炸理论与背景辐射预言成为现代宇宙学的基石。
