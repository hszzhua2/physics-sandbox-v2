# 维亚切斯拉夫·穆哈诺夫 (Viatcheslav Mukhanov)

## 1. 生平
1956年生于莫斯科，在莫斯科物理问题研究所获博士学位，任慕尼黑大学教授。1981年与奇比索夫提出暴胀量子涨落产生原初扰动。

## 2. 主要贡献
- 与奇比索夫提出[[暴胀理论]]的量子涨落产生原初微扰
- 推导原初功率谱

## 3. 关键公式与思想
- $P_\mathcal{R}(k)\propto k^{n_s-1},\;n_s\approx1$（原初扰动谱）

## 4. 历史影响
穆哈诺夫的暴胀量子涨落理论解释了[[宇宙微波背景辐射]]各向异性的起源。
