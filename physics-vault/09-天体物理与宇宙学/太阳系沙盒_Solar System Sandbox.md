# 太阳系沙盒 (Solar System Sandbox)

## 1. 实验目的与背景

太阳系沙盒是一种基于[[牛顿万有引力定律]]与[[开普勒定律]]的数值模拟实验平台，旨在通过N体引力计算再现行星、卫星、小行星及彗星的轨道演化。其历史根源可追溯至1609年[[约翰内斯·开普勒]]基于[[第谷·布拉赫]]的精密观测数据发表的《新天文学》，其中首次阐述了行星运动三定律。该沙盒实验的目的是：验证经典引力理论在多体系统中的适用性，探索轨道长期稳定性，并为航天器轨道设计（如引力辅助与拉格朗日点驻留）提供理论依据。

在数值模拟框架下，太阳系被视为一个N体引力系统，每个天体在其余所有天体的合引力作用下运动。当 $N$ 较大时，系统一般不存在解析解，必须依赖数值积分方法（如Runge-Kutta法、Symplectic积分器）逐步推进。该实验亦与[[三体问题_Three-Body Problem]]的混沌特性密切相关——即便是三体系统，其长期行为也表现出对初始条件的极端敏感性。

## 2. 实验装置与运行机制

### 2.1 N体引力数值模拟

太阳系沙盒的核心计算引擎基于N体引力运动方程。对于 $N$ 个质量为 $m_i$ 的天体，第 $i$ 个天体的运动方程为：

$$m_i \ddot{\mathbf{r}}_i = -G \sum_{j \neq i} \frac{m_i m_j (\mathbf{r}_i - \mathbf{r}_j)}{|\mathbf{r}_i - \mathbf{r}_j|^3}$$

其中 $G$ 为[[万有引力常数]]，$\mathbf{r}_i$ 为第 $i$ 个天体的位置矢量。该方程组为 $6N$ 维一阶常微分方程组，需通过数值积分求解。

常用的数值积分方法包括：
- **Symplectic积分器**（如Leapfrog/Verlet法）：保持哈密顿量结构的辛性质，在长期积分中能量漂移极小，适合行星轨道演化模拟。
- **高阶Runge-Kutta法**：精度高但非辛结构，适合短期高精度计算。

### 2.2 开普勒三定律

在二体近似（太阳质量远大于行星）下，[[开普勒定律]]给出：

1. **第一定律（椭圆定律）**：行星轨道为椭圆，太阳位于一个焦点上。
2. **第二定律（面积定律）**：行星与太阳的连线在相等时间内扫过相等面积：

$$\frac{dA}{dt} = \frac{1}{2} r^2 \dot{\theta} = \frac{L}{2m} = \text{const}$$

其中 $L$ 为角动量。

3. **第三定律（调和定律）**：轨道周期 $T$ 的平方与半长轴 $a$ 的立方成正比：

$$T^2 = \frac{4\pi^2}{G(M_\odot + m)} a^3 \approx \frac{4\pi^2}{GM_\odot} a^3$$

### 2.3 轨道要素

完整描述一条开普勒轨道需要六个轨道要素：
- 半长轴 $a$：确定轨道大小与能量 $E = -\frac{GMm}{2a}$
- 偏心率 $e$：确定轨道形状（$e=0$为圆，$0<e<1$为椭圆，$e=1$为抛物线，$e>1$为双曲线）
- 轨道倾角 $i$：轨道面与参考面的夹角
- 升交点经度 $\Omega$：参考方向到升交点的角度
- 近心点幅角 $\omega$：升交点到近心点的角度
- 平近点角 $M_0$（或过近心点时刻 $t_0$）：确定天体在轨道上的位置

### 2.4 拉格朗日点

在限制性三体问题中，存在五个拉格朗日点 $L_1$ 至 $L_5$，其中小质量天体所受引力与离心力平衡。对于共线点 $L_1, L_2, L_3$（不稳定平衡），其位置可通过求解以下方程获得：

$$\frac{GM_1}{(r - r_1)^2} - \frac{GM_2}{(r_2 - r)^2} - \omega^2 r = 0$$

其中 $\omega$ 为系统旋转角速度。$L_4$ 和 $L_5$ 位于两个大质量天体连线的等边三角形顶点上，为稳定平衡点。太阳与地球系统的 $L_2$ 点是詹姆斯·韦伯太空望远镜（JWST）的运行位置。

### 2.5 引力辅助（弹弓效应）

引力辅助是航天器利用行星引力场改变自身速度的轨道机动技术。在行星静止参考系中，航天器的进入与离开速率相等（弹性散射），但在日心参考系中，航天器可从行星轨道运动中获得或损失动能：

$$\Delta \mathbf{v} \approx 2 \mathbf{v}_p \sin\left(\frac{\delta}{2}\right)$$

其中 $\mathbf{v}_p$ 为行星速度，$\delta$ 为偏转角。该技术被广泛应用于旅行者号、卡西尼号等深空探测任务。

### 2.6 轨道共振

当两个天体的轨道周期之比为简单整数比时发生轨道共振。轨道共振可以稳定（如木卫Io-Europa-Ganymede的1:2:4拉普拉斯共振）或不稳定（如木星与火星之间的小行星带柯克伍德空隙）。共振条件为：

$$\frac{T_1}{T_2} = \frac{n_1}{n_2}$$

其中 $n_1, n_2$ 为整数。

## 3. 观测数据与数学模型

### 3.1 开普勒定律的验证

开普勒基于[[第谷·布拉赫]]对火星位置的观测数据（精度约1弧分），发现椭圆轨道模型比圆轨道模型更好地拟合数据。现代数值模拟在引入相对论修正后，可再现水星近日点进动的观测值：

$$\Delta \phi_{\text{GR}} = \frac{6\pi G M_\odot}{c^2 a(1-e^2)} \approx 43''/\text{世纪}$$

### 3.2 长期轨道稳定性

太阳系沙盒的长时间数值积分（如Laskar的工作，积分时间达 $10^9$ 年量级）显示：
- 内行星（水星至火星）轨道偏心率和倾角存在长期混沌，李雅普诺夫时间约为 $5 \times 10^6$ 年。
- 水星轨道在数十亿年尺度上有小概率偏心率达到足以与金星交叉的程度。

### 3.3 Tisserand参数

在彗星轨道与引力辅助分析中，Tisserand参数近似守恒：

$$T = \frac{a_p}{a} + 2\cos i \sqrt{\frac{a(1-e^2)}{a_p}}$$

其中 $a_p$ 为扰动行星的半长轴。该参数用于识别穿越行星轨道的彗星的来源。

## 4. 实验结论与影响

太阳系沙盒的数值模拟实验验证了以下关键结论：

1. **经典力学的有效性**：在太阳系尺度下，[[牛顿万有引力定律]]与[[开普勒定律]]以极高精度描述行星运动，仅在强引力场（如水星近日点）处需要[[广义相对论]]修正。

2. **混沌与稳定性共存**：太阳系在短期内（$10^6$年量级）表现出准周期稳定性，但在长期（$10^9$年量级）呈现混沌特性。这一发现与[[混沌双摆_Chaotic Double Pendulum]]和[[洛伦兹吸引子_Lorenz Attractor]]所揭示的对初始条件敏感性原理一致。

3. **轨道共振的结构塑造作用**：轨道共振既维持了行星卫星系统的稳定构型，也通过柯克伍德空隙塑造了小行星带的分布结构。

4. **引力辅助的工程价值**：弹弓效应使探测器能够以极少的燃料到达外太阳系，是深空探测的基石技术。

5. **拉格朗日点的战略意义**：$L_1$ 和 $L_2$ 点成为太阳观测与深空天文观测的理想驻留位置。

该实验平台与[[三体问题_Three-Body Problem]]的混沌分析、[[闵可夫斯基时空_Minkowski Spacetime]]的相对论框架以及[[太阳系沙盒_Solar System Sandbox]]自身的参数化探索密切相关，是经典天体力学与现代计算物理的交汇点。
