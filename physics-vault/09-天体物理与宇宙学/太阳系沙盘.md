# 太阳系沙盘 (Solar System Sandbox)

> [🌐 打开交互实验](../exp-solar-system.html)

## 1. 实验目的
通过三维可视化沙盘展示太阳系八大行星、矮行星与主要卫星的轨道运动，以及深空探测器的飞行路径，直观呈现 [[开普勒定律]] 所描述的行星轨道规律与 [[牛顿万有引力定律]] 的普适性。

## 2. 实验装置与方法
基于 Three.js 三维引擎构建实时轨道力学沙盘，包含：
- **八大行星**：水星、金星、地球、火星、木星、土星、天王星、海王星
- **矮行星**：冥王星、谷神星、阋神星、鸟神星、妊神星
- **主要卫星**：月球、木卫一至四（伽利略卫星）、土卫六、土卫二、海卫一
- **深空探测器**：旅行者1/2号、新视野号、帕克太阳探测器、朱诺号等10个
- **历史人物**：17位天文学家标签，从苏美尔人到克莱德·汤博

距离按15×AU压缩，轨道倾角真实呈现黄道面分布。支持速度控制（0.5x/1x/2x/5x）和暂停。

## 3. 关键结果
沙盘清晰展示开普勒三定律：
- 轨道为椭圆，太阳位于焦点之一：$r = \frac{a(1-e^2)}{1+e\cos\theta}$
- 面积速度恒定：$\frac{dA}{dt} = \frac{L}{2m} = \text{const}$
- 周期与半长轴的3/2次方成正比：$T^2 \propto a^3$

外行星轨道周期远大于内行星，直观验证 $T \propto a^{3/2}$ 关系。

## 4. 历史意义
- **古代**：苏美尔人、古巴比伦人最早记录行星运动周期
- **1543年**：[[哥白尼]]提出日心说，将地球从宇宙中心移除
- **1609-1619年**：[[开普勒]]分析第谷·布拉赫的观测数据，发现行星运动三定律
- **1687年**：[[牛顿]]在《自然哲学的数学原理》中用万有引力定律解释开普勒定律
- **1957年至今**：人类进入太空时代，旅行者号（1977）飞越四颗外行星，新视野号（2015）飞掠冥王星

## 5. 现代复现
现代天文学使用数值积分（如RK4、Symplectic积分器）精确计算行星轨道，NASA JPL的HORIZONS系统提供太阳系所有天体的实时星历表。本沙盘采用简化开普勒轨道（忽略摄动），适合教学演示。
