# 爱德华·莫雷 (Edward Morley)

## 1. 生平
1838年生于新泽西，在威廉姆斯学院学习后长期任凯斯西储大学化学教授。以与迈克尔逊合作的光速实验闻名。

## 2. 主要贡献
- 与迈克尔逊合作完成[[迈克尔逊-莫雷实验]]
- 精确测定氧与氢的原子量比

## 3. 关键公式与思想
- $\Delta\Phi=\dfrac{2Lv^2}{\lambda c^2}$（实验预期条纹移动，实测为零）

## 4. 历史影响
莫雷协助完成的零结果实验成为否定以太、催生[[狭义相对论]]的关键证据。
