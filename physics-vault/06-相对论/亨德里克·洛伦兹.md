# 亨德里克·洛伦兹 (Hendrik Lorentz)

## 1. 生平
1853年生于阿纳姆，1875年获莱顿大学博士学位并在此任教。发展经典电子论，提出洛伦兹变换，为狭义相对论铺路，1902年获诺贝尔物理学奖。

## 2. 主要贡献
- 建立[[洛伦兹力]]的电子论
- 导出[[洛伦兹变换]]，成为[[狭义相对论]]的数学基础
- 提出局域时间与长度收缩概念

## 3. 关键公式与思想
- $\mathbf{F}=q(\mathbf{E}+\mathbf{v}\times\mathbf{B})$（洛伦兹力）
- $t'=\gamma(t-vx/c^2)$（洛伦兹变换）

## 4. 历史影响
洛伦兹的电子论与变换为爱因斯坦的相对论提供了直接数学框架。
