# 大卫·希尔伯特 (David Hilbert)

## 1. 生平
1862年生于哥尼斯堡，1885年获博士学位，1895年起任哥廷根大学教授，将哥廷根建成世界数学中心。1943年卒于哥廷根。

## 2. 主要贡献
- 几乎与爱因斯坦同时导出[[爱因斯坦场方程]]的作用量形式
- 提出希尔伯特23问题引领20世纪数学
- 发展[[积分方程]]与[[希尔伯特空间]]理论

## 3. 关键公式与思想
- $S=\int R\sqrt{-g}\,d^4x$（爱因斯坦-希尔伯特作用量）

## 4. 历史影响
希尔伯特在数学与物理的交叉处贡献卓著，其作用量形式成为[[广义相对论]]的标准表述。
