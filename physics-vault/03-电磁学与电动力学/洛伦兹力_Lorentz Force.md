# 洛伦兹力 (Lorentz Force)

## 1. 核心定义

[[洛伦兹力]]是带电粒子在[[电场]]和[[磁场]]中所受的电磁力，由[[亨德里克·洛伦兹]]于1895年给出完整表述，是[[电动力学]]中连接场与粒子运动的基本定律，描述了电磁场对电荷的作用机制。

## 2. 数学表达

### 洛伦兹力公式

电荷量为 $q$、速度为 $\vec{v}$ 的粒子在[[电场]] $\vec{E}$ 和[[磁场]] $\vec{B}$ 中受力：

$$\vec{F} = q\left(\vec{E} + \vec{v} \times \vec{B}\right)$$

其中：
- $q$：粒子电荷量（含符号）；
- $\vec{v}$：粒子速度；
- $\vec{E}$：电场强度；
- $\vec{B}$：磁感应强度；
- $\vec{v} \times \vec{B}$：[[叉积]]，方向由右手定则确定。

电场力 $q\vec{E}$ 对电荷做功，改变粒子动能；磁场力 $q\vec{v} \times \vec{B}$ 始终垂直于速度，不做功，仅改变粒子运动方向。

### 带电粒子在均匀磁场中的运动

在均匀磁场 $\vec{B} = B\hat{z}$ 中（无电场），粒子运动分解为平行和垂直于磁场的分量：

**回旋运动**（垂直分量）：

粒子以[[回旋频率]]（[[拉莫尔频率]]）做圆周运动：

$$\omega_c = \frac{|q|B}{m}$$

回旋半径（[[拉莫尔半径]]）：

$$r_L = \frac{mv_\perp}{|q|B}$$

**螺旋运动**：平行分量 $v_\parallel$ 不受磁场影响，粒子沿磁场方向匀速运动，整体轨迹为螺旋线。

### 漂移运动

在叠加电场或其它力场时，粒子产生**漂移运动**。$\vec{E} \times \vec{B}$ 漂移速度：

$$\vec{v}_E = \frac{\vec{E} \times \vec{B}}{B^2}$$

该漂移与电荷符号无关，是[[等离子体物理]]中的基本输运机制。

引力场 $\vec{g}$ 引起的漂移：

$$\vec{v}_g = \frac{m\vec{g} \times \vec{B}}{qB^2}$$

该漂移与电荷符号有关，导致正负电荷分离。

### 相对论性洛伦兹力

[[狭义相对论]]中，洛伦兹力的四维形式为：

$$\frac{dp^\mu}{d\tau} = q F^{\mu\nu} u_\nu$$

其中 $p^\mu = (E/c, \vec{p})$ 为四维动量，$u^\nu$ 为四维速度，$F^{\mu\nu}$ 为[[电磁张量]]，$\tau$ 为固有时。

### 洛伦兹力密度与功率

对连续电荷分布（电荷密度 $\rho$，电流密度 $\vec{J} = \rho\vec{v}$），力密度和功率密度为：

$$\vec{f} = \rho\vec{E} + \vec{J} \times \vec{B}$$

$$p = \vec{J} \cdot \vec{E}$$

磁场部分不做功，故功率仅含电场贡献。

## 3. 物理学意义与推论

- 洛伦兹力是[[电动力学]]的基本力的定律，与[[麦克斯韦方程组_Maxwell's Equations]]共同构成经典电动力学的完整框架：麦克斯韦方程描述场如何由电荷和电流产生并演化，洛伦兹力则描述场如何作用于电荷。
- 洛伦兹力的相对论协变形式 $\frac{dp^\mu}{d\tau} = qF^{\mu\nu}u_\nu$ 是[[狭义相对论]]中最早被认识的协变方程之一。这一协变性启发了[[爱因斯坦]]发展[[狭义相对论]]——电磁理论天然满足相对论不变性，而牛顿力学需要修正。
- **回旋运动**是[[等离子体物理]]的基础。在[[托卡马克_Tokamak]]磁约束聚变中，强磁场使带电粒子绕磁力线做回旋运动，从而被"束缚"在磁力线附近，实现等离子体的磁约束。回旋频率 $\omega_c = eB/m$ 决定了磁约束的效率。
- **漂移运动**是磁约束等离子体中输运损失的主要机制之一。$\vec{E}\times\vec{B}$ 漂移是中性漂移（不引起电荷分离），而梯度漂移和曲率漂移会引起电荷分离，导致[[托卡马克_Tokamak]]中的[[玻姆扩散]]等反常输运现象。
- **[[回旋加速器]]**（[[劳伦斯]]，1932，获1939年诺贝尔物理学奖）直接利用洛伦兹力原理：均匀磁场使粒子回旋，交变电场在每次穿越间隙时加速粒子。这是洛伦兹力在实验技术中的里程碑应用。
- **[[质谱仪]]**利用洛伦兹力对不同荷质比 $q/m$ 的粒子进行分离，是化学分析和同位素分离的基本工具。
- **极光现象**：太阳风带电粒子在地球磁场中的洛伦兹运动导致粒子沿磁力线进入极区大气，激发大气分子发光，是洛伦兹力在自然现象中的宏观体现。
- 在[[量子电动力学]]中，洛伦兹力对应于带电粒子与光子场的耦合顶点 $q\gamma^\mu A_\mu$，是[[费曼图]]中电子-光子顶点的经典极限。

## 4. 相关验证实验

- **[[阴极射线管]]实验**（1897，[[约瑟夫·约翰·汤姆孙]]）：通过电场和磁场偏转阴极射线（电子束），测定电子荷质比 $e/m$，首次直接验证了洛伦兹力的电场和磁场偏转效应，并发现了电子。[[汤姆孙]]因此获1906年诺贝尔物理学奖。
- **[[回旋加速器]]实验**（1932，[[欧内斯特·劳伦斯]]）：利用洛伦兹力原理加速带电粒子至高能，直接验证了回旋频率 $\omega_c = qB/m$ 的预言。[[劳伦斯]]获1939年诺贝尔物理学奖。
- **[[塞曼效应]]**（1896，[[彼得·塞曼]]）：原子光谱在磁场中分裂的现象，本质是电子轨道磁矩在磁场中的洛伦兹力效应。[[洛伦兹]]与[[塞曼]]共获1902年诺贝尔物理学奖。
- **[[霍尔效应]]**（1879，[[埃德温·霍尔]]）：导体中电流在垂直磁场作用下产生横向电压，直接验证了洛伦兹力的磁场分量对载流子的偏转作用。[[量子霍尔效应]]（1980，[[冯·克利青]]）是其量子推广。
- **[[托卡马克_Tokamak]]磁约束实验**：在聚变实验装置中，带电粒子的回旋运动和漂移行为由洛伦兹力精确描述，是磁约束可行性的基础。
- **[[同步辐射]]**：相对论性电子在磁场中因洛伦兹力做加速运动而发出辐射，是现代[[同步辐射光源]]的基础。

## 5. 历史上下文

- 1865年[[麦克斯韦]]建立方程组后，电磁场如何作用于电荷的问题尚不明确。麦克斯韦本人的表述较为复杂，涉及以太力学模型。
- 1892-1895年，[[亨德里克·洛伦兹]]在发展[[电子论]]的过程中给出了简洁明确的力公式 $\vec{F} = q(\vec{E} + \vec{v}\times\vec{B})$。洛伦兹的电子论将以太与物质分离，将电磁力归因于电荷在电磁场中的受力，是现代电动力学观念的关键一步。
- 1897年[[约瑟夫·约翰·汤姆孙]]发现电子后，洛伦兹力的对象从抽象电荷变为具体的电子，使电子论获得实验基础。洛伦兹因电子论和[[塞曼效应]]的理论解释获1902年诺贝尔物理学奖。
- 1905年[[爱因斯坦]]在[[狭义相对论]]论文《论动体的电动力学》中，指出洛伦兹力天然满足洛伦兹协变性，是相对论性力的典范。这表明电磁学不需要"以太"作为参考系，彻底解决了以太疑难。
- 1932年[[欧内斯特·劳伦斯]]发明的[[回旋加速器]]是洛伦兹力的工程化里程碑，开辟了高能物理实验的新时代。[[同步加速器]]、[[对撞机]]等后续发展均以洛伦兹力为基础。
- 洛伦兹力的概念在[[量子力学]]中推广为最小耦合原理 $\vec{p} \to \vec{p} - q\vec{A}$，是[[规范理论]]的物理基础。这一推广直接导致了[[量子霍尔效应]]、[[AB效应]]（[[阿哈罗诺夫]]-[[玻姆]]效应）等量子现象的发现。
- 在[[标准模型]]中，洛伦兹力对应于带电费米子与光子的[[规范耦合]]，是 $U(1)$ 规范相互作用的具体体现，是连接经典电动力学与量子场论的基本节点。
