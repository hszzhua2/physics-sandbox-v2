# 点电荷群 (Point Charges)

## 1. 实验目的与背景

1785年，[[夏尔·奥古斯丁·库仑]] 利用精密扭秤装置定量测量了两个带电小球之间的电力，建立了静电力的平方反比定律。库仑实验是电学史上从定性观察走向精确测量的转折点，为 [[库仑定律]] 的确立提供了直接实验证据。点电荷群的研究从库仑扭秤实验出发，推广至多电荷系统的叠加原理、电场与电势的多体计算、静电平衡条件与高斯定理，构成整个经典电动力学的实验与理论基石。

## 2. 实验装置与运行机制

**库仑扭秤**：一根水平银丝悬挂一根绝缘横杆，杆的一端装有带电金属小球，另一端配以平衡配重。另一个带电球被固定在可调节位置处，接近悬挂球时产生电力。电力使横杆旋转，银丝产生恢复扭矩。通过标定银丝的扭转常数 $\kappa$，测量平衡时的扭转角 $\theta$，即可确定电力大小。

平衡条件为静电力矩等于扭丝恢复力矩：

$$F_e \cdot l = \kappa \theta$$

其中 $l$ 为力臂长度。通过改变两球间距 $r$ 测量 $F_e(r)$ 的距离依赖关系，库仑发现 $F_e \propto 1/r^2$。

扭秤的灵敏度极高，可探测约 $10^{-8}\,\text{N}$ 量级的微弱力，与 [[卡文迪什实验]] 测量万有引力常数所用的扭秤原理相同。

## 3. 观测数据与数学模型

### 库仑定律

两个点电荷之间的静电力为：

$$\mathbf{F}_{12} = \frac{1}{4\pi\varepsilon_0}\frac{q_1 q_2}{r^2}\hat{\mathbf{r}}$$

其中 $\varepsilon_0$ 为真空介电常数，$r$ 为两电荷间距，$\hat{\mathbf{r}}$ 为从源电荷指向受力电荷的单位矢量。库仑实验直接验证了力的大小与 $r^2$ 成反比，精度优于 $10^{-3}$。现代等效测量（基于光速与麦克斯韦方程组自洽性）将平方反比律的精度推至 $|\delta| < 10^{-16}$，直接限制了光子的静止质量上限。

### 叠加原理与多体电场

多电荷系统中，总电场为各电荷电场的矢量和（叠加原理）：

$$\mathbf{E}(\mathbf{r}) = \frac{1}{4\pi\varepsilon_0}\sum_{i=1}^{N}\frac{q_i}{|\mathbf{r}-\mathbf{r}_i|^2}\hat{\mathbf{R}}_i$$

总电势为标量叠加，计算更为简便：

$$\phi(\mathbf{r}) = \frac{1}{4\pi\varepsilon_0}\sum_{i=1}^{N}\frac{q_i}{|\mathbf{r}-\mathbf{r}_i|}$$

对于连续电荷分布，求和化为积分：$\phi(\mathbf{r}) = \frac{1}{4\pi\varepsilon_0}\int \frac{\rho(\mathbf{r}')}{|\mathbf{r}-\mathbf{r}'|}\,d^3r'$。

### 高斯定理

由库仑定律结合叠加原理可导出静电场的高斯定理。积分形式：

$$\oint_S \mathbf{E}\cdot d\mathbf{A} = \frac{Q_{\text{enc}}}{\varepsilon_0}$$

微分形式（利用 [[斯托克斯定理]] 与散度定理）：

$$\nabla\cdot\mathbf{E} = \frac{\rho}{\varepsilon_0}$$

这是 [[麦克斯韦方程组]] 的第一方程，表明电荷是电场的源。

### 静电平衡

导体在静电平衡条件下内部电场为零（$\mathbf{E}_{\text{内}} = 0$），电荷仅分布于导体表面。表面电场垂直于导体表面：

$$\mathbf{E}_{\text{表面}} = \frac{\sigma}{\varepsilon_0}\hat{\mathbf{n}}$$

其中 $\sigma$ 为面电荷密度，$\hat{\mathbf{n}}$ 为外法向单位矢量。整个导体为等势体。

## 4. 实验结论与影响

1. **平方反比定律的确立**：库仑实验确立了电力与距离的平方反比关系，是 [[麦克斯韦方程组]] 与经典电动力学的起点。平方反比律的精度直接决定了光子静质量的上限，是检验电磁理论基本对称性的关键。
2. **叠加原理的验证**：多电荷系统实验验证了电场叠加原理，使任意电荷分布的电场可通过积分求解，为计算电磁学奠定了基础。
3. **高斯定理的导出**：库仑定律结合叠加原理导出高斯定理，是电动力学规范不变性的体现，也是 [[麦克斯韦方程组]] 积分形式的核心组成部分。
4. **现代应用**：点电荷模型是 [[特斯拉线圈_Tesla Coil]] 高压放电分析的基础，也是计算电磁学与分子动力学模拟（如蛋白质静电相互作用）的基本计算单元。库仑扭秤的精密测量思想延续至现代的 [[卡文迪什实验]] 和引力波探测中干涉仪的精密测量理念。
