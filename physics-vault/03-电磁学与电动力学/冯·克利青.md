# 冯·克利青 (Klaus von Klitzing)

## 1. 生平
1943年生于施罗达，在维尔茨堡大学获博士学位，任马普固体研究所所长。1980年发现量子霍尔效应，1985年获诺贝尔物理学奖。

## 2. 主要贡献
- 发现整数量子[[霍尔效应]]
- 利用[[洛伦兹力]]下的朗道能级量子化

## 3. 关键公式与思想
- $R_H=\dfrac{h}{e^2}\approx 25812.8\,\Omega$（量子化霍尔电阻）

## 4. 历史影响
量子霍尔效应揭示了[[拓扑学]]在凝聚态中的角色，精确测定了精细结构常数。
