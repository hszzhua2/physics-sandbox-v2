# RLC谐振 (RLC Resonance)

## 1. 核心定义

[[RLC谐振]]是含电阻 $R$、电感 $L$、电容 $C$ 的二阶电路系统中电荷与电流的振荡现象，由[[亥姆霍兹]]在力学-电学类比框架下分析，是[[电路理论]]的核心内容，与机械[[弹簧振子_Spring Oscillator]]在数学上完全等价。

## 2. 数学表达

### 基本方程

由[[基尔霍夫电压定律]]，串联RLC电路中电容器电荷 $q(t)$ 满足二阶常微分方程：

$$L\ddot{q} + R\dot{q} + \frac{q}{C} = \mathcal{E}(t)$$

其中：
- $L$：[[电感]]（惯性项，对应动能）；
- $R$：[[电阻]]（阻尼项，对应摩擦）；
- $1/C$：[[电容]]的倒数（恢复力项，对应弹簧劲度系数）；
- $\mathcal{E}(t)$：外加驱动电动势。

### 自由振荡（$\mathcal{E} = 0$）

定义参数：

$$\omega_0 = \frac{1}{\sqrt{LC}}, \quad \alpha = \frac{R}{2L}, \quad \beta = \frac{\alpha}{\omega_0} = \frac{R}{2}\sqrt{\frac{C}{L}}$$

$\omega_0$ 为[[固有角频率]]，$\alpha$ 为[[阻尼系数]]，$\beta$ 为[[阻尼比]]。根据阻尼大小分三种情形：

**欠阻尼**（$\alpha < \omega_0$，即 $R < 2\sqrt{L/C}$）：

$$q(t) = A e^{-\alpha t} \cos(\omega_d t + \phi), \quad \omega_d = \sqrt{\omega_0^2 - \alpha^2}$$

**临界阻尼**（$\alpha = \omega_0$，即 $R = 2\sqrt{L/C}$）：

$$q(t) = (A + Bt) e^{-\alpha t}$$

**过阻尼**（$\alpha > \omega_0$，即 $R > 2\sqrt{L/C}$）：

$$q(t) = A e^{(-\alpha + \sqrt{\alpha^2 - \omega_0^2})t} + B e^{(-\alpha - \sqrt{\alpha^2 - \omega_0^2})t}$$

### 受迫振荡与共振

正弦驱动 $\mathcal{E}(t) = \mathcal{E}_0 \cos(\omega t)$ 下稳态电流：

$$I(t) = I_0 \cos(\omega t - \varphi)$$

电流幅值：

$$I_0 = \frac{\mathcal{E}_0}{\sqrt{R^2 + \left(\omega L - \frac{1}{\omega C}\right)^2}}$$

相位差：

$$\tan\varphi = \frac{\omega L - 1/(\omega C)}{R}$$

**共振频率**（电流幅值最大）：

$$\omega_{\text{res}} = \omega_0 = \frac{1}{\sqrt{LC}}$$

共振时 $\varphi = 0$，电路呈纯电阻性，感抗与容抗完全抵消：$\omega_0 L = 1/(\omega_0 C)$。

### 品质因子

[[品质因子]] $Q$ 描述共振峰的锐利程度和能量损耗速率：

$$Q = \frac{\omega_0 L}{R} = \frac{1}{\omega_0 RC} = \frac{1}{R}\sqrt{\frac{L}{C}}$$

共振时电感和电容上的电压可远大于驱动电动势，放大倍数即 $Q$ 值：

$$V_L = V_C = Q\,\mathcal{E}_0$$

带宽（半功率全宽）：

$$\Delta\omega = \frac{\omega_0}{Q}$$

### 能量分析

电路总能量：

$$E = \frac{1}{2}L\dot{q}^2 + \frac{q^2}{2C}$$

自由振荡时电阻消耗能量速率 $P_R = I^2 R$，衰减时间常数 $\tau = 2L/R = 2Q/\omega_0$。

## 3. 物理学意义与推论

- RLC谐振是[[电路理论]]的核心内容，其数学结构与机械[[弹簧振子_Spring Oscillator]] $m\ddot{x} + b\dot{x} + kx = F(t)$ 完全等价。对应关系为：$L \leftrightarrow m$（惯性）、$R \leftrightarrow b$（阻尼）、$1/C \leftrightarrow k$（劲度）、$q \leftrightarrow x$（位移）。这一力-电类比由[[亥姆霍兹]]首先指出，是物理类比方法的典范。
- **共振现象**在RLC电路中表现为驱动频率等于固有频率时电流幅值极大增强，感抗与容抗完全抵消。这一原理在[[无线电调谐]]、[[滤波器]]设计中是基础工具。
- 品质因子 $Q$ 是衡量谐振系统性能的核心参数：$Q$ 越大，共振峰越尖锐，能量损耗越慢，选频能力越强。在[[特斯拉线圈_Tesla Coil]]中，通过高 $Q$ 值谐振实现电压的巨大放大，可产生数百万伏的高压。
- RLC电路的**三种阻尼区**在工程中有不同应用：欠阻尼用于振荡器和滤波器；临界阻尼用于快速无振荡响应（如测量仪表的阻尼设计）；过阻尼用于平滑无超调的响应。
- 在[[特斯拉线圈_Tesla Coil]]中，初级和次级线圈各自构成RLC谐振回路，当两者共振频率匹配时，通过互感实现高效的能量传输和电压放大。这是电磁感应在高频高压下的精妙应用。
- RLC谐振的量子对应是**量子LC电路**：电荷 $q$ 和磁通 $\Phi$ 满足[[对易关系]] $[\hat{q}, \hat{\Phi}] = i\hbar$，能级等间距 $\hbar\omega_0$，是[[量子电动力学]]的电路实现，在[[超导量子比特]]中有重要应用。
- RLC电路是研究[[线性响应理论]]和[[涨落耗散定理_Fluctuation-Dissipation Theorem]]的具体模型：阻抗 $Z(\omega)$ 的实部（电阻）对应耗散，虚部（电抗）对应储能，二者通过[[Kramers-Kronig关系]]关联。

## 4. 相关验证实验

- **LC振荡实验**（[[赫兹电磁波实验]]，1887）：[[赫兹]]通过LC振荡电路产生高频电磁波，其振荡频率 $\omega_0 = 1/\sqrt{LC}$ 是RLC谐振的最直接验证。
- **[[特斯拉线圈_Tesla Coil]]实验**（1891，[[尼古拉·特斯拉]]）：利用高 $Q$ 值RLC谐振产生高频高压，是谐振放大效应的典型演示。
- **无线电调谐电路**：收音机的选频电路基于RLC共振，通过调节 $C$ 或 $L$ 改变共振频率以选择不同电台，是RLC谐振在工程中的日常应用。
- **频谱分析仪实验**：扫描频率测量RLC电路的幅频和相频特性曲线，验证共振峰的 $Q$ 值和带宽关系 $\Delta\omega = \omega_0/Q$。
- **量子LC电路实验**（1990s，[[超导量子比特]]）：在超导电路中实现量子化RLC谐振，观测到离散能级和真空涨落，是[[量子计算]]硬件的基础。
- **机械-电学类比实验**：用RLC电路模拟机械振动系统（如建筑抗震分析），验证力-电类比的正确性。

## 5. 历史上下文

- 1847年，[[亥姆霍兹]]在《论力的守恒》中首次指出力学与电学方程的数学等价性，建立了力-电类比的基本框架。[[开尔文勋爵]]（[[威廉·汤姆孙]]）随后发展了详细的类比关系。
- 1853年，[[开尔文勋爵]]分析了莱顿瓶放电的振荡现象，给出 $T = 2\pi\sqrt{LC}$ 的周期公式，是RLC谐振的早期理论预言。
- 1887年[[赫兹电磁波实验]]中，[[赫兹]]使用的火花间隙发射器本质上是一个高阻尼RLC电路，其振荡频率由 $L$ 和 $C$ 决定。这是RLC谐振在产生电磁波中的关键应用。
- 1891年[[尼古拉·特斯拉]]发明[[特斯拉线圈_Tesla Coil]]，利用双调谐RLC谐振（初级和次级线圈共振频率匹配）实现高压产生和无线能量传输的探索。特斯拉的无线电力传输构想虽未成功，但其谐振原理在无线电工程中广泛应用。
- 20世纪初，[[古列尔莫·马可尼]]将RLC调谐电路应用于无线电收发，实现了跨大西洋无线电传输（1901）。调谐电路的引入使不同频率信号得以分离，是无线电通信的基础。
- 现代电路理论中，RLC谐振的[[拉普拉斯变换]]分析（[[海维赛德]]运算微积）已成为标准方法。[[亥维赛]]将微分方程转化为代数方程的运算微积法是电路分析的重要简化。
- 1990s以来，[[超导量子比特]]（如[[传输子量子比特]]，[[transmon]]）将RLC电路量子化，利用超导约瑟夫森结的非线性电感实现可调谐振和量子比特操控，是[[量子计算]]硬件的主流路线之一。
